# 如何以多种方式在 Apple Mac 上录制屏幕

> 原文：<https://www.xda-developers.com/how-to-record-screen-mac/>

苹果电脑在台式机用户中越来越受欢迎。无论你是 T2 的学生、老师还是办公室职员，苹果电脑都是生产力、学习和工作的绝佳选择。此外，苹果的教育折扣只会让购买新 Mac 更有诱惑力。但你一定想知道——我如何在我的新 Mac 上录制屏幕？

录制 Mac 的屏幕不是一个简单的过程，至少与 iOS 的简单方法相比是这样的。您有大量的选项，每个选项都是为特定的需求和使用情形量身定制的。例如，如果您想要录制视频教程，所选择的模式不同于对您正在观看的任何内容进行静态截图的模式。我们将探索在 [macOS](http://xda-developers.com/macos-monterey) 上录制您的屏幕的所有可用选项。

苹果让选择哪种模式变得足够容易。它们都被捆绑在一个位置，并附有各自的简要描述。首先，你的应用程序是**截图**。你应该可以在**的 Launchpad** 中找到它，在其他的第一方预装应用中。或者，你可以通过按住**命令**按钮，然后点击**空格键**来调用**聚光灯搜索**。一旦**聚光灯搜索**出现，输入**截图**，点击第一个结果。

截图应用有一个基本的用户界面(UI)。这是一个位于屏幕底部的半透明小条，上面有几个按钮代表可供选择的模式。剩下的都是透明的，可以查看截图 app 背后的内容，决定记录什么/哪里。

## 选择

在浏览可用模式之前，最好先检查选项，选择最适合你的模式。与录制(视频)模式相比，选择捕捉(静态屏幕截图)模式时的选项有所不同。

### 当选择捕获模式时

*   **保存到:**此选项允许您选择您喜欢的截图目的地。这些目的位置包括桌面或文档文件夹、拷贝到剪贴板、直接共享到另一个应用程序(如“邮件”或“信息”)或您自己选取的另一个位置。
*   **定时器:**此选项允许您在截图前设置 5 秒或 10 秒的定时器。
*   **其他选项:**显示浮动缩略图、记住上一次选择(您的首选模式和选项)、显示鼠标指针。

### 当选择记录模式时

*   **保存到:**与捕获模式类似，但不包括保存到剪贴板选项。
*   **定时器:**类似于抓拍模式。
*   **麦克风:**可选择让您使用 Mac 的内建麦克风或其他连接的麦克风录制外部音频。
*   **其他选项:**与抓图模式类似，但将**显示鼠标指针**选项替换为**显示鼠标点击**。

## 模式

### 捕获模式

捕捉屏幕截图时有不同的选项可供选择。

*   **截图整屏:**这个选项，顾名思义，就是对整个 Mac 屏幕进行截图。这包括菜单栏和 dock，如果它们在采集时可见的话。

*   **捕获选定的窗口:**该选项可以捕获您自己选择的打开窗口的屏幕截图。在最终图像中，其他窗口、菜单栏和 dock 都不可见。

*   **捕获选定部分:**这将显示一个可调整大小的虚线矩形，您可以在屏幕上移动它。这允许您选择独立于 windows 的特定屏幕部分来拍摄屏幕截图。

### 记录方式

除了**记录选择窗口**选项不可用之外，记录模式几乎与捕捉模式相同。

*   **录制整个屏幕:**此选项会对整个 Mac 屏幕进行视频录制。

*   **录制选定部分:**该选项允许您录制显示在我上面提到的虚线矩形内的内容的视频。

一旦你设置了选项来匹配你的偏好，你就可以点击**捕捉** / **记录**，屏幕截图选择栏 UI 右侧的最后一个按钮。截图将立即保存到您自己选择的目的地。对于视频录制，菜单栏中会出现一个停止按钮。一旦你完成了记录，点击那个按钮就可以保存了。

## 让生活更轻松的捷径

*   截图:**换挡** + **指令** + **3**
*   截取部分屏幕: **Shift** + **命令** + **4**
*   抓取一个窗口或菜单: **Shift** + **命令** + **4** + **空格键**
*   启动截图 app: **Shift** + **命令** + **5**

你都准备好了！现在，无论出于何种目的，您都可以轻松录制 Mac 的屏幕。苹果在 macOS 中有各种选项和模式，每个人和每个可能的用例都有一个组合。
