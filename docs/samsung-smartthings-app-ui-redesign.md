# 三星正在彻底检查其老化的 SmartThings 应用程序

三星刚刚宣布了其 SmartThings 平台的另一项重大更新:SmartThings Android 和 iOS 应用程序的新设计。

SmartThings 是三星的智能设备平台，类似于亚马逊 Alexa(没有虚拟助手)或谷歌 Home。它将三星自己的产品和数以千计的第三方设备和平台结合在一起，但这个应用程序需要一些关注已经有一段时间了。这款应用的设计从来都不适合单一的用户界面，如果没有标签式的组织，寻找特定的设备可能需要一段时间。谢天谢地，三星终于对 SmartThings 应用程序进行了急需的彻底改革。

应用程序更新预计将于今天开始在 Android 设备上推出，iOS 应用程序也将很快收到类似的更新。三星放弃了汉堡菜单，采用了新的标签式设计，包括收藏夹、设备、生活(推荐的服务和设备)、自动和菜单(用于附加设置)按钮。不再需要多次点击来检查您的电视在做什么。

“同名的移动和 Windows 桌面应用程序现在有了增强的更新，”三星在声明中说[。“作为 SmartThings 简化互联家庭体验的持续承诺的一部分，新界面通过全新的界面提供了更轻松、更强大的用户旅程和更快的加载时间。"](https://www.samsungmobilepress.com/featurestories/samsung-smartthings-unveils-new-interface-offering-customers-a-more-dynamic-connected-home-experience)

新设计只是过去几个月来 SmartThings 平台的最新变化。Android Auto 支持是在 1 月添加的[，允许人们从他们的汽车上快速访问智能家居设备(例如，当你下班开车回家时，打开家里的恒温器)。三星还在 4 月份开始公开测试用于 SmartThings](https://www.xda-developers.com/samsungs-smartthings-android-auto-integration) 的[网络仪表盘，并且在 5 月份与 Galaxy Book Pro 一起发布了](https://www.xda-developers.com/samsung-smartthings-web-dashboard/)[新的 Windows 10 应用](https://www.xda-developers.com/samsung-smartthings-app-windows-10/)。
