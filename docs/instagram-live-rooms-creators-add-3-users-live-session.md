# Instagram Live Rooms 允许创建者在直播会话中添加 3 个用户

Instagram 正在印度和印度尼西亚推出直播间，允许在社交媒体平台上进行四场参与者直播。请继续阅读！

Instagram 已经成为影响者和内容创作者事实上的平台，这要归功于它更加专注于照片和视频分享服务。但 COVID 改变了我们创建和消费内容的方式，现在的重点转向了共享的现场体验和互动。视频通话现在是个人通话的标准，多人虚拟会议是事件的标准。因此，Instagram 正在调整其直播策略，因为它现在允许创作者通过直播间在直播会话中添加最多三个人。

Instagram 的最新功能叫做 Live Rooms。简而言之，它是现有 Live 功能的扩展，可以灵活地在会话中比以前多添加两个人。实时会议现在可以在同一个窗口中显示四个人，允许在这些会议中探索更大的主题灵活性。以前，一个实时会话只能容纳创建者和一个额外的来宾-如果您需要添加任何其他人，您需要删除以前添加的来宾。

Live Rooms 在印度与一些创作者进行了测试，多参与者视频聊天格式得到了他们的好评。Live Rooms 的首次展示已经开始，并将很快在印度和印度尼西亚向所有人开放。Instagram 没有提到这项功能是否以及何时会在世界其他地方推出，但我们估计全球推出也会很快，就像它对[消失模式](https://www.xda-developers.com/facebook-vanish-mode-disappearing-messenger-instagram/)和 [BTS 主题](https://www.xda-developers.com/facebook-messenger-instagram-bts-theme-watch-together/)等功能所做的那样。

### 如何使用 Instagram 直播间

要在 Instagram 上直播:

*   点击故事托盘左上角的加号，或点击主页导航栏右上角的“创建”加号图标
    *   注意:当你在 Instagram 主屏幕上时，你也可以向左滑动！
*   滚动并点击底部的实时摄像机选项
*   如果您愿意，可以使用左边的菜单添加标题
*   点击圆形按钮开始直播

要将客人添加到您的实时流:

*   当您已经开始直播时，点击摄像机/房间图标
*   现在，您将看到有人请求加入您的直播。您也可以搜索客人的姓名，向他们发送加入您的 live 的请求。
*   只需点击客人的手柄即可添加
*   您可以一次添加所有三位客人，也可以在以后添加客人
