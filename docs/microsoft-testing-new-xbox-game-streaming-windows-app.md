# 微软测试新的 Xbox 游戏流 Windows 应用程序

> 原文：<https://www.xda-developers.com/microsoft-testing-new-xbox-game-streaming-windows-app/>

游戏流媒体从一个只能被想象为遥不可及的未来概念的东西变成了实际存在的东西，甚至在用户中获得了一些动力(尽管一些公司[做得并不太好](https://www.xda-developers.com/google-stadia-failed-to-meet-expectations/))。每个人都想从游戏流媒体游戏中分得一杯羹，从 Nvidia 这样的公司到甚至微软，微软也有自己的游戏流媒体服务，称为 XboxCloud Gaming，以前称为 x Cloud(为了简单起见，我们仍将其称为 x Cloud)。鉴于你现在无法购买 Xbox Series X([或者也许你可以](https://www.xda-developers.com/xbox-series-x-restocks/))，这可能是次好的选择，但它有一些局限性。好消息是，随着新功能的出现和改进，大多数限制都将消失。

这些改进中的第一项是微软正在开发一款名为 Xbox Game Streaming 的 Windows 应用。这款 Windows 应用不仅可以让你从云中传输游戏，并将 xCloud 体验传递给运行 Windows 10 的设备和电脑，如果你有 Xbox Series S 或 Xbox Series X，它还可以让你从自己的主机上传输游戏。然而，这最后一个功能在这个(非常粗糙)未发布的版本中还没有功能，这个版本是 *The Verge* [在](https://www.theverge.com/platform/amp/2021/2/27/22304762/microsoft-xcloud-app-windows-xbox-touch-controls-gyro-support)上得到的。

这个应用程序并不是什么新东西，因为它是去年在一个更糟糕的状态下首次被发现的，这意味着微软正在积极地开发它。它不仅可以为 Windows PCs 提供 xCloud 体验，还可以兼容 ARM 设备，这意味着您也可以在始终连接的笔记本电脑或平板电脑上玩游戏。这个应用程序还将打包诸如触摸支持(这将允许你在没有控制器的情况下玩)和陀螺支持(尽管这个似乎也坏了)等功能。

第二个改进是 1080p 流媒体支持，据 *Windows Central* 报道。目前，xCloud streaming 的上限是 720p，这对于较小的显示器和智能手机来说可能是好的，实际上在带宽和响应能力方面更好，因为在网络上移动 720p 流更容易，但在较大的显示器上可能会很快变得非常模糊。只要您的互联网连接允许，1080p 流媒体可以为您的游戏提供更好的质量。

谷歌 Stadia 等其他竞争对手已经支持 1080p 流媒体，因此微软在这方面正在迎头赶上。同时，微软的大部分服务器基础设施都是基于 Xbox One 架构的。微软将在今年转向 Xbox Series X 架构，因此 1080p 流媒体可能是这些变化的首批结果之一。
