# 脸书的最新功能让你可以自定义你在订阅源中看到的内容

> 原文：<https://www.xda-developers.com/facebooks-latest-feature-lets-you-customize-what-you-see-in-your-feed/>

Meta 正在推出一项新功能，帮助用户在他们的脸书订阅源上看到更少的无关内容。虽然 Meta 声称这个新功能可以让你定制你的 feed，但它实际上并不能让你做出任何实质性的改变。相反，你可以用它来训练脸书的算法，在你的订阅源上显示更少的无关内容。

在一篇关于此事的博客文章中，Meta 透露脸书的用户将会在他们的订阅源的帖子下面看到新的“显示更多”和“显示更少”按钮。选择“显示更多”按钮将*“暂时增加该帖子和类似帖子的排名分数。”*另一方面，选择“显示较少”会降低其排名分数。脸书的算法将利用这些反馈来改善你在订阅源上看到的帖子类型。

请注意，这些按钮不会出现在您的脸书订阅源中的所有帖子上。Meta 表示，它们会定期出现，但它计划在未来将它们添加到帖子右上角的三点菜单中。除了脸书的 feed，Meta 还在 Instagram 上测试这个功能。

值得一提的是，YouTube 也提供类似的功能，帮助用户改善平台上的视频推荐。但是，根据最近的一项研究， [YouTube 的工具大多无效](https://techcrunch.com/2022/09/20/youtubes-dislike-and-not-interested-options-dont-do-much-for-your-recommendations-study-says/),因为用户不断收到糟糕的推荐。观察脸书的实施效果会很有趣，但我们不抱太大希望。

除了这项新功能，Meta 还在开发一个选项，帮助用户[使用同一个账户在脸书和 Instagram 上创建多个个人资料](https://www.xda-developers.com/facebook-create-multiple-profiles/)。此举背后的想法是帮助用户根据他们希望在平台上联系的群体来区分个人资料。
