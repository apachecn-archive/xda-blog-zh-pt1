# 微软在 Outlook 手机应用中加大广告力度

> 原文：<https://www.xda-developers.com/microsoft-more-ads-outlook-ios-android-apps/>

微软 Outlook 是 iOS 和 Android 上的一个可靠的电子邮件应用，但微软可能在其中的一部分变得有点过于激进。令他们沮丧的是，用户已经注意到微软正在向 Outlook mobile 推出更多的广告，作为一种尝试和推广微软 365 订阅的可能方式。

Outlook 手机版已经有两个布局选项，可以设置带有两个类别(聚焦和其他)的聚焦收件箱，也可以设置一个包含所有电子邮件的收件箱。在这些情况下，没有付费订阅微软 365 的用户总是在“其他标签”中看到看起来像真实电子邮件的广告你可以在下面的推文中看到一个例子。

然而，现在看来，微软似乎也在用单一收件箱向观众推送广告。这一增加使得在没有订阅微软 365 的情况下，避免在 Outlook 中看到广告变得更加困难。The Verge 最初报道了这一消息，微软发言人的官方回应证实了这一变化。公司提到了一种避免看到广告的方法。

> 对于 Outlook 的免费用户，广告显示在他们的收件箱中，如果他们只想在“其他”收件箱中看到广告，他们可以选择启用“聚焦收件箱”功能

如果你愿意，你可以轻而易举地把这些收件箱广告扫走，但它们最终会回来。但是，尽管变化看起来很简单，但在电子邮件应用程序中出现这类广告时，微软并不是唯一的一家。谷歌和雅虎也在他们基于网络的体验中这样做，你可以在收件箱中看到小横幅广告。

要获得无广告的 Outlook mobile 体验，您可以订阅 Microsoft 365。家庭可以购买 100 美元/年的年费，个人账户可以购买 70 美元/年的年费，或者分别以 10 美元/月或 7 美元/月的费率购买年费。如果订阅不是一个选项，那么其他没有广告的电子邮件应用程序也可以在 iOS 和 Android 上使用。
