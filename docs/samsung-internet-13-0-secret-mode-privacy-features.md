# 三星互联网 13.0 引入了增强的隐私和安全功能

三星发布了三星互联网 13.0，其中包括的功能将为用户提供更安全的浏览体验。

三星已经开始推出三星互联网 13.0，该公司表示，该版本引入的功能将为用户提供更安全的浏览体验。

其中一个大的新功能被称为“秘密模式”这种新模式允许用户在所有秘密模式标签关闭后立即自动清除他们的浏览历史。当启用保密模式时，用户将看到一个图标。

更新[还会在用户](https://news.samsung.com/global/samsung-internet-13-0-takes-your-browsing-experience-to-the-next-level?utm_source=rss&utm_medium=direct)访问潜在恶意网站时发出警告。“三星互联网 13.0 提供了一个新的权限请求用户界面，如果一个网站似乎是恶意的，并试图欺骗用户允许通知，它将显示一条警告消息。”

*权限请求 UI–之前(左)和三星互联网 13.0(右)*

除了三星互联网 13.0 改进的隐私和安全功能，更新还为用户提供了更好的用户体验:

*   您现在可以使用*高* *c* *对比度* *模式*并打开*暗* *m* *ode* ，使字体和其他组件更加突出。
*   三星互联网 13.0 还引入了一个*E**x 可扩展**A**PP**B**ar*用于书签、保存页面、历史和下载等菜单。
*   通过隐藏状态栏获得更多屏幕空间，让自己沉浸在浏览的内容中。
*   用*视频* *助手* *助手*全屏观看视频时，双击屏幕中间暂停。
*   轻松编辑书签的标题，以便更容易识别和搜索。

最后，三星表示，新的 API 模块现在可用于三星互联网 13.0，这将使开发者有机会使用它们来构建扩展。一些新模块包括 WebRequest、Proxy、Cookies、Types、History、Alarms、Privacy 等等。感兴趣的开发者可以通过访问[此链接](https://developer.samsung.com/internet/android/extensions-dev-overview.html)了解更多关于开发第三方扩展的信息。
