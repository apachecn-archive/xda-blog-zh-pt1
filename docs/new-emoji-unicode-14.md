# 以下是 Unicode 14.0 版手机上的所有新表情符号

> 原文：<https://www.xda-developers.com/new-emoji-unicode-14/>

Unicode Consortium 正式发布了 Unicode 14.0，它将为您的手机带来 37 种新的表情符号。最新发布的版本有一些期待已久的角色，包括巨魔，食指和拇指交叉的心(又名手指心)，敬礼脸，心手，等等。这些新表情符号将在今年年底或明年年初登陆所有主要平台。

根据 [*表情百科*](https://blog.emojipedia.org/whats-new-in-unicode-14-0/) ，Unicode 14.0 包括以下新表情:

*   熔化面
*   睁着眼睛，手捂着嘴
*   偷看的脸
*   敬礼脸
*   虚线面
*   斜嘴脸
*   忍住眼泪的脸
*   右手
*   左手
*   手掌朝下
*   手掌朝上
*   食指和拇指交叉的手
*   指向查看者的索引
*   心手
*   咬嘴唇
*   戴皇冠的人
*   怀孕的男人
*   孕妇
*   钓鱼
*   珊瑚
*   莲花
*   空巢
*   用蛋筑巢
*   豆子
*   倾倒液体
*   冲突
*   游乐场滑梯
*   车轮
*   救生圈
*   哈萨
*   镜子球
*   低电量
*   拐杖
*   x 射线
*   气泡
*   身份证
*   粗等号

此外，该版本还包括这些新表情符号的变体和肤色，使新设计总数达到 112 个。下图展示了 Unicode 14.0 中包含的所有新表情符号，以及所有变体。

Unicode 14.0 将表情符号的总数[增加到 3633 个](https://emojipedia.org/stats/)，包括性别或肤色序列、旗帜以及用于创建键帽、旗帜和其他序列的组件。

值得注意的是，最新版本增加了去年与 [Unicode 13.0](https://www.xda-developers.com/unicode-13-62-new-emoji-2020/) 一起发布的越来越多的性别包容性表情符号。 *Emojipedia* 注意:*“在此版本中，使用新的代码点添加了怀孕的性别选项。改变了通常使用表情符号序列来表示性别差异的格式。“怀孕的人和怀孕的男人都是新的代码点，这是 Unicode 不断努力使所有表情符号的性别选项一致的一部分。”*

此外，Unicode 14.0 将为所有平台带来对带有肤色组合的[握手](https://emojipedia.org/handshake/)表情符号的支持，并修复带有手捂嘴表情符号的[面部的跨平台显示问题。](https://emojipedia.org/face-with-hand-over-mouth/)
