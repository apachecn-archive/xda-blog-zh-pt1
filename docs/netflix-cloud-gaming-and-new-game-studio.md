# 网飞着眼云游戏服务，开设新游戏工作室

> 原文：<https://www.xda-developers.com/netflix-cloud-gaming-and-new-game-studio/>

今天，网飞宣布了其季度收益，结果实际上相当可观。该公司实现了转型，超过了目标，增加了更多用户，也创造了更多收入。在展望未来的同时，该公司已经开始采取新的行动，因为它将在 11 月推出一个新的[广告支持层](https://www.xda-developers.com/netflix-ad-supported-next-month/)，并且它还将更多地致力于其游戏细分市场，计划为[平台带来更多的游戏](https://www.xda-developers.com/netflix-announces-55-games-in-development/)。

显然，该公司计划更深入地进入游戏领域，在 TechCrunch Disrupt 大会上宣布，它正在研究云游戏，最令人惊讶的是，它正在开设一个新的游戏工作室。在 TechCrunch Disrupt 期间，网飞游戏副总裁 Nike Verdu 分享说，该公司正在“认真”考虑提供云游戏服务。虽然大多数公司目前都将这项服务作为独立产品提供，但如果网飞提供这样的服务，它将成为当前订阅服务的一部分，或者像 Verdu 所说的那样，是一种“增值服务”

像亚马逊这样的竞争对手已经开始涉足，它目前提供自己的游戏流媒体服务 [Luna](https://www.xda-developers.com/amazon-luna-official-launch-us/) 。就在最近，谷歌宣布将[关闭](https://www.xda-developers.com/google-stadia-is-shutting-down/)服务，因为它无法吸引用户。从我们所看到的来看，很明显，这个空间并不容易，所以如果网飞要介入，它必须非常谨慎。

除了云游戏的野心，Verdu 还透露网飞将在南加州开设一个全新的游戏开发工作室。此前，网飞收购了游戏工作室，但这将是该公司第一次从零开始。新工作室将由业内资深人士查科·桑尼(Chako Sonny)执掌，他曾是《守望先锋》(Overwatch)的执行制片人。虽然这是一个惊人的消息，但看看网飞的游戏之旅能走多远将会很有趣。
