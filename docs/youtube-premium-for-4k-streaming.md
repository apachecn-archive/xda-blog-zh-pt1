# YouTube 要求一些用户为 4K 流媒体付费

> 原文：<https://www.xda-developers.com/youtube-premium-for-4k-streaming/>

在过去的几个月里，YouTube 增加了许多新功能，为用户提供了观看和创作的新方式。该公司还一直在扩展 YouTube 短片，从明年开始为创作者提供一种新的赚钱方式。虽然这些都是很好的补充，但最近几周出现的一些变化引起了人们的关注。看起来，YouTube 要求一些用户如果想在 4K 观看内容，必须进行付费订阅。

目前还不知道这是某种测试还是新功能的缓慢推出。对某些类型的设备也可能有限制。不幸的是，这些细节目前还不得而知，但在过去的几周内，通过 Reddit 和 Twitter 等在线社区，有许多报告正在增长。大多数关心 4K 的人是那些在电视上看 YouTube 的人，而智能手机用户似乎对 1440p 或 1080p 的质量没问题。其他用户表达了他们对 YouTube 在未来某个时候将其他解决方案置于付费墙之后的担忧。YouTube 并未对这一变化做出公开声明，但这是可以理解的。

那么你从 YouTube Premium 得到了什么？最重要的是，你可以获得无广告的视频，还可以下载视频离线播放。您还可以在后台或屏幕关闭时播放视频，支持画中画，访问 YouTube 原件和 YouTube Music Premium。现在，所有这些都不便宜，每月 11.99 美元，每年 119.99 美元。虽然有些人会同意获得所有这些功能是值得的，但其他人会认为有其他方法可以获得无广告的体验。这将是一件有趣的事情，看看当它向公众推出时会发生什么，以及在 4K 是否会有新的替代观看 YouTube 的方式。
