# 华硕 ROG 手机 3，ZenFone 7 系列获得新的固件更新

> 原文：<https://www.xda-developers.com/asus-zenfone-7-pro-rog-phone-3-update-security-patches-bug-fixes/>

# 华硕发布了 ZenFone 7 和 ROG Phone 3 的新更新，包括安全补丁和漏洞修复

华硕已经开始为 ROG Phone 3 以及 ZenFone 7 和 ZenFone 7 Pro 智能手机推出新的更新。继续阅读，了解更多信息！

华硕已经开始为 ZenFone 7、 [ZenFone 7 Pro](https://www.xda-developers.com/asus-zenfone-7-pro-review-a-flipping-fantastic-flagship-smartphone/) 以及其最新的游戏智能手机 [ROG 手机 3](https://www.xda-developers.com/asus-rog-phone-3-review/) 推出新的更新。在我们看到该公司推出 ROG Phone II 的[新更新后一周，新更新终于在 PUBG Mobile 上推出了 90fps。](https://www.xda-developers.com/asus-rog-phone-ii-latest-update-90fps-pubg-mobile-volte-t-mobile-bsnl/)

ZenFone 7 和 ZenFone 7 Pro 的新固件更新带有版本号[29.14.53.14](https://zentalk.asus.com/en/discussion/42188/201218-zenfone-7-pro-7-ww-29-14-53-14)。根据 ZenTalk 论坛上分享的补丁说明，新的更新带来了 11 月份的 Android 安全补丁和相机部门的改进。

**华硕 Zen fone 7/7 Pro 29.14.53.14 更新发布说明:**

*   更新的 Android 安全补丁(2020/11/5)
*   提高相机质量

另一方面，ROG 手机 3 的新更新版本号为 [17.0823.2012.122](https://zentalk.asus.com/en/discussion/42188/201218-zenfone-7-pro-7-ww-29-14-53-14) 。它为智能手机带来了最新的 12 月 Android 安全补丁，以及改进的 5G 连接稳定性，并修复了旁路充电功能，该功能偶尔会不起作用。新的更新还支持 Maxis 上的 VoLTE(马来西亚)、Telia(立陶宛)、Rakuten(日本)和 CSL/Smartone/3HK/CMHK(香港)上的 VoWIFI。至于游戏功能，新的更新为游戏精灵带来了自动锁定触摸功能。据说这个功能可以在屏幕闲置一定时间后打开锁定触摸模式。光环灯功能也得到一些稳定的修复时，使用手机与 ROG 装甲照明案件。

**华硕 ROG 手机 3 17.0823.2012.122 更新发布说明:**

*   更新至 2020 年 12 月 Android 安全补丁
*   启用 VoLTE on Maxis(马来西亚)、Telia(立陶宛)、Rakuten(日本)
*   已在 CSL/数码通/3HK/CMHK(香港)上启用 VoWIFI。
*   游戏精灵增加了自动锁定触摸功能。当屏幕闲置一段时间后，此功能会打开锁定触摸模式
*   改善了显示器的灰度性能
*   改进了 ROG 电话装甲照明盒的光环灯功能稳定性
*   修复了在清除一长串通知后系统 UI 没有响应的情况
*   修正了旁路充电功能偶尔不工作的问题
*   提高了 5G 连接稳定性
*   修复了用 SMULE 应用程序录制音乐时的音频爆裂声和静态噪音。
*   默认情况下，在“系统更新”中打开了“通过 Wi-Fi 自动下载和安装”设置

ZenFone 7 系列和 ROG Phone 3 的新 OTA(空中下载)更新正在分批推出。这意味着您可能需要几天时间才能收到更新通知。您可以通过转至“设置”>“系统”下的“系统更新”来手动检查更新。目前无法通过抓取固件文件来手动更新。
