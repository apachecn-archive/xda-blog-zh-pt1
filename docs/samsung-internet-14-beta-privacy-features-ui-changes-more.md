# 三星互联网 14.0 测试版推出了改进的隐私功能、用户界面变化等

> 原文：<https://www.xda-developers.com/samsung-internet-14-beta-privacy-features-ui-changes-more/>

在过去的几个月里，三星互联网浏览器增加了一系列新功能。仅在最近两次更新中，三星就推出了[额外的隐私和安全功能，新的高对比度模式](https://www.xda-developers.com/samsung-internet-13-0-secret-mode-privacy-features/)，可扩展的应用程序栏，观看视频的沉浸式模式，[重定向保护，网址快速访问](https://www.xda-developers.com/samsung-internet-adds-redirect-protection-url-quick-access-better-dark-mode/)，等等。现在，该公司正在测试频道推出另一个浏览器更新，带来了额外的隐私和安全功能，一些用户界面的变化和其他小的改进。

三星互联网 14.0 测试版[引入了](http://www.samsungmobilepress.com/pressreleases/samsung-internet-14-0-beta-is-now-available)一种新的智能反跟踪功能，可以检测网站何时试图跟踪你并自动阻止它。它还带来了一个新的安全控制面板，让您可以轻松地监控您的隐私设置。新的隐私控制面板还显示一些有趣的每日统计数据，如浏览器阻止的弹出窗口和跟踪器的数量。

*新的隐私和安全功能*

除了新的隐私和安全功能，三星互联网 14.0 测试版还支持三星可折叠智能手机上的 Flex 模式。得益于此，你现在可以在屏幕的上半部分观看视频，并在下半部分访问所有视频控件。每当你在 Galaxy Fold 或 [Galaxy Z Fold 2](https://www.xda-developers.com/samsung-galaxy-z-fold-2/) 上全屏播放视频时，这项功能就会自动开启。

*应用配对支持*

此外，三星互联网 14.0 测试版支持三星的应用程序对功能，这将允许您在分屏模式下打开浏览器的两个实例。它还引入了对自定义字体的支持，在 [Galaxy Tab S7](https://www.xda-developers.com/samsung-galaxy-tab-s7-review/) 系列上的 S Pen 手写识别，对阅读器模式的改进，以及新的翻译扩展。

*Galaxy Tab S7 系列上的 S Pen 手写识别*

如果你是三星互联网测试计划的一部分，你应该可以从今天开始从谷歌 Play 商店或三星 Galaxy 商店下载最新的测试版。但是如果你在稳定频道，你必须等到四月才能收到所有前述的稳定更新。
