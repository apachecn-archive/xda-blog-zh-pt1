# 最新的 Bixby 更新带来了几个新的增强

> 原文：<https://www.xda-developers.com/bixby-update-compact-ui/>

就受欢迎程度而言，三星的 Bixby 助手比谷歌助手或 Alexa 更接近微软的 Cortana。早在 2017 年推出 Bixby 时，三星就为它制定了宏伟的计划。该公司甚至曾经在 Galaxy Note 10 系列之前的旗舰智能手机上包括一个专用的 Bixby 按钮。尽管三星尽了最大努力，但它并没有成为谷歌助手或亚马逊 Alexa 的真正第三选择。然而，Bixby 仍然是三星移动和物联网生态系统不可或缺的一部分，三星致力于保持它的活力。4 月早些时候，三星推出了 Bixby 3.0 的大更新，增加了对印度英语和其他几个印度特有功能的支持。现在，它得到了另一个更新，进一步改善了用户体验，并使服务使用起来不那么烦人。

随着最新的更新(通过[*)SamMobile*](https://www.sammobile.com/news/bixby-big-update-improvements-check-out/#commentsSectionPlaceholderAnchor))，Bixby 在显示查询结果时不再占据整个屏幕。不占用太多空间的结果显示在一个紧凑的卡片上，就像谷歌助手一样。查看截图前后的内容:

Bixby 现在还能随着时间的推移学习你的喜好。如果您启用了“个性化 Bixby”选项，现在当您召唤服务时，它会向您显示个性化的基于应用程序的提示。

Bixby 最讨厌的部分之一是训练助手识别你的声音，这需要你大声说五次热门词，嗨 Bixby。令人欣慰的是，新的更新使培训部分完全可选，允许新用户立即开始使用 Hi Bixby 命令。

最后，在 Bixby 的主屏幕上，你会注意到一个新的“向主页添加快速命令”卡片。这可让您将快速命令图标添加到主屏幕，以便更容易地访问自定快速命令。

新的 Bixby 更新已经开始推广到受支持的 Galaxy 手机。如果你已经切换到使用谷歌助手，并想再给 Bixby 一次机会，去 Galaxy 商店买最新版本。
