# Windows 应用程序的交叉兼容层在 Mac、Linux 和 Chrome 操作系统上获得重大更新

CrossOver 21 现已推出，更好地支持在 Mac、Linux 和 Chrome OS 上运行 Windows 应用和游戏。

交叉是在其他操作系统上运行 Windows 应用程序最流行的方式之一。它基于开源的 [Wine 项目](https://www.winehq.org/) (CrossOver 的开发者是 Wine 代码库的一些最重要的贡献者)，但更容易安装数百个应用程序和游戏。CodeWeavers 现在发布了 CrossOver version 21，对所有三个操作系统进行了一些有益的改进。

[CrossOver 21](https://www.codeweavers.com/blog/jschmid/2021/8/3/hello-crossover-21) 基于 Wine 6.0，[于 1 月份首次发布](https://www.winehq.org/news/2021011401)，其中包括一个新的 Vulkan 后端，用于 Direct3D、DirectShow 和 Media Foundation 支持，以及一个更新的文本控制台。默认情况下，CrossOver 现在还包括 [Wine Mono](https://wiki.winehq.org/Mono) ，这是微软的开源实现。NET 框架。

除了 Wine 6.0 中存在的所有东西，新的更新对 Mac 进行了一些改进。现在支持黑暗模式，Xbox Series X/S 和 PlayStation 5 DualSense 控制器在 macOS Big Sur 11.3 上支持蓝牙。在 Linux 上，Office 365 修复了各种登录和多显示器问题，CrossOver 现在在 Chrome OS 上的启动时间更快了。

CrossOver 的主要卖点(没错，是付费软件)是它在 Wine 之上增加了一个易于使用的界面，为数百个流行的应用程序和游戏预先配置了安装程序和设置。在这方面，它类似于 [Lutris](https://lutris.net/) 或 PlayOnLinux/PlayOnMac，但有实际的客户支持。
