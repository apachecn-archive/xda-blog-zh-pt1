# Chrome OS 终于有了一个垃圾文件夹来恢复删除的文件

> 原文：<https://www.xda-developers.com/chrome-os-trash-folder-undelete-files/>

谷歌的 Chrome 操作系统得到了一个垃圾文件夹。虽然这听起来像是一个小细节，但问问任何一个拥有 Chromebook 的人，他们都会告诉你这真的很令人兴奋。运行 Chrome 浏览器的其他操作系统的用户从未遇到过这个问题，因为他们可以使用原生的回收站/回收站功能。

尽管绝大多数 Chrome OS 文件都保存在云中，但有很多情况下，你可能想恢复或“恢复”收到的物理文件，却发现你做不到——一旦你点击了删除按钮，它们就不见了。

然而，在过去的两周，Chrome OS 的金丝雀版本中出现了一面新旗帜。叫做*#文件-垃圾桶*，标签为 *" **启用文件垃圾桶。**在“文件”应用程序中启用“我的文件”宗卷的废纸篓。- Chrome OS"* 。据谷歌 报道，该标志似乎是由一名投稿人提交的，而不是谷歌的官方功能，但后来被一个官方账户重新上传。

如果你在 Chrome OS 的金丝雀频道上，请注意，垃圾桶文件夹的测试版将在未来几天内移动，从我的文件的子文件夹变成它自己的虚拟卷，与下载文件夹的操作方式相同。稍后，将会添加第二个 Trash 文件夹，供 Chrome OS 中运行的 Linux 应用程序使用，这应该允许它在没有大量配置的情况下与 Linux 文件管理器一起工作。

*图片积分:9 比 5 谷歌*

目前，即使标志已经被激活，找到垃圾文件夹的唯一方法是按下 *Ctrl+。*显示隐藏的文件夹和文件。我们希望当 Trash 文件夹到达稳定通道时，访问起来会容易得多。据推测，删除文件会触发一个带有“撤销”选项的祝酒词，就像在 Gmail 中发生的一样。

尽管这一切令人兴奋，但它绝对是一项正在进行的工作。当前的金丝雀版本是 88 版，即使开发完成，我们最早也要到一月份才能在稳定的渠道中看到。目前的稳定版本- [ChromeOS 86](https://www.xda-developers.com/google-chrome-86-rolls-out-with-flags-for-menu-icons-back-forward-cache-and-more/) -不包含该标志。
