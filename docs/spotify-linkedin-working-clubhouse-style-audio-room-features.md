# Spotify 和 LinkedIn 也在开发俱乐部风格的音频室功能

> 原文：<https://www.xda-developers.com/spotify-linkedin-working-clubhouse-style-audio-room-features/>

Clubhouse 是一个社交媒体应用程序，人们可以在那里创建语音室或收听现有的语音室，它的突然流行导致其他几个应用程序复制其功能。现在看来，下一个拥有俱乐部式音频室的应用可能是 Spotify 和 LinkedIn。

[Spotify 今天宣布，它已经收购了贝蒂实验室](https://newsroom.spotify.com/2021-03-30/spotify-acquires-locker-room-and-announces-plans-for-a-new-live-audio-experience/)，这是一个主要为体育迷设计的 iOS 专属平台，是直播音频应用[更衣室](https://apps.apple.com/us/app/id1517524960)的开发者。它的功能已经非常类似于 Clubhouse，Spotify 计划扩大其覆盖范围和功能。

“在未来几个月里，”该公司在一篇博客文章中说，“Spotify 将发展和扩大更衣室，为更广泛的创作者和粉丝提供增强的现场音频体验。通过这种新的现场体验，Spotify 将提供一系列体育、音乐和文化节目，以及一系列互动功能，使创作者能够与观众实时联系。我们将为职业运动员、作家、音乐家、歌曲作者、播客和其他全球之声提供机会，以主持实时讨论、辩论、向我提问(AMA)会议等。”

每个人最喜欢的社交媒体网络 LinkedIn 也计划实现音频室。应用程序开发人员 Alessandro Paluzzi 在 LinkedIn 的代码中发现了该服务正在开发该功能的证据。音频室还没有功能，但目前的设计看起来类似于 Clubhouse 选择的人有演讲的角色，而其他人可以作为听众加入房间(稍后请求加入扬声器)。

在 Paluzzi 发布这些截图后不久，LinkedIn 的一位高级产品总监[回复了](https://twitter.com/szetopia/status/1376971412368318466)这条调侃即将推出的功能的推文。

其他几个应用和平台目前正在开发类似俱乐部会所的功能，包括 [Slack](https://www.xda-developers.com/slack-add-clubhouse-style-audio-rooms-stories/) 、 [Discord](https://www.xda-developers.com/discord-is-working-on-their-own-clubhouse/) 、[脸书](https://www.xda-developers.com/facebook-telegram-clubhouse-clones/)和 [Telegram](https://www.xda-developers.com/telegram-voice-chat-2-0/) 。音频室变得像故事一样受欢迎可能只是时间问题(Spotify 和 LinkedIn 已经这样做了)。
