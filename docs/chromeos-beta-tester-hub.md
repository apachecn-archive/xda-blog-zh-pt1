# 在新的 ChromeOS Beta 测试中心发现即将推出的 ChromeOS 功能

该中心列出了每个新的 beta 里程碑版本中的所有 beta 特性

为了帮助用户轻松跟踪 ChromeOS beta 版本中的所有新功能，谷歌现在已经建立了一个 ChromeOS Beta 测试中心。该中心列出了 ChromeOS beta 里程碑版本中正在开发的功能，便于用户在稳定推出之前在 Chromebooks 上试用。

ChromeOS Beta 测试中心向公众开放，因为谷歌旨在鼓励更多用户测试 Beta 功能，并与 ChromeOS 开发团队分享反馈。但是，hub 的一部分仅限于金牌+产品专家。这一专属部分提供了关于早期开发阶段特性的信息。

在 Chromebook 支持论坛上的一篇帖子中，谷歌表示*的目标是“这个平台是为了提供一个空间，让所有 ChromeOS 用户都可以轻松看到和访问 ChromeOS 测试版功能，无论你是想加入测试工作，还是只是想先睹为快，看看你可以期待未来的 ChromeOS 更新。”*但很明显，该公司希望更多用户测试测试版，因为它向首批 500 名用户提供了谷歌赠品，供他们下载最新的 ChromeOS 测试版更新。

那些加入测试社区的人，无论动机如何，都将经常与 ChromeOS 产品团队见面，以了解正在开发的功能。此外，beta 测试人员将有一个平台直接与团队分享反馈。您可以通过下面的链接访问 ChromeOS Beta Tester Hub，了解最新测试版中的所有新功能。

**[查看 ChromeOS Beta 测试中心](https://chromeos.betacommunity.com/)**

就在 ChromeOS Beta Tester Hub 发布前几天，谷歌[将 ChromeOS 上的 Steam 支持扩展到测试频道](https://www.xda-developers.com/steam-chromeos-beta-channel-expansion/)，让更多用户有机会在 Chromebooks 上玩 Steam 游戏。通过查看我们之前的报道，了解更多关于 ChromeOS 上的 Steam 支持。
