# 以下是火狐的“质子”重新设计将如何调整浏览器用户界面

> 原文：<https://www.xda-developers.com/firefox-proton-redesign-browser-ui/>

Mozilla 正在对其桌面 Firefox 浏览器进行重大的重新设计。但是在最终产品向消费者发布之前——在五月中旬的某个时候——我们可以先睹为快更新后的浏览器会是什么样子。

代号为“质子”的火狐改版在本月初首次被 Mozilla 取笑。现在，该公司已经分享了一些模型(通过 [*Techdows*](https://techdows.com/2021/01/an-early-look-at-firefox-proton-user-interface.html) )，揭示了当版本 90 发布时 Firefox 将经历的微妙视觉变化。

*图片:Mozilla*

据 Techdows 报道，Mozilla 正计划调整一些 Firefox 用户界面元素，包括地址栏、工具栏、标签栏、汉堡菜单、信息栏和门挂钩。在其中一个模型中，我们可以看到 Mozilla 将使定制新标签页的外观比以往任何时候都更容易。用户可以选择他们喜欢的主题，背景，以及热门网站和口袋推荐。

*图片:Mozilla*

据称，Mozilla 还将改变 Firefox 中的汉堡菜单。最大的变化之一将是使菜单不那么令人眼花缭乱。在当前版本中，当你点击汉堡菜单时，大约有 20 个选项出现，从打开新标签到改变缩放比例等等。今后，你的火狐菜单将只显示你经常使用的项目。如果没有看到您想要的设置，您可以展开菜单查看更多。

*图片:Techdows*

最后，新的 Firefox 更新似乎也将引入圆形标签，这改善了浏览器的整体外观。当前版本的 Firefox 使用了更尖锐的边角。

如果你对这些变化感兴趣，你可以尝试一下这些新功能。Firefox 通过在地址栏中输入“about:config”可以很容易地调整开发者设置。然后搜索“质子”并将 browser.proton.enabled 选项切换到“真”Mozilla 警告说，调整这些功能可能会对性能产生负面影响。

你可以在 Mozilla 的 wiki 上关注火狐的质子更新。如果你对测试仍在开发中的功能感到不安，我们预计 Firefox 90 将于 5 月中旬发布，因此重新设计的版本不会太久。与此同时，你可以使用[其他已经正式发布的新功能](https://www.xda-developers.com/mozilla-firefox-83-pinch-to-zoom-https-only-mode/)，包括捏合缩放和仅 HTTPS 模式。
