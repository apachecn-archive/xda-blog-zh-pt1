本周科技:WhatsApp 新功能，无根主题回归 Android，等等

原文：https://www.xda-developers.com/this-week-in-tech-october-16/#0001-01-01

虽然上周是科技世界的多事之秋，但过去的一周相对缓慢。除了一些关于最新软件发布的新花絮，一些 WhatsApp 的新功能，以及一些 Pixel 6 的泄露，本周没有发生任何重大事情。然而，这只是暴风雨前的平静，因为未来一周将充满苹果和三星的一些重大公告。尽管如此，如果你错过了我们本周的任何报道，这里有一个本周技术世界所有重大发展的简要回顾。

WhatsApp 的新功能

脸书旗下的人气信使 WhatsApp 本周收到了几条更新。稳定频道的更新之一引入了一个已经开发了一年多的新功能——端到端加密备份。如果你是 WhatsApp 的普通用户，你现在可以从 Play Store 安装最新的更新，并使用唯一的密码或 64 位密钥对你的聊天备份进行加密。

WhatsApp 也收到了一些测试版频道的更新，让我们看到了一些正在开发的功能。我们在 APK 的应用程序拆除中了解到即将推出的社区功能，这可能是更好地组织 messenger 中的群组的另一种方式。我们还了解到，WhatsApp 正在努力改进语音记录功能，增加了一个新选项，让用户暂停语音记录。

无根主题回归安卓

由于新的人造覆盖 API，无根主题在 Android 12 中卷土重来。正如 XDA 的 Zachary Wander 在本周的一篇文章中强调的那样，开发者可以利用这个 API 中的漏洞来实现主题，而不需要 root 权限或签名级别的许可。如果你有兴趣为运行 Android 12 的设备开发一个无根主题，你可以通过上面的链接了解更多关于 API 的信息。

随着这一发现，我们本周首次看到了 Android 12 的兼容性定义文件(CDD)。虽然它不包括任何有趣的面向消费者的变化，但它确实概述了面向 OEM 厂商运行 Android 12 的设备的新性能类别。你可以通过查看我们之前的报道来了解这些课程。

Windows 更新和性能问题

微软本周推出了第一个 Windows 11 发布后更新。此次更新引入了几个错误修复。然而，据报道，它没有 AMD 处理器的性能问题。事实上，一些报告表明更新加剧了问题，所以我们建议如果你有一台基于 AMD 的 PC，不要安装它。不过，你可能不用等太久就能安装更新，因为微软已经开始测试 Windows Insider 测试版和发布预览版的补丁。

微软还在开发频道推出了新的 Windows 11 版本，重新设计了表情符号，支持更新堆栈包，等等。然而，据报道，用户对新表情符号的重新设计不满意。此外，微软在微软商店上发布了 Windows 10 build 19043.1288 和 Windows 子系统 for Linux。我们还简要了解了 Windows 11 上运行的 Android 应用。

甚至更多的像素 6 泄漏

谷歌似乎很难隐瞒其即将推出的 Pixel 6 系列的信息。本周，我们看到了更多关于 Pixel 6 和 Pixel 6 Pro 的泄露，包括更多高分辨率渲染，关于 Pixel Pass 的信息，以及一则泄露的设备广告。此外，我们在 Pixel Stand 应用程序的最新更新中发现了关于第二代 Pixel Stand 的更多细节，并了解到 Pixel 6 系列可能会获得 Android 16 和一年的额外安全补丁。
