OneDrive 终于有了 ARM PCs 的原生应用[更新]

原文：https://www.xda-developers.com/microsoft-onedrive-sync-arm-preview/#0001-01-01

微软终于发布了【OneDrive 的一个版本，旨在原生运行在 ARM 电脑上，包括基于 Windows 的 ARM 设备和苹果硅 MAC 电脑。如果你有 Windows 或 macOS 的 OneDrive 应用程序，这意味着你现在可以选择获得原生 ARM 应用程序，而不是模拟版本。反过来，这意味着您可以让同步功能以稳定的性能正常工作。微软很久以前就已经宣布了 OneDrive 的 ARM 版本，所以有些人可能会惊讶地发现它还没有上市。

OneDrive 的文件同步已经有很长一段时间了，直到最近，还没有太大的问题。然而，大多数电脑使用英特尔或 AMD 处理器，现在，我们在市场上有一些基于 ARM 的设备。该应用程序的 x86/x64 版本在技术上支持同步，但在 ARM 设备上运行会非常慢，因为不是本地运行的。通过今天的更新，你可以获得适用于 ARM 设备的 OneDrive 应用程序的原生版本，这意味着同步性能现在应该会好得多。

考虑到微软在两年前发布了第一款基于 ARM 的设备，你可能会期望这种能力来得更早一点，特别是对于 Windows。不过，它终于来了，虽然现在还在预览中。如果你想尝试新版本的应用程序，你需要进入 OneDrive 应用程序的设置，并启用该选项以获取 OneDrive 的预发布微软版本，它应该会在下周推出。你应该在这个页面上看到一个按钮，显示你也可以获得 arm 64(Windows)或 Apple Silicon(MAC OS)版本的应用程序。

这是让基于 ARM 的设备获得更合法体验的又一步，特别是在 Windows 上，在许多层面上，它们仍然无法与英特尔或 AMD 设备相提并论。 Windows 11 已经在这方面取得了一些进展，增加了对 ARM 上 x64 仿真的支持，微软也使开发者有可能逐步调整他们应用程序的部分内容，以便他们可以逐步采用原生 ARM 技术。

由于新的高通骁龙 8cx Gen 3 ，ARM 设备上的 Windows 也将在 2022 年上半年获得显著提升，该产品承诺比当前一代产品性能提升 85%。你可以阅读我们使用骁龙 8cx Gen 3 参考设计的芯片的早期经验。

【UPDATE 12/6/2021 @ 14:48 ET】由于沟通有些不清楚，这篇文章最初声称同步功能正在添加到一个预先存在的应用程序中。文本已被更正，以表明微软只是推出了一个原生 ARM 版本的应用程序，以提供更高的性能。我们对造成的混乱表示歉意。
