未经修改的 Windows PC 游戏可能会进入谷歌体育场

原文：https://www.xda-developers.com/google-stadia-unmodified-windows-games/#0001-01-01

如果即将举行的谷歌游戏开发峰会上的一次会议有任何迹象的话，谷歌体育场可能很快就会成为未经修改的 PC 游戏的家园。为该活动列出的一个会议名为“如何从头开始为 Linux 编写一个 Windows 模拟器？”这意味着强调谷歌如何做到这一点。

Stadia 是一个基于云的游戏平台，其核心是基于 Linux 的强大服务器。因此，虽然性能限制并不是真正的障碍，但让 PC 游戏在 Stadia 上运行并不像看起来那么容易。这就是为什么当你考虑到有多少 PC 游戏时，Stadia 上可用的游戏库很小。

这次会议表明谷歌已经知道如何在 Linux 上运行 Windows 游戏而不需要修改它们，这都要归功于仿真。理论上，这意味着任何 Windows 游戏都可以在 Stadia 上出现。

然而，会议的描述方式引发了一些问题。它是这样说的:

详细概述了谷歌在 Stadia 上运行未经修改的 Windows 游戏的解决方案背后的技术。这是一些核心概念的深度技术演练，目的是让好奇的程序员更好地理解这些技术，并有可能构建他们自己的技术。

如果谷歌已经找到了在 Linux 上运行 Windows 游戏的方法，那么在 Stadia 上，它会分享该解决方案的技术细节，而不是让开发者随意使用，这很奇怪。另外，该描述指出开发人员可以基于这些知识“潜在地构建他们自己的”解决方案。这可能意味着谷歌打算让开发者创建他们自己的仿真工具，以便将游戏带到体育场。这也可能是向合作伙伴出售 Stadia as a service(Stadia 即服务)的计划的一部分，为他们提供所需的工具，以最少的努力通过云提供游戏。

另一方面，这一知识可以让 Linux 粉丝更容易在他们喜欢的操作系统上玩 Windows 游戏。这甚至可能包括 Chrome 操作系统，因为它也是基于 Linux 的。该会议将于 3 月 15 日举行，所以我们很快就会知道谷歌是否有任何切实的计划将 Windows 游戏带到 Stadia。

随着 Steam Deck 最近也预装了基于 Linux 的操作系统，看起来 Linux 上的游戏在不久的将来会有显著的增长。
