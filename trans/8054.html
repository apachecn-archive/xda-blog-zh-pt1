Chrome 开始测试侧边搜索和旅程，以改善搜索体验

原文：https://www.xda-developers.com/chrome-side-search-journeys-experiment/#0001-01-01

Chrome 开始测试侧边搜索和旅程，以改善搜索体验

谷歌已经开始测试 Chrome 的两个新功能，名为 Journeys 和 Side Search，以帮助改善搜索体验。

我们最近得知 Chrome 团队正在开发微软 Edge 风格的“侧边搜索”功能。Chromium Gerrit 的一项功能承诺显示，它将使用户更容易获得相关的搜索结果。然而，它没有强调任何进一步的细节。谷歌现在已经开始在 Chrome OS 开发频道测试该功能，该公司还分享了其他信息。

根据最近在 Chromium 博客 上的一篇文章，新的侧面板旨在帮助用户比较搜索结果。它出现在浏览器的左侧，显示最近搜索的结果。侧面板本质上让用户可以同时查看搜索结果和其他结果。

谷歌解释说，这个实验的“目标是探索 Chrome 如何更好地帮助用户轻松比较结果，”就像上个月为 Android 推出的 Chrome 94 测试版的连续搜索功能。正如你在附件 GIF 中看到的，你可以通过点击搜索栏旁边的 G 图标来访问新的侧面板。

谷歌目前正在通过谷歌搜索在 Chrome OS 开发频道测试该功能。该公司计划将其引入更多平台，并在未来增加对更多默认搜索引擎的支持。

谷歌还在测试一项名为“旅程”的新功能，让你可以在一个标签页中轻松访问你的浏览器历史中的所有相关页面。谷歌解释说，旅程会将与某个关键词相关的所有页面聚集到 Chrome 历史页面的一个标签中，让你“盘点已完成的研究，重温那个有用的金块，并继续探索某个主题。”

Journeys 在最新的 Chrome Canary 桌面版本中可用。要访问它，你可以前往 chrome://history，然后选择新的旅程标签。如果您不想使用它，您可以通过点击同一页面上的“关闭旅程”按钮轻松禁用它。在目前的形式下，Journeys 只能处理设备上的历史记录，不会与你的谷歌账户同步任何信息。然而，谷歌可能会根据用户反馈在未来版本中提供该功能。
