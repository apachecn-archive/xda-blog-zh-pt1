Chrome 操作系统上的计算器现在是一个渐进式网络应用程序

原文：https://www.xda-developers.com/chrome-os-calculator-pwa/#0001-01-01

谷歌一直在慢慢远离传统的 Chrome 应用程序，现在平台无关的渐进式网络应用程序是一个有能力的替代品。Chrome OS 上的 Calculator 应用程序就是这样一个传统应用程序，但是从最近的 Chrome OS 97 更新开始，它现在已经成为一个可以在任何浏览器上使用的 PWA。

新的渐进式网络应用程序位于 calculator.apps.chrome (通过9 to 5 GoogleT5)。它有一个带有图标和离线缓存的 web 清单文件，所以基于 Chromium 的浏览器可以将它安装到您的设备上。它也可以在任何网络浏览器上正常工作，功能几乎和旧的计算器一样。基本的加法、减法、乘法和除法是可用的，还有一些更复杂的函数，如平方根、对数和指数。

尽管计算器现在可以在任何带有网络浏览器的设备上使用，但它显然仍打算在计算机上使用——将窗口调整为较小的尺寸或纵向会导致一些控件被隐藏。

计算器的样式仍然与旧的材质设计相匹配，当您单击“清除”按钮时，还会完成一个边角擦拭效果。谷歌还没有更新它来匹配更新的材料你设计语言，但它只是一个计算器，所以它真的不需要来。如果默认计算器不能满足你的所有需求，Chrome OS 上还有很多其他数学和计算器工具。Wolfram Alpha 是我个人的最爱。

这是在过去几个月 Chrome OS 的许多其他变化和改进之后发生的，例如开发中的货架共享功能、摄像头改进、与 Android 手机的增强集成，以及更多选择说话的声音。该平台还面临(略微)更多的竞争，因为微软在 K-12 教室推广 Windows 11 SE，过去十年的大部分时间 Chromebooks 一直在 K-12 教室占据主导地位。
