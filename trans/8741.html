如何在 Windows Media Player 中免费播放 HEVC 编码的媒体

原文：https://www.xda-developers.com/play-hevc-encoded-media-windows-media-player/#0001-01-01

微软几十年来一直在 Windows 中包含视频和音频播放器，但 Windows 10 和 11 在没有额外(付费)包的情况下无法播放一些媒体格式。一些音频和视频编解码器需要付费许可证才能在商业产品(如 Microsoft Windows)中使用，因此，在播放一些文件时，您可能会看到指向媒体编解码器的 Microsoft Store 页面的链接，而不是将所有这些费用集中到每个 Windows 副本中。幸运的是，至少有一些编解码器包可能对你是免费的。

高效视频编码，也称为 HEVC 或 H.265，是一种常见的媒体编解码器，不能在商业产品中免费使用。然而，正如在 Reddit 上的 Froggypwns 指出的，许多显卡都附带了使用 HEVC 的许可证，这意味着你可能不必在你的电脑上再次购买。要找到答案，只需打开 Windows 运行对话框(同时按 Windows/super 键和“R”，或者右键单击开始菜单并单击“运行”)，粘贴以下 URL，然后按键盘上的 Enter 键:

 ms-windows-store:

微软商店应该打开一个“来自设备制造商的 HEVC 视频扩展”的页面如果您的电脑硬件附带许可证，您只需单击“安装”即可下载编解码器。很简单。

XDA 的亚当·康威对此进行了测试，点击 Windows 10 顶部的“安装”按钮成功了。

如果您的电脑不包含 HEVC 许可证，有许多其他方法可以在您的 Windows 电脑上使用该编解码器(和其他专有媒体格式)播放媒体。最受欢迎的选择可能是 VLC 媒体播放器，这是一个免费的开源媒体播放器，可以处理几乎任何可以想象的视频或音频格式。你可以从官方网站下载 Windows 版本，虽然 VLC 也可以在微软商店买到，但商店版本是功能较少的 UWP 应用程序。你可能最好使用普通的桌面 VLC 应用。

https://apps.microsoft.com/store/detail/9nblggh4vvnh
