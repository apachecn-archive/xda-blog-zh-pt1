质子邮件被迫释放用户的 IP 地址，但它有一个很好的理由

原文：https://www.xda-developers.com/protonmail-forced-to-release-user-ip-address/#0001-01-01

ProtonMail 是一项因致力于隐私保护而广受赞誉的服务。这是记者们特别喜欢的，因为它可以被告密者等安全地使用，而不用担心暴露他们的身份。然而，ProtonMail 最近被瑞士当局强迫公开它记录的一个用户的 IP 地址，尽管它有一个很好的理由。

几个月前，瑞安航空 FR4978 航班从希腊雅典飞往立陶宛维尔纽斯。然而，在明斯克国家机场和国有企业立陶宛机场收到炸弹威胁后，该航班转向白俄罗斯的明斯克机场。根据国际民航组织的事实调查，这些电子邮件是从一个 ProtonMail 地址发出的，特别有趣的是，用于创建帐户的 IP 地址也被 ProtonMail 发布给了当局。

飞机在明斯克降落后，记者罗曼·普罗塔塞维奇和他的女友索菲亚·萨佩加被捕。立陶宛的调查人员对此事展开了审前调查，这就是为什么 ProtonMail 被迫通过“法律互助机制”向当局公布 IP 地址。事件发生后，美国最近还指控白俄罗斯政府“空中海盗行为”，指控四名白俄罗斯政府官员。

ProtonMail 在 2021 年 5 月的一份声明中称瑞士政府的正式要求迫使其透露了所掌握的信息，尽管它明确表示邮件本身是由调查记者发布的。“我们支持欧洲当局的调查，因为根据瑞士政府的正式请求，我们有法律义务这样做。”

然而，许多人可能会感到惊讶的是，ProtonMail 能够在第一时间公布帐户创建者的 IP 地址。该公司对隐私的承诺(如完全加密的电子邮件和邮箱的内容所示)会让许多人认为它无法恢复 IP 地址。我联系了 ProtonMail 询问这怎么可能，我被告知该公司不对具体案例发表评论。然而，我得到了以下关于公司隐私政策的回复。

"与开户相关的数据"详细说明了为防止滥用和保护用户而采取的步骤。具体来说:“提供的 IP 地址、电子邮件地址和电话号码会被临时保存，以便向您发送验证码并用于反垃圾邮件目的

似乎当立陶宛政府意识到炸弹威胁是一个骗局时，就联系了 ProtonMail。被迫降落在白俄罗斯的瑞安航空公司航班于 2021 年 5 月 23 日起飞，这意味着该公司在那个时间点保留了一周多一点的 IP 地址日志。目前还不清楚 ProtonMail 会在 IP 地址、电子邮件地址和电话号码最后一次使用后保留多久。

不过，有一点是清楚的，那就是该公司没有违背对电子邮件内容进行加密的承诺。所有调查人员能够从 ProtonMail 收集到的是账户创建日期和时间，以及用于创建账户的 IP 地址。同样值得注意的是，电子邮件发件人的 IP 地址似乎没有被记录，只有该 IP 地址被用来创建帐户。
