Stardock 的 Start11 可以让你修复 Windows 11 中的开始菜单

原文：https://www.xda-developers.com/stardocks-start11-lets-you-fix-the-start-menu-in-windows-11/#0001-01-01

你知道你讨厌窗户上的什么东西，你真的希望你能改变吗？对，那个的东西。这是 Stardock 旨在通过其广泛的定制产品解决的问题，Start11 试图修复 Windows 11 开始菜单。如果你是认为新操作系统的开始菜单不好的人之一，那你来对地方了。自 8 月以来一直处于测试阶段，Start11 候选版本现已推出。

对于初学者(没错；我去了)，你可以把开始菜单改成 Windows 7 或者 Windows 10 的设计。抱歉，Windows 8 粉丝。

还有其他定制选项。例如，你可以改变周围的任务栏。您可以添加织物、深色木材、碳纤维等纹理。但这还不是全部，因为你实际上可以改变任务栏的大小和位置。

没错；Windows 11 中取消了将任务栏放在屏幕侧面或顶部的功能，Start11 又恢复了该功能。你甚至可以把它放在不同显示器的不同面上。它还解决了另一个大问题，允许你在任务栏上右键单击时使用 Windows 10 上下文菜单。

当 Stardock 发布了 Start8 作为 Windows 8 的补丁时，它在这一特定领域获得了一定的知名度。你可能还记得 Windows 8 是最有争议的操作系统版本之一，它完全取消了开始菜单，支持开始屏幕，并取消了开始按钮。Start8 提供了一个实用程序来返回每个人都满意的开始菜单。

但 Stardock 实际上有一系列定制产品，属于其 Object Desktop 应用程序套件。Groupy 是一款可以让你将应用程序分组到标签中的应用程序，类似于微软当年的设置。窗帘能让你更深入地定制主题。还有很多，你可以点击这里查看。
