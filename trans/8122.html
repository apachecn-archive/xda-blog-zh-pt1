AMD 处理器在 Windows 11 上运行速度较慢，但修复即将到来

原文：https://www.xda-developers.com/amd-cpus-slower-windows-11-fix-is-coming/#0001-01-01

在与 Windows 内部人士进行了三个多月的公开测试后，Windows 11 于本周开始在 T2 推出。虽然推出的操作系统带有几个已知的问题，但似乎有一个可能被人不知不觉地漏掉了。AMD 已经证实其处理器受到 Windows 11 升级的负面影响，在某些情况下导致性能下降。谢天谢地，决议已经在路上了。

AMD 指出了其处理器和 Windows 11 的两个明显问题。第一个与处理器上的 L3 缓存的延迟有关，在 Windows 11 上可能会高出三倍。这将特别影响可能依赖于内存子系统访问时间的应用程序的性能，包括一些快节奏的电子竞技游戏。AMD 表示，在非常特定的游戏中，性能可能高达 15%，但一般来说，性能影响在 3%至 5%之间。

对于这一点，AMD 表示，随着本月晚些时候 Windows 11 的更新，可能会在下周的周二补丁更新中修复。这将是 Windows 11 上市以来的首次累积更新，因此我们应该会看到大量的错误修复与此同时推出。

另一个问题与 UEFI 系统中的“首选核心”功能有关，该功能不会为给定场景中最快的可用核心调度任务。这可能会影响主要依赖单个或几个内核的应用程序的性能，因为多线程任务无论如何都会使用所有可用的内核。因此，在具有高内核和线程数的处理器中，性能影响可能更加显著，特别是那些具有 65W TDP 或更高的处理器。AMD 没有明确说性能影响可能有多大。

不管怎样，解决这个问题的方法也在工作中。AMD 表示，它将通过“软件更新”来实现，而不是 Windows 更新，因此它可能会以 Windows 11 新驱动程序发布的形式出现。这个修复也有望在本月的某个时候完成。

Windows 11 正在逐步推出，很可能许多用户还没有升级，所以这应该不是一个大问题。像这样的重大更新后出现问题并不是没有听说过，至少修复会相对较快。我们还听说了一个基于虚拟化的安全性问题，该问题可能会影响游戏性能，目前微软正在对此进行调查。如果你还没有，请阅读你的 Windows 11 评论，看看我们为什么向所有使用 Windows 10 的人推荐升级。
