在 Windows 11 中，将时间和日期添加到第二台显示器的任务栏中

原文：https://www.xda-developers.com/windows-11-taskbar-time-date-second-monitor/#0001-01-01

与 Windows 10 相比，Windows 11 在设计上是一股清新的空气，我们在的 Windows 11 评论中全力推荐这一升级。但说实话，并不是每一个变化都是积极的，你可能会觉得讨厌的一件事是，你再也无法在 Windows 11 的第二个或第三个显示器上看到任务栏上的日期和时间。只有主显示器显示它，但幸运的是，有办法解决这个问题。

好消息是微软已经在努力改变这种行为，你现在就可以通过加入 Windows Insider 计划来尝试这些改变。如果你不想这样做，有一些应用程序也可以恢复这种功能。或者，如果你不介意等待，这项功能应该会在 3 月份向所有 Windows 11 用户推出，所以你什么都不用做。以下是在 Windows 11 中，如何将时间和日期添加到第二台显示器的任务栏中。

选项 1:加入 Windows Insider 计划

我们会说，如果你想在 Windows 11 中的第二台显示器的任务栏上添加时间和日期，这是最简单的选择。微软最近在测试版和发布预览版渠道向内部人士发布了 Windows 11 build 22000.526 ，这意味着试用它是相当安全的。正如你可以在我们的Windows Insider 项目的解释者中读到的，发布预览频道是微软在发布稳定版之前测试更新的地方。这意味着它们通常实际上已经完成，你不应该遇到任何大问题。

要以这种方式在任务栏上恢复时间，请执行以下操作:

  • 打开设置应用，进入 Windows 更新部分，然后选择 Windows Insider 程序。
  • 点击开始，选择一个微软账号链接到 Insider 计划，然后在被问及加入哪一个时选择发布预览频道。任何其他选择都可能意味着你会得到不稳定的 Windows 11 版本，所以发布预览版是你最好的选择。
  • 出现提示时，重新启动您的电脑。
  • 电脑重新开机后，请返回 Windows Update 并检查更新。您应该会看到标记为 KB5010414 的累积更新。
  • 让更新安装并重新启动你的电脑。现在，您应该可以在所有显示器的任务栏上看到时间和日期。

收到此更新后，您可能需要取消注册 Windows Insider 计划。回到相应的页面，取消您的电脑的预览版本注册，您将会收到每月定期更新。

选项 2:使用 ElevenClock 自定义 Windows 11 任务栏上的时间和日期

如果你不想加入 Windows Insider 计划，另一个将时间添加回任务栏的选择是使用一个名为 ElevenClock 的应用程序。这个应用程序可以在你所有的显示器上添加时间和日期，但如果你感兴趣，它还可以做更多的事情。

你需要做的就是从 GitHub 下载最新版本的 ElevenClock 并运行安装程序。一旦安装完成，您将立即在所有显示器的任务栏上看到时间和日期。如果你想自定义你的设置，你可以去你的开始菜单，打开 11 点钟设置，这给了你大量的选项。

您可以将时钟对齐任务栏的左侧，隐藏通知标记，等等。甚至还有一个为任务栏时钟添加秒数的选项，你可能会发现这很有用，因为微软也从 Windows 11 中删除了该功能。有很多选择，所以在你空闲的时候探索它们。

为了让这些功能发挥作用，你需要确保启动时 ElevenClock 正在运行。这是默认情况下，当你安装应用程序时启用，所以你只需要确保你没有禁用它。

这就是全部了。这是一个非常简单的修复，但如果你不想做这些，你可以等待微软发布 Windows 11 的下一个累积更新。下一个补丁星期二将在 3 月 8 日，这是更新将到来的时间，很可能会增加这个功能，虽然它也应该是一个可选的更新。如果出现任何问题，这种情况可能会改变，但不太可能发生。除了在第二台显示器上添加一个时钟之外，您还可以期待一个新的 Widgets 按钮在任务栏上显示当前天气，团队的新特性等等。
