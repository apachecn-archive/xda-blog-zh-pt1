谷歌的 Chrome OS Flex 将个人电脑和苹果电脑变成了 Chromebooks

原文：https://www.xda-developers.com/chrome-os-flex/#0001-01-01

尽管 Chrome OS 只打算在 Chromebooks 上使用，但多年来已经有许多项目旨在将该操作系统引入传统 PC。一个例子是 CloudReady OS，这是 Neverware 推出的 Chromium OS 的一个分支，主要面向希望为现有电脑提供更新软件体验的学校和企业。 Neverware 于 2020 年末被谷歌收购，现在谷歌正在将 CloudReady OS 转变为“Chrome OS Flex”。

谷歌在周二的一篇博客文章中宣布，“为了帮助更多的组织采用现代计算解决方案，谷歌在 2020 年收购了 Neverware。作为基于 Chromium OS 的操作系统 CloudReady 的开发者，他们已经帮助无数企业和学校实现了 PC 和 MAC 的现代化，包括 Nordic Choice Hotels、ABN AMRO 和 MyGate。从那以后，我们一直在努力将 CloudReady 的优势集成到新版本的 Chrome OS 中。”

就像最初的 CloudReady 产品一样，Chrome OS Flex 是 Chrome OS 的修改版本，可以安装在大多数 PC 和 MAC 上，与 Chromebook 上的界面相同。谷歌表示，Chrome OS Flex 拥有“与 Chrome OS 相同的代码基础和发布节奏”，运行 Flex 的计算机可以像 Chromebooks 一样进行远程管理——这是学校和企业的一个重要卖点。Chrome OS Flex 还包括谷歌助手、对家庭链接帐户的支持、智能锁、即时共享和附近的共享。

谷歌在一篇新的支持文章中指出了 Chrome OS Flex 和 CloudReady 之间的一些差异。CloudReady 家庭版(可以免费下载)允许命令行访问和禁用 rootFS 验证的选项，但这些功能在 Chrome OS Flex 中都不存在。然而，你显然可以使用标准 Chromebooks 上的相同 Linux 虚拟机。支持页面中的一个部分说“未来的改进计划允许用户选择性地重新启用这些开发模式风格的功能”，而同一篇文章说“如果管理策略允许，可以安装 Linux 环境”，因此尚不清楚该功能是否已经准备好。谷歌还用一个可下载的图片取代了所有以前版本的 CloudReady。

Chrome OS 和 Chrome OS Flex 之间仍然有一些重要的区别。谷歌在另一个支持页面的中表示，Android 应用支持和 Parallels Desktop 缺失，安全启动不如真正的 Chromebooks 强大。谷歌也不支持 ARM PCs，所以像 T4 Surface Pro X 和 2021 MacBook Pro 这样的机器无法启动 Chrome OS Flex。

谷歌正在与一些“大客户”和内部员工一起测试 Chrome OS Flex，一旦准备好黄金时间，CloudReady 设备将自动更新到 Chrome OS Flex。该公司尚未证实是否会允许普通人在自己的电脑上下载并安装 Chrome OS Flex，或者是否会限制管理员的访问权限(更新:谷歌已向 XDA 开发者确认 Chrome OS Flex 将可供个人使用)。
