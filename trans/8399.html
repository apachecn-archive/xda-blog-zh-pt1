AdDuplex 表示，超过 5%的 Windows 电脑已经安装了 Windows 11

原文：https://www.xda-developers.com/adduplex-windows-11-october-2021/#0001-01-01

根据来自 AdDuplex 的最新调查，Windows 11 已经在 5%的现代 Windows 电脑上运行。对于那些不熟悉的人来说，AdDuplex 是微软商店应用程序的交叉推广网络——也就是广告平台，它可以在超过 5000 个应用程序中找到。该公司从其平台支持的应用程序中收集使用数据，其中包括约 6 万台个人电脑，诚然这是一个相对较小的样本规模。数据收集于 10 月 26 日。

ad duplex 2021 年 10 月的报告显示，在内部人员和稳定版本的人员之间，超过 5%的用户已经在运行 Windows 11 的某个版本。具体来说，4.8%的用户运行的是 Windows 11 的稳定版本，build 22000，0.3%是 Insider 计划的一部分。另有 0.3%的 PC 也在 Insider 计划中，但卡在了 Windows 10 上。

根据调查，使用最多的 Windows 版本仍然是 Windows 10 2021 年 5 月更新版，即版本 21H1，有 37.6%的用户。排在第二位的是 Windows 10 年 10 月更新，占 34%，2020 年 5 月更新紧随其后，占 14.1%。这些都是目前支持的 Windows 版本，正如你所料，绝大多数用户都在运行这些版本。然而，大约 9%的用户仍在运行 Windows 10 版或更早版本。

看起来 Windows 11 似乎只在一小部分个人电脑上运行，但有很多因素使得 AdDuplex 的数字更加令人印象深刻。首先，微软没有像 Windows 10 首次发布时那样强迫用户更新。更新是可选的，即使这样，如果你在等待它出现在 Windows Update 中，它也会逐渐推出。同样值得指出的是， Windows 11 系统要求将许多个人电脑抛在后面，使得用户更难升级。当然，Windows 11 正式推出才三周左右，所以它还有时间进一步发展。

如果你还不是 Windows 11 的一员，你可以跟随我们的指南安装新的操作系统。如果你愿意冒这个险，你甚至可以在不支持的电脑上安装 Windows 11。如果你不确定是否应该升级，请查看我们的 Windows 11 评测，看看为什么我们认为这是一次值得升级的升级。
