OPPO 的 Find N foldable 比三星 Galaxy Z Fold 3 更小巧，更便宜

原文：https://www.xda-developers.com/oppo-find-n-foldable-preview/#0001-01-01

经过几个月的传言和上周的官方调侃，OPPO 终于公布了其首款可折叠手机 OPPO Find N 的全部细节。该声明是在该公司一年一度的 Inno Day 活动上发布的，该活动还展示了一副“辅助现实”智能玻璃和一款新的图像信号处理芯片。

OPPO Find N 是一款向内折叠的设备，类似于三星引领潮流的 Galaxy Z Fold 系列，最显著的区别在于显示屏的长宽比。例如，三星的 Galaxy Z Fold 3 在折叠时具有细长的电视遥控器形状，这导致了非常高和瘦的屏幕外纵横比为 25:9。Find N 的外部屏幕(5.5 英寸，1972 x 988)具有传统的 18:9 纵横比。这意味着 Find N 在折叠时会感觉更接近“普通手机”(5.5 英寸的尺寸会让它感觉非常紧凑)。然而，这也意味着当展开时，该设备将会特别宽。事实上，Find N 的 7.1 英寸内部屏幕(1792 x 1920)默认情况下是横向的。

OPPO 的工程师表示，这完全是设计使然:他们想要一个感觉不太拥挤的外部屏幕和一个适合视频和游戏的内部屏幕。到目前为止，内部可折叠屏幕在纸面上看起来令人印象深刻:这是一个 120 赫兹的 LTPO 面板，刷新率在 1 赫兹到 120 赫兹之间变化，和三星的可折叠屏幕一样，它被 UTG(超薄玻璃)覆盖，感觉不太塑料。但与三星的可折叠手机不同，Find N 的屏幕没有明显的褶皱，因为 OPPO 采用了华为和摩托罗拉用于其可折叠手机的“水滴”铰链技术。铰链也可以完全折叠，没有间隙，OPPO 表示，它已经过测试，可以承受超过 20 万次折叠。

OPPO Find N 的铰链还可以在任何角度保持中间折叠，这使得该设备可以在类似笔记本电脑的外形中使用。OPPO 将这项功能称为“FlexForm”，与三星的“Flex Mode”计费相差不远。

Find N 重 275 克，折叠后的厚度为 15.9 毫米，这两个数字都比 Z Fold 3 的 271 克和 14.8 毫米略重，但它们击败了更重更厚的华为 Mate X2 和小米 Mix Fold 。

在光学方面，Find N 配备了一个三摄像头主系统，由一个 50MP、1/1.56 英寸的索尼 IMX766 传感器以及一个 16MP 超宽和 13MP 长焦摄像头组成，两者都是索尼传感器。还有一对 32MP 的传感器。不幸的是，OPPO Find N 运行在高通骁龙 888 SoC 上，而不是新的骁龙 8 Gen 1 (OPPO 表示，Find N 在新 SoC 上市之前就已经在开发中)，这意味着 Find N 也将而不是使用 OPPO 的新 MariSilicon X 成像芯片，相反，成像将由骁龙 888 的内置 ISP 处理。

还有一个 4500 毫安的电池，可以以 33W 有线和 15W 无线充电，它还支持 10W 反向无线充电。不幸的是，OPPO Find N 没有官方的防水评级，但公司代表声称这款手机可以承受水的飞溅。

对于 foldables 的粉丝来说，游戏中另一位玩家的引入令人兴奋，尤其是因为 OPPO Find N 似乎结合了三星和华为的最佳设计。但是有好消息也有坏消息。

坏消息:Find N 将暂时只在中国销售，尽管公司代表在发布会前被媒体问及此事时并不排除将来会在国际上发布。

优点:这是迄今为止最便宜的可折叠大屏幕手机，零售价为 7699 元人民币(约合 1200 美元)。相比之下，三星的 Galaxy Z Fold 3 起价 1799 美元，小米的 Mix Fold 售价约相当于 1500 美元。
