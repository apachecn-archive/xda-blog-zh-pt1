链接清洁 PWA 可以帮助您快速清洁任何设备上的链接

原文：https://www.xda-developers.com/link-cleaner-pwa/#0001-01-01

与他人分享网络链接通常会很烦人，跟踪代码和其他不必要的变量通常会被应用程序或推荐网站添加到末尾。在台式电脑上清除多余的垃圾通常很容易，但在手机上这个过程可能会更困难和耗时，因为触摸屏上的文本选择并不总是很好。不过，有一款网络应用可能会有所帮助:链接清理器。

Link Cleaner 是我创建的一个渐进式网络应用程序，可以通过任何浏览器在 linkcleaner.app 访问，只需点击一下就可以删除网络链接中不必要的部分。粘贴一个链接会立即给你一个干净的版本，一键选择将新链接复制到剪贴板，将其共享到设备上的其他应用程序，或显示二维码，以便周围的人轻松扫描。

除了打开网站，还有一些其他方法可以缩短链接。由于它是一个渐进式的网络应用程序，你可以将它作为一个系统应用程序“安装”在 Chrome 和 Microsoft Edge 等浏览器上。在 Android 上，安装 Link Cleaner 也会将其添加到系统共享菜单中，这样你就可以直接从任何一个有共享按钮的 app 上发送链接到 Link Cleaner。这样，它可以在共享链接时用作快速过滤器:首先点击链接清理器，然后点击链接清理器的共享按钮，最后选择所需的目的地。iOS 上的 Safari 不支持所需的 Web 共享目标 API，但 Siri 快捷方式具有相同的功能。

我最初创建 Link Cleaner 是为了减少来自任何网站的链接，但最近，我最常使用它的时候是在从 Twitter Android 应用程序共享推文时，它总是在 URL 的末尾添加跟踪变量，这些变量总是需要几秒钟才能手动删除。当我在平板电脑上写文章时，我几乎随时都会使用它，因为这可能意味着需要尽可能多地清理几十个链接，而用 Android 的文本选择一遍又一遍地这样做并不有趣。

Link Cleaner 在 GPLv3 许可下是开源的，所有代码都可以在 GitHub 上获得。
