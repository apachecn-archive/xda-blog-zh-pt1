零日影响 Windows 10、Windows 11 和 Windows Server，任何人都可以获得管理员权限

Windows 零日允许任何人获得 Windows 10、11 和 Server 的管理员权限。点击此处了解更多关于该漏洞的信息。

影响 Windows 10、 Windows 11 和 Windows Server 的新 Windows 零日将允许任何人获得设备的管理权限。它会影响所有受支持的 Windows 版本，并允许对设备具有有限访问权限的攻击者轻松提升其权限，以便在网络中传播。

出血计算机 已经在 Windows 上测试了该漏洞，并能够使用它从只有“标准”权限的帐户以系统权限打开命令提示符。该漏洞是微软为回应 CVE-2021-41379 而推出的补丁的旁路，由安全研究员 Abdelhamid Naceri 发现。该补丁已在本月的补丁周二发布中应用。Naceri 在 GitHub 上发布了一个概念验证，展示了如何利用该漏洞，而 BleepingComputer 展示了 Naceri 的“InstallerFileTakeOver”漏洞如何在几秒钟内获得系统权限。它在 Windows 10 21H1 内部版本 19043.1348 上进行了测试。

“这种变异是在分析 CVE-2021-41379 补丁时发现的。然而，错误没有被正确修复，而是放弃了旁路，”Naceri 在 GitHub 上解释道。“我选择放弃这个版本，因为它比原来的版本更强大。”当bleeding computer问他为什么公开披露零日漏洞时，他说他这样做是因为对微软减少其 bug bounty 项目的支出感到失望。“自 2020 年 4 月以来，微软的奖金一直被废弃，如果 MSFT 没有决定降低这些奖金，我真的不会这样做，”他说。

纳塞里并不是第一个对微软减少 bug 奖金表示担忧的研究人员。较低价值的支付鼓励黑客将漏洞留给自己，或者更糟的是，将它们卖给其他可能恶意使用它们的人。

我们预计微软将在未来的周二补丁更新中尝试修补此漏洞。Naceri 说，最好的解决办法是等待微软发布针对受影响的不同 Windows 版本的安全补丁。
