骁龙开发者工具包让你花 219 美元就能在 ARM 应用上测试 Windows

原文：https://www.xda-developers.com/qualcomms-snapdragon-developer-kit-for-219/#0001-01-01

早在五月份微软的 Build 开发者大会之前，高通发布了骁龙开发者工具包。这个想法是给开发者一个在 ARM 应用上测试他们的 Windows 的方法，而不需要花费一大笔钱(双关语)。

现在，它终于以 ECS LIVA 迷你箱 QC710 的形式出现了。该产品似乎是第一代骁龙 7c，而不是承诺的骁龙 7c Gen 2，这两者之间没有太大的区别。它还拥有 4GB 内存和 64GB 存储空间，搭载 Windows 10 Home 而不是 Windows 11 ，尽管你可以免费升级。

对于端口，有一个 USB 3.2 Gen 1 Type-A 端口，一个 USB 2.0 Type-A 端口，HDMI，以太网，还有一个 USB Type-C 端口用于充电。

“我们在与微软合作开发有用的开发工具方面有着令人自豪的历史，骁龙开发者工具包是这种合作的最新成果，”高通的 Miguel Nunes 在五月份说道。“该开发套件为其他消费和商业设备提供了一种经济实惠的替代方案。凭借更小的台式机配置，该套件为开发人员提供了比笔记本电脑更大的灵活性，并且价格更低。我们将继续致力于帮助开发人员解决客户的需求，同时降低总体部署成本。”

显然，这并不意味着是一个高端钻机。这意味着允许开发者测试应用程序，而且成本低廉。它甚至不允许你构建应用程序，特别是因为它缺乏 64 位仿真，如果你将它升级到 Windows 11，你就会得到这种仿真。

整个套装售价 219 美元，这是最便宜的了。骁龙开发者套件重约半磅，采用紧凑的无风扇设计，便于随身携带。

商店列表中没有提到的一件事是预装软件。当它被宣布的时候，骁龙开发者套件将会被提供。NET 5.0、Visual Studio 代码、FFMPEG、LLVM、Chromium、Wix 等等。当然，你可以下载那些东西。
