Galaxy S10 用户现在可以试用一个基于 Android 12 的 UI 4.0

原文：https://www.xda-developers.com/galaxy-s10-series-one-ui-4-beta-program-korea/#0001-01-01

虽然 Galaxy S21 系列的用户已经在享受 One UI 4.0 和 Android 12 的好处，但许多 Galaxy 手机正在等待轮到他们体验新软件。三星上个月为 Galaxy S20 和 Galaxy Note 20 系列开放了 One UI 4.0 测试版计划，现在这家韩国巨头正在将测试版计划扩展到另一个传统旗舰系列:Galaxy S10。

One UI 4 Beta 概述:动手体验三星 Android 12 更新中的每一项功能！T3】

三星已经为 Galaxy S10、Galaxy S10 Plus 和 Galaxy S10e、启动了 One UI 4.0 beta 计划，允许感兴趣的用户在公开发布之前试用 Android 12 的早期版本。该计划目前仅限于韩国，但我们预计该公司将很快将其扩展到全球模型。如果你是居住在韩国的 Galaxy S10 用户，并且想试用 Android 12，你可以使用三星成员应用程序注册测试程序。

三星尚未透露计划何时推出 Galaxy S10 系列的稳定更新的任何细节。无论如何，我们不会屏息以待最终版本在 2022 年前发布。

One UI 4.0 是对 One UI 3.0/3.1 的重大升级，带来了许多新功能和视觉变化。此次更新的主要亮点包括重新设计的快速设置面板、改进的黑暗模式、额外的充电效果、重新设计的主屏幕小部件、改进的多任务体验等等。您可以在我们对 One UI 4.0 的完整评测中了解更多关于新的最新软件的信息。

除了银河 S10 系列，三星还在测试银河 S20 和银河笔记 20 系列的 One UI 4.0 更新。事实上，这两个阵容都在今天早些时候收到了第二次 One UI 4.0 测试版更新，带来了多个错误修复、2021 年 12 月的安全补丁和稳定性改进。
