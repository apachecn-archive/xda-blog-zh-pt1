Firefox 94 开始在 iOS 和 Android 上推出新主页

Mozilla 发布了 Firefox 94，它在 Android 和 iOS 上有更新的主页布局，在 Android 上有新的标签管理功能。

近二十年来，Mozilla Firefox 一直是最好的网络浏览器之一，移动版本继承了桌面浏览器的大部分功能(带有一些扩展说明)。Firefox 94 今天开始在桌面和移动平台上推出，尽管供 web 开发人员使用的新 API 总是很有趣，但这一次最令人兴奋的变化可能是在 iOS 和 Android 上。

在这次新的更新之前，Firefox 移动浏览器的主页只有你经常访问的网站的链接(类似于 Chrome 新标签页的顶部)和一个收藏空间，你可以在那里保存标签供以后整理。Mozilla 正在 Firefox 94 中重新组织主页，除了相同的经常访问的网站之外，还为最近的标签和最近的书签增加了新的部分。火狐应用程序设置中的每个部分都有选项，以防你不想要其中的一些(或全部)。

“有了这个新的火狐主页，”Mozilla 在其声明中说，“我们希望解除记住完成那些中途阅读的文章或假期研究的沉重心理负担，直到你准备好跳回去完成那些任务。我们已经将您的移动体验简化并组织到一个中心位置，并提供了新的功能来帮助您忽略在线噪音，专注于对您重要的事情。”

除了更新的主页之外，Firefox 94 手机版还有一个标签管理的新功能。14 天没有查看的标签现在被移到非活动区，这样它们就不会弄乱你实际使用的标签。Firefox(至少在 Android 上)已经有了一个在一天、一周或一个月后自动关闭标签页的选项，但是这样你实际上不会丢失任何东西。新功能现在将出现在 Android 上，稍后将出现在 iOS 上，如果你喜欢收藏标签，你可以关闭它。
