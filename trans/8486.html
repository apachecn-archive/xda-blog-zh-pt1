Start11 让你让 Windows 10 的开始菜单看起来像 Windows 11

原文：https://www.xda-developers.com/start11-is-out-letting-you-make-the-windows-10-start-menu-look-like-windows-11/#0001-01-01

在发布候选版本近一个月后，Stardock 发布了 Start11 1.0。像它的前辈 Start10 和 Start8 一样，这个工具允许你定制你的开始菜单。事实上，Start8 最初是为了解决 Windows 8 的混乱，让你回到旧的 Windows 7 开始菜单。

如果你喜欢开始菜单，你仍然可以使用它。Start11 让你可以让 Windows 11 中的开始菜单看起来像在 Windows 7 或 Windows 10 中一样。它甚至有一个 Windows 11 选项，有更多的定制选项，让你按类别等对应用程序进行分组。你还可以用它来将任务栏移动到屏幕的不同侧，这个选项在 Windows 11 中被取消了。

“自从微软首次宣布 Windows 11 以来，我们一直在努力为新的操作系统优化 Start11。“我们让用户能够轻松定制他们电脑体验的几乎每个细节，包括能够从几个新的开始菜单布局中进行选择，重新定位任务栏，以及其他许多事情。”

这还不是全部，因为你实际上不需要在 Windows 11 上使用 Start11。也可以在 Windows 10 上使用。这意味着，如果你的电脑不符合新操作系统的严格 CPU 要求，你仍然可以让你的开始菜单看起来像它，坦率地说，视觉大修是升级的一大部分。

在 Windows 10 上，你将获得新的开始菜单，Start11 提供的新的定制选项，如固定文件和文件夹，它将位于中心。

Start11 从今天开始提供，价格为 5.99 美元，像往常一样，它包含在 Object Desktop 工具套件中。Start8 和 Start10 的用户也可以选择升级。
