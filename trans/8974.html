维瓦尔第指出微软对 Edge 的滥用营销

原文：https://www.xda-developers.com/vivaldi-criticizes-microsoft-edge-abusive-practices/#0001-01-01

过去几周，我们报道了微软向 Windows 用户推广其 Edge 浏览器时使用的一些恶意策略。从公开嘲笑竞争对手到强迫某些链接在 Edge 中打开，微软似乎在尽一切努力说服用户继续使用它的浏览器。最近，与之竞争的浏览器制造商 Vivaldi 指责微软滥用 Edge 的营销行为，利用微软在个人电脑市场的主导地位来增加其浏览器的市场份额。

Vivaldi 的 Jon von Tetzchner 提到了微软采用的一些策略。首先，当你在默认搜索引擎为 Bing 的 Edge 中搜索一款与之竞争的浏览器时，你会看到一个巨大的横幅，上面写着你不需要下载另一款浏览器。还有一个事实是，即使你安装了另一个浏览器，也不容易切换你的默认浏览器，尤其是在 Windows 11 上(尽管微软显然在最近的预览版中纠正了这一点)。这还没有提到 Windows 中的某些网页链接只能在 Edge 浏览器中打开，或者一些用户最近在试图下载 Chrome 时发现的弹出窗口。

最重要的是，如果你在 Edge 不再是默认设置后再次打开它，你会看到一个提示窗口，告诉你使用微软的“推荐浏览器设置”。那有什么用？其中，它将你的默认浏览器切换回 Edge。除了这些更明显的推动，如果你使用 Edge，微软还会为你提供更多的微软奖励积分——本质上是给你钱来使用它的浏览器。

这些都不是什么新鲜事，但很明显，微软正在滥用其市场主导地位，将越来越多的用户推上其浏览器。这是微软过去一直被抨击的方法，例如在 90 年代末和 21 世纪初，将 Internet Explorer 与 Windows 捆绑在一起，并禁止原始设备制造商在他们的个人电脑中捆绑任何其他浏览器。这最终导致微软被迫在 Windows 中为用户提供浏览器选择屏幕，在一些市场中，它不得不销售没有某些预装软件的 Windows 版本(称为 N 版)，以便不会被视为反竞争。然而如今，在大多数市场中，这种情况已经不复存在，部分原因是人们的注意力已经转移到了谷歌和苹果这样的竞争对手身上，这些公司以这样或那样的方式有自己的反竞争做法。

至于你能做些什么来改变现状，Tetzchner 给出了一些建议。首先，你可以在社交媒体和其他反馈渠道上指出微软的滥用行为。不保证公司会回应，但如果没有，你可以随时联系代表和监管机构调查微软，以便改变可以合法执行。

当然，你可以随时向微软表明，把你的默认浏览器改成你想要的样子是行不通的。就我个人而言，就在微软 Edge 开始显示试图下载竞争浏览器时取笑它们的消息后，我换成了 Vivaldi(巧合)。微软的做法显然不受业界欢迎，只有用户大声疾呼，变革才会到来。
