OPPO Find N 让我相信可折叠手机就是未来

原文：https://www.xda-developers.com/oppo-find-n-foldables-future/#0001-01-01

随着时间的推移，安卓手机变得越来越单调，尽管这不一定是件坏事。在传统的玻璃板中可以找到一种舒适感，各公司旨在完善外形，以生产出业内最符合人体工程学和最漂亮的智能手机。可折叠手机在某种程度上改变了这种模式，随着像三星 Galaxy Z Flip 3 这样的设备越来越流行，智能手机的未来似乎已经明朗。自从 OPPO Ireland 把 OPPO Find N 发给我测试之后，我就一直把它作为我的日常驱动。它让我相信，可折叠外形不仅实用，而且很可能是智能手机的未来。这是一款适合放在口袋里的平板电脑，有什么不喜欢的呢？

关于这个意见片 : OPPO Ireland 在 2022 年 2 月 4 日给我发了 OPPO Find N。该公司在这份意见书上没有任何意见。

Android 并不完全适合平板电脑

OPPO Find N 将 Android 平板电脑与 Android 手机结合在一起，虽然这不是一个评测(我的同事 Ben 有一个 OPPO Find N 评测，你应该查看一下)，但已经很容易看到好处了。Android 平板电脑不是很好，通常它们最大的问题是缺乏高端规格和质量差的软件。不过，到目前为止，可折叠手机几乎拥有最高端的规格，解决了这个问题。

你会看到一个奇妙的软件和硬件的组合，它展示了可折叠手机的真实功能

第二，因为应用程序很少是为平板电脑开发的(更不用说那些可以随时变成平板电脑的设备了)，所以软件必须在操作系统层面上支持这些应用程序。OPPO 已经在其软件中加入了大量的工作，以使应用程序变得舒适，当手机打开时，许多应用程序默认在一个肖像风格的盒子中启动。你可以很容易地从多任务菜单中启用应用程序使用全屏，但这表明 OPPO 知道很多应用程序只是没有针对大屏幕进行优化。 Android 12L 可能会在解决这个问题上大有作为。

当你将 OPPO 对可折叠手机的强硬态度与上述结合起来时，你会得到一个奇妙的软件和硬件网络，展示可折叠手机真正能够成为什么样的产品。我甚至不再考虑如何使用我的手机——我走来走去，像平常一样使用折叠状态的 60Hz 屏幕的手机。当我坐在火车上、飞机上，甚至只是在咖啡店里，我可以打开它，享受两倍大的显示屏。

可折叠是未来

一旦安卓解决了平板电脑的困境，我看不出可折叠手机会成为安卓的未来。大多数应用程序都反应灵敏，OPPO Find N 完全改变了我使用智能手机的方式。它更小的外形使它在展开之前可以单手使用，如果你正在展开它，那么几乎可以肯定的是，使用双手不是什么大问题。

如果我们已经知道如何使用笔记本电脑，并知道在不使用时关闭它们，那么我们就可以知道如何使用可折叠智能手机

至于“脆弱”，在实际使用该设备后，我还没有任何特别担心的原因。它肯定不会比其他设备更脆弱，而且它的保护程度与笔记本电脑相同。如果我们已经知道如何使用笔记本电脑，并知道在不使用时关闭它们，那么我们就可以知道如何使用可折叠智能手机。

我是否担心这些设备中难以置信的复杂铰链机制？当然，但这就是为什么我认为他们是未来而不是现在。我仍然认为，并不是每个人都想使用可折叠智能手机，老实说，如果我花自己的钱，我仍然可能坚持一两代人，然后再购买一部我自己可以使用的可折叠智能手机。

可折叠手机已经出现，而且很可能会继续存在

然而，真正让我认为可折叠手机是未来的最后一个方面是，至少根据马克·古尔曼 T2 T3 的消息，甚至 T4 苹果 T5 也对这个领域感兴趣。苹果采用新技术的速度是出了名的慢，如果该公司至少在关注可折叠产品，那就意味着它看到了这个领域的价值。

那是不是意味着苹果一定会发布可折叠的？不会，因为像苹果这么大规模的公司可能会测试数百种不同类型的产品，而这些产品永远不会问世。不同的是，可折叠手机已经出现，而且很可能会一直存在。很有可能有一天，一部可折叠的 iPhone(或者甚至以一个不同的名字，以真正的苹果方式)会到达大众手中。

统治所有这些的一个外形因素

从我自己的经验来看，这是统治他们的一个因素。这是一部手机，这是一部平板电脑，它拥有你能期待的所有好处。在像 OPPO Find N 这样没有折痕的设备中，很容易看到吸引力。有真正的使用案例，它比电话更有意义，也比我想象的更有意义。

例如，有一天晚上，我和几个朋友出去吃饭，我们试图解决如何分摊账单的问题。我们每个人都在一个共享的群聊中发送了一条消息，显示账单上的金额相当于我们订购的金额，我在展开的 OPPO Find N 上的一个浮动窗口计算器中将其相加，然后我就可以截屏显示总价值，并将用于相加的数字叠加在对话上。这是令人难以置信的无缝和方便，无疑比试图在普通手机上的应用程序上安装浮动窗口更好的体验，或者更糟糕的是，在 Facebook Messenger 和我的计算器应用程序之间不断切换。

正是这些小事让我意识到为什么我喜欢可折叠手机，也是为什么到目前为止，它给人的感觉是比普通玻璃板优越得多的体验。一部不会折叠的手机没有任何问题，但你可以用一部折叠的手机做更多的事情。我之前持观望态度，但现在我相信了:可折叠手机是未来，我迫不及待地想知道接下来会发生什么。
