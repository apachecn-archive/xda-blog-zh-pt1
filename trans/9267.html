微软发布 Windows 11 build 22526，更好地支持苹果的 AirPods

微软正在发布 Windows 11 Insider 预览版 build 22526，它增加了对苹果 AirPods 的更好支持，等等。

今天，微软发布了其第一个 Windows 11 Insider 新年预览版，使内部版本号达到 22526。像往常一样，它可以在开发频道获得，这是为了测试新功能，并不依赖于任何特定的操作系统版本。这意味着新功能可能会在今年秋天的功能更新中出现，也可能不会。

特别是对于这个版本，变化非常小。以下是变化和改进的完整列表:

  • 我们正在尝试为一些内部人员将 ALT + TAB 显示为窗口，而不是全屏。
  • 我们在使用苹果 AirPods 产品(AirPods、AirPods Pro 或 AirPods Max)时增加了对宽带语音的支持，提高了语音通话的音频质量。
  • 从内部版本 22518 开始，凭证保护现在默认情况下在加入企业的 Windows 11 企业版(E3 和 E5) 许可电脑上启用。
  • 我们正在尝试索引更多的文件位置，以便使用搜索在文件资源管理器中更快地找到重要文件。

如你所见，这里没有什么真正疯狂的事情发生。不过，这也说得通。尽管事实上已经有几周没有新的 Windows 11 Insider 预览版出现在 Dev 频道上，但值得记住的是，这是因为团队放假了。他们从星期一才开始工作。

为数不多的变化是值得注意的。ALT + TAB 被窗口化而不是全屏，这当然是行为上一个很大的视觉变化。然而，它只适用于那些为了通过加入 Dev 频道来尝试新特性而牺牲了系统稳定性的人。其余的变化，如 AirPods 的宽带语音，每个人都可以使用。

和往常一样，这里有一个很长的修复和已知问题列表:

Windows 11 构建了 22526 个修复程序

【文件浏览器】

  • 做了一些工作，以帮助解决一个问题，如果你使用搜索文件浏览器，然后 explorer.exe 崩溃，下次你试图在文件浏览器搜索它不会工作。

【搜索】

  • 我们做了另一个修复来帮助解决最近搜索弹出按钮卡在屏幕上的问题(除了边框以外都是透明的)。
  • 当显示比例设置为大于 100%时，提高了搜索结果中显示的应用程序图标的分辨率。

【聚光灯集合】

  • 如果您正在使用 spotlight collection，当前映像应该会在现在升级时迁移(如果您正在升级的版本是版本 22523 或更高版本)。

【微件】

  • 修正了一个问题，当鼠标悬停在第二个显示器上的入口点时，插件板可能没有正确的分辨率。
  • 解决了一个问题，插件板会暂时空白，只显示一个添加插件按钮(也打开一个空白对话框)。

【其他】

  • 修复了远程桌面用户在尝试关机或重启时遇到的 KMODE_EXCEPTION_NOT_HANDLED 错误检查。
  • Microsoft Store 不应再显示尝试安装最新 Web 媒体扩展更新返回错误代码 0x80073CFB 的消息。
  • 解决了上次航班上影响一些内部人员的 DWM 崩溃问题。

注意:在积极开发分支的 Insider Preview 版本中提到的一些修复可能会成为 Windows 11 发布版本的服务更新，该版本将于 2021 年 10 月 5 日日正式发布。

Windows 11 内部版本 22526 已知问题

【常规】

  • 您可能无法登录某些应用，如反馈中心。重新启动你的电脑应该可以解决这个问题。
  • 我们正在调查一些内部人员在最近的版本中看到驱动程序和固件更新失败的报告，错误为 0x8007012a。

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【任务栏】

  • 切换输入法时，任务栏有时会闪烁。
  • 网络图标有时在任务栏中不见了，而它应该在那里。如果您遇到这种情况，请尝试使用任务管理器重新启动 explorer.exe。
  • 如果你有多台显示器连接到你的电脑上，右击主显示器任务栏上的日期和时间，explorer.exe 就会崩溃

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【设置】

  • 查看可用 Wi-Fi 网络列表时，信号强度指示器不能反映正确的信号强度。
  • 进入系统>显示> HDR 时，设置可能会崩溃。
  • 蓝牙和设备下有一个空白条目。
  • 我们正在修复一个影响一些内部人员的问题，该问题导致在尝试访问已安装的应用程序、启动应用程序和默认应用程序页面时设置崩溃。如果你是受影响的人之一，需要卸载一个应用程序，使用 winget 应该可以。

【微件】

  • 更改任务栏对齐方式会导致小工具按钮从任务栏上消失。
  • 当有多台显示器时，任务栏上的小工具内容可能会在不同显示器之间不同步。
  • 任务栏左对齐时，温度等信息不会显示。这将在未来的更新中得到解决。

【语音接入】

  • 某些文本创作命令，例如“选择该内容”或“删除该内容”，可能无法在 Windows 应用程序中正常工作。
  • 对某些标点符号和符号(如@符号)的识别不准确。

【旁白】

  • 下载语音后，必须重新启动讲述人设置，才能在语音选择组合框中看到“Jenny”和“Aria”。
  • 在连续阅读的情况下，当按下 CTRL 键时，用户可能会看到在语音停止之前有一段较长的时间。

与往常一样，您可以在设置应用程序中的 Windows Insider 预览设置中注册您的电脑，然后点击“开始”。登录后，您可以通过检查 Windows Update 中的更新来获取新版本。
