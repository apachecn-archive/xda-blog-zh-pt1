安卓 12L 公布

在本周的 Android Dev 峰会上，谷歌正式发布了 Android 12L——一款专为大屏幕设备打造的功能 drop。我们之前在野外见过 Android 12L，但当时我们称之为 Android 12.1。然而，谷歌现在已经证实，该功能将被称为 Android 12L，但该公司尚未透露“L”代表什么。

Android 12L 带来了许多新功能，以增强可折叠平板电脑和 Chromebooks 等大屏幕设备上的 Android 体验。它包含了几个 UI 改进，改善了操作系统如何利用额外的屏幕空间，一个新的任务栏让用户可以快速访问应用程序，分屏模式的改进，以及信箱的改进。

谷歌已经推出了第一个 Android 12L 开发者预览版。该公司计划在未来几个月内发布三个测试版更新，并在 2022 年 Q1 奥运会结束前发布一个稳定版本。虽然这一更新是为大屏幕设备专门打造的，但它也将推广到谷歌的像素阵容。

通过 Android 12L，谷歌旨在从整体上改善大屏幕设备上的 Android 体验。因此，它还发布了各种工具来帮助开发人员构建在可折叠、平板电脑和 Chromebooks 上运行良好的响应应用。该公司为 Android Studio 添加了参考设备、布局验证工具和可调整大小的仿真器。

谷歌还推出了新的 WindowManager APIs，帮助开发人员为他们的应用程序构建响应迅速的用户界面，以适应任何屏幕尺寸。WindowManager 库包括活动嵌入、窗口大小类和一个用于窗口功能(如折叠和铰链)的公共 API 表面。

高通 7、6 和 4 系列芯片

高通本周宣布了四款新的中端芯片——骁龙 778G 加 5G、骁龙 695 5G、骁龙 680 4G 和骁龙 480 加 5G。骁龙 778G Plus 5G 比骁龙 778G 高一小步，具有更快的 CPU 和 GPU 性能。骁龙 695 5G 继承了去年的骁龙 690，支持毫米波 5G，图形渲染速度提高 30%，CPU 性能提高 15%。

骁龙 680 是一款面向 4G 智能手机的新型中端芯片组。它基于高通的 Kryo 265 CPU 内核，采用 Adreno 610 GPU。最后，骁龙 480 Plus 5G 是今年早些时候骁龙 480 的略有改进的版本，比以前的型号具有更好的 CPU 和 GPU 性能。

在本周的 Connect 活动中，脸书的马克·扎克伯格证实了公司的新品牌。“我们正处于互联网新篇章的开端，这也是我们公司的新篇章...为了反映我们是谁以及我们希望建设的未来，我很自豪地告诉大家，我们的公司现在是 Meta，”扎克伯格在博客中说。
