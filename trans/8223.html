Ubuntu Linux 21.10 到来了，带来了以开发者为中心的特性和 GNOME 40

原文：https://www.xda-developers.com/ubuntu-21-10-arrives/#0001-01-01

Ubuntu 是最流行的桌面 Linux 发行版之一，它是无数其他桌面操作系统的基础，如 Linux Mint 和 Elementary OS。Ubuntu 每年有两个主要发布版本，按照计划，Ubuntu 21.10“Impish Indri”刚刚发布。Ubuntu 上一个主要版本是 21.04，被称为“多毛的河马”。

Ubuntu 21.10 不是一个长期支持(LTS)版本，所以它只会在接下来的九个月内更新。如果你不想定期升级或者更喜欢稳定性而不是新功能， Ubuntu 20.04 是当前的 LTS 版本，将在 2025 年 4 月前接收更新。Impish Indri 是计划于 2022 年 4 月发布的下一个 LTS 版本之前的最后一个临时版本。开发人员可以使用 Ubuntu 21.10 为下一个 LTS 提供面向未来的工作，该版本将至少支持到 2032 年。

“随着开源成为新的默认，我们的目标是将 Ubuntu 带到企业的所有角落和开发者想要创新的所有地方，”Canonical 首席执行官马克·舒托沃尔斯说。“从最大的公共云到最小的设备，从 DGX 服务器到 Windows WSL 工作站，开源是新思想的跳板，Ubuntu 使这个跳板安全、可靠且一致。”

Ubuntu 21.10 的变化

GNOME 40 和快照统计

GNOME 40 桌面环境的到来，获得了动态工作空间和触摸板手势。还有作为快照应用程序的 Firefox，Canonical 表示，商店中发布的快照数量增长了 25%。snap store 现在每天为超过 1000 万个系统提供服务。在用于 Linux 的 Windows 子系统上也有对图形应用程序的现成支持。

Ubuntu 21.10 带来了 PHP 8、GCC 11 和其他以开发者为中心的特性

Ubuntu 主要关注开发者，全新的 PHP 8 和 GCC 11 在这个版本中开箱即用。PHP 8 于今年 6 月发布，其最令人兴奋的特性是包含了一个实时(JIT)编译器。从理论上讲，JIT 的加入在某些情况下应该可以显著提高性能。以前，PHP 使用 Opcache，你可以在线阅读更多关于包含 JIT 编译器对 PHP 意味着什么的内容。

Ubuntu 21.10 提供了 Apache Cassandra 作为 snap 包， MicroK8s 可以用来快速方便地安装 Kubernetes。

Linux 内核 5.13

Ubuntu 21.10 带来了 Linux 内核的 5.13 版本，增加了对内核电围栏(KFENCE)的支持。KFENCE 是一种新的运行时内存错误检测器，专为生产环境设计，在检测最常见的内存错误的同时保持较低的开销。默认情况下它是启用的，Ubuntu 21.10 会在 amd64 和 arm64 架构上的每个系统调用入口随机化内核堆栈的内存位置。
