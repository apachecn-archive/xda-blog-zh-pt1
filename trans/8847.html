JetBrains 发布了用于创建 Kotlin 应用程序的多平台 1.0

原文：https://www.xda-developers.com/jetbrains-releases-compose-multiplatform-1-0-for-creating-kotlin-based-android-desktop-and-web-apps/#0001-01-01

JetBrains 为用 Java、PHP、Kotlin 和其他编程语言创建软件开发了一些最流行的集成开发环境(ide)。该公司还一直在研究 JetBrains Compose multi platform(T1)，这是一个基于 Kotlin 语言创建跨平台应用程序的框架(并与谷歌的 Jetpack Compose(T3)相连)。多平台已经在不同平台上处于不同的开发阶段有一段时间了，现在 JetBrains 表示，它已经准备好供每个人在 Android、桌面和网络上使用。

Jetbrains 周四表示，“通过 JetBrains 的 Compose Multiplatform，开发人员可以在 Kotlin 中轻松创建带有 UI 的桌面应用程序。该框架是声明性的，因此代码反映了 UI，开发人员无需担心 UI 更新逻辑，它允许他们预览 UI 的一部分，而无需构建整个应用程序，并与 Google 的 Jetpack Compose 共享公共 API，以简化采用。[...]开发人员可以在 Android、桌面和 Web 之间共享他们的 UI 和状态管理代码，并轻松地将基于 Android Jetpack Compose 的应用程序移植到桌面。”

这意味着现在可以用一个 Kotlin 代码库创建跨 Windows、macOS、Linux、Android 和 web 的应用程序。桌面应用程序可以使用菜单、键盘快捷键和通知，通过使用 Skia 图形库进行硬件加速渲染。用多平台构建的网络应用程序可以使用特殊的 CSS 功能与页面元素交互。

在这一点上，不缺乏创建跨平台软件的框架，其中 Electron 可能是目前最受欢迎的(为 Discord、Slack、微软团队、Skype 和许多其他软件提供动力)，尽管 Electron 根本不支持移动。谷歌目前正在开发 Flutter ，它使用 Dart 编程语言，目前支持桌面平台、移动和网络——与 Jetpack Compse 多平台相同。

更多关于 Compose Multiplatform 的信息可以在 JetBrains 的网站上找到。
