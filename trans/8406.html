GitHub 刚刚在 GitHub Universe 上宣布了大量新功能

原文：https://www.xda-developers.com/github-announces-web-command-palette-revamped-issues-and-much-more/#0001-01-01

GitHub 是世界上最大的代码库之一，为世界各地成千上万(如果不是数百万)的软件项目托管源文件和管理工具。它是维护开源软件的最佳服务之一，具有强大的问题管理、持续集成支持等等。今天是 GitHub Universe 的开始，这是该公司的年度开发会议，有一些令人兴奋的消息。

正如 GitHub 环球新闻综述中所强调的，GitHub 网站正在进行一些奇妙的改进。新的 GitHub Issues experience 作为公开测试版提供，引入了项目板和具有高级过滤功能的动态表格。它更接近吉拉和 Asana 这样的平台，能够创建自定义字段并在私人和公共观看之间切换项目(点击这里查看演示)。GitHub Discussions 还为 GitHub 移动应用程序添加了标签和支持，在接下来的几个月内将会推出民意调查和洞察仪表盘。

也许对 GitHub 网站最好的补充，尤其是对于习惯于在 ide(如 Visual Studio 代码)中工作的人来说，是一个新的命令面板。现在，在 Mac 上按 Command+K 或在 Windows 和 Linux 上按 Control+K 会打开调色板，它可以快速将您带到任何项目、存储库、拉式请求或问题，而无需将手从键盘上拿开。它还可以用于运行选择命令。

GitHub 也为平台的持续集成工具 Actions 推出了几项改进。现在可以从单个存储库中重用工作流，新的 API 可用于自动扩展自托管操作，并且添加了对开放 ID 连接(OIDC)身份验证支持的支持。

你可以在 GitHub 公司的 GitHub 宇宙综述中读到 GitHub 的一切。现在，如果你不介意的话，我要去试试新的项目板了。
