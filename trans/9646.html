Chrome OS 98 推出 Unicode 14.0 表情符号等

原文：https://www.xda-developers.com/chrome-os-98-unicode-14-emoji-desk-keyboard-shortcuts/#0001-01-01

虽然 Chrome OS 要想赶上 Windows 和 macOS 还有很长的路要走，但它的情况比几年前好得多。随着时间的推移，它的新特性和新增功能越来越好。随着最新的更新，它变得更好了。

Chrome OS 98 已经开始在稳定频道上推出(通过 Android Police )，它带来了大量的改进和功能，包括更新的表情符号选择器、彻底改革的语言设置、新的虚拟桌面快捷方式、改进的滚动性能等等。

最值得注意的是，Chrome OS 98 增加了对 Unicode 14.0 的支持。最新的 Unicode 标准版带来了 37 个新的表情符号，包括敬礼，融化的脸，忍住眼泪的脸，手指，等等。

最新版本引入了新的虚拟桌面键盘快捷键。早在 2019 年 8 月，Chrome OS 76 就推出了虚拟桌子。在 Chrome OS 78 中，谷歌增加了键盘快捷键，使得使用虚拟桌面更加方便。

Chrome OS 98 还增加了全面的语言设置菜单，并支持特定设备上的“基于网络的恢复”。此外，用户应该注意到使用鼠标在 ARC 应用程序中滚动时性能的提高。在其他地方，Chrome Files 应用程序正在获得完全的黑暗主题支持，而新的噪音抑制标志允许用户减少视频通话中的外部噪音。

最新版本还修复了已知的错误，如 Linux 中的拖放随机崩溃，智能卡连接器和读卡器通过 WebUSB 获得自动授权的 USB 访问。

Chrome OS 98 已经开始向支持 Chromebooks 和其他 Chrome OS 的设备推出。如果你连接到 Wi-Fi 网络，你的 Chromebook 会自动下载更新，当它准备好安装时，你会收到通知。

Chrome OS 还在开发一项名为自适应充电的新功能。据报道，该功能将使用机器学习来学习你的充电程序，并相应地调整充电速度，以帮助你延长 Chromebook 的电池寿命。
