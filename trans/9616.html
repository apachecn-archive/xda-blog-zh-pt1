Windows 11 可以获得背景贴纸和更多新功能

原文：https://www.xda-developers.com/windows-11-desktop-stickers-focus-sustainability/#0001-01-01

【UPDATE 2/8/2022 @美国东部时间下午 4:33】更多 Windows 11 的功能已经在网上分享，包括对任务栏上任务管理的改进。我们已经在文章末尾添加了关于它们的信息。

Windows 11 将于本月获得一些新功能，但我们已经开始期待接下来可能会发生什么。来自 Twitter 用户 Albacore 的新消息分享了 Windows 11 在未来某个时间点的新功能。新功能包括桌面贴纸，为 Windows 11 增加了一个新的个性化层。

据一条推文称，Windows 11 将会附带一个新的贴纸编辑器应用程序，允许你在桌面背景上添加贴纸，即使你改变背景，这些贴纸也会持续存在。然而，如果您的背景是幻灯片或您有多个显示器，该功能将不起作用，这使得它们不太有用。这可能是该功能早期实现的一个限制，也许微软会让它在多显示器设置中使用成为可能。不管怎样，如果你是定制的粉丝，这对你来说可能是个好消息，尽管我们真的不知道贴纸会是什么样子。

微软似乎正在做出的另一个重大改变与 Windows 11 中的通知和对焦辅助功能有关。焦点辅助页面被重命名为焦点，它有一些新的选项可以切换。例如，您可以在时钟应用程序中启用或禁用焦点会话功能，隐藏任务栏应用程序的通知徽章和闪烁，以及将通知静音，以便它们直接进入操作中心。该页面还包括一个新链接，告诉您如何在 Outlook 中设置焦点时间。您可以使用 Outlook 为您的电脑安排关注时间，但这也可能只是一个额外的工具，以便您可以在所有设备上静音 Outlook 通知。

似乎缺少的一个选项是在聚焦辅助的仅优先级模式和仅警报模式之间进行选择。事实上，您不能再从焦点页面自定义您的优先级列表，这将被移动到常规通知设置页面。该页面现在将为您提供一个选项来静音(除了禁用之外)所有通知，因此它们会直接进入操作中心，而不会在屏幕上弹出。目前，这需要打开聚焦辅助，但它现在只是一个正常的通知设置。如果你想让一些应用程序突破你的静音设置，你也可以在这个页面自定义优先级通知列表。

Windows 11 设置应用的最后一个新增功能是一个新的可持续发展页面。此页面允许您启用最佳电源选项，以使您的计算机节能，它还为您提供了一个了解如何回收您的电脑的链接。在页面顶部，有树叶图标，显然是为了象征你的电脑的“生态分数”，但该功能还没有功能。

我们不知道微软计划何时为 Windows 11 用户带来贴纸等功能，但几乎可以肯定的是，Windows 内部人士将首先尝试这些功能。微软最近表示，它计划对开发频道进行更多的实验,并且如果你想获得更稳定的版本，它将很快给内部人员一个窗口来转移到测试频道。这表明开发频道的 Windows 内部人员在不久的将来可能会有一些大的变化，这可能包括这些功能。

更新:更多 Windows 11 功能泄露

Albacore 已经分享了未来某个时候 Windows 11 会有更多功能的图片。一个特别大的改进与任务栏有关，如果你的应用空间不够了，任务栏会显示一个溢出菜单。目前，任务栏一旦填满就会停止显示更多的应用程序，最后一个图标只会在你切换到当前未显示的应用程序时发生变化。这将使你更容易在任务栏上找到所有的应用程序。

在更多的推文中，Albacore 还展示了新的英雄控件，它将出现在设置应用程序的时间和语言部分的顶部。此外，如果用户在不支持的 PC 上安装了不符合最低要求的 Windows 11，应用程序的系统部分将显示警告。
