Microsoft Edge 96 增加了价格通知和简单的密码更新

原文：https://www.xda-developers.com/microsoft-edge-96-price-notifications/#0001-01-01

微软本周推出了 Edge 浏览器的新升级版本，随着假日季节的到来，重点是网上购物。微软 Edge 96 是最新版本，它提供了一些新功能，包括价格历史记录(也将用于 Android ),更容易获得微软奖励，等等。

从价格历史特性开始，它几乎和它听起来的一样。当您在线查看产品时，Microsoft Edge 可以开始跟踪其价格，并在价格变化时通知您，以便您可以省钱。Edge 会通知您最近查看的产品的价格变化，因此您不必一直查看购物清单上的每个产品。这增加了 Edge 中已经提供的价格比较和价格历史功能。

在这一点上，微软终于将价格比较和价格历史带到了 Android 版 Edge 和 96 版。这些功能只限于桌面版本的 Edge 已经有一段时间了，所以很高兴看到它们也适用于智能手机。微软表示，它也在努力将其他 Edge 桌面功能引入手机。

Microsoft 提供的另一种帮助您省钱的方式是 Microsoft Rewards，通过此次更新，您将更容易获得这种体验。从省略号菜单中，您现在可以将 Microsoft Rewards 固定到您的 Edge 工具栏，以便您可以更快地访问它。

除了省钱，微软还为保存的密码增加了一个新的简单更新功能。当你的一个密码被泄露时，你现在可以进入 Edge 设置中的密码管理器并点击转到网站按钮，这将直接带你到正确的页面来更新该网站的密码。这项功能最初只适用于少数网站，但微软表示将很快扩展这项功能。另一方面，Edge 还可以自动生成新密码，省去了你自己设置密码的麻烦。

另一个新功能是 Edge 设置中提供的新效率模式。该功能旨在通过降低浏览器的 CPU 和内存使用量来帮助您延长笔记本电脑的电池寿命。微软警告说，这样做的好处可能会因您的典型使用场景而异，但它仍然是一个受欢迎的功能。

最后，微软还宣布其 Bing 搜索引擎正在与 Good On You 合作，Good On You 是一个根据各种可持续发展相关指标对时尚品牌进行评级的平台。Bing 会让你知道什么时候你在购买来源可靠的商品，这样你也可以根据可持续性做出购买决定。除此之外，Edge 本身很快就会增加一个新功能，即在合格网站上每购买 10 件商品，就会向伊甸园重新造林项目捐款，帮助在最需要的地方植树。这两个功能都只有在英国才有，至少目前是这样。

Edge 版推出了新的价格跟踪和密码功能。你可以通过打开 Edge 中的省略号菜单，点击关于微软 Edge 来检查更新。
