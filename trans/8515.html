苹果公司修补了一个 macOS Monterey bug，该 bug 使一些 MAC 电脑变成了砖块

原文：https://www.xda-developers.com/apple-patched-macos-monterey-bricking-bug/#0001-01-01

苹果公司修补了一个 macOS Monterey bug，该 bug 使一些 MAC 电脑变成了砖块

马科斯·蒙特雷正在用砖砌一些旧的苹果机。苹果终于修补了导致问题的漏洞，用户现在可以安全地更新了。

苹果上个月向公众发布了 MAC OS 12 Monterey T1。虽然这个新的更新来到挤满了大量的商品，它带来了一些不愉快的惊喜。一些使用旧苹果电脑的用户——尤其是那些使用 T2 芯片的用户——在更新后，最终只能使用死机的机器。谢天谢地，苹果已经修补了 macOS Monterey 的砌砖错误，用户现在可以安全地更新他们的 MAC 电脑了。

上周，一些老 Mac 用户在 macOS Monterey 更新后无法重启设备时，在 Twitter 上向苹果公司寻求支持。这个漏洞影响了一小部分使用 T2 芯片 MAC 的用户。虽然这个问题相对来说并不普遍，但它仍然使受影响用户的工作流程陷入瘫痪。据 9to5Mac 报道，苹果公司终于推出了针对这一漏洞的修复程序，但 Mac 电脑被屏蔽的用户需要联系苹果支持部门。

新补丁包括苹果 T2 安全芯片的固件更新，防止 MAC 电脑在更新到蒙特利时被拦截。但是如果你的苹果电脑已经坏了，你必须去苹果商店修理。考虑到 macOS 12 操作系统自 6 月以来一直处于 beta 测试阶段，尚不清楚如此重大的一个错误是如何出现在 MAC OS 12 的公共版本中的。

T2 芯片设计用于加密存储和安全引导功能，增强图像信号处理和触控 ID 数据的安全性。任何 M1 的苹果电脑都没有这种功能，因为苹果 M1 的芯片内置了这些功能。根据 Apple 支持，以下是采用 T2 安全芯片的 Mac 型号:

  • iMac(视网膜 5K，27 英寸，2020 年)
  • iMac Pro
  • Mac Pro (2019 年)
  • Mac Pro(机架，2019 年)
  • 苹果迷你电脑(2018)
  • MacBook Air(视网膜，13 英寸，2020 年)
  • MacBook Air(视网膜，13 英寸，2019 年)
  • MacBook Air(视网膜，13 英寸，2018 年)
  • MacBook Pro (13 英寸，2020，两个雷电 3 端口)
  • MacBook Pro (13 英寸，2020，四个雷电 3 端口)
  • MacBook Pro (16 英寸，2019 年)
  • MacBook Pro (13 英寸，2019，两个雷电 3 端口)
  • MacBook Pro (15 英寸，2019 年)
  • MacBook Pro (13 英寸，2019，四个雷电 3 端口)
  • MacBook Pro (15 英寸，2018)
  • MacBook Pro (13 英寸，2018，四个雷电 3 端口)
