动手实践:联想 ThinkPad Z 系列是一台 ThinkPad 应该有的样子

原文：https://www.xda-developers.com/lenovo-thinkpad-z13-z16-hands-on/#0001-01-01

在 CES 上，联想宣布了其全新的 ThinkPad Z 系列，包括 ThinkPad Z13 和 Z16。这里有很多新东西。一方面，联想在这方面专门与 AMD 合作；没有英特尔型号。

但更重要的是，这些笔记本是为那些通常不会购买 ThinkPads 的人设计的，我去看看。联想所做的正是你所期望的。它组织了焦点小组并与顾客交谈。它向没有购买 ThinkPads 的人询问原因。

鉴于我们在过去看到的品牌，Z 系列是一个彻底的变化。事实上，如果你听说过任何 ThinkPad，你可能已经在想象一台黑色长方形的笔记本电脑，仅此而已。它们是标志性的，但这并不意味着这种外观适合所有人。

联想 ThinkPad Z13 有三种颜色，不过我只看到了其中两种。缺少的其实是黑色的，所以我看到了银色的，也是我个人最喜欢的，带有青铜色口音的黑色皮革。这些服务于不同的人。请记住，Think 是一个商业品牌，所以普通的银色笔记本电脑确实会吸引那些人，而黑色的选择会吸引 ThinkPad 用户。

联想 ThinkPad Z16 只有银色，这并不奇怪。13 英寸的笔记本电脑卖得更多，而且通常有更多的颜色。

正如我上面提到的，联想确实与 AMD 在这方面进行了合作，所以他们在 Z13 和 Z16 中采用了锐龙 6000 处理器和 U 系列，在 Z16 中采用了专用镭龙显卡。这就是 ThinkPad Z13 和 Z16 最大的区别。

你可以看到顶部有一个标签，我认为这增加了设备的风格。在里面，它是一个反向的凹槽，里面有一个 FHD 相机。我在那里的时候确实偷拍了一张照片。

我确实认为网络摄像头在照片质量方面还有待提高，但我期待在视频通话中测试它。

正如你所看到的，它们的四周都有金属边缘，这使得笔记本电脑更加时尚。这也是 ThinkPads 普遍缺乏的东西。很高兴看到。

有两个 USB Type-C 端口，有一点很重要，那就是它们是 USB4，这意味着它们可以与雷电 3 端口相媲美。购买 AMD 笔记本电脑时，缺少 Thunderbolt 是一个很大的缺点，所以这将会带来很大的不同。还有一点需要注意的是，联想 ThinkPad Z16 还增加了一个 SD 读卡器。

现在，该说说键盘了。是的，中间仍然有一个红色的跟踪点，这有点令人惊讶，因为这个产品更现代。TrackPoint 是 Windows 触摸板时代的遗留物。你会发现每个 ThinkPad 键盘上都有一个。

然而，你会注意到通常位于触摸板上方的物理按钮不见了。这些按钮旨在与 TrackPoint 配合使用，但您仍然可以使用触摸板的顶部。基本上，联想让你选择使用顶部作为按钮或作为一个更大的触摸板。这是一个触觉触摸板，允许公司进行这种定制。

键盘本身有 1.35 毫米的按键，与 ThinkPad X1 Nano 和 ThinkPad X1 Titanium Yoga 上的按键相同。与 ThinkPad X1 Carbon 上的 1.5 毫米按键甚至其他一些 ThinkPad 上的 1.8 毫米按键相比，这是这款设备另一个令人感觉现代的地方。较短的键程感觉很好，但也很自然，因为联想做了大量工作来确保按下一个键所需的力度相同。

这是我最喜欢的键盘之一。我真的希望球队在某个时候把它带到整个阵容中。

联想 ThinkPad Z13 和 ThinkPad Z16 都将于 5 月上市，起价分别为 1，549 美元和 2，099 美元。
