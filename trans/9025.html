Realme GT Neo 2 获得基于 Android 12 的 Realme UI 3.0 的早期访问版本

Realme 已经为 Realme GT Neo 2 开放了 Realme UI 3.0 早期访问计划，让用户有机会在稳定推出之前体验 Android 12。

Realme 已经开始向其设备推出基于 Android 12 的 Realme UI 3.0 的早期版本。上周晚些时候，该公司发布了 Realme X7 Max 5G 和 Realme GT 的版本，让用户有机会在稳定发布之前体验 Android 12。该公司现在已经为 Realme GT Neo 2 启动了类似的早期访问计划。

根据 Realme 社区论坛上最近的帖子，Realme GT Neo 2 的 Realme UI 3.0 早期访问计划现已上线。如果您有兴趣在手机上试用 Realme UI 3.0，您可以按照以下步骤注册测试计划:

  • 确保您的设备运行的是 RMX3370_11。A.05 稳定发布
  • 前往设备设置中的软件更新部分
  • 点击右上角的齿轮图标
  • 点击试用版并填写您的详细信息
  • 点击立即应用按钮

然后 Realme 将审查您的申请，一旦获得批准，您应该可以通过 OTA 更新在您的设备上接收 Realme UI 3.0 early access build。为防止任何数据丢失，请确保在设备上安装更新之前进行备份。此外，请注意，早期访问版本可能会对您的手机产生“不可预测的影响和对日常使用的影响”，因此我们不建议在您的日常驱动程序上安装它。
