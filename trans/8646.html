以下是如何立即安装 Windows 10 版本 21H2 的方法

原文：https://www.xda-developers.com/how-to-get-windows-10-version-21h2/#0001-01-01

微软刚刚开始推出 Windows 10 版本 21 H2(T1)，这是针对仍在使用 Windows 10 的用户的下一个功能更新。尽管 Windows 11 已经发布，微软仍然在为 Windows 10 提供更新，特别是针对商业用户。现在的重点是 Windows 11，尤其是对消费者来说，但如果你想马上获得 Windows 11 版本 21H2，有几种方法可以做到。

Windows 10 版本 21H2 中的新功能几乎完全专注于安全和业务用例。UWP 应用中的 VPN APIs 有一些变化，支持带有 WPA3 哈希到元素协议的 Wi'-Fi 6，等等。微软今天还宣布，它将改变 Windows 10 功能更新的发布频率，改为每年一次，类似于 Windows 11。

那么，如果你想马上获得 Windows 10 版本 21H2，你能做些什么呢？最简单的方法是使用 Windows Update，但这将是一个逐步推出的过程，因此您可能需要找到替代方法。下面是要做的事情。

使用 Windows Update 下载 Windows 10 版本 21H2

获得 Windows 10 版本 21H2 更新的最简单方法是使用设置应用从 Windows Update 下载。随着微软逐渐退出，你可能不会马上看到它，但你可以简单地打开设置应用程序，然后点击更新&安全。点击检查更新，如果你是推广的一部分，你会看到一个选项来下载 Windows 10 版本 21H2 的功能更新。

这是一个可选的更新，至少只要您当前的 Windows 版本受支持。您需要运行 Windows 10 版或更高版本才能看到更新。

请注意，如果您的设备受支持，您可能还会看到升级到 Windows 11 的选项。不想要可以放心忽略，只安装新的 Windows 10 更新。

通过 Insider 计划下载 Windows 10 版本 21H2(如果您的设备不支持 Windows 11)

如果您不在 Windows 10 version 21H2 的初始推广范围内，并且您现在想要它，您可以通过加入 Windows Insider 计划立即获得它。然而，要实现这一点，你需要一台不支持 Windows 11 的电脑。如果是这样，并且您加入了 Insider 计划，您将获得 Windows 11。

要加入 Windows Insider 计划，请按照下列步骤操作:

  1. 打开设置 app，进入更新&安全，然后点击左侧菜单的 Windows Insider 程序。
  2. 点击开始
  3. 您需要链接您的 Microsoft 帐户才能加入 Insider 计划。
  4. 选取接收预览构件的通道。发布预览频道是这里最好的选择。
  5. 点击确认后，会提示重启电脑。这样做，并回到设置应用程序。
  6. 像往常一样检查更新，你应该会看到 Windows 10 版本 21H2。

获得 Windows 10 版本 21H2 更新的最后一种方法是下载媒体创建工具。这是最慢的，但可能是最安全和最有效的升级 Windows 10 版本的方法。它是最慢的，因为这种方法实际上会下载完整版本的 Windows 10 并进行安装，而 Windows Update 只下载与您当前安装不同的文件。如果你想快速做事，我们推荐另外两种方法中的一种。

如果你想使用这种方法，请前往 Windows 10 下载页面，下载媒体创建工具。运行它并按照屏幕上的步骤下载最新版本并升级您的电脑。
