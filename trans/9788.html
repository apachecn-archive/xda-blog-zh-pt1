如果用户的电脑不受支持，Windows 11 将很快警告用户

原文：https://www.xda-developers.com/windows-11-warning-users-unsupported-hardware/#0001-01-01

在不受支持的硬件上运行 Windows 11 的用户可能很快就会看到一个警告，这是基于 Dev 频道向内部人士发布的最新版本。当打开设置应用程序时，Windows 11 会让用户知道他们的 PC 是否满足运行该操作系统的最低要求。然而，这不会阻止用户运行操作系统。

这并不完全出人意料，因为 Twitter 用户 Albacore 已经发现了这个功能隐藏在以前的 Windows Insider 版本中，但用户现在可以在他们的机器上看到它，而没有任何黑客行为，正如最初由 Windows 最新 指出的那样。该警告也显示在桌面上的 Windows Insider 水印旁边。微软在针对 Windows 11 build 22557 的博客帖子中没有提到这一变化，但这可能只是为了让用户知道会发生什么。

当微软宣布 Windows 11 的最低要求时，它在 Windows 用户中引起了一点骚动。除了需要双倍的内存外，许多人面临的主要问题是微软限制了与新一代处理器的兼容性，留下了大量仍在使用的机器。微软后来解释说，旧的 CPU 经历系统崩溃的频率要高得多，这意味着体验不会是最佳的。

然而，虽然升级通过典型的方法被阻止，但总是有可能通过其他方式在不受支持的硬件上安装 Windows 11。您可以使用媒体创建工具、升级助手或下载 ISO 来安装 Windows 11。当然，这也带来了在不受支持的硬件上性能不稳定或不可靠的风险。设置应用程序中的警告可以帮助用户了解他们可能遇到的任何问题背后的原因，如果他们喜欢的话，总是可以回到 Windows 10 。

目前，这一警告仅显示给在不受支持的硬件上运行 Windows 11 的内部人员，没有迹象表明它何时会面向普通公众。这似乎是一个足够小的变化，可以通过简单的累积更新来推出，但它可能会随着下一次功能更新而推出，预计将于今年晚些时候推出。
