微软发布面向 K-8 的 Windows 11 SE，Surface Laptop SE

原文：https://www.xda-developers.com/microsoft-windows-11-se-surface-laptop-se/#0001-01-01

今天，微软宣布推出 Windows 11 SE，这是其剥离操作系统并瞄准教育市场的最新尝试。换句话说，Windows 11 SE 是微软公司与 Chromebooks 竞争的最新尝试。

Windows 11 SE 是专门针对 K-8 学生的。如果你给一个五岁的孩子一台 Windows 笔记本电脑，这可能有点复杂。这是新 SKU 背后的理念。这意味着更容易和更直观。

Windows 11 SE 更简单，因此更易于儿童使用

有很多不同之处，尽管也有很多相同之处。在 Windows 11 SE 中，应用程序默认全屏打开，尽管你仍然可以在传统的窗口模式下使用它们。Snap Layouts 仍然存在，但它只会一次显示两个应用程序的布局，因为分屏应用程序仍然是必要的。

新的以学生为中心的操作系统也删除了一些功能。与之前强调微软商店的尝试不同，这一次完全删除了它。另一个你在这里找不到的 Windows 11 功能是 Widgets 面板。

另一个关键区别是，这是一个以云为中心的操作系统。所有文件将自动保存到云中，因此学生不会丢失他们的文件。此外，OneDrive 仍将同步文件，以便他们可以在需要时离线工作。这来自于 1600 万 K-12 学生在校外没有互联网接入。

显然，这不是微软第一次试图渗透到课堂中，从谷歌的 Chrome 操作系统那里抢走空间。这个想法是，如果学生在成长过程中使用一个平台，那么他们在以后的生活中也会继续使用这个平台。最后一次尝试是 Windows 10 S，这是 Windows 10 Pro 的锁定版本，只提供来自微软商店的应用程序。有趣的是，最初的 Surface 笔记本电脑是一个发射设备。

然而，有一点已经改变了，那就是 Edge。对于 Windows RT，你必须使用 Internet Explorer，而对于 Windows 10 S，你必须使用基于微软内部 EdgeHTML 引擎的 Edge。现在，Edge 是由 Chromium 构建的，所以在 Windows 11 SE 中，你可以更有信心，你正在使用的任何 web 应用程序都将工作。

Surface 笔记本电脑 SE 是迄今为止价格最低的 Surface

回到它作为教育设备的根源，微软宣布了 Surface Laptop SE。它的起价仅为 249 美元，是有史以来 Surface 的最低价格，配备了双核英特尔赛扬处理器、11.6 英寸 16:9 高清屏幕和高清网络摄像头。它也是为可修复性而设计的，这意味着显示器、电池、键盘和主板等组件都可以更换。

它还使用桶式充电器代替 Surface Connect，并且它有 USB Type-A 和 Type-C 端口。基本型号配有赛扬 N4020、4GB 内存和 64GB eMMC 存储，而高端 SKU 配有赛扬 N4120、8GB 内存和 128GB eMMC 存储。这款笔记本电脑重 2.45 磅，厚度不到四分之三英寸。

这种颜色来自冰川，你只能通过教育渠道获得。事实上，获得 Windows 11 SE 的唯一途径是获得一台新的笔记本电脑，它可以来自宏碁、华硕、戴尔、Dynabook、富士通、惠普、JK-IP、联想和 Positivo 等合作伙伴。

Windows 11 SE 不会取代微软阵容中的任何其他产品，所以你仍然会看到针对学生的 Windows 11 Pro。你还会看到 S 模式。这是 K-8 特有的东西，所以其他产品将用于年龄较大的学生。
