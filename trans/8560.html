Windows 11 build 22499 已经发布，它还带有新的 iso

原文：https://www.xda-developers.com/windows-11-build-22499-new-isos/#0001-01-01

今天，微软向开发者频道发布了 Windows 11 build 22499。像往常一样，没有太多新的东西，但本周比我们前几周看到的要多一点。此外，该公司正在发布新的 ISO 映像，所以如果你想全新安装最新的 Windows 11 Insider 预览版，你可以。

所以，这里有什么新的。时钟应用程序现在支持通过工作和学校帐户登录。在博客文章中的 bulletpoint 说，“你问了，我们听了，”所以推测起来，这是一个非常高需求的东西。Windows 11 的时钟应用程序中的一个新功能是 Focus Sessions，它可以让你专注于特定的任务，集成 Spotify 和 To Do。

另一个新功能是，你现在可以直接从任务栏共享应用程序窗口给团队。这意味着，如果你将鼠标悬停在任务栏中的一个图标上，你会在预览下看到一个写着“共享此窗口”的按钮，旁边还有一个微软团队的小图标。共享时，该按钮会变为“停止共享”。显然，这应该只发生在你打电话的时候。

不幸的是，尽管这个版本中的每个人都注册尝试最新和最棒的新功能，而且他们冒着系统稳定性的风险来测试目前没有提供多少测试的东西，但从任务栏向团队共享您的窗口的能力只面向 windows 内部人员的子集。如果你还没看出来，那就是原因。

像往常一样，有一长串的修复和已知问题。

Windows 11 内部预览版内部版本 22499 修复

【输入】

  • 一旦你使用了这个版本，剪贴板历史应该被备份并正常运行。
  • 与之前的版本不同，现在点击表情面板中的 gif 会将它们插入到支持应用程序中。
  • 更新了几种语言的后台词典——触摸键盘的文本建议和自动更正现在应该更准确了。
  • 如果你改变你的口音颜色，表情面板中的口音应该跟着改变，而不是停留在旧的颜色上。
  • 修复了选择使用以前版本拼音 IME 的用户有时会出现的 IME 崩溃问题。
  • 减轻了 explorer.exe 崩溃，有时最近发生的触摸键盘使用有关。

【开窗】

  • 作出了另一个修复，以解决一个问题，即 explorer.exe 将崩溃，如果你按下 ALT + F4，而 ALT + Tab 是打开的。

【设置】

  • 修复了在通过远程桌面访问电脑时尝试检查远程声音属性时可能发生的设置崩溃..

【其他】

  • 修正了在某些超宽显示器上通过 OOBE 时导致剪辑/意外缩放的问题。
  • 当您的指纹在登录屏幕上未被识别时，错误消息中的撇号现在应该可以正确显示了。
  • 当通过在截图工具中按下新按钮来截取 UWP 应用程序的截图时，截图工具应该在截图完成后出现在前台。
  • 修复了一个问题，导致一些内部电脑在最近从睡眠中醒来时进行错误检查，错误消息为“SYSTEM _ THREAD _ EXCEPTION _ NOT _ HANDLED”。
  • 修复了可能发生的与MediaPlaybackCommandManager相关的死锁，导致某些应用程序有时无法播放媒体。
  • 解决了在查看更多信息时，可靠性监视器中的报告意外空白，只有一个空矩形的问题。
  • 做了一些工作，以帮助解决一个问题，导致某些游戏有一些滞后，只有当窗口处于焦点时才明显。作为一个提醒，如果需要，请参考以下文档以提交可操作的游戏性能反馈。

注意:在积极开发分支的 Insider Preview 版本中提到的一些修复可能会成为 Windows 11 发布版本的服务更新，该版本将于 10 月 5 日日正式发布。

Windows 11 内部预览版内部版本 22499 已知问题

【常规】

  • 从版本 22000.xxx 或更早版本更新到使用最新开发频道 ISO 的较新开发频道版本的用户可能会收到以下警告消息:您尝试安装的版本是 Flight Signed。要继续安装，请启用航班签名。如果您收到此消息，请按下启用按钮，重启电脑，然后重试更新。
  • 我们正在调查一些电脑无法安装新版本或其他更新的问题。PC 可能会报告错误代码 0x80070002。如果您遇到此问题，请重新启动您的电脑，然后重试。
  • 安装此版本时，某些设备可能会执行错误检查，错误代码为 0xc1900101-0x4001c。如果你点击这个，在设备回滚到以前的版本，你可能要暂停更新，直到我们发布一个修复。

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。
  • 我们正在调查此版本中的一个问题，即任务栏中的时钟可能会停滞不前且无法更新，尤其是在通过远程桌面访问电脑时。

【开窗】

  • 在任务视图中的不同桌面之间来回悬停鼠标将导致显示的缩略图和内容区域意外缩小。

【输入】

  • 我们正在努力解决一些内部人士的报告，他们的键盘上的灯，例如 caps lock，在升级到以前的版本后无法正常工作。

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【快速设置】

  • 我们正在调查来自内部人士的报告，即音量和亮度滑块在快速设置中显示不正确。

与往常一样，您可以从 Windows Update 获得版本，或者如果您不在开发人员渠道，您可以通过“设置”中的“Windows Insider 计划”选项卡注册。如果想下载 ISO，可以在这里下载 Windows 11 build 22499 。
