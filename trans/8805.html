JetBrains 揭示舰队编辑器，它的答案 Visual Studio 代码

原文：https://www.xda-developers.com/jetbrains-fleet/#0001-01-01

JetBrains 是 IntelliJ 平台的开发者，该平台支持 Android Studio、IntelliJ IDEA Ultimate、PyCharm Professional、GoLand、PhpStorm 和其他 ide。然而，ide 是大型应用程序，对于简单的项目来说可能是多余的。JetBrains 现已发布了其新的轻量级编辑器 Fleet，旨在实现模块化并兼容几乎所有东西。

Fleet 被宣传为一个更轻便的“下一代 IDE”，就像微软创造了 Visual Studio Code 作为 Visual Studio 的一个更轻便的替代品一样。JetBrains 在的博客文章中说，“对于那些有时只想要一个编辑器但也想要一个成熟的 IDE 的人来说，Fleet 是一种不同的体验，对于那些想要使用单一工具而不是专用工具的人来说，当然对于那些我们现有的一些 IDE 可能无法满足分布式开发的特定场景来说。”

JetBrains 表示，Fleet 可以与 Java、Kotlin、Python、Go、JavaScript、Rust、TypeScript 和 JSON 一起工作。该公司还计划增加对 PHP、C++、C#和 HTML 的支持。还有一个集成的终端(可以在协作会话中共享)，对 Java 项目的 Maven 和 Gradle 支持，Git 集成，主题，以及远程机器的端口转发。桌面版本将适用于 macOS、Windows 和 Linux。

Fleet 正在被打造为一个模块化框架，有前端(所见)、后端(索引、分析、导航、高级搜索等。)和文件系统作为不同的组件。JetBrains 表示，这将有助于 Fleet 在许多不同的环境中运行，从简单的本地应用程序到基于云的网络服务。

JetBrains Fleet 听起来可与 Visual Studio 代码媲美，后者也支持多种语言、远程会话、主题和协作编辑。微软也刚刚推出了基于网络的 Visual Studio 代码编辑器的第一个版本。不过，竞争总是好的，它有助于鼓励微软不断改进 Visual Studio 代码。

JetBrains Fleet 还没有公开，但是你可以在公司网站上注册提前预览。
