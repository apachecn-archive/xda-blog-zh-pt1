这些是 Windows 11 初始版本中的已知问题

原文：https://www.xda-developers.com/known-issues-initial-windows-11-release/#0001-01-01

今天，经过几个月的测试，微软发布了 Windows 11。该公司将采取“追随太阳”的方式发布，因此它将在你居住的地方发布，无论何时都是 10 月 5 日。即使在那之后，它也将是一个缓慢的、分阶段的推出，将持续到 2022 年年中。基本上，你的电脑越新，你就会越早得到新的操作系统。但和往常一样，Windows 11 也有一些需要注意的已知问题。

毕竟，你可能倾向于强制更新，这是你完全有权做的事情。您可能已经通过 Windows Insider 程序运行了 Windows 11，或者您可能已经通过媒体创建工具升级了您的电脑。不管你为什么这么早就开始升级，只要记住每个新功能更新都有一些已知的问题可能会阻碍你的升级，Windows 11 也不例外。

以下是 Windows 11 中的已知问题:

对于 VirtualBox 和 Cốc Cốc 的问题，微软正在采取保护措施，所以你的机器将不会提供 Windows 11。然而，它只是建议你不要试图通过媒体创建工具来升级，所以你还是应该远离它。

至于带有黑仔网络软件的设备，这是一个非常简单的方法。微软表示，它计划在本月的安全更新中准备好一个补丁，这离现在只有一个星期了。不要忘记，已经有几周没有新的 Windows 11 更新了，所以在即将到来的周二补丁更新中会有大量的修复。

对于其他两个更新，这只是微软正在研究的东西。请记住，随着时间的推移，新的问题可能会添加到列表中。
