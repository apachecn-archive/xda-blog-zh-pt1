Firefox 95 带来了增强的安全性、macOS 的改进等等

原文：https://www.xda-developers.com/firefox-95/#0001-01-01

Firefox 是少数几个拥有完全自定义渲染引擎的 web 浏览器之一，这使得 Mozilla 能够尝试新的 web 功能和低级性能变化，而大多数基于 Chromium 的浏览器无法做到这一点(无论如何，本质上不会成为一个分支)。Firefox 版本 95 现在正在推出，它在桌面和移动平台上有几个重要的变化。

这次更新中最好的改进( via Techdows )可能是 RLBox ，这是 Mozilla 与加州大学圣地亚哥分校和德克萨斯大学的研究人员合作开发的。RLBox 现在可以在桌面和移动平台上使用，并使用 WebAssembly 来隔离潜在的错误代码。这应该减少浏览器中潜在的未来漏洞。

桌面上的 Firefox 95 还包括 macOS 上减少的 CPU 使用量，macOS 上减少的解码视频的功耗，以及 Slack 的 web 应用程序的用户代理覆盖(这样你就可以访问更多的功能)。火狐的站点隔离安全特性现在也为大家全面铺开。

Firefox 95 桌面变更日志

新功能

  • RLBox 是一种新技术，它强化了 Firefox 对第三方库中潜在安全漏洞的防御，现已在所有平台上启用。
  • 在事件处理过程中，我们减少了 macOS 上 Firefox 和 WindowServer 的 CPU 使用。
  • 我们还降低了 macOS 上软件解码视频的功耗，尤其是在全屏模式下。这包括网飞和亚马逊 Prime Video 等流媒体网站。
  • 您现在可以将画中画切换按钮移动到视频的另一侧。只需寻找新的上下文菜单选项移动画中画切换到左(右)侧。
  • 为了更好地保护 Firefox 用户免受 Spectre 等旁道攻击，现在为所有 Firefox 95 用户启用了站点隔离。
Fixes
  • 启动 Firefox 后，JAWS 屏幕阅读器和 ZoomText 放大镜的用户将不再需要切换应用程序来访问 Firefox。
  • 您会发现使用 ARIA 开关角色的控制状态现在可以由 Mac OS VoiceOver 正确报告。
  • 你会看到 macOS 上的内容处理启动速度更快。
  • 我们还改进了内存分配器。
  • 我们通过提前推测性地编译 JavaScript 提高了页面加载性能。
  • 各种安全修复
Changes
  • 我们为 Slack.com 添加了一个用户代理覆盖，允许火狐用户使用更多的通话功能，并可以进入秘密会议。

Android 版火狐浏览器的更新日志并不那么激动人心，但是仍然有一些亮点。Mozilla 在上个月发布的 Firefox 94 中引入了一个新的移动主页，从 Firefox 95 开始，有更多的设置选项可用。当你打开 Firefox 时，你现在可以选择总是打开主页，打开最后一个标签，或者在不活动的情况下四个小时后才打开主页(最后一个是默认的)。

火狐 95 安卓变更日志

新功能

  • 你现在可以在设置菜单中找到新的“主页”部分。
  • “跳回”部分现在显示英雄图像。
  • 当用户从标签托盘启用自动关闭时，您会发现小吃店“自动关闭已启用”的确认。
  • 口袋(发人深省的故事部分)现在在加拿大得到了支持。

修复

  • “跳回”部分现在将总是显示标题或网址。
  • 关闭提示后，非活动选项卡提示将不再重新显示。
  • 删除文件后，下载列表将会正确刷新。
  • 我们还修复了以下崩溃:
    • 打开历史记录组时
    • 当“按主题的故事”下的选定主题不再存在时)
  • 各种稳定性和安全性修复
