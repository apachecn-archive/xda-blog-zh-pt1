Chrome 操作系统将获得新的“自适应充电”功能

原文：https://www.xda-developers.com/chrome-os-adaptive-charging-feature-in-the-works/#0001-01-01

Chrome OS 将通过“自适应充电”帮助您延长 Chromebook 的电池寿命

Chrome OS 将获得新的自适应充电功能，这将有助于 Chromebook 用户延长笔记本电脑的电池寿命。

由于受热和老化等因素，所有锂离子电池都容易随着时间的推移而退化。为了解决这个问题，手机和笔记本电脑制造商使用各种技术来延长电池的寿命。例如，苹果在 MacBooks 上提供了一项名为“优化电池充电”的功能，一旦电池充电到 80%，充电速度就会明显变慢。如果你的笔记本电脑长时间插着电源，这是很有帮助的。在智能手机方面，谷歌在其 Pixel 手机上提供了“自适应充电”功能。根据在 Chrome 上发现的新提交的，Chrome OS 可能会在未来的更新中获得类似的功能。

正如9 to 5 谷歌 所指出的，Chrome OS 将获得一个新的自适应充电功能，这将帮助 Chromebook 的用户延长他们笔记本电脑电池的寿命。基本上，该功能将确保你的 Chromebook 不会在充满电的情况下花费太多时间。据报道，该功能将使用机器学习来学习你的充电程序，并相应地调整充电速度。这与 Pixel 手机上的自适应充电功能不同，Pixel 手机只有在夜间充电时才能工作。

最大限度地缩短设备充满电的时间，以延长电池寿命。

Chrome OS 会显示一个通知，让你知道自适应充电何时激活。谷歌报道称，也有可能完全禁用这一功能。

自适应充电功能正在积极开发中，但尚不清楚它何时会准备好。我们也不知道它是否将只适用于新的和即将推出的 Chromebook 型号，或者也适用于现有的和旧的型号。我们会保持关注，如果我们了解到关于这个功能的任何新细节，我们会让你知道。
