没有 Apple ID 的 iPhone 如何恢复出厂设置

原文：https://www.xda-developers.com/how-to-factory-reset-iphone-without-apple-id/#0001-01-01

有许多情况可能会导致 iPhone 需要重置，但您可能不知道 Apple ID 帐户信息。执行工厂重置可以将手机恢复到默认状态，准备好与新用户进行设置。注册了 Apple ID 的手机很难在没有电子邮件和密码的情况下删除 iPhone。

有几种不同的方法可以尝试将 iPhone 恢复到出厂设置。在本文中，我们将向您展示如何通过设置对 iPhone 进行出厂重置，如何在没有 Apple ID 的情况下使用特殊软件，以及如何使用 iTunes 重置 iPhone。

如何通过设置重置没有 Apple ID 的 iPhone

要使用这种方法，您需要有 iPhone 的密码。您还需要禁用“查找我的 iPhone”功能。Apple 有几种不同的方法可以让你重置 Apple ID 的密码，这将让你进行工厂重置。

要查看您可以用来重置 Apple ID 密码的所有方法，请访问此处的 Apple 支持页面。

如何使用 iTunes 重置没有 Apple ID 的 iPhone

如果上面的方法不起作用，你可能会在 iTunes 上找到一些运气。如果您以前将手机与 iTunes 同步过，您可以使用该软件来执行恢复出厂设置。

1.在装有 macOS Catalina 或更高版本的 Mac 上，打开 Finder。在装有 macOS Mojave 或更早版本的 Mac 上，或者在 PC 上，打开 iTunes。

2.使用 USB 电缆将 iPhone 连接到电脑。

3.如果您看到要求您输入设备密码或信任这台电脑的信息，请按照屏幕上的步骤操作。

4.当您的设备出现在您的计算机上时，请选择它。

5.单击恢复[设备]。如果您已登录“查找我的”,则需要先注销，然后才能单击“恢复”。

6.再次单击恢复进行确认。您的电脑会抹掉您的设备并安装最新的 iOS、iPadOS 或 iPod 软件。

7.设备恢复出厂设置后，会重新启动。现在，您可以将其设置为新的。

使用本文中提到的方法之一，您应该能够对您的 iPhone 执行完全的出厂重置。重置完成后，您的 iPhone 将恢复到出厂默认设置。此时，为新用户设置 iPhone 或将其出售给其他人是安全的。
