Windows 11 build 22000.493 推出了更多修复

原文：https://www.xda-developers.com/windows-11-patch-tuesday-february-2022/#0001-01-01

又到了微软周二发布补丁的时候了，届时所有受支持的 Windows 版本都将获得安全和可靠性更新。这个月，Windows 11 将更新到构建 22000.493，如果你希望这是我们获得 Android 应用支持的一天，那似乎不是这样。与大多数其他累积更新一样，这一次集中在安全性和一般改进上。

事实上，微软上个月调侃的功能似乎都不在这里，比如新的 Widgets 按钮。这些变化可能会在以后发布，并会有单独的公告。

更新标签为 KB5010386 ，你可以在这里手动下载。这是一个累积更新，这意味着它将包括所有属于 Windows 11 build 22000.469 的更改和修复。这是一月份的可选更新，但这些变化现在包含在强制更新中，其中包括设置应用程序中新的你的微软账户页面(在账户下)。还有一个新的 HelpWith 功能，它使用 Bing 在设置应用程序的特定区域显示某些有用的建议。

微软在其 changelog 视频中强调的所有变化——没错，这次是视频——都包含在之前的更新中。这包括对任务栏自动隐藏功能的修复，应用程序图标在应用程序关闭后显示在任务栏上，图标不显示在辅助显示器上，等等。
