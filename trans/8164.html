Android 版 Chrome 正在测试新的插件和 RSS 订阅

原文：https://www.xda-developers.com/chrome-for-android-is-testing-new-widgets-and-rss-subscriptions/#0001-01-01

Chrome 网络浏览器最近受到了很多批评，特别是关于浏览器的 cookie 替代计划，但它仍然在不断增加新功能。今年早些时候，谷歌开始测试一项新功能，用于跟踪你最喜欢的网站的新闻(由 RSS 源提供支持)，现在这项功能正在更广泛地推广。与此同时，Chrome 的更新插件已经进入测试版。

Android 上的谷歌 Chrome 已经提供了相同的基本搜索小工具一段时间了，它主要是作为俄罗斯反垄断诉讼的结果而创建的，外加另一个带有书签列表的小工具。谷歌终于在测试频道更新了 Chrome 的 widgets(通过 Android Police )，尽管如果你没有看到它们，你可能需要在 chrome://flags 菜单中激活# enable-quick-action-search-widget-Android和# enable-quick-action-search-widget-Android-dino-variant。

Chrome Beta 中仍然存在两个现有的小部件，但增加了三个:一个带有匿名/相机/语音按钮的紧凑搜索小部件，一个带有 Chrome Dino 游戏按钮的更大的搜索小部件，以及一个专门针对 Dino 的专用小部件。点击前两个小工具中的搜索框，你就会进入 Chrome 的搜索栏。

与此同时，谷歌开始在有限的测试中推出 Android 平台 Chrome 的稳定分支“开始表面”新标签页的重新设计。新页面保留了通常的 Google Discover feed，但增加了使用 RSS feeds 跟踪网站的功能。当你访问一个有有效 RSS 源的网站时，一个新的关注按钮会出现在 Chrome 主菜单中。你关注的网站上的新文章会显示在“关注”标签上。

Chrome 团队的 Adrienne Porter Felt】说这一功能正在为 iOS 浏览器开发，但 Chrome 在桌面上的计划“有点遥远”以下功能无法与 Inoreader 或 Feedly 等专用 RSS 阅读器相媲美，但肯定比只查看算法推荐内容要好。
