AiryxOS 希望成为运行 Mac 应用程序的开源 macOS

原文：https://www.xda-developers.com/airyxos-macos-clone/#0001-01-01

有几十个(也许几百个)优秀的桌面 Linux 发行版可以替代 Windows 或 macOS，但也有一些系统是专有软件产品的完全克隆。其中一个是 AiryxOS，它旨在将 macOS 的界面和软件设计与 FreeBSD 的多功能性结合起来，目前处于早期开发阶段。

有许多开源操作系统旨在克隆专有系统，通常具有某种程度的二进制支持——ReactOS是 Windows 的克隆(具体来说，现在是 Windows Server 2003)；Haiku是 BeOS 的延续， FreeDOS 是 MS-DOS 的克隆，等等。AiryxOS 项目仍处于早期阶段，但它有一些崇高的目标。zo Knox 是主要开发者，目前在 OpenNMS 集团担任工程副总裁。

自述文件概述了与 macOS 应用程序具有源代码和二进制兼容性的计划(例如，您可以在 Airyx 上编译 macOS 应用程序或运行现有的 Mac 应用程序)，类似 Mac 的界面和文件夹布局，以及“易用、安全、稳定和高性能”的设计。它还与 helloSystem 共享一些代码和资源，后者是一个带有类似 Mac 界面的 FreeBSD 发行版。

对现有 macOS 软件的完全支持很可能还有很长的路要走——有成千上万的开发者在为 Wine 项目做贡献，它仍然不是 Windows 应用程序的完美兼容层。然而，FreeBSD 基础应该会使任务变得更容易一些(macOS 中的 XNU 内核部分基于 BSD)，并且还为运行原生 FreeBSD 应用程序和一些 Linux 软件打开了大门。例如，包含的 Firefox 应用程序是 FreeBSD 版本。

对于当前的 v0.3.0 版本，你能做的并不多，但该项目在相对较短的时间内已经取得了很大的进展——第一个开发者版本是在去年 5 月。如果你想尝试一下，请点击下面的链接进入官方网站。
