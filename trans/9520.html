Redmi Note 11S 将在一些市场上作为 POCO M4 Pro (4G)推出

原文：https://www.xda-developers.com/poco-m4-pro-4g-rebranded-redmi-note-11s/#0001-01-01

周三，小米正式向国际市场推出红米 Note 11 系列。该阵容由四款设备组成:红米 Note 11、红米 Note 11s、红米 Note 11 Pro 和红米 Note 11 Pro 5G。如果你还记得的话，小米还有一个名为 POCO 的子品牌，其产品组合主要由更名后的 Redmi 和小米智能手机组成。因此，新推出的红米 Note 11 手机之一现在将穿上新衣服，以不同的身份在其他市场推出，这并不令人惊讶。

根据我们从情报人员处获得的信息，我们认为全球 Redmi Note 11S 将在一些市场上以 POCO M4 Pro (4G)的名义推出。两台设备共享相同的设备代号: fleur 。注意已经有一款 POCO M4 Pro 5G ，去年 11 月上市。这是一款中国红米 Note 11 的改装版。

正如我们在之前的 Redmi-POCO 更名中看到的那样，POCO M4 Pro (4G)几乎肯定会保持与 Redmi Note 11S 相同的内部设计，但会有一些外观上的变化，以将其与 Redmi Note 11S 区分开来。我们不知道 POCO M4 Pro (4G)将于何时上架，以及将登陆哪些市场。至少，我们希望这两款手机不会在同一个市场共存——我们已经在 Redmi Note 10T 和 POCO M3 Pro 5G 上看到了这种情况。

另外，我们还了解到(通过 小米 ui)POCO 正在开发一款名为 POCO F4 GT 的设备，它将作为去年 POCO F3 GT 的直接继任者。

POCO M4 Pro (4G)可能会与 Redmi Note 11S 搭载相同的硬件。这意味着我们已经知道手机的规格了。它将配备 6.43 英寸 FHD+ LCD，刷新率为 90Hz，联发科 Helio G96 SoC，6GB/8GB RAM，最高 128GB 存储空间，由 108MP 主摄像头领导的四摄像头设置，5,000mAh 电池，支持 33W 快速充电，以及侧装指纹扫描仪。
