PNY GeForce RTX 3050 台式机 GPU 变体将于本月上市

PNY GeForce RTX 3050 桌面 GPU 变种将于本月晚些时候上市

PNY 是宣布推出 GeForce RTX 3050 桌面 GPU 的最佳制造商之一。这是我们所知道的一切。

紧随英伟达的产品展示，PNY 宣布推出其 GeForce RTX 3050 变种。GeForce RTX 3050 是桌面市场上最新的 GPU 之一，它将 Nvidia 的 Ampere 架构带给了比以往更多的游戏玩家。PNY 现在已经证实，它将有三个版本的 RTX 3050 GPU 上市，最早将于本月底上市销售。

如果你不知道，英伟达的新 RTX 3050 GPU 是 GeForce GTX 1650 的继任者。新的 GeForce RTX 3050 GPU 取代了 3 年前的 GTX 1650，通过提高吞吐量和效率，实现了性能的巨大飞跃。RTX 3050 也是首款 50 系列级桌面 GPU，能够以超过 60FPS 的速度为游戏进行光线跟踪。

这是对老化硬件的一次急需的升级，必将改善入门级游戏玩家的游戏体验。GeForce RTX 3050 GPU 带来了高达 8GB 的 GDDR6 VRAM，这本身就是对即将推出的硬件的一次升级。随着 Nvidia 及其合作伙伴的新 RTX 3050 桌面 GPU 的到来，可以肯定地说，我们将看到我们的最佳显卡系列发生一些变化。

PNY GeForce RTX 35050 起义和狂欢显卡

PNY 是首批宣布推出 GeForce RTX 3050 GPU 的制造商之一。该公司将提供三种不同版本的 RTX 3050 GPU——PNY GeForce RTX 3050 XLR 8 起义，PNY GeForce RTX 3050 XLR8 REVEL 单/双风扇，带 EPIC-X RGB 照明。它们将作为起义和狂欢显卡家族的一部分，并具有单风扇和双风扇冷却器。该系列的部分型号还配有 RGB 灯，可进行定制。来自 PNY 的所有三种变体主要在散热器设计方面不同。虽然这些 GPU 的整体规格保持不变，但由于冷却性能的差异，它们的整体性能可能会有一定程度的差异。

下面快速浏览一下 PNY GeForce RTX 3050 GPU 的产品规格:

  • 8GB GDDR6 (192 位)
  • 单风扇和双风扇配置
  • PCI Express 4.0
  • 显示端口 1.4a 和 HDMI 2.1
  • 狂欢版 EPIC-X RGB

定价和可用性

PNY 没有提到其 RTX 3050 GPU 变种的价格。然而，该公司表示，新的 GPU 将于 1 月 27 日开始通过百思买、亚马逊及其 PNY 官方商店出售。
