Stardock 的 Start11 1.1 允许你在开始菜单中放置文件夹

原文：https://www.xda-developers.com/stardocks-start11-1-1-lets-you-put-folders-in-your-start-menu/#0001-01-01

几个月前， Stardock 发布了 Start11 ，这是一个实用程序，可以让你的 Windows 11 开始菜单看起来更像 Windows 10 或 Windows 7 上的菜单。反过来也一样，让你的 Windows 10 开始菜单看起来像 Windows 11 的开始菜单。该公司在发布修复微软开始菜单错误的实用程序方面有着悠久的历史，可以追溯到 Start8 和 Windows 8 中开始菜单的删除。

今天，该公司发布了 Start11 v1.1，为该产品带来了更多改进。最值得注意的是，你可以在 Windows 10 和 Windows 11 风格中添加文件夹，这是微软在其新操作系统中已经删除的内容。要创建一个文件夹，你只需按住 Shift 键，将一个图标拖到另一个图标上。

这也包括围栏整合。如果你使用这个特殊的 Stardock 工具——它也包含在 Object Desktop 套件中——来组织那些通常会把你的桌面弄得乱七八糟的应用程序和文件，你也可以把它们放到开始菜单中。

你会发现的另一件事是，你可以右键单击一个图标，弹出一个上下文菜单，让你改变它。

“自微软发布 Windows 11 以来，我们一直在为新的操作系统优化 Start11，”Stardock Software 副总裁兼总经理布拉德·萨姆斯(Brad Sams)表示。“我们在 1.1 版中的一个优先事项是将文件夹功能引入 Windows 11 风格的菜单，并增强与我们最受欢迎的应用程序 Fences 的集成。”

另一件新的事情是，如果它是 Windows 10 上的新安装，它将导入您现有的设置。现在，该应用程序支持文件夹，它可以从开始菜单中导入那些你已经在使用的文件夹。

Start11 v1.1 现已推出。该工具本身售价 5.99 美元，但 Start8 和 Start10 用户有升级价格，它是 Stardock 的 Object Desktop 套件的一部分。
