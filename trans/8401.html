Windows 11 微软商店现已面向 Windows 10 内部人员开放

原文：https://www.xda-developers.com/windows-11-microsoft-store-comes-windows-10/#0001-01-01

几周前，Windows 11 发布了，它引入了许多新东西，其中之一就是改版的微软商店。然而，新的微软商店从未打算成为 Windows 11 的专属，现在，它终于来到了 Windows 10。新微软商店的首席设计师鲁迪·胡恩在分享了这个消息，他说 Windows 10 内部人员现在可以尝试新的界面。

新的微软商店有许多变化，尤其是全新的用户界面，在视觉上更具吸引力。在界面的侧面可以看到不同的部分和匹配的图标，每个部分都展示了大图标，一个大横幅用于高亮显示的项目，以及流畅的动画。如果你有兴趣了解更多关于变化的信息，我们有一个关于 Windows 11 微软商店的深度潜水。

除了吸引眼球之外，新的微软商店也有新的开发人员条款，因此平台上支持和提供更多的 Win32 应用程序，因为它们不再需要由微软管理。此外，像 Disney+这样的服务可以与这个新的店面集成，所以如果你在 Disney+上搜索电影，你可以直接从微软商店打开应用程序，而不必在不同的地方搜索。

当然，Windows 10 没有的一点是对 Android 应用程序的支持，因为这不仅仅需要一个新的商店。你需要 Android 的 Windows 子系统，这是 Windows 11 的专属。

内部人士测试新功能并不是什么新鲜事，但运行 Windows 10 的内部人士并不多，因为他们中的大多数人已经升级到了 Windows 11。大多数运行 Windows 10 的内部人员被踢出了该计划，因为他们的个人电脑不支持新的操作系统。然而，微软也表示，新的微软商店将很快向所有 Windows 10 用户开放。希望他们不用等太久就能获得大幅改善的体验。
