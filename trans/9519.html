谷歌 Chrome 开始测试标签页上的一键静音按钮

原文：https://www.xda-developers.com/google-chrome-mute-button-tabs/#0001-01-01

Chrome 已经允许你通过右键点击标签并选择静音选项来静音来自标签的任何音频。然而，谷歌现在正在为这项功能试验一种不同的设计，这种设计消除了打开右键菜单的要求。

正如在/r/Chrome(viaChrome Story)上看到的，Chrome 金丝雀的最新版本包括一个名为“Tab audio muting UI control”的新标志 Chrome 已经在标签播放音频时显示了一个声音指示器，但如果启用了这个标志，当你用鼠标悬停在它上面时，图标会变成一个静音按钮。这与现有的静音功能完全相同，但它只需单击一次鼠标，而不是打开右键菜单。方便。

这个功能和 Firefox 已经提供了一段时间的功能是一样的。现在标签静音按钮已经在 Chrome 中了，所有没有麻烦自己实现它的基于 Chrome 的浏览器都应该接收到它，只要它还在。提交消息(正如 Leopeva64-2 在 Reddit 上指出的)提到，这是一个决定静音按钮是否应该保留在标签中或回到全局媒体控件的实验。

如果你想亲自尝试新功能，下载 Chrome Canary ，打开 chrome://flags，搜索“Tab audio muting UI control”，设置为 Enabled。Chrome 可能会要求重启，重启后，你应该会看到静音按钮。该标志在 macOS、Windows、Linux 和 Chrome OS 上可用——Android(甚至平板电脑上的 Android)被排除在外，至少目前如此。

在过去的几个月里，谷歌一直在测试 Chrome 的其他改进，包括地址栏中的设置搜索、Android 12 上的滚动截图支持，以及 Windows 上的增强的强调颜色支持。谷歌还刚刚宣布了取代第三方 cookies 的修订提案，称为主题 API ，可能会在未来几个月内开始测试。
