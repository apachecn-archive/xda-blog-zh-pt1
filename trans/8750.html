谷歌 Chrome 致力于将 Windows 强调色支持扩展到用户界面的更多部分

原文：https://www.xda-developers.com/google-chrome-working-on-expanding-windows-accent-color-support-to-more-parts-of-the-ui/#0001-01-01

上周，谷歌在稳定频道推出了 Chrome 96，为 Windows 11 设备带来了更新设计。设计更新将菜单上的尖角和浏览器中的其他 UI 元素替换为圆角，以符合 Windows 11 的审美。Chrome 团队似乎正在研究另一个针对 Windows 的视觉变化，这将允许用户在浏览器的不同部分应用 Windows 强调色。

Chrome for Windows 目前允许你在标题栏应用 Windows 强调色。该功能是可选的，可以通过导航到设置>个性化>颜色>显示以下表面的强调色来访问。但看起来 Chrome 可能很快会将 Windows accent color 支持扩展到 UI 的更多部分，包括按钮、聚焦环、文本选择背景等。

一向可靠的 Chrome tipsterleopeva 64已经发现了Chrome Gerrit 中的一个新补丁，这暗示 Chrome for Windows 正在努力将强调色支持扩展到浏览器的更多表面。

为以编程方式在 Windows 上设置强调颜色做准备工作。

*根据所提供的最小值，更改 PickGoogleColor()以尝试并保留输入对比度。

*以编程方式从 accentcolor 计算项目突出显示颜色。

*强制各种前景色的最小对比度。

这一变化似乎是一项正在进行的工作，还没有在 Chrome Canary 中上线。

谷歌 Chrome 也在测试一项新功能，该功能将允许用户在 Chrome 侧面板中阅读文章。在支持 Chrome 阅读器模式的网站上，你可以打开侧面板，点击阅读器标签，在一个小窗口中打开那个页面，同时继续浏览其他网站。该功能已在 Chrome Canary 中上线，但尚未推出稳定版。
