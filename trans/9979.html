Mac 工作室占多大空间？出奇的少

原文：https://www.xda-developers.com/how-big-mac-studio/#0001-01-01

苹果公司最近推出了最新的台式电脑 Mac Studio T1，并立即带来了一些重大的声明。据苹果公司称，Mac Studio 可以让顶级 Mac Pro 相形见绌。但 Mac Pro 是一台大型电脑，所以 Mac Studio 肯定不会太远，对吧？不完全是。Mac Studio 并不占用太多空间，实际上它更接近 Mac mini 的大小。

事实上，Mac Studio 的宽度和长度与 Mac Mini 完全相同，为 7.7 英寸见方。唯一的区别是高度，Mac Studio 的高度为 3.7 英寸。作为参考，Mac mini 高 1.4 英寸，因此 Mac Studio 大约高 2.6 倍。这仍然是令人难以置信的紧凑，你可以很容易地把它放在你的显示器旁边或稍微后面的桌子上。如果你在显示器的下面明确地指出它与对齐的是什么，这可能会有问题，但这听起来很容易解决。

Mac Studio 与 Mac Pro 相比如何？

在发布声明期间，苹果公司一直在比较 Mac Studio 和 Mac Pro 的性能。有了新的苹果 M1 超处理器，Mac Studio 的 CPU 性能可以提高 60%，图形速度比最强大的 Mac Pro GPU 快 80%，视频转码速度快 5.6 倍。这些数字出现在如此小的封装中，这有点令人难以置信。

它小了多少？嗯，目前的“奶酪刨丝器”Mac Pro 深 17.7 英寸，宽 8.58 英寸。它的高度也是 20.8 英寸，包括手柄和脚，或者 21.9 英寸，带可选的轮子。它几乎在每个维度上都要大得多，它真的是一种可以放在地板上的东西，或者如果你有一张足够大的桌子，它还可以放在旁边。Mac Studio 需要的空间少得多，因此您可以更容易地使用电脑及其所有 I/O，这是一件大事。

这里有一个尺寸差异的可视化，有 Mac Pro 的前视图和侧视图供参考。我们没有包括 Mac 工作室的侧视图，因为它本质上是一个正方形。我们尽最大努力将产品缩小到实际尺寸。

除了出色的性能，Mac Studio 还在这个紧凑的机箱中集成了相当多的 I/O。四个 Thunderbolt 4 端口，两个 USB Type-A 端口，10Gbps 以太网，HDMI 和一个 3.5mm 耳机插孔在背面，前面有两个额外的 USB Type-C 端口和一个 SD 读卡器。如果你使用 M1 超处理器，前面的 USB-C 端口也支持 Thunderbolt 4。

部分原因是苹果 M1 超是一个系统级芯片(SoC)，而 Mac Pro 使用更传统的 PC 组件。在 Mac Pro 上，你有一个英特尔酷睿或至强 CPU，一个 AMD 显卡，一个后燃卡选项，等等。它的可升级性比 Mac Studio 强得多，后者根本无法升级。Mac Studio 上的 CPU、GPU 和 RAM 都是 SoC 的一部分，存储也焊接在主板上，这使得它可以节省大量空间。Mac Pro 确实给了你一些空间，至少可以在以后增加更多的内存和存储空间。

如果缺乏可升级性对你来说不是问题，或者如果你认为节省空间和性能值得在可升级性方面进行权衡，你可以订购下面的新 Mac Studio，M1 Max 处理器的起价为 1999 美元，M1 Ultra 的起价为 3999 美元。如果你还没有显示器，你还需要一个显示器，苹果的 Studio Display 是一个完美的搭配。如果价格太高，你可以随时查看你现在能买到的最好的苹果电脑。对于新的或老的 Mac 用户来说，也许值得看看苹果硅 Mac 的最佳应用，以及你需要知道的关于 macOS Monterey 的一切。
