Realme GT Neo 3 可能会推出更便宜的 80W 快速充电版本

原文：https://www.xda-developers.com/realme-gt-neo-3-leaked-specs/#0001-01-01

Realme 最近在巴塞罗纳 MWC 展示了其新的 150W 有线快速充电技术。在演示之后，该公司确认新技术将在今年晚些时候与即将到来的 Realme GT Neo 3 一起亮相。尽管 Realme 在活动中没有透露有关该设备的任何进一步细节，但之前的泄露显示，它将采用 6.7 英寸 FHD+ AMOLED 显示屏，一个未指定的八核 2.85GHz 芯片组，最高 512GB 存储和 5G 支持。现在，由于 XDA 公认的开发者 mlgmxyysd，我们可以获得关于该设备的更多细节。

正如之前在 TENAA 列表中发现的那样，Realme 正在准备 Realme GT Neo 3 的两个版本，型号分别为 RMX3560 和 RMX3562。根据 mlgmxyysd 的消息，型号为 RMX3562 的 Realme GT Neo 3 将采用联发科 MT6895 SoC。这与 Realme 在最近的联发科天玑 8000 系列发布后的声明一致，在发布会上，Realme 证实该设备将采用新的 Dimensity 8100 SoC。我们还了解到，该设备将运行基于 ColorOS 12.1 ( Android 12 )的 Realme UI 3.0，其内核版本为 5.10.66。

此外，新的泄漏显示，Realme GT Neo 3 可能会有两种不同的快速充电技术。Realme 可能会推出带有 USB 2.0 Type-C 端口的设备，支持 80W 或 150W 快速充电。目前，我们不确定 Realme 为什么要采用这种方法，因为它已经确认 GT Neo 3 将提供 150W 快速充电支持。然而，我们有理由相信，该公司可能会推出更便宜的 80W 型号和更大的电池，以及 150W 型号和略小的电池，让买家可以根据自己的要求选择。

虽然新泄露的信息没有透露 Realme GT Neo 3 的其他信息，但上个月泄露的手机渲染图表明，它将在背面安装一个三摄像头，位于一个矩形摄像头岛中。渲染图也证实了该设备将采用带圆角边缘的普通背部，并在左下角显示 Realme 品牌。

目前，这些是我们关于即将到来的 Realme GT Neo 3 的所有细节。一旦我们对这款设备有了更多了解，我们会尽快通知您。

更新:之前的文章提到，Realme 可能会随 Realme GT Neo 3 配备 80W 的充电砖。然而，事实并非如此。我们现在已经证实，Realme 可能会推出两款设备，一款支持 80W 快速充电，另一款支持 150W。这篇文章已经更新，以反映这些变化。

感谢 XDA 公认的开发者 mlgmxyysd 的提示！
