Apple 的 Studio Display 在与 Windows 配合使用时功能有限

原文：https://www.xda-developers.com/apple-studio-display-limited-functionality-windows/#0001-01-01

在本周的苹果活动上，该公司展示了一个全新的工作室显示器来配合 Mac 工作室。起价 1599 美元，这不仅仅是的显示器。这是一个 27 英寸的 5K 屏幕，基本上内置了 iPhone 11。它拥有自己的 A13 仿生处理器和 12MP 超宽摄像头，具有中央舞台等功能。

根据苹果的规格表，它只兼容现代 MAC 和几款新 iPads。然而，你可以在 Windows 上使用它，尽管容量有限。在给 The Verge 的一份声明中，苹果发言人证实了什么可行，什么不可行。

简短的回答是，这将像为特定产品制造的大多数配件一样工作。当你在 Windows PC 上使用苹果的 Studio Display 时，它很可能会是一个愚蠢的产品。这就像使用任何带有 1200 万摄像头和扬声器的 5K 显示器一样。这些事情都将工作，一个警告是，你需要雷电 3 或 4 连接它，并获得完整的决议。

不起作用的关键功能是中央舞台、空间音频，当然还有“嘿，Siri”。很明显，苹果只是为 macOS 开发了这些东西，它根本没有为使它与 Windows 兼容做任何工作。Cupertino 公司可能甚至没有预料到会有很多 Windows 用户购买工作室显示器，特别是考虑到苹果硅 MAC 电脑没有 Boot Camp。

毕竟，如果你是一名 Windows 用户，正在寻找一款带有网络摄像头和扬声器的高分辨率显示器，你可能会想要看看 Dell UltraSharp 会议显示器，它是 31.5 英寸，内置 4K 网络摄像头，可以像 Center Stage 一样将视野移动到你身上。它没有嘿，Siri，但如果你作为一个 Windows 用户真的想要它，你可以在你的桌子上放一个 HomePod 什么的。
