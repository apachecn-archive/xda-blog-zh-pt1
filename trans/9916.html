Microsoft Office web apps 将很快使转换帐户变得更加容易

原文：https://www.xda-developers.com/microsoft-office-web-apps-switch-accounts/#0001-01-01

微软正在让网上办公用户的生活变得更加轻松，尤其是那些拥有多个账户的用户。由 Neowin 首次发现的微软 365 路线图的一项新内容表明，你很快就可以在账户之间切换，而无需先注销。

现在，当你在网上使用 Microsoft Office 应用程序时，无论是 Word、Excel、Outlook 等，右上角都有一个徽章，它只允许你查看当前帐户。比如你有学校账号和个人账号，你就看不到两个。切换帐户的唯一方法是从当前帐户注销，然后用另一个帐户登录。

就谷歌而言，它已经让用户很容易转换账户一段时间了。当您点按您的个人资料图标时，您可以在浏览器上轻松看到您已登录的其他帐户，并立即在它们之间切换。大多数人都会同意，微软早就应该做出这样的改变了。

Microsoft 365 路线图中列出了在 Office web apps 中的多个帐户之间切换的功能。身份证号码是 70801。它是今天添加的，微软的目标是下个月全面上市。如果你好奇，完整的描述如下:

用户可以在同一个浏览器中登录 Microsoft 365 web 应用程序的多个工作和个人帐户。这还允许他们在帐户之间无缝切换，而不必注销并重新登录。

令人惊讶的是，微软花了这么长时间来实现这一功能，因为该公司已经将如此的重点放在混合和远程工作上。许多人在工作和个人使用同一台笔记本电脑，所以这似乎应该是一个明显的补充。尽管如此，看到它姗姗来迟总比没有好。
