Windows 11 的 EdgeDeflector block 现在在公共版本中

原文：https://www.xda-developers.com/windows-11-blocks-edgedeflector/#0001-01-01

我们过去已经报道过几次微软关于其内部 Edge 浏览器的滑稽动作，从让你在 Windows 11 中更难改变默认浏览器到下载竞争对手的浏览器时一些非常露骨的营销信息。我们最近看到该公司做的事情之一是屏蔽 EdgeDeflector ，这是一款旨在将定制 Windows 11 链接重定向到默认浏览器的软件——这些链接只能在 Edge 中打开。现在，该区块已经作为 build 22000.376 的一部分向所有 Windows 11 用户推出。

最初，这个 EdgeDeflector 块出现在 Windows Insider 程序的 Dev 通道中，所以很明显微软已经在计划这么做了，但很难说什么时候会发生。据 HowToGeek 报道，本周早些时候推出的最新补丁周二更新将该块带到了更广泛的 Windows 11 用户群，这意味着更多的用户现在被迫使用 Edge。

这只适用于 Windows 11 本身的自定义链接，比如当你在窗口小部件窗格中点击一篇新闻文章或使用 Windows Search 搜索网页时。这些链接使用一个自定义处理程序将它们重定向到 Edge 浏览器，而不管你的默认设置是什么，这也是 EdgeDeflector 试图改变的。然而，它并不是唯一一个做到这一点的软件，因为 Mozilla 的 Firefox 浏览器本身也做了类似的事情。

值得庆幸的是，在 Windows 11 Insider 版本中 EdgeDeflector 被阻止后不久，一个替代解决方案出现了。这款名为 MSEdgeRedirect 的新应用使用了一种不同的方法，“过滤并传递微软 Edge 进程的命令行参数到你的默认浏览器中”。截至目前，还没有被封杀，如果你想不惜一切代价避免 Edge，还是可以用的。然而，与其他解决方案不同，这种解决方案要求应用程序一直运行，以将链接重定向到默认浏览器。

微软也很有可能试图阻止这个变通办法，所以我们不知道你还能依赖它多久。不过，目前你可以从 GitHub 下载 MSEdgeRedirect。

浏览器制造商维瓦尔第最近指责微软与 Edge 的反竞争行为，我们只能希望该公司听取反馈，放松其咄咄逼人的 Edge 营销。开发频道最近的 Windows 11 版本显示出一些进步的迹象，微软在设置应用中增加了一个按钮来改变默认浏览器。
