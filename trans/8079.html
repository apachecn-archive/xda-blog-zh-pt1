三星配备 AMD GPU 的下一代 Exynos 芯片将支持光线追踪

原文：https://www.xda-developers.com/samsung-exynos-chip-amd-gpu-ray-tracing/#0001-01-01

三星配备 AMD GPU 的下一代 Exynos 芯片将支持光线追踪

三星宣布，其下一代带 AMD GPU 的 Exynos 芯片将提供光线跟踪支持。请继续阅读，了解更多信息。

虽然距离三星的下一代旗舰智能手机系列还有几个月的时间，但关于这些手机的泄露已经开始在网上出现。在过去的几周里，我们已经看到了即将到来的 Galaxy S22 系列中所有三款设备的泄露渲染图，并了解了它们的规格。最近的一个泄露消息透露Galaxy S22 Ultra 将采用 Exynos 2200 芯片组，具有一个主频为 2.9GHz 的 ARM Cortex-X2 核心，三个主频为 2.8GHz 的核心，四个主频为 2.2GHz 的核心，以及一个主频为 1250MHz 的 AMD GPU。

尽管三星尚未确认这些规格，但它已经之前宣布其下一款旗舰 Exynos 芯片将采用 AMD GPU。这增加了上述规范的可信度。三星下一款旗舰 Exynos 芯片中的 AMD GPU 可能会比 Exynos 2100 上的 Mali-G78MP14 提供更大的性能提升。此外，三星声称将提供光线追踪支持。最近在 微博(viaVideocardz)上，该公司表示:

光线跟踪技术是桌面 GPU 支持的高级图形显示技术。通过计算光线的位置和反射的面积，将位置区域的像素逐一渲染，产生逼真的效果。这项技术也即将通过三星 Exynos GPU 搭载在智能手机的移动终端上，将手机游戏的体验提升到一个新的高度。(机器翻译)

除了这篇文章，三星还分享了一张图片，让我们一窥光线跟踪在即将推出的 Exynos GPU 上的优势。然而，该图像仅用于演示目的，实际上并不是由真正的 GPU 渲染的。它只是用来展示光线追踪如何更好地照亮物体。
