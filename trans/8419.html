微软 Edge 正在为微软 365 获得新的企业功能

原文：https://www.xda-developers.com/microsoft-edge-new-features-microsoft-365/#0001-01-01

微软的 Edge 浏览器通过微软 365 为商业用户提供了一些新功能。这个想法是通过提供无缝的上下文切换和其他改进来帮助用户“更聪明地工作”。这些功能将在未来六个月内推出。

首先，当谈到上下文切换时，微软试图解决的一个痛点是必须用同一个帐户登录不同的应用程序。很快，当你在微软 365 应用程序(如 Outlook to Teams)中收到一个链接时，Edge 将使用与收到该链接的帐户相匹配的用户配置文件自动打开该链接。这样，即使您在 Edge 中有多个 Microsoft 帐户，您的链接也将始终在保存了您的密码和登录信息的正确配置文件中打开。

将 Edge 和微软 365 结合在一起的另一个新功能叫做共享链接，它相当简单。这是 Edge 历史页面中的一个新部分，它收集了所有通过 Microsoft 365 服务与您共享的链接。如果你想找到一周前团队中有人发给你的链接，你不必翻遍整个对话，你可以直接在 Edge 中看到与你分享的链接。即使您从未打开过链接，它们也会保存在此页面中。

当您打开链接时，Microsoft Edge 还能够提供额外的上下文。如果您正在访问一个与您共享的页面，但您忘记了它的重要性，Edge 会在提到该链接的地方显示团队和 Outlook 邮件，为您提供额外的背景信息，说明您最初为什么要打开它以及您需要做什么。

企业新标签页也得到一些消息。这是 Microsoft Edge 中新选项卡页面的特殊版本，提供您的 Microsoft 365 组织内的相关内容。这包括 Outlook 事件和 SharePoint 网站，但在未来，卡片也将用于任务列表和 Outlook 电子邮件。

最后，微软编辑器将原生地来到 Edge，这是微软已经宣布的事情。编辑器是微软的高级写作工具，可以纠正拼写错误，并帮助解决其他写作问题。它通常以扩展的形式提供，类似于 Grammarly，但是很快，你不需要安装任何东西，它就可以工作了。这实际上已经可以在美国的 Edge 内部人员中使用了，但是推测起来在将来会有更多的语言版本。
