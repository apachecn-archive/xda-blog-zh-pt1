TaskbarXI 将你的 Windows 11 任务栏变成一个类似 macOS 的 dock

TaskbarX 的开发者新推出的实用工具 TaskbarXI，旨在为 Windows 11 上的任务栏带来新的定制选项。

微软彻底改革了 Windows 11 的任务栏和开始菜单，但并不是每个人都对新的外观和感觉感到满意。 Stardock 是一个实用程序，它用更接近 Windows 7 或 Windows 10 的方式取代了更新的开始菜单，现在又有了一个新的第三方工具来定制任务栏。

TaskbarXI(viaGhacks)是一个新的应用程序，为 Windows 11 任务栏增加了额外的定制选项，是作为流行的 Windows 10TaskbarX 工具的继任者开发的。它仍处于开发的早期阶段——甚至还没有一个图形界面——但它确实可以工作。目前的主要功能是它可以将任务栏收缩成一个紧凑的 dock，就像 macOS 和 iPadOS 上的 dock 一样，当窗口最大化时，它可以拉伸回全尺寸。

该实用程序还支持多显示器、单独的 DPI 缩放，以及将任务栏分为左右两个窗格。改变任务栏背景风格和颜色的工作正在进行中，同时还有一个简单的图形界面来更新设置。没有将任务栏移动到屏幕两侧或顶部的选项，但是确实有人要求将其作为一项功能——如果你有 GitHub 帐户并希望看到该功能被添加，请单击原始评论上的大拇指按钮。

如果你想尝试一下 TaskbarXI，你可以从 GitHub 下载最新版本。当前版本“仅用于测试目的”，正如开发人员所指出的，您必须使用任务管理器关闭 TaskbarXI 并重新启动 Windows 资源管理器才能应用更改。

微软仍在致力于 Windows 11 任务栏和开始菜单的更新，尽管该公司不太可能添加 TaskbarXI 等第三方工具正在开发的一些定制选项。早在 7 月预览版中就提供了对任务栏的多显示器支持，以及开始菜单中的新搜索框。
