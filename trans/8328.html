Windows 终端预览版 1.12 推出一系列新功能

原文：https://www.xda-developers.com/windows-terminal-adds-session-restore-and-more-profile-features-in-latest-preview/#0001-01-01

Windows 终端是微软几年来一直致力于开发的新型一体化终端应用程序。这是与 PowerShell、Linux 的 Windows 子系统和旧的命令行应用程序一起工作的新的中央方式，并且自从第一次发布以来，微软一直在继续推出新的功能。Windows 终端预览版 1.12 现已推出，为 Windows 10 和 11 用户提供了新功能。

微软宣布的主要新功能是 Windows 终端现在可以被设置为你系统的默认终端模拟器，只要你使用的是 Windows Insider 程序开发频道或普通的 Windows 11。这意味着打开任何命令行应用程序都将在 Windows 终端上运行，而不是 Cmd.exe 或 PowerShell。这个新选项可以从 Windows 终端设置页面的“启动”部分，或者 Windows 设置应用程序的“隐私&安全”部分访问。

另一个附加功能是配置文件匹配，这意味着如果你打开一个与你定制的配置文件同名的可执行文件，这个可执行文件将在那个配置文件中运行。还有一个新的窗口恢复设置，允许你在重新启动 Windows 终端后恢复以前的标签和窗格——类似于网络浏览器多年来提供的恢复功能。

其他变化包括完全透明的背景选项，为 Visual Studio 开发人员命令提示符和 Visual Studio 开发人员 PowerShell 自动生成的配置文件，为提升的终端窗口提供新的盾形图标，以及一些错误修复。你可以从微软商店或者 GitHub 发布页面下载 Windows 终端预览，如果你在稳定的终端频道，这些更新中的大部分应该会在几周内推出。
