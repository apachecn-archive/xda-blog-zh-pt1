最初的 Raspberry Pi 个人电脑今天 10 岁了

原文：https://www.xda-developers.com/raspberry-pi-10-years/#0001-01-01

最初的 Raspberry Pi 微型计算机非常成功，结合了开放的 Linux 驱动的 ARM 系统和超便宜的价格(大约 35 美元)。从那时起，已经有了许多模型，但是到今天(2 月 28 日)为止，启动这一切的微型计算机已经 10 岁了。

Eben Upton 是 Raspberry Pi 基金会的创始人之一，他在博客中写道，“差不多就在十年前的今天，成千上万的人定好了闹钟，在闰日早上醒来，发现我们已经开始销售 Raspberry Pi 电脑。到那天晚上我们的全志愿者团队聚集在酒吧喝庆祝酒的时候，我们的被许可方 Farnell 和 RS Components 已经接受了超过 100，000 个订单(尽管在负载下努力保持他们的网站在线)；我们(短暂地)超越了 Lady Gaga 树莓派正在变得比我们计划的要大一点。”

最初的 Raspberry Pi 是 Pi Model B(Model A 后来出现)，它使用 BCM2835 片上系统，256MB RAM，以太网，根本没有无线网络，以及 26 引脚 GPIO。随后很快出现了更便宜的 A 型，以及内存更大的“A+”和“B+”型。Raspberry Pi 2 于 2015 年上市，配备了高达 1GB 的 RAM 和更新的芯片组，偶尔会有硬件改进的新机型延续至今。尽管 Pi 最初是为教育用途而设计的，但作为 BBC Micro 的精神继承者，Pi 计算机现在被用于无数的嵌入式和低功耗系统中。

树莓派的软件这些年也在持续改进。默认的 Raspberry Pi 操作系统(之前称为“Raspbian”，基于 Debian)刚刚在本月早些时候发布了正式的 64 位版本，许多其他 Linux 发行版和操作系统也可用于 Pi 板。
