三星向Galaxy Tab S7推出了带有一个UI 3.1的Android 11更新

原文：https://www.xda-developers.com/samsung-galaxy-tab-s7-android-11-one-ui-3-1/#0001-01-01

回到2020年8月，三星推出Galaxy Tab S7(我们的回顾)。凭借高通骁龙865 Plus SoC、流畅的120Hz显示屏和高达8GB的RAM，Galaxy Tab S7系列过去是、现在仍然是韩国OEM厂商最强大的Android平板电脑产品。Galaxy Tab S7系列在Android 10之上推出了三星的One UI 2.5皮肤，但现在，Android 11的更新正在推出。令人惊讶的是，更新还带来了最新的UI 3.1软件，具有过多的新功能。

三星Galaxy Tab S7 / S7 Plus XDA论坛T3】

据 SamMobile 报道，韩国Galaxy Tab S7 LTE ( SM-T875 )、S7 Plus LTE ( SM-T975N )和S7 Plus 5G ( SM-T976N )设备的更新正在推出。新版本的软件版本是 TxxxNKOU1BUA8 ，它配备了2021年1月Android安全补丁级别。

三星第一个基于Android 11的软件版本是 One UI 3.0 ，但正如你所见，它已经被Galaxy Tab S7跳过，取而代之的是One UI 3.1。小版本仍然基于Android 11，就像为许多其他Galaxy设备推出的一个UI 3.0更新一样，但它包括了三星做出的一些最新变化，因为它是新推出的 Galaxy S21系列随附的最新版本的软件。

前述更新的完整变更日志可在这里(韩语)获得，它突出了刷新的UI设计、股票应用程序改进、内存分配增强等。这次更新还带来了三星在发布会上宣布的承诺的第二屏幕功能。此功能可让您将平板电脑用作Windows PC的辅助显示器。

如果你有Galaxy Tab S7的韩国版本，但还没有收到更新通知，你可以前往设置应用程序的软件更新部分，检查OTA提示是否适用于你的设备。考虑到最近的趋势，三星可能会在未来几周内在其他地区发布更新。
