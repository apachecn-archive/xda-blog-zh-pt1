微软终于可以在 Windows 11 的文件浏览器中加入标签了

原文：https://www.xda-developers.com/microsoft-bringing-tabs-file-explorer-windows-11/#0001-01-01

这可能是 Windows 中最受欢迎的功能之一，但微软可能会在 Windows 11 中给文件浏览器带来标签。该功能甚至还没有在 Windows Insider 程序的开发频道公开发布，但开发者 Rafael Rivera 在幕后发现了它。

如果出现这种情况，那就是大事了。如果你在 Feedback Hub 应用程序中搜索文件浏览器建议，当按 upvotes 排序时，顶部结果是 tabs，有 3，285 票。第二个跟版面有关，51 票。差别很大。

显而易见，选项卡极大地改善了浏览器的体验，而且它们会让任何其他应用程序变得更加高效。微软甚至一度有一个名为 Sets 的功能，允许你将应用程序放在标签组中。不幸的是，Sets 在发布到 Insider 计划之外之前就被扼杀了。据推测，它只是复制了太多已经存在于任务栏中的功能。

如果您喜欢选项卡式应用程序，可以使用第三方工具。明星制造了一个叫追星族的人。

同样值得注意的是，这可能永远不会见天日。即使它在 Dev 频道上市，也可能永远不会上市。微软很清楚这一点，开发频道不依赖于特定的版本，它是为了测试可能会或可能不会发布的新功能。而且这个功能也不是现成的。

尽管这是可能的。正如里维拉的推文中提到的，有一个可以打开的功能标志。当然，Microsoft 不建议您启用它没有为您启用的功能。
