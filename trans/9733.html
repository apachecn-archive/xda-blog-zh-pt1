Realme kicks 为 Realme GT Neo 2、Realme GT Master Edition 和 Realme X7 Max 启动 Realme UI 3.0 公测程序

Realme 已经为 Realme GT Neo 2、Realme GT Master Edition 和 Realme X7 Max 启动了 Realme UI 3.0 公开测试计划。请继续阅读，了解如何参与。

在去年年底向一些 Realme GT Neo 2 和 Realme X7 Max 用户推出基于 Android 12 的 Realme UI 3.0 早期版本后，Realme 现在终于准备好给更多用户一个体验软件发布的机会。今天，该公司已经为这两款设备启动了 Realme UI 3.0 开放测试计划，允许更多用户在稳定推出之前体验更新。

根据 Realme 论坛上最近的帖子，Realme UI 3.0 公开测试版计划现已面向 Realme GT Neo 2 、 Realme X7 Max 和 Realme GT Master Edition 用户开放。如果您拥有这些设备中的任何一款，并且希望在稳定推出之前试用基于 Android 12 的软件版本，您可以通过以下步骤注册测试计划:

  • 请确保将您的设备更新到公开测试版公告中为您的设备指定的软件版本。
  • 安装更新后，导航至设备设置中的软件更新部分，点击右上角的设置按钮
  • 点击下页的试用版，然后选择开放测试版选项
  • 点击下一页的立即申请按钮，然后提供所有必需的详细信息以完成您的申请
  • 如果您的应用程序通过了 Realme 的审查过程，您应该会通过 OTA 更新收到 Realme UI 3.0 开放测试版

在安装更新之前，请确保您的设备不是根设备，至少有 60%的电池剩余，并有大约 5GB 的可用存储空间。此外，在继续之前，请备份您的所有数据。请注意，由于这不是最终的稳定版本，您可能会遇到一些错误。如果您将设备用作日常驱动程序，我们不建议安装开放测试版更新。如果您仍然希望继续，请确保按照公告帖子中的说明报告您遇到的所有错误。
