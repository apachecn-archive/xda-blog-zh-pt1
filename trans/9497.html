Windows 11 上的安卓应用发布预览版

原文：https://www.xda-developers.com/windows-insiders-release-preview-android-apps-windows-11/#0001-01-01

如果你是注册了发布预览版频道的 Windows Insider，并且正在运行 Windows 11，你现在可以开始在你的 PC 上测试 Android 应用了。微软正准备在下个月向所有 Windows 11 用户推出这一功能——尽管它仍将被称为预览版——按照惯例，发布预览版频道中的 Windows 内部人士可以提前尝试一下。

如果你还不知道，Windows 11 上的 Android 应用程序运行在 Android 的 Windows 子系统上，该子系统是基于 Linux 的 Windows 子系统构建的，多年来一直是 Windows 10(和 11)的一部分。微软正在与亚马逊应用商店合作推出它们，亚马逊应用商店是你可以为你的安卓 PC 获得安卓应用的唯一官方来源。目前，那里只有大约 50 个应用程序可用，而且只在美国有效，但是你可以绕过这些限制。

通过改变一些设置，你可以在网络上的任何地方安装 Android 应用程序，只要不需要谷歌移动服务，它们应该可以工作。你可以在这里了解更多关于在 Windows 11 上侧载 Android 应用的信息，如果你使用一款名为 WSATools 的新第三方应用，这甚至会更容易。

对于 Beta 频道(以及后来的 Dev 频道)的内部人士来说，Android 应用程序已经推出几个月了，最近，我们的执行主编里奇·伍兹写了一篇关于体验如何演变的文章。它仍然不完美，但如果你需要一种在 PC 上运行智能手机应用程序的方法，它就能完成任务。

如果你不是 Windows Insider，Android 应用支持很可能会与下个月 2 月 8 日的补丁星期二更新一起到来，但这不是一个保证。除此之外，你还将在任务栏上获得一个新的天气小工具，一个新的麦克风静音图标和便于团队共享的窗口，以及新的记事本和媒体播放器应用程序。对于 Windows 11 来说，这将是一个重要的月份，尽管真正的功能更新预计将在今年下半年进行。
