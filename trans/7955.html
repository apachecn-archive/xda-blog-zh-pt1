在第三方屏幕修复后，Face ID 停止了 iPhone 13 的工作

原文：https://www.xda-developers.com/face-id-stops-working-on-iphone-13-after-any-third-party-screen-replacements/#0001-01-01

在第三方更换屏幕后，Face ID 停止了 iPhone 13 的工作

一个新的视频指出，Face ID 在未经授权的维修后停止工作，但不清楚这是一个错误还是预期的功能。

苹果本月早些时候刚刚发布了 iPhone 13 系列，有四款机型可供选择: iPhone 13 、 iPhone 13 Mini 、 iPhone 13 Pro 和 iPhone 13 Pro Max 。这款手机比以前的型号更上一层楼，显示屏更平滑，摄像头更强，但 iPhone 13 系列至少在一个方面似乎比早期的 iPhone 更差——在除了苹果(或苹果授权的维修中心)之外的任何人更换你的屏幕后，Face ID 将停止工作。

下面这段来自手机维修大师(通过MAC rumors)的视频展示了两部 iPhone 13 手机的显示屏被调换。尽管显示屏是真正的苹果零件，屏幕组件不包含任何与 Face ID 直接相关的组件，但结果是 Face ID 不再工作。

目前还不清楚这是一个软件错误，还是针对未经授权的 iPhone 维修的另一项措施。过去几年，苹果对第三方维修越来越不友好。苹果有自己的独立 iPhone 维修计划，为精选公司或第三方维修中心提供苹果正品零件和维修手册。然而，iFixit 去年的一份报告指出，维修中心加入该计划可能需要几个月的时间，苹果经常以高价向维修中心出售零件。在某些情况下，零件费用超过了苹果公司进行整体维修的费用。

苹果尚未发表关于 Face ID 和第三方维修的声明。如果 Face ID 有意违约，这可能只会给“维修权利”运动提供更多动力，该运动推动世界各国政府迫使电子产品制造商随时提供更换零件和维修手册。美国总统乔·拜登(Joe Biden)在 7 月签署了一项行政命令，呼吁联邦贸易委员会(FTC)制定设备维修指南，而世界上其他国家也在不同阶段起草类似的立法。
