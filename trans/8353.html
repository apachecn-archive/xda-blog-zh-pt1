小米红米 Note 11 系列确认提供 120W 快充

原文：https://www.xda-developers.com/xiaomi-redmi-note-11-series-confirmed-to-offer-120w-fast-charging/#0001-01-01

下一个 Redmi Note 阵容将把中端细分市场的快充大战提升到一个全新的水平。小米周一证实，其即将推出的红米 Note 11 系列将支持高达 120W 快速充电。

上周，红米总经理陆证实，红米 Note 11 系列将于 10 月 28 日与红米 Watch 2 一起推出。在正式发布之前，这家中国公司已经透露了(通过 小米手机 电报频道)其即将推出的智能手机系列的一些主要功能和设计。小米分享的预告暗示至少会有三款:红米 Note 11、红米 Note 11 Pro、红米 Note 11 Pro+。

小米 11T Pro 评测:120W 快充快得吓人！T3】

经小米证实，Redmi Note 11 Pro+将封装双芯电池，并支持 120W 快速充电。目前还不清楚 120W 快速充电器是包含在包装盒中还是单独出售。小米表示，这款手机的电池也通过了 TUV 莱茵的安全认证。

Redmi Note 11 Pro+的其他确认功能包括 3.5 毫米音频插孔、蓝牙 5.2、多功能 NFC、Wi-Fi 6 支持、X 轴线性电机和 JBL 调谐扬声器。最后，预告显示，Redmi Note 11 Pro+将在背面设置四摄像头。其他硬件规格，如 SoC、摄像头传感器、显示器等还不知道。

红米 Note 11 将接替去年的红米 Note 10 系列。Redmi Note 10 系列取得了巨大的成功，我们预计 Redmi Note 11 也将获得类似的成功。

120W 快速充电是否将仅限于中国车型，还是也适用于印度和全球车型，还有待观察。虽然中国的 Redmi Note 10 系列提供高达 67W 的快充，但海外型号的快充最高可达 33W。

距离正式发布只有几天了，我们不用等太久就能知道新的 Redmi Note 系列为我们准备了什么。
