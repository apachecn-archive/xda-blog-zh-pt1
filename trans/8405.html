你很快就能在 Windows 上体验三星的 One UI 设计

原文：https://www.xda-developers.com/samsung-one-ui-book-4-windows/#0001-01-01

与大多数 Android OEMs 不同，PC 制造商没有自由来广泛修改工厂安装的 Windows 实例的外观和感觉。这就是为什么你不会看到像小米的 MIUI 或 OPPO 的 ColorOS 这样的流行定制 Android 皮肤的成熟 PC 版本的主要原因。这种情况现在可能会改变——至少在某种程度上——因为三星计划将 One UI 4 界面设计引入其 Galaxy Book 系列 Windows 笔记本电脑。

正如在今年的三星开发者大会(SDC)上宣布的一样，微软和三星之间合作关系的下一步扩展意义重大，原因有几个，尽管看起来可能不像。这个想法是让基本的用户界面元素在皮肤之间保持同步，你可以在三星最新的 Android 手机和运行 Windows 11 的笔记本电脑上找到，这样最终用户可以很容易地在两者之间切换。韩国 OEM 厂商将这一新设计称为“One UI Book 4”，并将很快在 Galaxy Book Pro、Galaxy Book Pro 360 、Galaxy Book Flex2、 Galaxy Book Odyssey 和普通 Galaxy Book 上推出。

当涉及到实施设计变更时，最初将在三星的一套 Windows 应用程序中执行，如三星笔记、三星图库和三星设置(h/t:The Verge)。例如，画廊将获得重新设计的菜单栏和黑暗模式，以匹配 Windows 11 的主题设置。另一方面，Notes 应用程序将会有一系列新的图标、菜单布局等等。最后，设置应用程序将从其移动版本中获取一些设计线索，以更好地适应即将发布的 One UI 皮肤的主要版本中的设计变化。

值得注意的是，你也可以在非三星电脑上安装这些应用程序。这样做的时候，看起来前面提到的几个特性已经在最近的大部分版本中发布了。例如，最新版本的 Gallery 应用程序带有可定制的黑暗模式设置，尽管新的菜单栏尚未推出。

这不是我们第一次听说微软和三星之间的密切合作关系。微软 Office 套件的 Android 版本专门针对 Galaxy foldables 进行了优化，而 Windows 的 Your Phone 应用程序为三星旗舰产品提供了大量现成的独家功能。现在，你可以通过 Android 的 Windows 子系统在你的 PC 上运行 Android 应用程序，我们希望在不久的将来看到这两者之间如此丰富的集成。
