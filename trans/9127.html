如何用 Pojav Launcher 在 Android & iOS 上玩《我的世界》Java 版

原文：https://www.xda-developers.com/how-to-play-minecraft-java-edition-smartphone/#0001-01-01

《《我的世界》》是安卓系统上最受欢迎的游戏之一，尽管它不像你在桌面上看到的 Java 版游戏那样功能齐全。你可以玩的服务器非常有限，当然，Java 玩家也更容易修改他们的游戏。然而，它实际上可以在你的智能手机上玩 Java《我的世界》，而且设置起来出奇的简单。你可以在 Android，越狱的 iOS 设备和 Chromebooks 上玩。

我在我的谷歌 Pixel 6 Pro 上玩这个游戏取得了巨大成功，尽管你在任何一部配备 adre no GPU 的最好的手机上都会有更好的结果，比如一加 9 Pro。注意:玩《我的世界》1.17.1 也需要一个变通方法，最新的悬崖&洞穴第二部分更新(1.18.1 版本)也需要一些复杂的步骤。你可以查看 Pojav Launcher 官方网站了解更多信息，包括如何安装 Optifine 或 Forge。

尽管如此，这仍然令人惊讶地工作得很好，甚至还有一个鼠标指针和触摸控制，可以用来导航菜单。你可以使用 1.16.5 版本的 Forge Launcher 来获得更好的体验，同时也支持 Fabric。启动器在 GitHub 上是完全开源的。令人难以置信的是，我在谷歌 Pixel 6 Pro 上的性能比我在一些笔记本电脑上看到的更好，而且没有任何优化。体验中最讨厌的部分是，你不能通过点击来轻松点击——你需要按住屏幕才能使用武器。当在 PvP 服务器上玩的时候，这变得很复杂。

使用 Pojav Launcher 进行游戏时，有一些已知的问题，概述如下:

  • 由于新的 GLSL 用法，目前尚不支持《我的世界》21w10a或更新版本。启动器中提供并内置了一个解决方法。
  • 在 1.16 和更高版本中，产卵横幅是白色的(你可以通过将渲染器切换到gl4es 1.1.5来解决这个问题，只在 1.16 和更高版本中有效，不要在这个版本中使用，因为当遇到暴徒时纹理会出现问题)
  • 控制器模块不起作用。
  • 在游戏加载或加入世界期间，Android 5.x 上经常会发生随机崩溃。
  • 对于大的模块包，纹理可能会被搞乱

我们想提醒用户，您将被要求使用该应用程序登录他们的 Microsoft 或 Mojang 帐户，您这样做的风险由您自己承担。Pojav Launcher 的源代码可以在 GitHub 上获得，测试它最安全的方法是审计源代码，并按照官方说明自己构建。我一直用我的个人《我的世界》账户和从谷歌 Play 商店下载的启动器来使用它。

如何使用 Pojav Launcher 在 Android 智能手机上玩桌面《我的世界》

首先，您需要在谷歌 Play 商店上下载 Pojav Launcher。这是一个镜像桌面上的《我的世界》启动器，需要你登录你的 Mojang 帐户。它从 Mojang 的服务器上下载包括游戏资产在内的所有必要文件，甚至打包了自己的 Java 运行时环境(JRE)。

安装应用程序后，登录您的《我的世界》帐户。请务必点击“保持登录”，以便下次您可以点击“选择帐户”。选择您想要玩的游戏版本，然后点按“玩”。对于游戏的早期版本来说，这是一个非常简单的过程，然而，随着新版本的出现，这个过程变得更加复杂。

为了玩《我的世界》1.17，你需要为你的特定设备下载 JRE 17，并从右下角的设置中更改渲染器。选择运行时管理器，添加新的 JRE17 zip 文件，然后将渲染器更改为带有 OpenGL ES 3 的 gl4es 1.1.5。它会说“还不支持 1.17”，但如果你有 JRE 17，那么它是受支持的。您还可以从设置中增加指定给 JVM 的 RAM 数量，如果您遇到性能问题，尤其是在使用 mod 时，这可能是值得的。

然后你可以点击“播放”，如果你在一个对话框中得到一个警告，点击“仍然播放”。虽然您应该按照官方说明在 Pojav Launcher 中安装 Optifine，但是在尝试将文件复制到/SD card/Android/data/net . kdt . pojavlunch【时，您可能会遇到困难。调试]/文件/。minecraft 文件夹感谢 Android 11 的范围存储变化。你可以使用内置的 Android 文件浏览器或电脑将文件复制到其中，你需要根据你的配置修改不同的文件。安装 1.18 版本的自定义资源包需要导航到这个文件夹，为你的游戏安装某些 mod 也是如此。

至于安装 Optifine，也需要切换回内部的 Java JRE 和原始的 gl4es 渲染器。不切换将导致 Optifine 安装程序在启动时崩溃，尽管一旦安装，您需要再次切换回 JRE17 和 gl4es 1.1.5 以及 OpenGL ES 3 才能实际播放。这很复杂，很容易忘记你需要做什么，所以你最好的办法就是仔细按照说明去做，以确保你做对了。

如果您有性能问题，您可以尝试许多方法。最重要的是降低分辨率，这可以从启动器的设置中完成。如果做不到，你也可以遵循 Pojav Launcher 网站上的官方优化指南,其中有一个链接指向 Fabric 1.16.5 的一组 mods，可能会提高你的性能。虽然几乎所有的 mod都旨在提高性能，但包中也有一两个用于提高生活质量。包中的一些模块包括:

  • 钠，一个新的渲染引擎。
  • 星光，光引擎的返工。
  • FerriteCore，以减少内存使用。
  • Smoothboot，减少低端设备的加载时间。
  • OKZoomer，用于最佳变焦控制。
  • NotEnoughCrashes，报告更多关于崩溃日志的细节，并允许你继续游戏。

在低端笔记本电脑上运行《我的世界》Java 版经常是一款费力的游戏，但许多用户在智能手机上运行它的体验非常棒。虽然它并不完美，但如果你的朋友在 Java 版服务器上玩游戏，而你的电脑或笔记本电脑没有足够的能力运行游戏的 Java 版，这可能是你和他们一起享受游戏的最佳方式。
