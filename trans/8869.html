三星 Galaxy Note 20 系列获得第三个 UI 4 测试版，修复了几个错误

三星已经开始向 Galaxy Note 20 和 Galaxy Note 20 Ultra 推出基于 Android 12 的第三个 One UI 4 测试版。

三星正在努力向其一系列设备发布基于 Android 12 的 UI 4 测试版。在过去的几周里，该公司发布了两个 One UI 4 测试版给 Galaxy Note 20 系列。第一个测试版更新包括了 Android 12 中引入的所有新功能，而第二个测试版将 Android 安全补丁级别提升至 2021 年 12 月。三星现在已经开始推出 Galaxy Note 20 和 Galaxy Note 20 Ultra 的第三个测试版，为这些设备带来了几个错误修复。

One UI 4 Beta 概述:动手体验三星 Android 12 更新中的每一项功能！T3】

据知名三星消息人士@ frontron称，第三次 beta 更新(软件版本 ZUL1)已经开始向 Galaxy Note 20 和 Galaxy Note 20 Ultra 推出。此更新包括以下错误修复:

  • 已修复的错误
    • 修复了一些应用程序中的指纹认证问题
    • 修正了选择快速面板项目时无效的问题
    • 修复了关于音量监视器的数据显示错误
    • 修正了重启后屏幕亮度变暗的问题
    • 修正了锁屏不能快速解决时的休息问题
    • 修正了 Galaxy Watch 4 的崩溃问题
    • 修正了电池很快耗尽的问题
    • 改进了与 Windows PC 的 USB 连接的断开操作
    • 修复了一些小问题

目前，我们无法获得 Galaxy Note 20 系列最新 One UI 4 beta 更新的下载链接。如果你有一部 Galaxy Note 20，并且想尝试一下最新的 UI 4 测试版，你可以在三星会员应用程序中注册测试版计划。只需点击应用程序顶部的 One UI beta 横幅，然后点击以下屏幕上的注册按钮。等待几分钟，让应用程序处理您的注册，然后前往设备设置中的软件更新部分。如果您的设备有 beta 版本，当您点击“检查更新”时，它应该会显示出来。或者，您可以查看我们的 One UI 4 更新跟踪器，以便在 OTA 包可用时获取它。
