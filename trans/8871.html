Windows 11 将让你再次更容易地改变你的默认浏览器

原文：https://www.xda-developers.com/windows-11-change-default-browser-button/#0001-01-01

微软正在恢复 Windows 11 中的一项更改，该更改使得通过系统设置更改默认浏览器变得更加困难。随着本周早些时候发布的 Windows 11 build 22509，微软在设置应用程序中添加了一个按钮，允许你快速将默认浏览器更改为你喜欢的任何浏览器，这是 Twitter 用户拉斐尔·里维拉(通过 The Verge )发现的。

正如 Rivera 分享的图片所示，当你进入 Chrome 等浏览器的文件关联页面时，你会在顶部看到一个按钮，让你将其设为默认浏览器。这也是当浏览器询问您是否要将其设置为默认浏览器时，您将被带到的页面，因此现在只需点击几下鼠标就可以更容易地更改您的默认浏览器。微软已经证实这一变化是有意的，但它没有说我们是否需要等待明年秋天的完整功能更新，或者它是否将通过当前版本的 Windows 11 的累积更新来提供。

这一变化非常重要，因为这是微软试图强迫用户使用 Edge 浏览器的最具争议的举措之一。当你试图在 Windows 11 中打开一个浏览器之外的链接，并且你安装了多个浏览器时，你会被要求选择一个浏览器。但是除非你特别选择了一个不同于 Edge 的浏览器并且勾选了复选框使那个浏览器成为你的默认浏览器，否则 Edge 将会成为你的默认浏览器。雪上加霜的是，Windows 11 设置页面一般不再让你选择默认的浏览器应用；相反，您必须单独更改 HTTP 协议、HTML 文件等的关联，这使得更改默认浏览器更加繁琐。这促使 Mozilla 开发了一种黑客技术，允许 Firefox 绕过所有这些，并使自己成为默认浏览器。

然而，情况变得更糟，因为微软采取了其他违背用户选择的措施。该公司最近屏蔽了 Edge Deflector ，这是一款将原生 Windows 11 链接从 Edge 重定向到用户默认浏览器的应用。就在昨天，人们发现如果用户试图下载谷歌 Chrome 浏览器，该公司还会在 Edge 浏览器中显示令人沮丧的提示。希望我们能看到更多对消费者友好的变化，让用户更容易选择自己的浏览器。

除了这一变化， Windows 11 build 22509 还包括一些针对 Windows 11 的调整，即给用户更多新的开始菜单定制选项，允许他们看到更多固定项目或底部推荐列表中的更多项目。它还将控制面板中的更多设置引入了 Windows 11 设置应用程序，这是微软将其所有设置统一在一个界面下的过程的一部分，该过程已经持续了几年。
