随着 Windows 11 的推出，看看过去的 Windows 版本历史

原文：https://www.xda-developers.com/windows-version-history/#0001-01-01

Windows 11 已经发布一年多了，这是 Windows 多年来收到的最好的更新之一。如果你读了我们自己的 Rich Woods 写的评论，你会知道我们对微软新操作系统的发展方向非常满意。它很漂亮，在许多方面更有能力，而且使用起来感觉很棒。但是，每当我们向前迈出一大步，也是一个很好的时间来回顾是什么导致了我们的现状。所以，为了庆祝 Windows 11 的发布，让我们来看看 Windows 版本的历史，看看每个版本都带来了什么。

我承认，我没有经历过 Windows 的所有主要版本。我第一次使用电脑是在 Windows XP 上，我并不总是一个技术爱好者。那是很多年后的事了，我得说我真正开始关注新的 Windows 版本是在 Windows 8.1 刚刚发布的时候。所以，这是真的，我参与 Windows 世界的时间没有其他人长，但回顾事情开始的地方总是很有趣。让我们开始吧。

Windows 1.0

虽然我们今天知道 Windows 是自己的操作系统，但它实际上更多的是建立在微软磁盘操作系统(MS-DOS)之上的 GUI。Windows 的最初版本是——坐稳了——Windows 1.0，发布于 1985 年 11 月，它给用户提供的不仅仅是一个基于文本的界面来与他们的电脑进行交互。你可以在 windows 中打开应用程序，即使它们不能重叠，因为重叠窗口的概念并没有马上实现。相反，应用程序是平铺的，只能并排显示。

Windows 1.0 还包括一些应用程序，为用户提供了提高工作效率的方法。有计算器、日历、剪贴板查看器、时钟、记事本、画板、卡片文件、终端和书写工具。如果你想娱乐一下，你也可以玩几局黑白棋。

Windows 2.x

Windows 2.0 于 1987 年 12 月发布，作为 Windows 1.0 的继任者，它仍然基于 MS-DOS，但它给桌面带来了一些大的改进。它的整体性能更好，但你可能会注意到的最大变化是增加了对重叠窗口的支持。Windows 2.0 也发布了两个版本，分别针对不同类型的处理器，Windows/386 是更现代的版本，具有抢占式多任务处理等功能。

Windows 2.0 中包含的许多应用程序与 Windows 1.0 中的相同，但 Windows 2.0 后来获得了更多的应用程序支持。微软自己在 1989 年为 Windows 开发了 Word 和 Excel 的第一个版本，这是一件大事。Windows 2.1 仅在 Windows 2.0 之后六个月推出，它是第一个需要硬盘驱动器的 Windows 版本。

Windows 3.0

Windows 版本历史的下一步是 1990 年 5 月推出的 Windows 3.0，它对用户界面进行了重大的重新设计。虽然它在大多数方面仍然类似，但它看起来非常不同，用更多的 3D 感觉取代了平面外观。但它也包括一些大的新功能。比如说。MS-DOS 的执行程序，即文件管理器，被程序管理器、文件管理器和任务列表所取代。其他重大更新包括计算器应用程序中的科学计算器支持，Paint 成为改进的画笔应用程序，以及添加了空前经典的游戏 Solitaire。事实上，Windows 3.0 还改进了新处理器的内存管理。

Windows 3.0 获得了几项重大更新，但最值得注意的可能是多媒体扩展，增加了对录音和播放音频的支持。它还包括一个新的闹钟应用程序。Windows 3.0 是微软 Windows 家族的第一个巨大成功。

Windows 3.1

Windows 3.0 两年后，Windows 3.1 发布，这次发布也意义重大。它包括了第一种字体的 TrueType 字体系统，使得字体更易于阅读和扩展。我们今天仍然知道的标志性字体，如 Times New Roman 和 Arial，也是在这个时候推出的。许多图标也得到了改进，WIndows 3.1 增加了对拖放它们的支持，所以你可以将文件拖到应用程序图标或 WIndows 中，用该应用程序打开该文件。对于黑白棋的粉丝来说，一个很大的变化是这个游戏被扫雷取代了——从那以后 Windows 版本中的另一个主要内容。

Windows 3.1 还包括一些大的变化，你需要一个英特尔 80286 处理器和至少 1MB 的内存来运行操作系统。这使得微软能够解决用户在 Windows 3.0 中遇到的一些稳定性问题。

Windows NT 3.x

此时，微软已经开发基于 MS-DOS 的 Windows 版本有一段时间了，但在 1993 年，它终于把 Windows 变成了自己的完整操作系统。这是第一个基于 Windows NT 内核的版本，直到今天，它已经被更新并支持新的 Windows 版本。Windows NT 3.1 是一个 32 位操作系统，可以在 Intel x86 处理器以及 DEC Alpha 和 MIPS 处理器上运行。不过，它对系统的要求比 Windows 3.1 高得多，包括 12MB 内存和一个 VGA 显卡。

虽然它具有与 Windows 3.1 相同的功能，但该版本还包括一些独有的功能，如性能监视器、磁盘管理器、事件查看器和备份应用程序。

在接下来的几年中，Windows NT 3.5 和 3.51 也相继发布，增加了一些功能，如 Winsock 支持、FTP 和其他重要的商业功能。这些版本还具有改进的性能。WIndows NT 3.51 还增加了对 IBM Power PC 处理器的支持。

Windows 95 和 Windows NT 4.0

在接下来的几年里，微软不断发布基于 MS-DOS 和基于 Windows NT 的 Windows 版本，Windows 95 和 Windows NT 4.0 是每个版本的下一个主要版本。这些版本引入了重大的新概念，如任务栏和开始菜单。任务栏是显示当前正在运行的应用程序的地方——我们今天所知道的范例——而桌面被重新用于保存应用程序的快捷图标。当然,“开始”菜单是启动应用程序和打开文件的新方式。

文件管理器应用程序被替换为 Windows 资源管理器，它还首次包括了回收站。还添加了像文档这样的用户文件夹，自动运行是一项新功能，当 CD 驱动器连接到 PC 时，它允许计算机采取特定的操作。这仍然是今天的一个主要功能，但它现在被称为自动播放，它适用于所有类型的驱动器。其他新功能包括对外部设备的即插即用支持和对最多 255 个字符的文件名的支持。虽然最初它并不包含在 WIndows 95 中，但后来 Internet Explorer 作为一项功能被添加进来。

Windows NT 4.0 主要带来了从 Windows 95 到 Windows NT 的变化，但也包括一些默认的 Windows 95 软件包中没有的功能，如 Space Cadet，这是一个你可能已经知道的弹球游戏。

视窗 98

Windows 98 是 Windows 95 的继任者，它主要集中在改进前一版本的某些方面。与以前的版本相比，它具有更多基于网络的元素，不仅与 Internet Explorer 一起提供，还与 Outlook Exporess、FrontPage Express 和 Microsoft Chat 等应用程序一起提供。还有对界面的一般性改进，包括通过单击任务栏上的窗口来最小化窗口的能力，以及改进的 windows 资源管理器中的导航。

此外，还有一些改进，比如引入了 Windows 驱动程序模型(WDM)来帮助提供硬件支持。尽管有了重大更新，WDM 仍然在今天的 Windows 中使用。然而，这个版本没有与 Windows NT 对等的版本。

Windows Me 和 Windows 2000

最后一次回到双版本，Windows Me 和 Windows 2000 分别是基于 MS-DOS 和 Windows NT 的等效版本。Windows 2000 于 2000 年初首次发布，它为 Windows NT 带来了许多 Windows 98 的功能，包括许多尚未移植的辅助功能，以及像屏幕键盘和讲述人这样的新功能。除了英语，它还引入了对多种语言的支持。Windows 2000 还引入了计算机管理控制台，包括磁盘管理、设备管理器和磁盘碎片整理程序等工具。

Windows Shell 也在这个版本中得到了一些大的改进，支持透明和阴影等效果，任务栏增加了对气球通知的支持，可以用来引起用户的注意。Windows 资源管理器有可定制的工具栏，加上自动完成支持。在底层，还有对 NTFS 3.0 的支持，这是一个用于 Windows NT 的具有新功能的新文件系统，以及文件系统加密。

Windows Me 包括与 Windows 2000 相同的对外壳的改进，以及与以前版本相比的许多新功能，包括添加了视频编辑器 Windows Movie Maker，以及 Windows Media Player 和 Windows DVD Player 的更新版本。新的系统工具，如系统还原和自动更新，也增加了这个版本，屏幕键盘也从 Windows 2000 的方式。Windows Me 也改进了硬件支持。然而，最终 Windows Me 作为一个不稳定的版本被载入史册，它不是一个受欢迎的 Windows 版本。这也归功于我们接下来要讨论的内容。

Windows XP

可以说是有史以来最具标志性的 Windows 版本之一，Windows XP 于 2001 年首次发布，这是微软第一次完全放弃基于 MS-DOS 的操作系统。Windows XP 完全基于 Windows NT，面向消费者和商业用户，不像以前的 Windows NT 版本更侧重于后者。Windows XP 的特色是重新设计了 UI，称为 Luna，外观更加丰富多彩(还有标志性的 Bliss 壁纸)。Windows XP 还增加了像任务分组这样的新功能，因此同一应用程序的多个实例在任务栏中被分组在一起。

当然，Windows XP 中添加或大幅更新了大量功能和内置应用程序，因此我们不可能涵盖所有功能和应用程序，更多功能和内置应用程序是随服务包一起添加的，其中 Windows XP 获得了三个服务包。总的来说，Windows XP 的性能和稳定性比它的前辈好得多，正因为如此，它获得了消费者的巨大吸引力，成为历史上最成功的 Windows 版本之一。由于其受欢迎程度，它获得支持的时间远远超过了微软的生命周期政策，直到 2014 年才收到安全更新。然而，让用户难以离开 Windows XP 的部分原因是它的直接跟进。

Windows Vista

2006 年发布的 Windows Vista 是微软的另一个臭名昭著的版本，原因都是错误的。甚至我还记得当时对该操作系统的强烈反对，我记得几年后当我的家用台式电脑可以使用 Windows 7 时，我真的很想把它换成 Windows 7。Windows Vista 再次对操作系统进行了重大的重新设计，推出了名为 Windows Aero 的新界面。这个新的用户界面使用了新的透明效果、动画，总体感觉比以前的版本更生动、更漂亮，但这对性能有很大影响。例如，RAM 需求是 Windows XP 的八倍，即使这样，低端硬件的性能也不是很好。

另一个困扰 Windows Vista 一段时间的批评是用户帐户控制的实现，当试图在系统上做任何事情时，它被认为会带来太多的安全提示。这个问题后来得到了解决，但 Windows Vista 的声誉很快受到损害，未来的更新也不足以缓解这一问题，使其成为操作系统历史上最不受欢迎的 Windows 版本之一。

尽管如此，它为下一个主要的 Windows 版本奠定了基础，并且对系统应用程序也进行了一些重大改进。其中包括 Windows Media Player 11、Internet Explorer 7、Windows Search、Windows Mail 等等。

Windows 7

微软从对 Windows Vista 的批评中吸取了教训，并在 2009 年用 Windows 7 创造了另一个重大成功。从外观上看，Windows 7 肯定没有偏离 Vista 太远，更新了 Windows Aero 设计，增加了透明度和视觉效果，使其与 Vista 一样漂亮，如果不是更漂亮的话。然而，对任务栏等方面进行了一些大的更新，用任务栏上可锁定的应用程序取代了快速启动工具栏，以及应用程序在任务栏上仅由其图标表示，而不是在图标旁边添加应用程序的名称(尽管这可能会改变)。

同样，包含了大量新功能和改进功能，包括 Windows Media Player 的最新主要版本 12。计算器应用程序获得了多线支持和单位转换，Windows PowerShell 首次捆绑在一起，等等。

虽然系统需求再次上升，但与 Vistas 在 Windows XP 上的变化相比，这种增长并不显著。总的来说，Windows 7 因其显著提高的性能和稳定性而备受推崇，就像 Windows XP 一样，它成为历史上最成功的 Windows 版本之一，许多用户在其继任者的生命中继续使用它。

Windows 8

2012 年，微软意识到它已经有一段时间没有做出糟糕的 Windows 版本了，所以它决定发布 Windows 8。事实上，虽然 Windows 8 受到了大多数人的严厉批评，但我实际上是微软当时试图对该操作系统所做的事情的粉丝。随着智能手机和平板电脑等基于触摸的设备的兴起，微软试图基于磁贴和“Metro”UI 创建一个更友好的触摸界面，具有更大的 UI 元素、扁平的颜色和四周的锐角。“开始”菜单被“开始”屏幕取代，这是一个与 Windows 桌面完全不同的环境。

Windows 8 还推出了 Windows Store，专门为 Windows 8 设计了新的应用程序，但这些“现代”应用程序只能全屏或与另一个应用程序并排打开，这使它们比历史上任何其他 Windows 版本都更受限制。当并排拍摄时，应用程序只能以两种不同的大小使用，尽管桌面仍然可以像以前一样用来打开传统的应用程序。

不过，Windows 8 确实带来了一些不错的功能，比如 OneDrive 集成允许您从 Windows Explorer shell 轻松访问云文件，这是 Windows Explorer 等一些系统应用程序的新功能区 UI，使常见功能变得容易访问，而且在性能方面，它在许多方面实际上比 Windows 7 更好。Windows 8 也是 PC 和移动版 Windows(当时称为 Windows Phone)在设计方面开始更加一致的时候。但是新的用户界面非常难学的事实使得这对于大多数 PC 用户来说是一个不受欢迎的升级。

Windows 8.1

虽然 Windows 10 将普及“Windows 即服务”的理念，但 Windows 8 在 Windows 8.1 版本中引入了这一理念。Windows 8.1 是 Windows 8 用户的免费更新(尽管升级路径可能有点麻烦)，但它的范围比以前 Windows 版本的服务包大得多，更接近于完整版本升级，而完整版本升级通常是在那时付费的。Windows 8.1 解决了许多针对 Windows 8 的批评，改善了快照应用的工作方式，增加了新的大小调整选项和改进的多显示器支持，恢复了开始按钮(尽管它仍然会打开开始屏幕)，等等。

许多包含的应用程序也更新了新功能和改进的设计，包括大幅改进的 Internet Explorer 11 等等。微软还增加了一个“现代”计算器应用程序、录音机等等。Windows 8.1 更新进一步改进了 Windows 8.1，增加了将“现代”应用最小化到任务栏的功能和其他较小的改进。大约在这个时候，微软也开始倾向于 Windows PCs 和智能手机的“通用”应用的概念，这将在 Windows 10 中变得更加普遍。无论如何，Windows 8 作为一个整体作为迄今为止最糟糕的 Windows 版本之一被载入史册。

Windows 10

在 Windows 8.1 更新发布后不久，微软在 2014 年晚些时候宣布了 Windows 10，以及 Windows Insider 计划，首次向任何人开放 Windows 的 beta 测试。Windows 10 的第一次稳定发布发生在 2015 年 6 月。Windows 10 带回了 Windows 7 的许多受欢迎的元素，包括正确的开始菜单——现在也有了 Windows 8 的许多元素，如动态磁贴。它还使得在 windows 中打开“现代”应用程序而不是平铺它们成为可能，就像对待经典应用程序一样对待它们。Windows 10 还进一步推动了 Windows PCs 和智能手机的统一，尽管没过多久 Windows 手机就被完全抛弃了。

Windows 10 也带来了 Windows 即服务的想法，在 2015 年首次推出后发布了大量更新。事实上，如果你有 Windows 7 或 8.1，Windows 10 本身就是一个免费的更新。这些重大更新被称为功能更新，因为它们通常会添加新功能，而不是改变 Windows 品牌，微软在 Windows 10 之后开始添加版本号。因此，我们获得了 Windows 10 版本 1511、1607、1703、1709、1803、1809、1903、1909、2004、20H2 和 21 h1(21 H2 也与 Windows 11 一起为一些用户发布)。许多新功能都是以这种方式添加的，而有些功能已经停止了。最值得注意的一个特点，个人认为，是 1607 版本增加了黑暗模式，但还有更多。然而，许多添加的功能也在不久后被放弃，Paint 3D 或 Mixer 集成就是明显的例子。

总的来说，Windows 10 最终受到了欢迎，尽管在 Windows 8 期间坚持使用 Windows 7 或 XP 的用户最初有一些抵制。Windows 10 现在很容易成为主导操作系统，鉴于它对现代和经典应用一视同仁，公平地说，它在吸引每个人方面做得足够好。

Windows 11

所以，我们来到了今天，Windows 11 已经推出整整一年了。就像 Windows 10 一样，这是一次重大的免费更新，但这一次，系统要求被大幅提高，它带来的变化比大多数 Windows 10 功能更新通常带来的变化更多。这是一种全新的视觉体验，操作系统和内置应用都更新了新的外观和功能。如果你想了解新的重大变化，请查看我们对 Windows 11 的评论。

随着时间的推移，微软试图通过 Windows 10 将桌面和现代应用结合在一起，而 Windows 11 在这个方向上走得更远，将微软商店(以前的 Windows 商店)变成了每个应用都可以存在和被发现的地方。甚至一些安卓应用也是通过与亚马逊应用商店的合作来实现的。Windows 无疑已经走过了漫长的道路，虽然现在很难判断 Windows 11 的接受程度，但我们相信这对大多数用户来说都是一个受欢迎的更新。

当然，微软仍在不断改进。我们最近看到了 Windows 11 2022 更新版本 22H2 的发布，这是该操作系统的第一次重大更新。这带回了 Windows 10 的一些受欢迎的功能，但它也增加了许多新功能，并为视觉体验带来了更多的一致性。随着时间的推移，我们将看到更多的功能和更小的更新，因此 Windows 11 将不断发展，变得越来越好。
