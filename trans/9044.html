谷歌通过新的降价提醒让在 Chrome 上购物变得更加容易

原文：https://www.xda-developers.com/google-chrome-price-drop-prompt/#0001-01-01

谷歌通过新的降价提醒让在 Chrome 上购物变得更加容易

谷歌 Chrome 将向你发送降价提醒，提醒你关注的产品，只要在 Chrome 中保持产品页面作为标签打开。

现在是美国的假日季节，这意味着是每个人庆祝圣诞节购物的时候了。但是在网上购买各种各样的礼物可能会很麻烦。虽然你可以使用一些方便的 Chrome 扩展来找到最好的交易，但谷歌 Chrome 通过几个新功能让购买礼物变得比以往任何时候都更容易。其中最有用的一个会弹出提示，告诉你你看中的产品是否降价。

Android 版 Chrome 上的价格下降提醒

谷歌为谷歌 Chrome 浏览器引入了一些新功能，这些功能将增强用户在线购物的体验。如前所述，最有用的新功能可以帮助您跟踪某些产品的价格下降。如果你已经关注一款产品有一段时间了，但正在等待一个好的交易，Android 上的 Chrome 现在会在一个打开的标签中自动检查产品的更新价格，并显示在角落里。这样，你就不必反复打开产品页面来查看价格下降。

请确保在 Chrome 的标签页中打开产品页面，您将能够看到标签上方显示有当前价格的较小横幅。请注意，这项功能目前只在 Android 的 Chrome 上可用，所以如果你在 iOS 上，你将不得不等待更长的时间。目前，它也仅限于美国。

除了价格下降提醒，谷歌在最近的一篇博客文章中强调了其他一些功能，这些功能将进一步改善你的购物体验。其中包括:

  • 通过地址栏搜索图片:Chrome 上的地址栏集成了谷歌镜头，这意味着如果某样东西吸引了你的眼球，你想在网上购买它，你只需点击地址栏上的图片。这也将很快在桌面版 Chrome 上实现。
  • 新的你的购物车部分:谷歌在美国的 Windows 和 Mac 的 Chrome 上增加了一个新的你的购物车部分，它将在一个地方向你展示你在各个网站上添加到你购物车中的所有产品。因此，您不必访问每个单独的网站。
  • 保存的密码和地址: Chrome 能够保存你的地址和密码已经有一段时间了，你可以在购物时利用它们来更快地结账。

这些是谷歌试图改善你在 Chrome 上购物体验的一些方式。如果你还没有看到这些新功能，请确保将 Chrome 应用程序更新到最新版本。
