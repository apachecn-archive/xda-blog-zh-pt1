ElevenClock 使用 Windows 11 将时钟放在您的第二台显示器上

原文：https://www.xda-developers.com/elevenclock-utility-clock-back-windows-11/#0001-01-01

当 Windows 10 在 2015 年推出时，你无法在第二台显示器的任务栏中放置时钟。几年后，当这成为现实时，用户们欢欣鼓舞。这个简单的功能太有意义了，以至于从那里返回似乎很疯狂。然后 Windows 11 出现并移除了额外显示器上的任务栏时钟，ElevenClock 旨在解决这一问题。

ElevenClock 是 GitHub 上的一个开源应用，它只做一件事，但做得很好。事实上，它比你想象的要好得多。首先，它会将时间和日期添加到除主显示器之外的所有显示器上，这是因为主显示器已经有时钟了。

最重要的是，它支持黑暗和光明的主题，它支持所有的显示比例(100%，125%，150%等。).这意味着，在大多数情况下，11 点钟的时钟看起来和感觉上都很自然。当你点击时钟时，它甚至会打开你的日历和通知。

这一切都是天衣无缝的。它将在你启动电脑时启动，因此你不必每次打开电脑时都启动应用程序。它还会自动更新，这是你不用担心的另一件事。

如上所述，这一切都很简单。你所要做的就是从 GitHub 下载应用程序，安装并运行它。一旦你打开应用程序，它会将时钟放在你的任务栏中，你不必再考虑它。

据推测，微软将在某个时候将该功能添加回 Windows 11。新操作系统似乎缺少了很多东西，而这显然会在 Windows 11 的整个生命周期中被添加回来。幸运的是，你不必等待。
