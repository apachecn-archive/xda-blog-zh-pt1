谷歌的 Pixel Pass 似乎是它对苹果 One 的回应

原文：https://www.xda-developers.com/google-pixel-pass-leaked/#0001-01-01

谷歌似乎正在全力以赴开发 Pixel 6 系列。我们已经知道了关于它的几乎所有事情，但是仍然有一些细节还没有公布。虽然谷歌将于 10 月 19 日在正式发布这些设备，但一个大规模的泄露证实了已经泄露的一切，同时给了我们更多的信息。显然，该公司正在开发一种 Pixel Pass，可以与购买新的 Pixel 智能手机捆绑在一起，它包括 YouTube Premium、Google One、Play Pass 和 Google Fi。

这一泄露来自 YouTube m .李国豪，来自 This is Tech Today 频道，他已经泄露了很多关于 Pixel 6 系列的信息。我们已经看到了一个动手，一些照片样本，甚至潜在定价，这一切都要感谢李开复。Lee 泄露的图片显示了 Pixel Pass 的购买流程。它还解释了它的组成，以及如何销售给用户。

Pixel Pass 看起来可能是苹果 Apple One 订阅的明确竞争对手。就上下文而言，Apple One 订阅是一种包含主要 Apple 服务的一体化订阅服务，尽管不一定要与设备一起购买。还有三层提供更多服务或更多 iCloud 存储，最高一层提供 2TB 的 iCloud 存储。

从我们可以看到，Pixel Pass 有三个主要组成部分，用户获得的是:一部 Pixel 智能手机，该公司的主要第一方服务，以及延长的保修期。它可以通过 Google Fi(新用户或现有用户)购买，也可以通过 Google Store 购买运营商解锁的设备，该设备可以与美国所有主要运营商合作。必须同时购买 Pixel 设备，并且还可以选择以旧手机折价。用户还需要申请融资。

Pixel Pass 似乎是谷歌对 Apple One 订阅的回应，也可能是销售 Pixel 设备的主要推动力的一部分，特别是在美国。Pixel Pass 似乎不是一种国际产品，尽管它可能会在该公司的发布会上向国际买家宣布。
