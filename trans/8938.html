Galaxy Z Fold 3 和 Flip 3 的稳定 One UI 4.0 更新似乎有问题

原文：https://www.xda-developers.com/galaxy-z-fold-3-flip-3-buggy-one-ui-4-update/#0001-01-01

在上月向 Galaxy S21 系列推出基于 Android 12 的稳定 One UI 4.0 更新后，三星本周早些时候开始在向 Galaxy Z Fold 3 和 Galaxy Z Flip 3 播种新的更新。韩国和塞尔维亚目前正在为这两款手机稳定推出一个 UI 4.0，未来几周将有更多地区跟进。虽然许多所有者都期待着尝试新软件，但看起来更新充满了相当多的错误和问题。

三星 Android 12 Tracker:这里是所有官方 One UI 4.0 稳定版和 beta 版下载安装

根据三星论坛上的多个报道(via SamMobile )，Galaxy Z Fold 3 和 Galaxy Z Flip 3 的用户在安装了最新的稳定 One UI 4.0 更新后，面临着各种问题。一些用户声称他们的手机在更新期间进入了恢复模式，因此对设备进行了软屏蔽。

与此同时，那些能够成功安装更新的人正面临着一些问题，如屏幕闪烁，性能不佳，黑暗模式无法正常工作，双 Messanger 功能无法与一些应用程序一起工作，以及损坏的相机应用程序。也有报道称，用户无法截图或访问网飞或 YouTube。

据报道，在用户投诉后，三星已经停止了 Galaxy Z Fold 3 和 Galaxy Z Flip 3 的更新推出。如果你拥有一部 Galaxy Z Fold 3/Flip 3，并且居住在韩国或塞尔维亚，我们建议你推迟安装新的 One UI 4.0 更新，等到三星发布一个修复了错误的新版本。
