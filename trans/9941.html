Firefox 98 推出新的 Android 和 iOS 定制功能

原文：https://www.xda-developers.com/firefox-98/#0001-01-01

Firefox 是少数几个保留了非 Chromium 渲染引擎引擎的 web 浏览器之一，这使得 Mozilla 能够尝试新的 web 功能和低水平的性能变化，而这对于基于 Chromium 的 web 浏览器来说是不切实际或难以维护的。Firefox 98 现在在桌面和移动平台上推出，这一次，Mozilla 实际上专注于设计更新，而不是引擎优化。

桌面平台上的 Firefox 多年来一直支持主页(新的标签页)上的自定义背景，有一个大规模的主题库可供选择，但现在壁纸也开始出现在 Android 和 iOS 应用程序中。Mozilla UX 设计师 Melissa Chang 创作了第一批壁纸，还有一些基于皮克斯新电影变红的图像，Mozilla 在过去几个月里一直在进行其他品牌整合。

Firefox 96 桌面变更日志

  • Firefox 现在有了一个新的优化下载流程。现在文件会自动下载，而不是每次都提示。只需点击一下，就可以从下载面板打开它们。
  • 现在，您可以使用上下文菜单直接从下载面板和其他下载视图中删除下载的文件。webRequest 的使用会导致插件在 Firefox 启动时提前启动。这已更改为仅使用 webRequest 阻塞调用。非阻塞调用不再导致插件提前启动。

iOS 上的 Firefox 也可以将搜索栏移动到显示屏的顶部或底部，就像 Android 浏览器已经提供了一段时间一样。尽管在手机上把滚动条放在底部通常更好，因为你的手指不必在高屏幕上够得那么远，但有更多的选择总是好的。

桌面上的 Firefox 98 改变了处理下载的方式——不再默认显示弹出提示，文件现在会自动下载，就像 Chrome 和大多数其他网络浏览器的工作方式一样。也可以使用右键/上下文菜单从下载面板(以及显示下载内容的任何地方)删除下载的文件。

最后，Android 上的 Firefox Focus 浏览器将很快拥有仅 HTTPS 模式，该模式将阻止加载任何没有 HTTPS 支持的网站或资源。

你可以从 Mozilla 的网站下载桌面平台的 Firefox，安卓设备的 Play Store 也有。Mozilla 最近也在微软商店为 Windows 个人电脑提供了 Firefox。
