Windows 11 内置的视频编辑应用是一项订阅服务

原文：https://www.xda-developers.com/windows-11-built-in-video-editing-expensive/#0001-01-01

当微软昨天发布 Windows 11 内部预览版 22572 时，它向该操作系统添加了几个新的收件箱应用程序。其中一个叫做 Clipchamp，这是微软在 9 月份收购的一项服务。这是一项基于云的视频编辑服务，也是微软自 Windows Movie Maker 时代以来首次真正推动视频编辑器的发展。

但事实证明，并不便宜。如果你安装了 Windows 11 的最新开发频道版本，并决定使用 Clipchamp，你很快就会发现你的选择非常有限。在支付现金之前，你可以以 480 便士的价格输出视频，也就是所谓的标准清晰度。

之后，还有一个“创造者”计划，提供 720p(高清)输出。接下来是每月 19 美元和 39 美元的商务和商务高级计划，提供 1080 便士(FHD)出口，以及其他一些额外津贴。

值得注意的是，这不仅是一次追加销售，甚至不是一次竞争性追加销售。苹果免费赠送 iMovie 已经有很长时间了，它允许你在 4K 输出你的电影，这是 Clipchamp 甚至没有的。iMovie 连广告都没有；这只是在苹果生态系统中的额外津贴。

iMovie 并不是唯一免费或便宜的选择。即使是 Adobe Premiere Pro，现有的最强大的视频编辑解决方案之一，每月也要 20.99 美元。然后就是 DaVinci Resolve 这样的 app，还有无数 iOS 和 Android 的移动解决方案，比如 LumaFusion。

浏览创作者的风景从来都不是微软的优势之一。在扼杀了 Windows Movie Maker 之后，它有一个用于 Windows 8 的 Metro 应用程序，该应用程序从未退出测试版，然后 Story Remix 随着秋季 Creators 更新来到了 Windows 10。但是，创作者的一系列更新似乎从来没有关注创作者真正想做什么。该公司专注于像 Paint 3D 和 Windows 混合现实这样的东西，而不是为苹果过去称为 iLife 的东西提供竞争解决方案，包括 iMovie 和 Garage Band。

我们已经联系了微软，想知道这些价格是否会持续下去。这些层级完全有可能已经存在了一段时间，并且 Redmond 公司计划在面向公众发布之前对其进行改造。当我们有更多的信息时，我们将更新这个。

更新:微软已经确认这些价格不会改变。
