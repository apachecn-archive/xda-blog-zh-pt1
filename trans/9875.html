微软商店在网上有了新的外观，还有一个搜索栏

原文：https://www.xda-developers.com/microsoft-store-apps-new-web-portal/#0001-01-01

微软正在让在微软商店的网页版上找到 Windows 应用变得更加容易。该公司已经悄悄地推出了一个重新设计的 Windows 应用门户网站，它包括了一些重大改进，可以更容易地找到你正在寻找的应用。这包括一个新的搜索栏和改进的导航。

对于新手来说，是的，现在在这个新店面的屏幕左侧有一个搜索栏。需要明确的是，你可以使用微软网站顶部的通用搜索图标来搜索应用程序，但这样一来，搜索栏更具情境性，并且只关注应用程序的。在搜索栏下方，还有各种应用程序类别，因此您可以探索不同的部分。可以说，它甚至比 Windows 11 的微软商店应用程序更有条理，所以如果这最终也成为应用程序上的体验，我们也不会感到惊讶。

占据中心位置的是应用程序的主页，它仍然服务于同样的目的。你会看到一张幻灯片，顶部是一些主要的应用程序，当你向下滚动时，其他流行的应用程序也会显示出来。不过，整体设计看起来确实比之前的版本更干净、更一致。

新设计也延伸到了个人应用页面。当你点击一个应用程序的名称或图片时，你会看到新的页面，它现在看起来更干净、更有条理。在顶部，您会看到应用程序图标、名称和发布者，屏幕截图就在下方。在截图旁边，你还可以看到该应用程序上次更新的时间以及它的语言版本。值得注意的是，之前的版本根本没有提到应用程序上次更新的时间。

这种设计中不再有选项卡式导航。该应用的描述在截图的下方(并且不再显示两次)，然后评级和评论在它的下方。相关 app 可以在底部找到。有一点似乎从 app 页面上消失了，那就是系统需求。

最后一个重大变化是，微软商店的门户网站现在可以安装商店应用程序，这是微软去年 6 月与 Windows 11 一起宣布的一项功能。以前，如果你点击按钮下载应用程序，你会被带到 Windows 上的 Microsoft Store 应用程序，再次查看完整的应用程序列表，然后下载它。在这个新的网络版本中，当你点击“获取商店应用程序”按钮时，它只会打开一个小的弹出窗口来确认安装。

这种新体验仍处于预览阶段，因此如果你试图从微软网站导航到 Windows 应用程序，旧版本仍是默认版本。你可以点击这里查看新的微软商店门户网站。
