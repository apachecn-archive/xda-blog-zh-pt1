微软发布 Windows 11 build 22000.434——以下是新内容

原文：https://www.xda-developers.com/microsoft-releases-windows-11-build-22000-434/#0001-01-01

今天是这个月的第二个星期二，你知道这意味着什么。今天是补丁星期二，是每个受支持的微软产品获得更新的日子。在 Windows 11 的情况下，只有一个版本可以考虑，版本 21H2。

与大多数月份不同，但可能并不令人惊讶的是，这个月没有预览更新。这几乎肯定是因为球队放假回家了。星期二的补丁更新有一个相对较短的变更日志，简单地捆绑了预览版更新中的所有内容。你可能认为这个月会更长一点，因为没有预览版更新，但事实并非如此。

如果你在 Windows 11 版本 21H2 上，你将获得 KB5009566 ，使内部版本号达到 22000.434。这里可以手动下载，以下是亮点:

以下是修复的完整列表:

  • 解决了影响日语输入法编辑器(IME)的已知问题。当您使用日文 IME 输入文本时，在使用多字节字符集(MBCS) 的应用程序中，文本可能会出现乱序或者文本光标可能会意外移动。此问题影响 Microsoft 日语 IME 和第三方日语 ime。

与所有星期二补丁更新一样，这是强制性的。这意味着您可以通过在 Windows Update 中检查更新或手动安装来获得它，但如果您不这样做，它仍会在某个时候自动安装。

在此之后，本月晚些时候应该会有另一个预览更新。这些通常被称为“C”和“D”周更新，字母对应于该月的第三和第四周。与周二补丁更新不同，这些是可选的。它们出现在 Windows Update 中，但除非您明确要求，否则不会安装。
