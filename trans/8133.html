以下是如何禁用火狐地址栏中的广告

原文：https://www.xda-developers.com/disable-ads-firefox-address-bar/#0001-01-01

火狐是最好的网络浏览器之一。许多用户更喜欢它，而不是谷歌 Chrome 和 Edge，因为它使用自己的引擎，并强调用户隐私。Mozilla 也一直在为 Android 上的 Firefox 添加新功能，包括最近宣布的全系统密码管理器，这也使它成为智能手机用户的一个好选择。

然而，如果你在桌面上使用 Firefox，你会在地址栏中看到一些建议，有些人可能最终会把它们视为“广告”这是因为在这个列表的底部有一个标签为 Firefox 建议的部分。在这一部分，你会发现火狐对网站访问的建议。Mozilla 说说 Firefox 建议可以成为“更好网络的可靠指南”并提供相关建议。如果你不喜欢，你可以关掉它。这就是我们制作这份指南的原因。

如何关闭 Firefox 建议

正如你在上面的图片中看到的，浏览器在建议的底部显示了某些网站的链接。虽然这看起来并不碍事，因为它只占据了屏幕的一小部分，但对一些用户来说可能会很烦人。幸运的是，该功能在默认情况下是禁用的，安装 Firefox 92 更新后，Firefox 会询问您是否希望允许建议。

如果你错误地启用了这个功能，浏览器会给你一个选项来禁用 Firefox 建议并删除广告。只需在浏览器的设置页面点击几下，你就可以开始了。以下是如何禁用火狐地址栏中的火狐建议:

  1. 点击浏览器右上角的菜单图标，然后前往设置。或者，你可以点击菜单栏中的 Firefox ，然后选择首选项。
  2. 现在，选择隐私&安全选项。
  3. 在下面的窗口中，向下滚动并在地址栏- Firefox 建议下查找。
  4. 关闭来自网络的建议和来自赞助商的 建议的开关。你可能还想取消选中搜索引擎和快捷方式的复选框。
  5. 重启浏览器，享受你的无广告地址栏！

虽然这些类型的“广告”可能会打扰并影响用户体验，但 Mozilla 表示，他们已经添加了这一功能，以资助 Firefox 的开发和优化。如果您发现上下文建议很有帮助，或者不介意在地址栏底部看到建议的网站，您可以保留该功能。
