这是新 WhatsApp for Windows 的预览版[更新:现已推出]

原文：https://www.xda-developers.com/whatsapp-windows-11-leaked-videos/#0001-01-01

【更新 16/11/2021 @ 4:58AM】全新的 Windows 版 WhatsApp 测试版现已可供下载。我们在文章底部添加了该应用的链接。

一个新的 WhatsApp Windows 应用程序的图像和视频已经在网上浮出水面，为 Windows 11 和 10 用户展示了一个全新的 UI 和一些新功能。这些信息来自于Aggiornamenti Lumia，它经常在官方发布之前透露许多 Windows 应用和功能。

任何现在在个人电脑上使用 WhatsApp 的人都知道，这种体验并不理想。虽然你可以从微软商店下载一款应用，但它主要基于 WhatsApp 的网页版。它使用电子技术将网络技术引入更传统的应用程序格式，但这导致了不太出色的性能。这个 WhatsApp 的新版本似乎基于 UWP，使用 XAML UI 语言来适应原生 Windows 应用程序。它的启动速度也非常快，尽管我们看到的演示只在聊天列表中显示了一个聊天，所以这可能会改变。

新的 WhatsApp 应用程序将适用于 Windows 11 和 10，从用户界面来看，它似乎是为以前的 Windows 版本设计的。像搜索框和聊天气泡这样的一些元素仍然使用尖角，这是 Windows 11 大部分已经远离的。然而，一些 Windows 11 元素已经在这里了，比如新的云母材料，允许桌面背景在应用程序内部微微发光。随着时间的推移，或者在正式发布之前，可能会有更多适应 Windows 11 的调整。

除了消息传递，新应用似乎还支持语音和视频通话，这是基于 Aggiornamenti Lumia 分享的图片。此外，还有一个绘图功能，允许的 Windows Ink 粉丝绘制一张图片，并在 WhatsApp 内部分享。因为 UWP 应用程序总是可以在后台同步，所以这款应用程序的一个新功能是即使在应用程序不运行时也可以接收通知。
