采访:POCO Global 权衡为廉价设备解除 MIUI 屏蔽

原文：https://www.xda-developers.com/poco-global-interview-debloat-miui/#0001-01-01

在今年的 MWC 上，POCO 发布了 T2 POCO X4 Pro 和 M4 POCO Pro T3。对于€299 英镑的起价来说， POCO X4 Pro 是一笔令人难以置信的交易。它采用 6.67 英寸 FHD+ AMOLED 显示屏，峰值亮度为 1200 尼特，高通支持 5G 的骁龙 695 SoC，108MP f/1.9 主摄像头，以及高达 8GB 的 RAM 和 256GB 的存储空间。不过，在活动中，我们也有机会与 POCO Global 的总经理 Kevin Qiu 和 POCO Global 的产品营销负责人兼全球发言人 Angus Kai Ho Ng 坐在一起，讨论 POCO、MIUI 的未来以及接下来的展望。

在整个采访过程中，POCO 重申了其对发烧友社区的承诺，承诺仍然允许其设备组合上的引导加载程序解锁。我们还被告知，虽然该公司目前的重点是中端智能手机，但不排除未来推出折叠手机或超旗舰手机的可能性。虽然不是一个结论性的答案，但它确实让我们对主流可折叠手机的未来充满了兴奋。

当被问及像谷歌或三星这样的公司一样提供长期支持时，我们也被告知，该公司正在评估提供长期支持，尽管承认这是小米 MIUI 开发团队的要求，因为如果小米不提供扩展支持，它就不能提供支持。Angus 告诉我们，POCO 目前正在评估为一些现有用户提供更长时间的安全更新，如果这种情况发生，很可能会发生在 X 和 F 系列手机上。

当然，与小米的联系并不新鲜，因为该公司所有的智能手机都从小米的产品组合中更名而来。即使在最近发布的小米红米 Note 11 系列的情况下，这两款设备都已经在国际上以不同的名称发布。

  • 代号“peux”:
    • 红米 Note 11 Pro Plus 5G【印度】
    • POCO X4 Pro 5G[印度]
  • 代号“veux”:
    • 红米 Note 11 Pro 5G[全球]
    • POCO X4 Pro 5G[全球]
    • 红米 Note 11E Pro[中国]

然而，该公司与我们分享的一个有趣的花絮是，它正在考虑提供一个修改或去屏蔽的 MIUI，重点是提高预算设备的性能。

作为背景，我提到一些用户更喜欢智能手机上更干净或更简单的系统，而不是 MIUI。Kevin 提到“一些客户购买低端产品可能是为了某项基本功能”。安格斯随后插话说:“你所说的‘更清洁的系统或更简单的系统’，实际上是我们为低端设备和入门级产品考虑的一种方式。这就是他们如何在加工方面减轻一些重量。“Angus 还证实，POCO 正在评估 POCO 专属功能的纳入，并提到 POCO Launcher 最初是为了“尝试一些新的东西”而推出的，自 2020 年的大 2.0 更新以来，该团队一直在努力改进它。

目前还不清楚“去屏蔽”的 MIUI 会是什么样子，不过我们可以做出一些有根据的猜测。POCO 可能会取消一些预算设备上预装的膨胀软件，甚至可能会删除一些计算成本更高的软件功能。也不能保证真的进行下去，因为该公司只是在“考虑”这种可能性。它完全有可能以当前的格式保留 MIUI，或者也有可能这是小米正在为其廉价设备考虑的事情，POCO 只会为小米更名的设备继承这一点。

尽管如此，POCO 的未来似乎是光明的。我们将密切关注该公司的未来发展，因为 Kevin 也告诉我们，2022 年将是 POCO 迄今为止最重要的一年。
