Reddit 揭示了其俱乐部会所克隆的新功能，Reddit Talk

原文：https://www.xda-developers.com/reddit-talk-new-features/#0001-01-01

社交音频应用 Clubhouse 在 2020 年底和 2021 年初声名鹊起，不久之后，几乎所有其他消息和社交媒体服务都实现了自己版本的 Clubhouse 音频室。Reddit 去年发布了对这个想法的看法，名为“Reddit Talk”，现在更多的改进正在推出。

Reddit 在一篇博客文章 ( via The Verge )中强调了一些改进，称“今天我们为 Reddit Talk 推出了许多新功能，包括:一个录音功能，这样社区成员可以在 Reddit 谈话发生后听到它们，为网络用户带来 Reddit Talk 体验，在谈话过程中与文本和表情符号进行互动的能力，以及在 feed 顶部显示实时 Reddit 谈话何时发生的 live bar 实验。”

就像 Clubhouse、Twitter Spaces 和这个想法的大多数其他实现一样，可用性始于移动设备，并慢慢扩展到桌面。现在，桌面版 Reddit 上的人们可以在对话中听和说，剩下的就是创建新对话的能力，这显然是“在本月晚些时候到来”Reddit 也从其他俱乐部会所的复制品中获得灵感，特别是 Twitter，在屏幕顶部有一个新栏专门用于谈话。

音频聊天室是否仍将是一个常用功能，或者它们是否会像最初的俱乐部会所应用程序一样慢慢淡出人们的视线，还有待观察。 Instagram 、 Discord 、Twitter、、【脸书】等平台都不同程度的普及了公共音频室。

Reddit 也正在修改其平台上的广告，以在可能的股票上市前提高利润。该公司去年聘请了第一位首席财务官(CFO)，德鲁·沃勒罗(Drew Vollero)，他曾在 2017 年帮助 Snap(Snapchat 的所有者)上市。
