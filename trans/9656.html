web 上的 OneNote 正在改进墨迹和新功能

原文：https://www.xda-developers.com/onenote-web-teams-improved-inking-new-features/#0001-01-01

微软宣布了一些改进和功能，将在网络和团队中推广到 OneNote。其中一些功能已经向用户推出，而其他功能才刚刚开始推出。

从 OneNote 在 web 上已经可用的功能开始，有一个改进的复制和粘贴体验，允许您在 OneNote 页面上粘贴丰富的内容。如果需要，您也可以粘贴不带格式的文本。另一项改进是能够放大和缩小 OneNote 页面，以便您可以更近距离地查看事物或查看大图。已经实现的最后一项改进是能够调整页面上嵌入的 web 内容的大小，比如 YouTube 视频。

至于刚刚开始在网络上推广到 OneNote 的功能，有一个新的模式切换器，因此您可以在编辑模式和查看模式之间进行选择。事实上，这也适用于网络上的其他 Office 应用程序，这是为了防止你在滚动文档或页面时发生意外变化，如果你使用触摸屏，这可能特别有用。

就这一点而言，还有一种改进的手写体验，因此当您在 OneNote web 中使用活动手写笔(如 Surface Pen)时，您将自动开始绘图。这是您在 OneNote 桌面版上预期的行为，因此在 web 上也能看到这种行为也很好。如果您使用手指，您可以更轻松地滚动页面或放大和缩小，而不会意外墨迹。如果你想用手指画画，你也可以启用这个选项。

现在推出的几个新功能是将图片设置为页面背景，以及记住您在网上访问 OneNote 时使用的最后一页。与这些功能中的大多数一样，这是您在桌面版 OneNote 中所期望的，但这使网页版达到了同等水平。

最后，在不久的将来，您将在团队内部获得更好的文件共享体验。当您攻击 OneNote 笔记本的某页邮件时，如果其他人无权访问该文件，团队会通知您。此外，您还可以为链接设置新的权限。例如，您可以将 OneNote 笔记本设置为任何人都可以使用，这样您就不必考虑向每个用户授予权限。收到 OneNote 笔记本的用户也可以选择在团队中、在 web 上打开 OneNote，或者在他们的 PC 上打开 OneNote 桌面应用程序。

这些功能中的大部分仍在推出，所以你可能还看不到它们。您可以访问 web 上的OneNote来查看它们是否已经可供您使用。
