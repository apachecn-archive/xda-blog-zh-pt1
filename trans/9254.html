Linux Mint 20.3 现在可以使用更新的系统主题和新的应用程序

原文：https://www.xda-developers.com/linux-mint-20-3/#0001-01-01

Linux Mint 是最受欢迎的桌面 Linux 发行版之一，它有一个 Ubuntu 基础(给它足够的软件选择)和几个桌面环境选项。Linux Mint 20.3 已经在 beta 测试中使用了一段时间，但最终版本已经发布，今天可以下载。

Linux Mint 20.3 ( via omg！ubuntu！ )基于 Ubuntu 20.04.5 LTS(就像它之前的 Mint 20.2 一样)，采用相同的 Linux v5.4 内核版本，因此它将在 2025 年 4 月之前接收关键软件更新。你可以在 Cinnamon 、 MATE 或 XFCE 桌面环境中进行选择，它们都有类似 Windows 的界面和基于 GTK 的应用程序。

Mint 20.3 中所有版本的主要改进是新的“Mint-Y”主题，它具有更亮的灰色阴影，更大的标题栏/窗口控件，以及圆角窗口。窗口按钮以前肯定非常小，所以这是一个受欢迎的变化，即使它可能会与标准的 Gnome Shell 桌面环境相比较。如果你不喜欢新的外观，旧的主题仍然可以通过安装“mint-themes-legacy”包来访问。一些应用程序现在也默认使用深色主题，比如赛璐珞和图像浏览器。

Cinnamon desktop 在这个版本中有一些有用的调整，例如日历小程序中的多帐户/源支持，运行对话框的新样式，以及对 Nemo 文件管理器的改进。还有一个新的图书馆应用程序(也称为“Thingy”)，可以快速访问最近保存的阅读进度，有点像电子阅读器应用程序。

Linux Mint 还没有发布关于新更新的完整博文，主下载页面仍然指向 20.2 版本。然而，随着 omg！ubuntu！指出，该项目的许多镜像下载服务器都有适用于 20.3 的 ISO 镜像。其中两个链接如下。
