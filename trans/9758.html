Windows 11 Pro 将需要一个 Microsoft 帐户来设置

原文：https://www.xda-developers.com/windows-11-requiire-microsoft-account/#0001-01-01

根据最新开发频道 buid 的变更日志，在未来的更新中，WIndows 11 Pro 将在安装期间开始要求一个微软帐户。从 Windows 10 开始，家庭版就需要一个微软账户，但专业版用户仍然可以在不使用微软账户的情况下设置个人账户。

正如微软在 Windows 11 build 22557 的变更日志中指出的，Windows 11 Pro 用户现在将被要求在首次设置他们的 PC 时输入一个微软帐户。当然，这只适用于为个人用户设置笔记本电脑的情况，因此企业设备仍然可以使用组织帐户进行设置。博文中写道:

与 Windows 11 家庭版类似，Windows 11 专业版现在仅在初始设备设置期间需要互联网连接(OOBE)。如果您选择将设备设置为个人使用，设置时还需要 MSA。您可以预期在随后的 WIP 航班中需要 Microsoft Account。

看起来微软并不打算取消这一改变，这对一些用户来说可能是个问题。不过，和以前一样，你可以在设置应用中从电脑中删除你的微软账户，所以这应该不是什么大问题。

如果你想知道如果没有互联网连接会发生什么，那么，这也是必须的。第一次安装将需要互联网连接来链接您的 Microsoft 帐户。通常有一些漏洞可以用来在没有微软账户的情况下设置 Windows 11，但这在最新版本中也是不可能的(正如 Windows Central 所指出的)，所以看起来用户将被迫使用微软账户。

这一变化应该不会很快影响到普通大众。这一要求刚刚在开发频道为 Windows 内部人员推出，微软向其他用户推出它还需要一段时间。它应该是 Windows 11 下一个主要功能更新的一部分，我们预计在今年下半年进行，尽管这尚未得到证实。

用户不太可能对 Windows 11 的微软账户要求感到满意。许多人已经认为使用本地账户的能力是 Windows 11 Pro 相对于家庭版的最大优势之一，所以这肯定会引起一些反对。即使在像 Android 这样的移动平台上，使用谷歌账户并不是安装手机的严格要求，所以可以理解一些用户对这一要求感到不满。
