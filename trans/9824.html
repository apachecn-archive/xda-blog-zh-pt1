微软在二月概述微软 365 的新特性

原文：https://www.xda-developers.com/microsoft-365-february-2022/#0001-01-01

随着又一个月的临近，微软在 2 月份分享了对微软 365 套件所做的所有更改和改进的概述。这个月的变化列表相对较小，但这里仍然有一些受欢迎的消息。

本月一些最大的新闻与微软的云个人电脑服务 Windows 365 有关。微软正在让不需要 Azure 订阅就能供应云电脑成为可能。相反，管理人员可以在配置过程中使用 Azure Active Directory Join。此外，云电脑的首次运行体验现在可以本地化为 38 种不同的语言，这应该会使这项服务对世界各地的企业更加有用。这两项功能都在 2 月份进入了预览版，并且很快就会正式发布。

团队在 2 月份收到了一个相对较小的更新，即在会议期间隐藏或固定自己的视频的能力。这可能很有用，具体取决于您是希望在通话过程中能够看到自己，还是希望有更多的空间来看到其他参与者。

Viva Learning 是微软团队中的一个学习中心，是 Viva“员工体验平台”的一部分，它也正在与学习管理系统(LMS)集成，包括 SAP SuccessFactors、Saba Cloud 等。这样，员工可以更容易地看到他们在不久的将来需要完成的学习任务，并开始执行。

Yammer 有几个新功能，首先是一个“建议社区”面板，可以更容易地找到新的社区加入。另一个消息是，Yammer 社区现在可以设置为在有新活动时自动更新，因此如果人们仍在参与，管理员不必手动保持社区运行。最后，欧盟的 Yammer 用户将很快能够加入托管在美国的 Yammer 外部网络，同时仍然保持政策合规性。

就功能而言，2 月份对微软 365 来说是一个相对缓慢的月份，但其中一些改进肯定是受欢迎的。安装 Windows 365 的额外灵活性应该对世界各地的企业特别有用。
