如何在 Apple Mac 上编辑或合并 PDF 文件

在 macOS 上签署、注释或合并 PDF 文件不需要额外的软件。以下是如何在 Mac 上编辑或合并 PDF 文件。

苹果在 2021 年的释放 Mac 活动中发布了升级版 MacBook Pro 机型。如果你有一台 MacBook Pro，考虑用外壳保护它，以确保它尽可能长时间使用。毕竟，macOS 是一个伟大的操作系统，如果你妥善保管，你的电脑可能会在未来几年保持流畅。macOS 在生产力部门也很出色，尤其是处理文档的时候。然而，处理 PDF 文件不像 Pages 文档那样简单。以下是如何在 Mac 上编辑或合并 PDF 文件的方法。

如何在 Mac 上编辑或合并 PDF 文件

macOS 让用户可以选择做一些 PDF 编辑，而不需要下载额外的第三方软件。虽然对用户可以编辑的内容有一些限制，但内置编辑器仍然提供了不错的功能。无论您计划如何处理 PDF 文件，都有一些常见的第一步:

  • 从你的坞站打开 Finder 应用。
  • 导航到您想要编辑或合并的 PDF 文件。
  • 双击要处理的 PDF 文件，在预览应用中打开。
  • 点击标记图标显示其工具栏。这是搜索栏左侧的第一个图标。

如何在 Mac 上编辑 PDF 文件中的文本

  • 点击标记工具栏上的第三个图标。
  • 像通常选择文本一样，单击、按住并在您想要密文的文本上移动光标。
  • 一旦您退出 PDF 文档，更改将被保存，文本将保持编辑状态。

如何在 Mac 上绘制 PDF 文件

  • 点击标记工具栏上的第四个或第五个图标，分别绘制草图。
  • 单击、按住并在 PDF 周围移动光标。
  • 草图允许你画，然后有选择地把你的涂鸦变成一个精确的几何形状，比如一个圆。
  • 画画可以让你随意涂鸦。

如何在 Mac 上的 PDF 文件中插入形状和箭头

  • 点击标记工具栏上的第六个图标，这是一个下拉菜单。
  • 选择要插入的形状或箭头。
  • 拖动它，调整它的大小。
  • 您甚至可以插入一个放大圆圈来突出 PDF 的某一部分。

如何在 Mac 上将文本添加到 PDF 文件

填写 PDF 表单时，添加文本会很方便。以下是如何做到这一点:

  • 点击标记工具栏上的第七个图标。
  • 在出现的文本栏中键入您想要添加的文本。
  • 将它拖到您想要放置它的任何位置。
  • 您甚至可以使用标记工具栏最末端的按钮来更改字体大小、类型和颜色。

如何在 Mac 上签署 PDF 文件

macOS 提供了三种签署 PDF 的不同方式。为此:

  • 点击标记工具栏上的第八个图标。
  • 选择:
    • 触控板如果你想在 Mac 的触控板上追踪你的签名。

    • 摄像头如果你想用 Mac 的摄像头扫描白纸上的签名。

    • iPhone 如果你想用你 iPhone 的触摸屏来签名。

  • 点击完成。
  • 在签名下拉菜单中点击它。
  • 将它拖到各自的签名位置。

如何在 Mac 上给 PDF 文件添加便笺条

  • 点击标记工具栏上的第九个图标。
  • 会出现一个浮动的便利贴。键入您想要添加的内容。
  • 单击任意位置将其最小化。
  • 将最小化的便笺拖到您认为合适的位置。

如何在 Mac 上重新排列 PDF 文件的页面

  • 在边栏中，将您想要重新定位的页面缩略图拖到实际边栏中的所需位置。

如何在 Mac 上合并两个 PDF 文件

  • 打开您想要合并成一个文件的两个 PDF 文件。
  • 从一个 PDF 的边栏中选择所有或单个页面缩略图。
  • 将它们放在第二个 PDF 文件的边栏中。
  • 如有必要，重新排列它们。
  • 瞧啊。您的两个 PDF 文件将被合并到您放入的文件中。如果您只想保留一个 PDF，那么您可以删除从中拖动的那个。

macOS 提供了大量的工具来编辑 PDF 文件，不需要任何额外的软件。虽然它不支持所有类型的编辑，但它足以满足用户在日常生活中遇到的大多数基本需求。
