微软 Edge 有新的膨胀软件，绝对没人需要

原文：https://www.xda-developers.com/microsoft-edge-buy-now-pay-later/#0001-01-01

微软去年发布了全新的 Edge 浏览器，由开源的 Chromium 引擎驱动，而不是最初版本(和 Internet Explorer)中的微软定制渲染引擎。该浏览器普遍受到好评，因为它在快速的 Chromium 引擎上添加了大量有用的功能，但并非微软的所有想法都是赢家。最新的例子是一项新功能，将网上购物的利息支付添加到浏览器本身。

微软在其 Microsoft Edge Insider 论坛上宣布了新功能，在网上商店结账时，该功能为您的支付方式列表添加了一个“立即购买，稍后支付”选项。该功能由第三方公司 Zip (之前的 Quadpay)提供支持，允许你在六周内将 35-1000 美元的购物付款分成四期。假设你按时支付每期贷款，就没有利息费用。

Zip 已经有了一个移动应用程序和一个用于购物的 Chrome 扩展，但很难看到微软 Edge 的新集成是一个臃肿的软件。当我试图买东西时，我不想看到来自我的浏览器的支付公司的广告，至少现在，微软没有提到任何关闭它的方法。除了功能膨胀之外，类似于信用卡，建立“现在购买，以后支付”的服务是为了鼓励人们购买更多的东西。

令人震惊的是，使用微软 Edge 的人的反应似乎并不积极。BioTurboNick 在社区帖子上写道，“请停止用这些收入攫取来膨胀浏览器。这就像你在复述 90/00 年代最糟糕的 IE 浏览器扩展，并默认安装它们。”另一个来自 Jozef Izso 的评论写道，“这是一个直接嵌入浏览器的可怕的膨胀软件。”

Microsoft Edge 的 Zip 支付集成已经在 Canary 和 Dev 频道中提供，并将在 Microsoft Edge 96 的稳定版本中向所有人推出。如果你不想要一个鼓励不必要购买的浏览器，我推荐火狐。
