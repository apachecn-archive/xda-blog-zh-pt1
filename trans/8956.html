Realme X7 Max 5G 和 Realme GT 用户现在可以试用基于 Android 12 的 Realme UI 3.0

原文：https://www.xda-developers.com/realme-x7-max-5g-realme-gt-android-12-realme-ui-3-0-early-access-open-beta/#0001-01-01

Realme 以生产一些真正有趣的设备而闻名，这些设备有着 T2 吸引人的设计和极具竞争力的价格。除了用其强大的硬件产品扰乱智能手机市场，OEM 还一直在忙着准备Realme UI 3.0——基于 Android 12 之上的最新版本的定制皮肤。该公司现在邀请印度的 Realme X7 Max 5G 用户通过“早期访问”计划试用 Realme UI 3.0 的预发布版本。

Realme X7 Max 5G XDA 论坛

在官方 Realme 社区论坛上的一篇帖子中，Realme 已经分享了 Realme X7 Max 5G 的 Realme UI 3.0 早期访问计划的细节。感兴趣的人可以通过注册该程序来获得该软件，这只不过是某种阿尔法测试。

为了申请早期访问版本，请确保您正在运行 RMX3031_11。A.21 稳定构建。接下来，前往设置 > 软件更新 >点击右上角的齿轮状设置图标>点击试用版 >填写您的详细信息，然后点击“立即申请”按钮。一旦您的应用获得批准，您将通过 OTA 在您的设备上收到 Realme UI 3.0 预发布版本。

截图鸣谢:Realme 社区成员rahulsid 8888T5】

由于这一举措，Realme X7 Max 5G 用户现在可以有机会在他们的设备上提前看到 Android 12。除了标准的 Android 12 改进，Realme UI 3.0 还带来了大量新功能。一些主要亮点包括一个改进的黑暗模式，一个新的基于壁纸的主题引擎，支持自定义面部表情符号，等等。查看我们对 Realme UI 3.0 的实践，了解更多关于新软件的信息。

除了 Realme X7 Max 5G，该公司还为 Realme GT 推出了 Realme UI 3.0 公开测试版。在此之前，在 10 月开始了类似的手机早期接入版本。

Realme GT XDA Forums

与每个预发布软件一样，Realme X7 Max 5G 的早期访问版本和 Realme GT 的开放测试版本都可能比日常驱动材料稍逊一筹。因此，我们建议您在安装更新之前备份您的数据。如果您遇到任何 Bug，您可以使用官方反馈表和 Bug 报告日志包直接向软件开发团队报告。
