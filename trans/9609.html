原文：https://www.xda-developers.com/google-chrome-new-icon/#0001-01-01

谷歌 Chrome 将很快推出一个新图标，并针对特定平台进行修改

谷歌开始测试谷歌浏览器的新图标，有一个不太引人注意的阴影和特定平台的设计调整。

谷歌 Chrome 是现代史上最重要的软件项目之一，它的图标在桌面和移动设备上都可以立即识别。自 2008 年第一次发布以来，由三个风车状切片包围的蓝色圆圈的核心概念一直保持不变，但它多年来一直在发展，现在一个更新的图标正在路上。

谷歌的交互设计师 Elvin Hu 在一系列推文中分享了新的图标设计。他说，“你们中的一些人可能已经注意到了 Chrome 今天的金丝雀更新中的一个新图标。是啊！我们 8 年来第一次刷新 Chrome 的品牌图标。新图标将很快出现在您的设备上。[...]我们通过去除阴影、调整比例和增加颜色来简化主品牌图标，以符合谷歌更现代的品牌表达。”

新图标去掉了当前图标设计留下的阴影，但仍然保持红色、黄色和绿色之间的轻微渐变。每个平台上的图标看起来也不完全一样。Windows 将获得更渐变的外观，以适应微软新的流畅用户界面，Chrome OS 将获得一个完全纯色的版本，以匹配其他系统图标，Mac 图标将在现有的白色圆形正方形内有一个阴影。

谷歌的设计团队也考虑过在每个风车状片周围用白色衬垫的图标，但这个想法被放弃了，因为它会缩小整个图标——这是小屏幕上的重要因素。在接下来的几个月里，更新后的图标将会在各地慢慢推出。

热门应用的新图标经常受到争议，例如 Slack 从 2019 年开始的新标志和应用图标，但这种变化足够微妙，大多数人可能不会注意到差异。
