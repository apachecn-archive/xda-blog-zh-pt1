这个新的渐进式网络应用程序可以将图像转换成 SVG 矢量图形

SVGcode 是一个新的渐进式 Web 应用程序，可以自动将光栅图像(如 JPEG 和 PNG)转换为 SVG 图形文件。

你通常在网上看到两种不同类别的图像。光栅图像(如 JPG、PNG、WebP 等)是一系列固定分辨率的像素，而矢量图像(如 SVG)使用各种线条和形状。矢量图形对于徽标和其他简单的图像来说很受欢迎，因为它们在高分辨率下(或放大时)看起来并不可怕，但从头开始创建可能需要大量劳动力。现在有一个方便的 web 应用程序可以从标准图像创建 SVG:SVG code。

SVGcode 是一个新的渐进式网络应用程序，由在谷歌工作的开发人员关系工程师 Thomas Steiner 创建(尽管 SVGcode 本身不是谷歌的产品)。一旦你在浏览器的 svgco.de 打开它，你就可以从你的电脑上打开一个图像，它会被自动“绘制”成一个 SVG。大多数图像需要一些调整来创建一个看起来不错的 SVG，但它是一个令人印象深刻的 web 应用程序，如果你想制作一个更干净的矢量版本的光栅徽标，它可以派上用场。

SVGcode 由 Chromium 和 Firefox 等 web 浏览器添加的一些最新 API 支持。核心绘图功能基于 Potrace 项目，该项目被转换为该 web 应用程序的 Web 汇编代码。SVGCode 可以将完成的文件复制到您的剪贴板(使用异步剪贴板 API )，当右键单击图像时显示为 PC 文件管理器中的一个选项(使用文件处理 API )，并从您的设备打开/保存文件(使用文件系统访问 API )。它甚至使用新的窗口控件覆盖 API 来隐藏安装后的系统标题栏，让它感觉更像一个原生应用程序。

你可以在 Google 的 web 开发博客上的一篇博客文章中了解更多关于 SVGcode 的开发过程。因为它是一个 PWA，你可以在任何有现代网络浏览器的设备上使用和安装 SVGcode。
