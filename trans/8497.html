微软通过 Edge 浏览器让引用变得更加容易

微软正在 Edge 浏览器中测试一项新的引用功能，以使学生更容易在论文中添加引用

微软希望通过 Edge 浏览器的一项新功能，让学生更容易在他们的研究论文中添加引文。给论文添加适当的引用可能是一场噩梦，尤其是在不同机构使用不同格式的情况下。Microsoft Edge 中引用的目标是通过允许用户轻松获取特定页面的引用来简化这一过程。

Edge 中的引用功能与集合相关联，它允许你保存页面组以供以后参考。这个想法是，学生将使用收藏来保存他们在工作中用于研究的一系列页面，然后他们可以很容易地从这些页面中获取引用。

你需要在 Edge 的收藏菜单中启用引用，一旦完成，你会在收藏的每一页旁边看到一个引号符号。这让你既可以选择全文引用也可以选择全文引用，你可以选择你所使用的来源类型，还可以改变你所引用的作品的作者。您也可以选择您正在处理的风格，如 APA 或 MLA。

该功能目前在 preview 中可用，你需要使用 Edge 的 Dev 或 Canary 版本才能看到它。你还需要运行 Edge 版本 95 或更高版本，但考虑到 Edge 95 已经在稳定的渠道中，这应该是给定的。微软表示，Edge 中的自动引用目前只在某些学术网站上可用。对于 Edge 无法自动抓取所有信息的网站，可以像作者一样手动添加信息，生成合适的引用。

为了解决用户对该功能性能影响的任何担忧，微软表示，引用功能所需的文件仅在首次使用时下载。如果你不需要引用，你就永远不会下载那些文件，浏览器也会像以前一样轻巧。如果你有兴趣尝试一下，你可以下载最新的 Edge Insider 版本或者等到稳定版本中有引用的时候。
