如何忘记苹果 Mac 电脑上的无线网络

您可能想要从电脑中删除 Wi-Fi 密码有几个原因。以下是如何忘记 Mac 上的 Wi-Fi 网络。

Mac 电脑是苹果公司的个人电脑系列——iMac、Mac Mini、MacBook Air、 MacBook Pro 等等。它们提供了强大的功能，在某些情况下，竞争对手很难赶上。用外壳保护你的 MacBook Pro 只会让它更耐用，所以如果你还没有买的话，一定要买一个。有时您更改了 Wi-Fi 路由器、密码或网络名称，保存的网络列表会不断增加。或者，您可能正在搬迁，不再需要连接到您的旧网络。不管你有什么理由，以下是如何忘记 Mac 上的 Wi-Fi 网络。

如何忘记运行 macOS Monterey 或更早版本的 Mac 上的 Wi-Fi 网络

  • 从启动器或聚光灯搜索中启动系统偏好应用。或者，你可以点击左上角的苹果 Logo ，之后点击系统偏好。
  • 点击左下角的网络。
  • 要进入该屏幕，您也可以点击菜单栏中的 Wi-Fi 图标，然后点击网络首选项。
  • 确保在边栏中选择了 Wi-Fi 。
  • 点击前进，朝向右下角。
  • 在顶栏中，确保你在第一部分，标题为 Wi-Fi 。
  • 您会找到一个保存的 Wi-Fi 网络列表。点击你想忘记的人。
  • 点击列表正下方的减号( - )按钮。
  • 确认您要移除网络。
  • 瞧啊。您已经设置好了，网络将从列表中删除。值得一提的是，如果你启用了钥匙串 iCloud sync，那么 Wi-Fi 网络将在你的所有 Apple 设备上被遗忘。

在 macOS Ventura 或更高版本上

  • 点击菜单栏中的 Wi-Fi 图标。
  • 滚动到底部，点击高级。
  • 单击您想要忘记的 Wi-Fi 网络旁边的 3 点菜单，然后选择从列表中移除。

下次您想要连接到被遗忘的网络时，您必须再次输入您的密码。连接后，Wi-Fi 网络将再次存储在您所有的 Apple 设备上，前提是您在 iCloud 中使用的是“钥匙串”。
