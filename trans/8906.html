Windows 11 将获得支持黑暗模式的新记事本

原文：https://www.xda-developers.com/windows-11-notepad-dark-mode/#0001-01-01

微软正在开发频道向试用 Windows 11 的 Windows 内部人员推出经典记事本的新版本。正如你所料，新的应用程序使记事本更加符合 Windows 11 的设计语言，包括云母透明效果和新的图标。或许最重要的是，新应用支持黑暗模式。

如果你是记事本的粉丝，这是相当重要的消息。虽然微软在 Windows 10 的整个生命周期中增加了一些功能，但记事本已经很久没有重大的重新设计了。现在，它会看起来完全不同。你会在应用程序的标题中看到云母透明，菜单选项变得更大了。右上角还有一个新的设置按钮。

当然，最大的新闻可能是记事本支持黑暗模式，这是近年来许多内置 Windows 应用程序从未得到的。油漆也应该在不久的将来得到它，但是 T2 还没有发生。这将使使用记事本对眼睛更容易，尤其是如果你在低光环境下工作。

微软还增加了新的查找和替换体验，用位于文档顶部中心的小工具栏取代了典型的弹出窗口。这应该感觉更无缝一点，在不妨碍我们视线的情况下找到结果。另一个新特性是支持多级撤销。目前，如果你在记事本中按下 Ctrl + Z ，你将撤销你的最后一次操作，再次按下这些键将重做那个操作，所以你只能真正撤销你的最后一次更改。这一改变意味着你可以一次撤销多个或更大的错误。

有些人可能会担心，通过修改记事本，微软可能会毁掉一个人们依赖了多年的应用程序。然而，该公司重申，性能和生产力是“至高无上”的记事本，它将继续在这些领域做得很好。在这个预览版中有一些已知的问题，但公司正在努力解决它们，并改善对非常大的文本文件的支持。
