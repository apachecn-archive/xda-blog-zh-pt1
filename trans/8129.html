微软今年将推出一个新的反馈门户

原文：https://www.xda-developers.com/microsoft-has-a-new-feedback-portal-coming-this-year/#0001-01-01

今年早些时候，微软关闭了它的用户之声论坛。这有点奇怪，因为它没有提供任何形式的警告。最终，该公司表示，它确实有一些新的机制即将推出，事实证明，它被称为反馈门户，并将在年底前进行预览。

这一消息要感谢推特上的卡尔·克内希特。它将开始作为微软 Edge 和微软 365 的反馈机制，明年将支持 Windows 等其他服务。它将在所有市场上发售，但至少在开始时只有英文版。

微软确实有其他方法来发布反馈。例如，在微软 Edge 浏览器中有一个发送反馈的选项。虽然这项服务可能更像是来自不同产品的反馈中心，但它必须被称为反馈门户，因为反馈中心已被占用。这是内置于 Windows 中的应用程序，你可以在这里抱怨事情，并为操作系统建议新功能。

使用新的反馈门户，你当然可以留下关于产品的反馈，也可以投票支持其他用户的反馈。是的，这听起来确实像微软的其他反馈机制的工作方式。也许有一天，雷德蒙公司会整合所有这些东西。

同时，你可以选择投诉微软产品的方式。反馈门户将于今年推出预览版，这大概意味着它将于明年全面上市。还是那句话，微软 365 产品和 Edge 独占，Windows 随后。
