亚马逊在 Windows 11 上的应用商店将只提供少量应用

原文：https://www.xda-developers.com/amazon-appstore-windows-11-small-selection-android-apps/#0001-01-01

当微软宣布 Windows 11 时，它承诺支持 Android 应用程序，但至今仍未兑现。它被推迟到发布日期之后，官方的说法是内部人员很快就可以测试该功能。现在，这家位于雷德蒙德的公司已经发布了用于 Android 的 Windows 子系统的文档。

该页面有广泛的信息，从 Windows 上的 Android 应用程序开发到亚马逊应用商店的信息。然而，在最后有一个说明，并非所有的亚马逊 Appstore 应用程序都可以在 Windows 11 上使用。

“为了在 Windows 11 设备上可用，Android 应用程序必须发布到亚马逊应用商店。目前，只有微软和亚马逊选择的一小部分应用程序可用。”

更新:Android 应用程序现已正式在测试版渠道向 Windows 内部人员开放。它可以与 50 个精选应用程序一起工作，微软表示，随着时间的推移，它将增加更多应用程序。

显然，声明并没有说这是一个永久的东西。新的支持页面正准备向 Windows 内部人员，而不是普通大众推出。该公司可能计划在对所有人开放时点亮整个商店，甚至在以后的某个日期。当然，我们正在向微软询问此事。

如果它最终只是一个精心策划的应用列表，那就有点奇怪了。如上所述，那里有很多关于开发将在 Windows 11 上运行的 Android 应用程序的信息，所以任何人都可以制作一个。这也可能与亚马逊与其开发者的协议有关。

不管是什么原因，也不管当这种应用普遍可用时会发生什么，很明显，亚马逊应用商店在面向内部人士推出时只会提供有限的应用选择。看起来也很快了，因为这一页已经发表了。

页面上还有一些其他信息，比如要求屏幕至少为 9 英寸，分辨率为 720p 或更高。换句话说，我们将期待看到人们破解他们的旧 Windows 手机来让它工作。此外，微软表示，如果你试图安装亚马逊应用商店或其他安卓应用，它只会将亚马逊应用商店下载到你的电脑上。它不会自动出现。

该页面没有说明你需要什么样的 Windows 11 版本才能获得 Android 的 Windows 子系统。显然，它将首先提供给 Dev 频道。
