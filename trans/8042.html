Windows 11 深度探索:查看新的画图应用程序

原文：https://www.xda-developers.com/windows-11-paint/#0001-01-01

Windows 11 已经发布了大约一年，但对我们许多人来说还是新鲜事物。并不是每个人都已经升级了——许多人即使想升级也无法升级——所以如果你刚刚开始使用 Windows 11，并且想了解新功能，我们可以随时提供帮助。有相当多的增加和变化，其中之一是在油漆应用程序中，这是多年来第一次得到了重大的翻新。在这篇文章中，我们将看看 Windows 11 中的新画图应用程序，这是我们几十年来熟悉和喜爱的应用程序的视觉更新。

新的 Paint 应用程序主要是一个视觉刷新，这意味着它将与以前的工作方式非常相似。然而，视觉上的差异并不小，某些东西的工作方式也不同。让我们仔细看看。

当你打开新的 Paint 应用程序时，你会感觉既熟悉又陌生。一切都是原来的样子，但一些小的调整和新图标让一切焕然一新。让我们从头开始:保存、撤销和重做按钮都已经从应用程序的标题栏中移走，现在与文件和视图菜单选项放在一起。视图选项不再是一个单独的选项卡，而是在一个小的下拉菜单中提供所有选项。因此，也没有主页标签-所有的工具总是可见的。

那个工具栏看起来也很不一样。每个选项都被赋予了一个新的图标，选项也以更平衡的方式分布。在图像类别中，翻转和旋转功能已经分离，因此现在两种设置都更容易发现。颜色选择器现在使用彩色圆圈而不是正方形来表示颜色，选择自定义颜色的选项也是如此。您可能还注意到缺少了一些东西——在 Paint 3D 中打开图像的按钮不见了。微软似乎希望 Paint 成为 Windows 11 中的标准图像编辑工具。

正如我们上面提到的,“视图”选项卡已经被一个下拉菜单所取代，它允许您切换网格线、标尺和状态栏。放大和缩小选项已从该菜单中移除，因为您可以使用应用程序底部的状态栏来更改缩放级别。但是，您可以使用此菜单重置回默认的 100%缩放级别。

paint 中的所有下拉菜单都已经更新，使用了更干净、更紧凑的设计，但是选项仍然是一样的。文件菜单变得更小了，下拉列表中的项目有额外的选项。笔刷菜单现在列出了每一个笔刷的名称，以及笔刷外观的清晰样本，所以它使用了列表格式而不是网格。笔刷大小列表也已经更新，可以立即显示每个大小的像素厚度。之前，您必须将鼠标悬停在选项上，等待工具提示出现。

你从 Paint 中了解到的所有基本特征都还在，只是现在有了一个视觉上更愉悦的设计。

画布上没有看到 Windows 11 的大量变化，但有一些值得注意的变化。首先，在 stand 视图中，画布边界已经从应用程序窗口的边界移开。这确实会导致可用空间减少，但这种方式可能有助于防止用户在绘图时意外触摸其他 UI 元素。画布和应用程序空白区域之间的对比现在也不那么明显了，至少在光照模式下是这样。

大多数工具的工作方式完全相同，但微软也对文本工具的工作方式进行了重大改进。文本工具现在有了自己的工具栏，你可以改变字体、字体大小和相对于文本框的对齐方式，而不是接管主工具栏。您还可以选择是否在文本框中添加背景填充。

在功能上，它和以前一样，但现在看起来更现代，圆角是 Windows 11 设计的核心。

在画布的底部，您仍然会看到状态栏，但是现在看起来焕然一新。与 UI 的其余部分非常相似，它看起来更干净，并与 Windows 11 UI 相匹配。

奇怪的是，虽然这个新应用程序首次亮相已经有一年了，但有些东西已经承诺了很长时间，却还没有出现。例如，新的油漆应该支持黑暗模式，但现在还不是这样。微软还表示，将更新用户界面，使画布对齐窗口的中心，而不是左上角，但这也没有发生。希望不会太久。

Windows 11 中新的 Paint 应用程序也就这些了。所有相同的工具仍然在这里，毫无疑问，您对它们很熟悉，但是事情看起来确实有点不同。这是一个期待已久的变化，它肯定有助于 Windows 11 感觉更连贯，所以我们欢迎这次更新。
