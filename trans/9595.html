Windows 终端 1.13 预览版重新设计了设置，添加了自动提升功能

原文：https://www.xda-developers.com/windows-terminal-1-13-preview-settings-admin/#0001-01-01

微软今天发布了 Windows 终端 1.13 预览版，和往常一样，最新版本打包了一些新的变化。同时，Windows Terminal 1.12 现在是一个稳定的版本，包含了预览版中的所有更改。这包括从以前的会话中恢复窗口，以及应用程序的新透明模式。

不过，关注今天的新内容，Windows 终端 1.13 带来了一些显著的改进。首先，设置页面有一个全新的外观，这意味着要与 Windows 11 的整体外观保持一致。如果你使用过 Windows 11 设置应用程序，你会觉得新的外观很熟悉，这是一个受欢迎的升级，让应用程序在 Windows 11 上看起来更好一些。

更重要的消息是，微软增加了一个选项，当你启动终端配置文件时，自动授予管理员权限(提升)。这意味着，每当您想做一些需要管理员权限的事情时，您不必再专门寻找“以管理员身份运行”选项。如果您从非提升的终端窗口启动该配置文件，新的配置文件将在新窗口中启动。如果没有，它将在同一个窗口中打开。你也可以添加一个新的 elevate 参数到 split-pane 和 new tab 命令中，这样它们可以自动启动提升的概要文件。

微软还增加了一个新的实验性文本渲染引擎，你可以在终端的设置中启用它。虽然这对大多数用户来说不会有很大的不同，但微软表示，当处理大量不同颜色的文本时，你可能会注意到性能的提高。该团队为使用性能较低的 DirectWrite 渲染器道歉，因为它收到了用户的一些严厉批评，称渲染管道具有“糟糕的性能”。

最后，现在可以为特定的配置文件定制铃声。在较小的变化中，Windows 终端现在也支持 Windows 11 中的 Snap 布局，使窗口管理更容易。

如果你想下载 Windows 终端 1.13 预览版，可以从 GitHub 或者微软商店下载。如果你想要稳定版，它会预装在 Windows 11 上，你也可以通过微软商店进行更新。你也可以将设为你默认的主机。否则可以在这里下载。
