Android Studio 2021“大黄蜂”现已可供下载

原文：https://www.xda-developers.com/android-studio-bumblebee/#0001-01-01

Android Studio 是用于创建 Android 应用程序的主要集成开发环境(IDE ),具有功能丰富的代码编辑器、调试工具、仿真器等等。上一次重大发布是在去年 3 月，昵称为 Android Studio 北极狐。经过几个月的测试，现在又有了一个升级版本:Android Studio Bumblebee。谷歌显然以小型飞虫命名，但我更喜欢以变形金刚命名的想法。

安卓工作室大黄蜂，又称版本 2021.1.1，与安卓 Gradle 插件(AGP)的版本 7.1.0 重合。Android Studio 中有一个全新的设备管理器工具窗口，取代了旧的 AVD 管理器，这使得管理物理和仿真设备进行测试变得很容易。Android Studio 现在也有一个通过 Wi-Fi 连接 ADB 设备的简化流程，它使用二维码进行快速配对，Android Studio 4.1 中引入的仿真器工具窗口现在默认启用。

谷歌也花了一些时间来改进 Android Studio 的测试。更新后的 Gradle 插件有一个新的统一测试器，取代了现有的两个测试环境(一个由 Android Studio 提供，一个在 Gradle 中),应该会减少开发人员的头痛。Android Studio 的分析器也得到了改进，在 Android 11+设备上增加了新的 CPU 分析功能和一些其他有用的功能。

苹果硅处理器的原生支持也有了更多进展。谷歌最终更新了 Android SDK 平台工具包，其中包括像 ADB 和 Fastboot 这样的常用工具，以便在 macOS 上作为通用二进制文件(Silicon+x86_64)工作。这将加快苹果芯片 Mac 电脑上的应用安装、调试和许多其他工作流程——如 MacBook Pro 14 和 MacBook Pro 16 (2021) 和 iMac 24 英寸。

你可以在完整博文和发布说明中查看所有改进。安卓工作室可以从安卓开发者网站下载。现在对 Android 工作室擎天柱的等待开始了。
