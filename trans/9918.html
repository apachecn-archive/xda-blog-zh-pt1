Microsoft Edge 99 为自动填充和更多功能带来了自定义密码

原文：https://www.xda-developers.com/microsoft-edge-99/#0001-01-01

微软发布了另一个新版本的 Edge 浏览器，这次是 99 版。由于这是版本 100 之前的最后一个主要版本，微软再次警告 web 开发者，用户代理字符串将发生变化，可能会破坏一些网站。这也是 Chrome 和其他基于 Chrome 的浏览器正在发生的事情。

至于此次更新的新内容，微软增加了为 web 表单中的自动填充设置自定义主密码的功能。当你尝试使用自动填充功能在网站上输入用户名和密码时，Edge 已经要求用你的 Windows 密码或 Windows Hello 验证你的身份，因此即使其他人正在使用你的电脑，他们也无法轻松访问你的登录信息。使用 Edge 99，您可以设置自定义密码来保护您的自动填充信息-如果您愿意，也可以设置密码来保护您的密码。

在这方面，微软还增加了一项政策，允许管理员在某些网站上阻止密码管理器。如果您不希望用户能够保存密码并在特定网站上使用自动填充，您可以使用 PasswordManagerBlocklist 策略来做到这一点。

回到面向用户的功能，微软也在改进 Edge 中的 PDF 导航。现在，您可以在左侧窗格中看到页面的缩略图，这样您就可以更容易地找到您想要的内容。以前，Edge 只在 PDF 文件有目录时才提供导航窗格，所以这应该有助于没有目录的长文档。

最后，奇怪的是，微软再次提到了自动切换某些网站的用户配置文件的能力。这最初是在 Edge 98 的变更日志中提到的，但后来因为未知原因被删除了。它似乎又回到了 Edge 99，这意味着你可以在 Edge 中设置特定的网站，自动将你切换到不同的用户配置文件。这样，您可以立即从正确的配置文件访问保存的密码，或者立即登录，而无需手动切换配置文件。

Edge 应该很快就会自动更新，但你可以通过进入省略号菜单，然后选择 选择帮助&关于微软 Edge 的反馈>来检查更新。如果你还没有安装浏览器，你可以在这里下载 Edge。
