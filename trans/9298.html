这些都是你需要知道的 Windows 10 键盘快捷键

原文：https://www.xda-developers.com/windows-10-keyboard-shortcuts/#0001-01-01

对大多数人来说，使用带键盘和鼠标的电脑是最自然的方式。这就是大多数现代计算机的设计使用方式，一般来说，它可以更直观。使用鼠标可以让您轻松地看到并指向您想要点击、打开或移动的对象，因此任何人都可以拿起它并了解它是如何工作的。但对于熟练用户来说，键盘本身有时可能是一个更强大的工具。在本指南中，我们将向您展示您可以在 Windows 10 中使用的所有键盘快捷键，以便更快地完成工作。

你可以用键盘做很多你可能不知道的事情。无论是像复制文本到打开应用程序和某些功能这样简单的事情，在阅读本指南后，你可能会发现自己更多地使用键盘。

Windows 10 功能和应用的键盘快捷键

多年来，您可能都知道键盘上用于打开开始菜单的 Windows 键，但是您知道吗，您可以将它与其他键结合使用来做更多的事情？这里有一些使用 Windows 键启动功能和应用程序的快捷方式，以及其他一些快捷方式。

  • Windows 键+ A -打开行动中心。这种显示或通知以及 Wi-Fi、蓝牙、对焦辅助等的快速操作切换。
  • Windows 键 + C -启动 Cortana，Windows 10 内置的数字助理。您可以键入问题和其他查询来获得 Cortana 的帮助，或者配置 Cortana 在您使用快捷方式时开始监听您的语音。
  • Windows 键 + D -显示或隐藏桌面。显示桌面会隐藏所有打开的窗口和应用程序，再次按下键盘快捷键会将窗口恢复到正确的位置。
  • Windows 键 + Alt + D -显示时间和日历弹出按钮。
  • Windows 键 + E -打开文件浏览器。
  • Windows 键 + F -截图打开反馈中枢。这使您可以更容易地向 Microsoft 发送有关您可能遇到的问题的反馈。
  • Windows 键 + G -打开 Xbox 游戏栏。这可以让您在玩游戏时录制视频、截图和访问各种功能。
    • Windows 键+Alt+B-打开或关闭 HDR。这需要一个 HDR 兼容的显示器和一个最新版本的 Xbox 游戏吧应用程序。
    • Windows 键 + Alt + R -使用 Xbox 游戏栏开始录制您的应用程序或游戏的视频。
  • Windows 键 + H -启动听写，让你用声音输入文本，而不是打字。这仅适用于特定语言。
  • Windows 键 + I -打开设置 app。
    • Windows 键 +暂停 -在设置中打开关于页面。这包括有关您的计算机硬件和 Windows 本身的信息。
    • Windows 键 + U -打开辅助功能设置页面。
      • Windows 键+Ctrl+C-打开或关闭彩色滤镜。此功能需要首先在“辅助功能设置”中启用(在“颜色过滤器”标签中)，它允许您更改颜色的显示方式，以应对各种形式的色盲。
  • Windows 键 + K -打开连接面板，允许您连接到支持 Miracast 的无线显示器，以及其他无线设备，如蓝牙。
  • Windows 键 + L -锁定你的电脑。这将使你的所有应用保持打开，但需要你在再次使用它们之前解锁你的电脑。
  • Windows 键 + M -最小化你所有的窗口。
    • Windows 键 + Shift + M -恢复你最小化的窗口。这只有在你没有打开任何窗户的情况下才有效。
  • Windows 键 + O -锁定设备方位。对于平板电脑和敞篷车，这可以防止显示器在您旋转电脑时旋转。
  • Windows 键 + P -改变多台显示器的显示模式。如果您连接了多台显示器，您可以选取是将您的屏幕用作彼此的扩展、复制它们，还是一次只使用一个屏幕。
  • Windows 键 + Ctrl + Q -开启快速辅助。这是一个工具，允许您通过查看另一个 Windows 用户的屏幕或控制他们的 PC 来为他们提供远程帮助。
  • Windows 键 + R -打开运行对话框。这允许你在你的电脑上打开任何应用程序，只要你知道应用程序的可执行文件的名称。
  • Windows 键 + S 或 Windows 键 + Q -打开 Windows 搜索。你可以在这里搜索文件、应用和网页。
  • Windows 键+Shift+S——使用 Snip &草图截图。按键后，您可以选择您喜欢的捕获模式，包括自由格式、矩形、窗口和全屏(这包括所有连接的显示器)。
  • Windows 键 + V -打开你的剪贴板历史记录。这允许您查看您拷贝的多个项目，包括文本、链接和图像。此功能最初是禁用的，但您也可以在此窗口中启用它。
  • Windows 键 + X -打开快速链接菜单，相当于右击你任务栏上的开始图标。这包含系统功能的链接，如设置、文件资源管理器、任务管理器等。
  • Windows 键 + Y (仅限 Windows 混合现实设备)——在 Windows 混合现实和桌面之间切换输入。
  • 视窗 + 。 (期)或；(分号)-打开表情面板。这让你可以添加表情符号到任何文本字段。
  • Windows + ， (逗号)——暂时看一下桌面。你的桌面是可见的，只要你按住其中一个键，然后你的窗口被恢复。
  • Windows 键+Ctrl+F-搜索网络上的电脑。这意味着 Azure 活动目录域。
  • Windows 键 + 空格键 -在语言和输入法之间切换(如果安装了多个)。
    • Windows 键 + Shift + 空格键 -在列表中向后循环。
    • Windows 键 + Ctrl + 空格键 -切换到上次使用的输入法。
    • Ctrl + Shift -如果有多个可用的键盘布局，切换到不同的键盘布局(同一种语言)。
    • Ctrl + 空格键 -打开或关闭中文 IME(如果安装了中文)。
  • Windows 键+Ctrl+Enter-打开讲述人。这种辅助功能可以读取屏幕上的元素，使有视觉障碍的人更容易导航。
  • 视窗键 + + (加号)-打开放大镜，放大屏幕。
    • Windows 键+--(减号)-用放大镜缩小。
    • Windows 键 + Esc -关闭放大镜。
  • Windows 键 + / (正斜杠)-开始 IME 恢复。
  • Windows 键+Ctrl+Shift+B-从空白或黑屏中唤醒您的电脑。
  • Windows 键 + PrtScn (打印屏幕，在不同键盘上可能表现不同)——全屏截图，自动保存到文件。项目保存在图片库的一个名为“屏幕截图”的文件夹中。
    • Windows 键+Alt+PrtScn-截取活动游戏或窗口的截图，保存到文件中。该功能使用 Xbox 游戏栏，文件保存在视频库中，位于名为 Captures(默认)的文件夹中。
    • PrtScn——全屏截图，复制到剪贴板，这样就可以粘贴到别的地方，不用保存为文件。你也可以去设置 app >辅助功能>键盘设置 PrtScn 键打开剪贴工具(使其等同于 Windows 键 + Shift + S )
  • Ctrl + Esc -打开开始菜单。
    • Ctrl +(箭头)-调整开始菜单的大小(当它打开时)
    • Alt + Shift +(箭头)-当焦点设置到开始菜单上的一个磁贴上时，按箭头方向移动磁贴。
    • Ctrl + Shift +(箭头)-当焦点被设置到一个磁贴时，将其移动到下一个磁贴来创建一个文件夹。
  • Ctrl+Shift+Esc-打开任务管理器。

多任务和窗口管理的键盘快捷键

同时处理多个应用程序和窗口可能会很麻烦，鼠标并不总是找到你想要的应用程序的最快方法。在 Windows 10 中，这些键盘快捷键可以帮助您在应用程序之间切换，调整窗口大小，或使它们彼此相邻。

  • Windows 键+Tab-打开任务视图。这会将您所有打开的应用程序显示为磁贴，这样您就可以选择一个应用程序进行关注。它还显示您的虚拟桌面和时间线，包括以前打开的应用程序和文档。
  • Alt + Tab -切换到上一个活动窗口。按住 Alt 的同时反复按下键，选择切换到哪个窗口。
    • Alt + Esc -按照窗口打开的顺序在窗口间循环。
    • Ctrl+Alt+Tab-查看你所有打开的应用程序，这样你可以用键盘箭头选择一个。这类似于任务视图，但不显示您的虚拟桌面，仅在您当前的活动监视器中可见。
  • Alt + F4 -关闭活动窗口或应用程序。如果在桌面上使用，这将打开 Windows 电源菜单。
  • Windows 键 + 向上箭头 -最大化活动窗口，使其占据整个桌面。
  • Windows 键 + 向下箭头 -如果当前窗口最大化，将其设置为较小的尺寸。如果窗口没有最大化，将它隐藏到任务栏中。
  • 窗口键 + 左箭头 -将活动窗口贴紧屏幕的左半部分。
  • Windows 键 + 右箭头 -将活动窗口贴紧屏幕的右半部分。
    • 如果您在将应用程序抓取到半个屏幕后按住 Windows 键，您可以使用其他箭头之一将应用程序抓取到四分之一的屏幕。您也可以重复按下同一个箭头，将窗口移动到不同的显示器上(仍然占据半个屏幕)。
  • Windows 键 + Home -最小化除活动窗口以外的所有窗口。再次按下可恢复最小化窗口。
  • Windows 键 + Shift + 向上箭头 -拉伸活动窗口占据整个垂直空间，保持宽度不变(对于非最大化窗口)。带有向下箭头的同一快捷方式与此相反。
  • Windows 键 + Shift +右箭头或左 箭头 -将活动窗口移动到不同的监视器。
  • Windows 键 + Ctrl + D -创建新的虚拟桌面
  • Windows 键 + Ctrl + 左箭头或右箭头 -向左或向右切换虚拟桌面。
  • Windows 键 + Ctrl + F4 -关闭当前虚拟桌面。任何打开的应用程序都会被移动到下一个虚拟桌面。

Windows 10 中的文件浏览器键盘快捷键

文件资源管理器是 Windows 的核心功能之一，允许您查看和管理存储在 PC 上的所有文件。使用键盘而不是鼠标，有一些快捷方式可以让这种体验变得更容易、更快。

  • Alt + D -将焦点设置到地址栏。
  • Ctrl + E 或 Ctrl + F 或 F3 -将焦点设置到搜索栏。
  • Ctrl + N -打开一个新窗口(在一些网络浏览器中也适用)
  • Ctrl + W -关闭活动窗口(这也可以用来关闭网络浏览器中的活动标签)。
  • Ctrl +鼠标滚轮——改变文件夹和文件图标的大小。向上滚动使图标变大，向下滚动使图标变小。
  • Ctrl+Shift+N-新建一个文件夹。
  • Ctrl+Shift+E-如果当前文件夹没有子文件夹，则显示当前活动文件夹的父目录中的所有文件夹(通过展开侧边栏上的列表)。
    • Num lock + * (星号)或 + (加号)-在工具条上显示当前选中文件夹中的所有子文件夹。
    • Num lock + - (减)-折叠展开的文件夹。
  • Alt + Enter -查看所选文件或文件夹的属性。
  • Alt + P -显示预览面板。
  • Alt + 向左箭头或退格键 -返回上一页(也适用于一些应用程序，如网络浏览器)
  • Alt + 右箭头 -前进一页(也适用于网络浏览器等应用程序)
  • Alt + 向上箭头 -查看当前活动文件夹的父文件夹。
  • Ctrl + (箭头)-导航当前页面中的项目，而不选择它们。
    • Ctrl + 空格键 -导航时选择单个项目
  • Shift +(箭头)-从当前选中的项目开始选择连续的项目。在网格样式的布局中，上下箭头选择整行项目。如果您开始向相反方向移动，则会取消选择最后选定的项目。
  • 右箭头(在导航侧边栏上)-展开折叠的文件夹或切换到展开文件夹的第一个子文件夹。
    • 左箭头 -折叠展开的文件夹或选择当前所选文件夹的父文件夹。
  • 主页 -转到当前页面的顶部(适用于其他各种应用)。
  • End -转到当前页面的底部(在多个应用中也适用)。
  • F2 -重命名选定的文件或文件夹。
  • F4 在文件浏览器中显示地址栏列表。

对话框的键盘快捷键

对话框是小的 windows 应用程序，有时可以生成来显示菜单选项和设置。有几个键盘快捷键是专门为对话框设计的，但是它们会根据对话框的复杂程度而有所不同。以下是您可以在这些对话框中使用的一些键盘快捷键。

  • F4 或空格键 -显示活动下拉列表中的项目。
  • 箭头键 -选择一组按钮中的一个按钮或列表中的一个选项。
  • 空格键 -如果是复选框，选择或取消选择活动项目。
  • Ctrl + Tab -切换不同的标签页(也适用于网络浏览器)
    • Ctrl+Shift+Tab-通过 Tab 向后切换
  • 标签 -在当前页面的选项间移动
    • Shift + Tab -在当前页面的选项中向后移动。
  • Alt +(字母)-选择描述中带有相应下划线字母的选项。

Windows 10 任务栏快捷方式

任务栏是 Windows 10 的另一个核心部分，虽然我们很多人通常会使用鼠标与它进行交互，但你也可以通过键盘快捷键来完成。但在某些情况下，你也可以用键盘作为鼠标的助手，让某些动作更快。

  • Windows 键 + T -通过任务栏上的应用程序循环聚焦(打开或锁定)。然后，您可以通过按 Enter 键来选择它们。
  • Windows 键 +(数字)-在数字指示的位置启动钉在任务栏上的应用。如果应用程序已经在运行，请切换到该应用程序。如果一个应用程序打开了多个实例，您可以按住 Windows 键并再次按下相同的数字来切换到该应用程序的特定实例。
  • Windows 键 + Shift +(数字)-启动一个新的应用程序实例，该应用程序被固定在任务栏上由数字指示的位置，即使其中一个已经打开。
    • Windows 键+Ctrl+Shift+(数字)——以管理员身份打开位于任务栏给定位置的 app 新实例。
  • Windows 键 + Ctrl +(数字)-切换到钉在任务栏上由数字指示位置的应用程序的最后一个活动窗口。
  • Windows 键 + Alt +(数字)-在数字指示的位置打开钉在任务栏上的应用的跳转列表。
  • Shift +鼠标左键点击——打开所选 app 的新实例。
  • Ctrl + Shift +鼠标左键点击——以管理员身份打开选中的 app。
  • Shift +鼠标右键-打开所选应用的窗口菜单
  • Ctrl +鼠标左键点击(在一个打开了多个窗口的应用程序上)——在应用程序打开的窗口间循环
  • Windows 键+ B -高亮显示通知托盘区的第一个图标。这允许您使用箭头键导航图标，并使用 Enter 键选择它们。

Windows 10 中的通用键盘快捷键

除了特定的 Windows 10 功能，许多键盘快捷键在整个操作系统及其应用程序中都可用。这可能包括复制文本和项目、导航菜单等，但应用程序对这些功能的支持可能会有所不同。

  • Ctrl + A -选择文档、页面或窗口中的所有文本或项目。
  • Ctrl + D -删除选中的文本或项目。
  • Ctrl + X -剪切选中的项目或文本。
  • Ctrl + C -复制选中的项目或文本。
  • Ctrl + V -粘贴剪贴板中复制或剪切的内容。
  • Ctrl + Z -撤销你上一次的操作。
  • Ctrl + Y -重做一个撤消的动作。
  • F5 或 Ctrl + R -刷新活动窗口或页面(适用于各种应用，包括网络浏览器)。
    • Ctrl + F5 -在一些网络浏览器中，这通过强制浏览器重新加载文件来刷新活动窗口，即使这些文件之前已经被缓存。如果对页面进行了更改，但您无法在浏览器中看到这些更改，这将很有帮助。
  • F6 -在活动窗口或桌面的屏幕元素间循环。
  • F10 -激活活动窗口或应用程序中的菜单栏。
  • Alt + F8 -在 Windows 登录屏幕上显示你的密码
  • Alt +(字母)-当显示菜单时，选择文本中带有相应下划线字母的选项。
  • Alt + 空格键 -打开活动窗口的快捷菜单。
    • Shift + F10 -打开所选项目的快捷/上下文菜单。
  • 左箭头(菜单中)-在菜单上向左移动，或关闭一个打开的子菜单。
    • 右箭头(菜单中)-在菜单上向右移动，或打开选定的子菜单。
  • Ctrl + F4 -在允许你一次打开多个文档或标签的应用程序中关闭打开的文档或标签，例如网络浏览器。
  • Ctrl + E -打开搜索(部分 app 中)。
  • Ctrl + 右箭头 -在文档中，将文本光标移动到下一个单词的开头。
    • Ctrl + 左箭头 -将文本光标移动到前一个单词的开头。
  • Ctrl + 向上箭头 -在文档中，将文本光标向上移动一个段落。
    • Ctrl + 向下箭头 -将文本光标下移一段。
  • Alt + Shift +(箭头键)——当焦点被设置到开始菜单中一个被钉住的应用程序时，将被钉住的应用程序向箭头方向移动。
  • Shift +(箭头键)-在文档中，从文本光标的位置开始选择文本。
    • 左箭头 -选择前一个字符。向右箭头取消选择最后选定的文本。
    • 右箭头 -选择下一个字符。向左箭头取消选择最后选定的文本。
    • 向上箭头 -选择上一行。向下箭头取消选择最后一行。
    • 向下箭头 -选择下一行。向上箭头取消选择最后一行。
  • Ctrl + Shift +(箭头键)-在一个文档中，从光标所在的位置开始选择一个文本块:
    • 左箭头 -选择上一个单词。向右箭头取消选择最后选定的文本。
    • 右箭头 -选择下一个单词。向左箭头取消选择最后选定的文本。
    • 向上箭头 -选择上一段。向下箭头取消选择最后选定的段落。
    • 向下箭头 -选择下一个段落。向上箭头取消选择最后选定的段落。
  • Esc -停止或离开正在进行的任务(使用因应用程序而异)。

有了这些 Windows 10 键盘快捷键，你可能很快就会发现自己更喜欢键盘而不是鼠标。许多人的第一直觉可能是，这并不直观，但学习所有这些捷径是非常有益的，因为从长远来看，它们会节省你很多时间。对于其中的一些快捷方式，特别是基于 Windows 键的快捷方式，微软的 PowerToys 软件包括一个叫做快捷方式指南的工具。您可以按住 Windows 键来查看所有可用的快捷键。

Windows 10 仍然是最广泛使用的操作系统，但是如果你已经升级到了 Windows 11，我们也有一个关于 T2 的 Windows 11 键盘快捷键的指南。在两种操作系统中，大多数都是相同的，但有一些发生了变化，Windows 11 也有一些新的变化。如果你想知道你是否应该升级，请查看我们的 Windows 11 评测。
