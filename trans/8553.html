Firebase 现在为 tvOS 和 macOS 提供测试级支持

Firebase 现在提供了对 tvOS 和 macOS 的测试级支持，以及针对苹果用户的大量其他改进。

谷歌的虚拟 Firebase 峰会 2021 今天拉开帷幕，我们将获得 Firebase 的一些新功能。Google 的 Firebase 是一套非常全面的开发工具。它的目标是通过详细的崩溃报告、用户分析、身份验证和存储等功能，使应用程序的开发和维护变得更容易。谷歌一直在努力改进 Firebase，最近在 Google I/O 21 上宣布了 Firebase 的新扩展。现在，谷歌宣布 Firebase 为 tvOS 和 macOS 提供测试级支持，同时也支持 Apple TV 和 Apple Arcade。

这实质上意味着，开发者现在可以使用 Firebase 产品来构建和运行与苹果电视和 Macbooks 兼容的应用程序。它从单个代码库开始工作，为用户提供跨设备的体验，减少麻烦。这方面的一个例子是 Crashlytics SDK，因为开发人员可以直接从 Firebase Crashlytics 控制台识别关键崩溃，甚至按 Apple 设备类型或操作系统过滤崩溃。

游戏开发者在为 Apple TV 和 Apple Arcade 开发时也获得了一些相当大的改进。现在许多 Firebase c++ SDK 都支持 Apple TV，这意味着您可以使用 Firebase 开发 Apple 街机游戏！最重要的是，我们是谷歌宣布，它将通过使云 Firestore 可用于 Unity 和 C++来扩大对游戏框架和引擎的支持。这意味着您可以将云 Firestore 添加到您的游戏中，以便近乎实时地存储和同步您的游戏数据，添加离线支持，并将您的游戏体验扩展到大量用户。

如果您不熟悉，Cloud Firestore 是一个用于移动、web 和服务器开发的 NoSQL 数据库。它是跨平台的，开发者可以创建使用原生 SDK 从其中检索信息的 Apple、Android 和 web 应用程序。它简化了终端用户跨设备的数据同步。
