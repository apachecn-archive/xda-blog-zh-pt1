Microsoft Defender 预览版现在可用于 Windows 和 Android

原文：https://www.xda-developers.com/microsoft-defender-preview-windows-android/#0001-01-01

微软本周推出了适用于 Android 和 Windows 设备的新微软 Defender 预览版应用。目前，该应用程序仅在美国可用，但随着测试的进行，它可能会扩展到其他地区。

微软 Defender 是该公司的反恶意软件工具，它已经在 Windows 上可用了一段时间，它也在 Android 上推出了一些功能，带有微软 Defender ATP ，这仅适用于商业用户。Microsoft Defender 的新预览版让您不仅可以管理当前设备的安全性，还可以管理连接到同一 Microsoft 帐户的所有设备的安全性。在主页上，您可以看到所有设备上的防护状态摘要，如下所示。

您可以单击或点击该面板，查看有关每个设备状态的更多信息。在这里，您可以看到过去 24 小时内扫描的应用程序和链接(Android 上)或文件(Windows 上)的数量。在 Windows 上，您还可以查看您当前的病毒和威胁防护、勒索软件防护等设置。如果您想查看过去的威胁，也可以查看您的安全历史记录。

对于 Windows 用户来说，这似乎并不能完全取代 Windows Security，而是对它的补充，或者对 UI 的新看法，最终可能取代内置的应用程序。在 Android 上，防病毒应用的有用性值得怀疑，但它可以告诉你是否安装了潜在的恶意应用。

您可能从中获得的最有用的功能是能够在同一个地方查看所有设备的安全状态。如果你和别人分享你的电脑，这尤其有用，这样你就可以在你的手机上看到另一个用户是否安装或下载了恶意软件。

如果你感兴趣并且住在美国，你可以在微软商店下载微软视窗版，在谷歌 Play 商店下载安卓版。Windows 应用程序将在美国以外的地区安装，但它不允许您登录，至少目前不允许。
