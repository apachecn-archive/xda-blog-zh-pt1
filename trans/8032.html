三星从 Samsung Pay 和 Samsung Health 中移除广告

原文：https://www.xda-developers.com/samsung-pay-health-ads-removed/#0001-01-01

三星从 Samsung Pay 和 Samsung Health 移除广告，更多广告将跟进

三星终于开始从 One UI 股票应用中移除广告，首先是三星健康和三星支付。请继续阅读，了解更多信息。

三星终于开始摆脱 One UI 股票应用中的广告。今年早些时候，该公司承诺从专有应用中移除广告，如三星天气、三星支付和三星主题。但是，当时，它并没有分享一个明确的时间表。三星终于开始从 One UI 股票应用中移除广告，首先是三星健康和三星支付。

在三星社区论坛上的一份声明中(通过 TizenHelp )，一名版主透露，三星健康应用程序将从今天开始不显示广告。他们写道:

“这是三星健康运营经理。请注意，10 月 1 日起将不再提供三星健康应用顶部的横幅。”(机器翻译)

一名社区成员的独立帖子透露，该公司也从 Samsung Pay 上移除了横幅。包括以下截图。

三星终于开始从 One UI 股票应用中移除广告，这太好了。虽然该公司还没有任何关于何时将广告从其余应用程序中移除的信息，但我们预计在未来几周内将看到三星天气、三星主题等类似的更新。然而，目前还不清楚三星是否也会以类似的方式删除其设备上通过推送通知发送的广告。

值得注意的是，三星目前仅在韩国 One UI 固件发布中移除了三星健康和三星支付的广告。这些变化在其他地区生效可能还需要一段时间。一旦发生这种情况，我们会尽快通知您。
