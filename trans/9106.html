[更新:在印度推出] Realme 在中国向 Realme GT 推出稳定的 Realme UI 3.0

Realme 已经开始在中国推出基于 Android 12 的 Realme UI 3.0 的第一个稳定版本。请继续阅读，了解更多信息。

更新(2021 年 12 月 29 日@09:30ET):Realme GT 的稳定 Realme UI 3.0 更新现已在印度铺开。滚动到底部了解更多信息。下面保留了 2021 年 12 月 23 日发表的文章。

在测试频道对 Realme UI 3.0 进行了数周的测试后，Realme 已经开始向 Realme GT 推出第一个基于 Android 12 的稳定版本。此次更新带来了谷歌在 Android 12 中引入的所有新功能，以及 Realme 的一些独特新增功能。目前，稳定的 Realme UI 3.0 更新仅限于 Realme GT 的中国版本，但我们预计该公司将在未来几天发布全球版本的更新。

根据 Realme 社区论坛上最近的帖子，稳定更新(软件版本 RMX2202_11_C.10)已经开始在中国推出 Realme GT。该更新正在分阶段进行，将在未来 15 天内覆盖该地区的所有设备。请查看以下部分以获取完整的变更日志。

Realme UI 3.0 (RMX2202_11_C.10)变更日志(机器翻译)

  • 全新的设计
    • 新增智能助理卡，提供智能高效的应用卡，关键信息一目了然，关键功能一键到达
    • 重新调整了页面布局、降噪和空白，并通过颜色区分主次信息，核心信息更加集中
    • 增加了三维图标设计，引入了新材料，统一了光影和层次，体验了色彩的喜悦
    • 优化量子动画引擎 3.0，聚焦“品质”，模拟真实物理运动，优化动画细节 300+，带来更自然的交互体验
    • 优化更丰富的屏幕样式，支持齐针喵和人像阴影 AOD，让自我表现更加独特和独特
  • 便捷高效
    • 新增笔记涂鸦工具，个性化记录生活
    • 优化了智能侧栏
      • 支持视频、购物等场景下的智能推荐相关功能
      • 增加了小窗口速记，点击进入笔记的浮动窗口，快速完成记录
      • 增加了智能字幕，支持会议和通话语音识别，实时生成字幕，支持中英文互译
      • 新增背景音乐识别，快速识别短视频中的背景音乐，无需求助即可找到音乐
      • 增加了文章朗读，支持一键阅读微信官方账号、今日头条等资讯内容
      • 增加了商品比价，使用购物软件时，一键显示同样好的商品的低价
    • 优化 Shinda 窗口升级到自由浮动窗口
  • 安全性和隐私
    • 增加了对删除位置信息和拍摄照片或视频数据的支持
    • 优化了手机丢失后可以通过云端锁定使其无法关机或使用，降低了数据丢失的风险
    • 优化手机管理器，集成更多安全和隐私功能，如隐私替身、密码本、紧急呼叫等。
    • 优化状态栏图标提醒，使其在应用调用相机、录音、定位等敏感权限时更加醒目。
    • 优化医疗急救卡，支持在拨打 120 时自动将预设的医疗急救卡信息发送给急救中心
    • 优化地震预警，新增功能预览页面，新增预计烈度 2.0 及以下地震横幅通知提醒，时刻提供地震信息预警
    • 优化骚扰拦截，增加彩信拦截功能，减少垃圾信息的中断
  • 性能优化
    • 增加了对常用应用程序的智能预测，并提前加载，使应用程序打开更快
    • 添加了显示功耗记录的图表，以快速了解电池消耗详情
    • 针对切换无线局域网、蓝牙、飞行模式和 NFC 时的更快响应进行了优化
    • 优化第三方应用高频场景的加载速度，提高扫描微博的速度，看视频更过瘾
  • 比赛
    • 增加了游戏资源包更新时，游戏可以切换到后台进行更新(仅支持部分游戏)
    • 增加了游戏中的弹出游戏技巧和比赛回放，有助于快速掌握游戏玩法(仅支持部分游戏)
    • 优化团战等高负载场景下游戏的帧率性能，使其更加稳定
    • 优化异构计算，降低处理器的平均负载，降低游戏功耗
  • 照相机
    • 增加了自定义相机模式的新顺序，便于快速切换到常用的相机模式
    • 优化惯性变焦，后置视频拍摄模式，拖动滑动条匀速变焦
  • 系统
    • 优化自动亮度调节算法，优化智能场景检测，浏览屏幕更舒适
  • 易接近
    • 优化辅助功能模式
      • 添加了视觉介绍，并在功能页面中添加了图像介绍
      • 优化功能分类，按照视觉/听觉/交互/通用分类显示
      • TalkBack 适应更多的系统应用，如相册、通话、邮件、日历等。

目前，Realme 还没有分享任何关于全球推广的信息。然而，不久之后这个更新将会在更多的地区推出。一旦 Realme 开始为 Realme GT 的全球版本推出 Realme UI 3.0 的稳定版本，我们会尽快通知您。

更新:稳定的 Realme UI 3.0 现已在印度为 Realme GT 推出

根据 Realme 社区论坛上最近的一篇帖子【the Realme GT 的稳定 Realme UI 3.0 更新现在正在印度推出。更新(内部版本号 RMX2202_11。C.05)具有以下变更日志:

  • 新型设计
    • 全新的设计强调空间感，提供简单、干净、舒适的视觉和交互体验。
    • 根据减少视觉干扰和分隔元素的原则修改页面布局，并使用不同的颜色区分信息的优先级，以突出关键信息。
    • 使用新材料重新设计图标，赋予图标更多的深度和更强的空间感和质感。
    • 优化了量子动画引擎:量子动画引擎 3.0 贯彻了质量的概念，使动画更加逼真，优化了 300 多个动画，带来更自然的用户体验。
    • 更有创意的始终显示:增加真实喵和肖像剪影，让你自由表达自己。
  • 便利性和效率
    • FlexDrop 被重命名为灵活窗口，并经过优化:
      • 优化在不同大小之间切换浮动窗口的方法。
      • 现在，您可以将“我的文件”中的文件或“照片”应用程序中的照片拖到浮动窗口中。
  • 表演

    • 添加图表来显示您的电池使用情况。
    • 提高打开或关闭 WiFi、蓝牙、飞行模式和 NFC 的响应能力。
  • 比赛
    • 在团队战斗场景中，游戏在稳定的帧速率下运行更加流畅。
    • 降低 CPU 平均负载并降低电池使用量。
  • 照相机
    • 您现在可以决定在菜单栏上显示哪些相机模式以及它们的显示顺序。
    • 现在，当使用后置摄像头拍摄视频时，您可以拖动缩放滑块来平滑地放大或缩小。
  • 系统
    • 优化自动亮度算法，使屏幕亮度适应更多场景，带来舒适的屏幕阅读体验。

    • TalkBack 支持更多的系统应用程序，包括照片、电话、邮件和日历。
