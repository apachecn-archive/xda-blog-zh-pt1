微软开始用个人帐户测试“列表”任务管理器

原文：https://www.xda-developers.com/microsoft-lists-personal-accounts/#0001-01-01

多年来，微软的 To-Do 一直是一款简单的移动和桌面任务管理应用，但该公司还为企业和其他大型组织提供了一款更强大的替代产品。微软列表是一个超级充电的任务管理器，更注重协作，现在微软开始向个人账户开放它。

微软在一个社区帖子 ( via The Verge )中表示，“今天我们将推出微软列表- MSA 预览版，这是一款轻量级版本的微软列表应用，专为小型企业和个人设计，可与你的微软账户(MSA)配合使用。Microsoft List - MSA 预览版仅在线提供，仅限 200，000 个 Microsoft 帐户(MSA)免费试用，先到先得。”

微软的列表类似于 Monday.com 吉拉的 Asana 和其他协作任务管理系统。列表可以有多个可自定义的字段，比简单的待办事项列表应用程序提供了更多的排序和查看选项。您可以邀请其他人创建或编辑任务，带有截止日期的任务会添加到日历视图中。微软将此次预览限于参观 lists.live.com 的前 20 万人。

不过，还是有一些问题。iOS 和 Android 的微软列表还不支持个人账户预览，所以你只能使用 web 应用。它也不能像企业版那样连接到微软团队，至少目前是这样。最后，个人帐户被限制为 50 个列表，每个列表最多 2000 个项目，附件最大不超过 200MB。

微软是否会通过免费个人账户向所有人开放列表还有待观察，但这对 Asana 和竞争平台来说可能是一个重大打击。即使每个人都可以使用列表，对于不需要所有列表功能的人来说，微软的待办事项可能会是一个更简单的选择。
