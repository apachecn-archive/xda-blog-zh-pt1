谷歌称 Chrome 99 是 Mac 上最快的网络浏览器

原文：https://www.xda-developers.com/chrome-99-mac-fast/#0001-01-01

谷歌在本月初发布了 Chrome 99，与往常一样，它结合了性能改进和面向 web 开发人员的新 API。然而，谷歌表示，最新的更新比以往更能提升性能，现在显然已经摘得了 macOS 上最快浏览器的桂冠。

谷歌周一在一篇博客中写道，“我们很兴奋地宣布，在 M99 中，Mac 上的 Chrome 在苹果评估浏览器响应性的速度表基准中获得了迄今为止所有浏览器中最高的分数——300 分。基于过去一年的许多性能变化，我们在 M99 中启用了 ThinLTO，这是一种构建优化技术，它优先考虑专注于浏览器速度的代码。结果呢？一个额外的全面减速带，让 Chrome 比 Safari 快 7%。”

由于 Chrome 中的大多数引擎代码都是基于 Chrome 的浏览器，如 Microsoft Edge 和 Vivaldi，其他浏览器一旦更新了代码库，应该也会得到同样的改进。然而，基准并不代表一切— 谷歌在 2017 年要求旧的 Octane JavaScript 基准，因为网络浏览器优化了他们的代码，以现实世界的性能为代价获得了更高的 Octane 结果。谷歌正在使用速度计 2.0 来获得这些结果，它是由苹果的 WebKit (Safari)团队开发的，并在各种流行的框架中重建常见的网络应用任务。

谷歌还表示，由于上述 ThinLTO 改进和额外的解码器，Chrome 的图形性能现在比苹果 Safari 快大约 15%。这主要有助于处理图像和视频的网络应用程序，以及具有 3D 渲染元素的网站，如谷歌地球。

在过去的几个月里，Chrome 浏览器经历了许多其他变化。今年 2 月，谷歌增加了一个“旅程”功能，用于对浏览器历史进行排序，这一功能与 Android 版 Chrome 上以你为主题的小工具同时推出。一个稍微更新的 Chrome 图标也在工作中，它将在每个平台上看起来有点不同，以适应操作系统的设计语言。
