如何在 Windows PC 或 Mac 上调整图像的大小

原文：https://www.xda-developers.com/how-to-resize-an-image/#0001-01-01

我们都有过这样的经历:试图上传一张图片，有时是作为一个大表单的一部分，而你需要上传图片的网站对文件大小有一个恼人的限制。这可能是一个可以理解的限制——毕竟，图像占用了大量空间，因此压缩它们可能很重要。但这意味着你现在必须自己调整图像的大小，我们在这里向你展示如何在你的 Windows PC 或 Mac 上做到这一点。

需要说明的是，在 PC 上并不缺少调整图像大小的方法，但是在你下载一些你从未听说过的第三方工具之前，最好先看看那些现成的工具。让我们来看看你的选择。

如何在 Windows PC 上调整图像的大小

开箱即用，Windows 给你一个以上的工具来调整你的图像。另外，你的笔记本电脑制造商甚至可能包括他们自己的工具，但我们将重点放在任何版本的 Windows 都包括的工具上。

如何使用“照片”应用程序调整图像大小

照片应用程序可能是你调整任何图像大小的最简单的方法。它非常快速和简单，假设你只是想调整一对夫妇的形象。你需要做的是:

  • 使用“照片”应用程序打开您想要调整大小的图像。
  • 点击顶部菜单栏上的省略号按钮，点击调整大小。
  • 从提供的默认大小选项中选择。如果您想设置特定尺寸，也可以选择定义自定义尺寸。
    • 选取“自定尺寸”还允许您更改图像的长宽比并设置最终图像质量，质量越高，文件越大。
  • 选择保存新图像的位置。

如果你想使用 Photos 应用程序快速调整图片大小，这就差不多了。如果要覆盖现有文件，可以用相同的文件名保存图像，但也可以保留原始的高质量版本。

使用油漆

如果你真的因为任何原因讨厌照片应用程序，画图也可以让你相当容易地调整图像的大小。这样做的主要好处是，您可以在保存文件之前看到最终结果。只要应用程序打开，您也可以撤销更改。如果您想使用这种方法，应该这样做:

  • 在文件资源管理器中，右键单击要调整大小的图像，然后选择用打开，然后从下拉列表中选择画图。
  • 在顶部菜单栏中，选择调整大小选项(见下文)
  • 您可以选择将图像大小调整为原始大小的百分比，或者如果您想要更具体的大小，可以使用绝对像素值。您可以点按链按钮来选择是否要保持原来的纵横比。如果您出于任何原因想要扭曲图像,“画图”还为您提供了扭曲图像的选项。单击“确定”查看结果。
  • 如果您对图像满意，可以单击窗口顶部的保存图标来更新原始文件。如果您想保存副本，请单击文件，然后单击另存为，并选择不同的文件名或位置将新图像保存到单独的文件中。

这是另一种调整图像大小的简单方法，也是我个人长期以来最常用的方法。

批量调整图像大小

如果你只是想调整一两张图片的大小，上面的两种方法很有效，但是如果你有一个包含大量大图片的文件夹，过一会儿就会很累。如果你想一次调整多个图像的大小，你可以下载一些工具来帮助你。

我个人最喜欢的是 PowerToys，它也是由微软发布，并与开源社区合作开发的。PowerToys 实际上有比图像缩放器更多的工具，但这是我们现在关注的。你需要做的是:

  • 在 GitHub 上下载最新版本的 PowerToys，并安装。
  • 启动 PowerToys 并切换到图像缩放器选项卡。确保该功能已打开。
  • 如果您想要设置图像大小调整器，使它总是设置为使用特定的大小，您可以自定义它提供的大小预置。您可以更改现有预设或添加新预设。当您调整图像大小时，您也可以每次设定自定尺寸，这样您就可以更快地决定您想要的大小。注:当您使用此工具调整图像大小时，它将被调整到您指定的最大尺寸。如果图像的纵横比与您选择的大小不同，则宽度或高度将小于您选择的大小。
    • 您还可以在此页面上更改高级编码设置和文件名格式设置。
  • 一旦你在这里设置好一切，你就可以关闭 PowerToys 窗口，转到你的图片所在的文件夹。选择所有要调整大小的图像，并右键单击它们。
  • 如果你使用的是 Windows 11，点击上下文菜单中的显示更多选项按钮，然后点击调整图片大小。在 Windows 10 上，该选项立即可用。
  • 选取一个预设大小或设定自定尺寸。您还可以选取是要替换现有图像还是创建副本、移除图像元数据等等。
  • 点击调整大小，图像将被调整大小。

这就是在 Windows PC 上调整图像大小的方法。如果您在上下文菜单中看不到调整图片大小的选项，您可能需要确保 PowerToys 正在运行。启动应用程序，如果你愿意，设置它在启动时运行，这样你就不必再考虑它了。

如何在 Mac 上调整图像的大小

如果你使用的是 Mac，事情实际上要简单得多，因为你需要的唯一工具是内置在 macOS 中的。苹果的操作系统附带了一个名为 Preview 的应用程序，这可以让你单独或批量调整图像的大小。

在此之前需要注意的一点是，当您调整图像文件大小时，预览将始终替换原始图像文件。如果您想要保留原始文件，请先复制它们，然后调整副本的大小并保持原始图像不变。

如果您想要调整单个图像的大小，请遵循以下步骤:

  • 在 Finder 中，打开您想要调整大小的图像。如果预览不是您打开图像的默认方式，请右键单击图像文件，然后选择用打开，并选择预览应用。
  • 打开预览，点击顶部菜单栏中的工具选项，然后选择调整大小。
  • 您将看到一个窗口，为您的图像选择目标尺寸。有一些预设可用，但您也可以选择自定大小。您可以按原始分辨率的百分比调整大小，或者使用下拉菜单切换到绝对值。如果您喜欢这种方法，也可以选择最终图像的像素密度。
  • 点击确定，您的图像将以新的分辨率保存。

如果你想批量调整图片的大小，过程是相似的，但是有一些事情你需要注意。首先，在 Finder 中选择您想要的所有图像，并使用预览打开它们。然后，在“预览”中，确保在使用“调整大小”选项之前选择了所有要调整大小的图像的缩略图。

大概就是这样。不可否认，在 macOS 上完成这项工作要容易一些，但无论如何，这都不是一个困难的过程。作为一名 Windows 用户，PowerToys 是我现在调整图像大小的首选，因为我只需点击几下就可以处理多个图像。第一次设置确实有点麻烦，但从长远来看，它会为您节省大量时间。如果你想使用第三方工具，Windows 和 macOS 都有很多其他选项，但这对大多数人来说可能更容易。
