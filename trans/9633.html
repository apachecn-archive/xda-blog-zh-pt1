谷歌很可能已经结束了 2018 款 Pixel 3 和 Pixel 3 XL 的官方更新

Pixel 3 和 Pixel 3 XL 可能会在本月 2022 年 2 月的安全补丁发布后收到官方更新

Pixel 3 和 Pixel 3 XL 于 2018 年 10 月发布，一直是谷歌的旗舰手机，直到一年后 Pixel 4 系列取代它们。尽管 2018 年 10 月可能看起来像是很久以前，但它只是三年多一点，现在 Pixel 3 系列(可能)已经收到了它的最终更新。

谷歌最初承诺 Pixel 3 和 Pixel 3 XL 将在 2021 年 10 月前获得主要的 Android 更新和每月安全补丁。然而，谷歌后来在 1 月份推出了紧急呼叫漏洞的修复程序，其他 Android 设备在 2022 年 1 月的安全补丁中收到了该漏洞。现在，Pixel 3 和 Pixel 3 XL 又推出了一个更新，澳大利亚运营商的版本号为 SP1A.210812.016.B1，其他版本的版本号为 SP1A.210812.016.C1。

如果在 Android 中发现另一个严重问题，谷歌可能会对 Pixel 3 和 Pixel 3 XL 进行另一次更新，但这可能是这条路的尽头。三年(又四个月)的支持肯定是不够的，尤其是对于一家在每一个可能的机会谈论关于减少环境破坏的公司来说。让手机远离垃圾填埋场的最好方法是确保它们尽可能长时间地保持功能和安全。

好消息是，Pixel 3 和 Pixel 3 XL 可以轻松解锁，允许任何人安装定制的 rom 和替代的操作系统。LineageOS 和其他项目可能会让 Pixel 3 和 Pixel 3 XL 在未来许多年内继续存在，特别是考虑到最早的官方版本的谷歌设备是 2014 年的 Nexus 6。不应该由定制 ROM 社区来让这些设备存活更长时间，但这就是我们目前的情况。

谷歌还在去年的某个时候将 Pixel 3 从其软件支持帮助文章中移除。该页面现在在一条消息下列出了 Pixel 3，该消息写道:“以下 Pixel 手机不再接收 Android 版本更新和安全更新。”
