谷歌用 Chrome 100 稳定更新杀死“精简模式”

原文：https://www.xda-developers.com/google-chrome-lite-mode-being-killed/#0001-01-01

Chrome 最古老的功能之一“精简模式”将很快被取消。该功能最初被称为“数据保护程序”，自 2014 年以来一直是 Android 版 Chrome 的一部分。但随着移动数据在全球变得越来越便宜、越来越容易获得，再加上谷歌的 AMP 计划，Lite 模式不再像过去那样有用。谷歌认为告别这一功能是安全的，不会对移动用户的浏览体验产生负面影响。

随着稳定频道即将到来的里程碑 Chrome 100 发布，Lite 模式将被移除。这一变化将影响最新的 Chrome 版本以及旧版本。

“2022 年 3 月 29 日，随着 Chrome M100 在稳定频道的发布，我们将关闭 Lite 模式，这是我们在 2014 年作为 Chrome Data Saver 推出的一项 Android Chrome 功能，旨在帮助人们在手机上使用更少的移动数据，更快地加载网页，”Chrome 支持经理 Craig 在最近的一篇帖子中写道。

Lite 模式可以从 Chrome 的“高级”设置页面中进入。当它打开时，Chrome 使用谷歌服务器来加快页面加载速度和保存数据。沉重且加载缓慢的页面被识别出来，并通过谷歌服务器进行“简化”(读取压缩)。Lite 模式仅在 Android 设备上可用。

正如谷歌的支持页面所解释的:

当您使用 Lite 模式时，您的一些网络流量可能会在下载到您的设备之前通过 Google 服务器。如果页面加载缓慢，谷歌服务器可能会对其进行简化，从而减少下载到您设备上的数据。

Chrome Canary 已经退出了精简模式，Chrome Beta 和 Chrome stable 将在未来几周内跟进。
