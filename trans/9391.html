微软强调采用 Windows 11 SE 的新型教育笔记本电脑

原文：https://www.xda-developers.com/microsoft-new-windows-11-se-laptops/#0001-01-01

微软去年年底宣布 Windows 11 SE 是面向 K-8 教育的 Windows 11 的轻量级版本。新操作系统与 Surface 笔记本电脑 SE 一起首次亮相，但微软的各种合作伙伴也加入了新 Windows 11 SE 笔记本电脑的竞争。今天，微软分享了更多关于即将上市的 Windows 11 SE 设备财富的信息。

首先，宏碁将推出两款新设备，宏碁 Travelmate Spin B3 和 Travelmate B3。这些 11.6 英寸的设备具有军用级别的耐用性，能够承受日常生活中的跌落、碰撞和泼溅。它由英特尔奔腾银或赛扬处理器驱动，支持 Wi-Fi 6，并承诺长达 10 小时的电池寿命。它还有一个网络摄像头，如果需要，可以进行远程学习。旋转版自然是敞篷车，而普通版 Travelmate B3 则是翻盖式。

华硕推出了 BR1100F，这也是一款小型 Windows 11 SE 笔记本电脑，采用可转换设计，支持触摸和笔。由于采用了全方位的橡胶保险杠，除了其他颠簸之外，它还可以承受桌子高度的跌落而不会出现问题。它也是防溅的。正如你所料，这款笔记本电脑支持 Wi-Fi 6，但如果你需要在任何地方连接，它也可以选择 4G LTE。该笔记本电脑采用模块化设计，易于维修，并且大部分笔记本电脑都经过抗菌处理，以保持清洁。最后，除了网络摄像头，华硕 BR1100F 还有一个面向世界的摄像头，可以拍照和录制视频。

戴尔还推出了配备 Windows 11 SE 的 Latitude 3120 和 3120 二合一笔记本电脑的新版本。这些 11 英寸笔记本电脑采用四核英特尔赛扬或奔腾处理器，除了电池续航时间长之外，它们还具有 ExpressCharge 功能，因此可以快速充电。笔记本电脑用户使用激光蚀刻来提供更好的握持，并防止笔记本电脑轻易地从孩子手中滑落。这款 2 合 1 机型还配有两个摄像头——一个前置摄像头和一个面向世界的网络摄像头——加上 Dragontrail Pro 玻璃，以防止划伤。

微软也提到了 Dynabook E10-S，尽管它之前是和 Windows 11 SE 一起发布的。这是另一款 11.6 英寸笔记本电脑，采用加固设计，以抵御年轻学生的日常生活。它有入门级规格，如 1366 x 768 显示器和英特尔赛扬处理器，但它有丰富的连接和端口。Dynabook 还推出了 Satellite Pro E10S，如果你需要这些额外的功能，可以使用完整版的 Windows 11。

惠普今天宣布了大量 Windows 11 SE 设备，其中最值得注意的是惠普 ProBook Fortis 14 英寸 G9，也可用于 Windows 11 Pro。顾名思义，这是一款 14 英寸的笔记本电脑，和其他笔记本电脑一样，它采用了加固设计，可以承受跌落、碰撞和泼溅。充电和 USB Type-C 端口也经过加固，可以承受拉扯的电缆，键盘有防撬键，防止儿童拔出。除了 Wi-Fi 6，如果您需要在旅途中保持连接，惠普还提供可选的 4G LTE 连接。惠普还推出了这款笔记本电脑的 11 英寸版本，包括一个 5MP 的面向世界的摄像头和一个可转换的设计，称为 HP Pro x360 11 英寸 G9。

这些笔记本电脑还有其他款式。两者都有 G10 版本，将处理器升级到第 12 代英特尔酷睿 i3 或酷睿 i5，并运行常规的 Windows 11 Pro，还有 ProBook Fortis 14 英寸 G10 的 Chromebook 变种，称为惠普 Fortis 14 英寸 G10 Chromebook。在这方面，惠普还宣布了 Fortis 11 英寸 G9 Q Chromebook，采用骁龙 7c 芯片组。

惠普 ProBook Fortis 14 英寸 G9(配有 Windows 11 Pro)现已上市，售价为 369 美元，搭载 Windows 11 SE 的版本将于 4 月推出。ProBook Fortis 14 英寸 G10 也将于 4 月推出，但 Fortis G10 Chromebook 已经上市，起价为 349 美元。惠普 Pro x360 Fortis G9 现在也可以用 399 美元购买 Windows 11 Pro，Windows 11 SE 型号将于 4 月推出。Pro x360 Fortis G10 也将于 4 月推出。Fortis 11 英寸 G9 Q Chromebook 将于 6 月上市。

除了笔记本电脑，惠普还推出了两个新的独立全高清网络摄像头，惠普 320 FHD 网络摄像头和 325 FHD 网络摄像头。前者将于 2 月推出，售价 39.99 美元，后者将于 3 月推出，售价 39 美元。

接下来是 JP。IK 的 Leap T304 笔记本电脑，采用坚固的设计和旋转的 2MP 网络摄像头，也可用于在笔记本电脑后面拍照。它配有防眩光显示屏、超长电池续航时间和稳定的端口供应。此外，其包装通过了森林管理委员会的认证，来源于可持续森林，使用可回收塑料。J.IK 还宣布了起价为 269 美元的 Turn T303，外加两款 Windows 11 笔记本电脑，即 Slide S301 和 Leap Connect T304。

最后，联想公布了搭载 Windows 11 的新款 10w 平板电脑和 13w Yoga。联想 10w 平板电脑采用高通骁龙 7c 芯片组，提供始终连接的体验。它有一个 10.1 英寸的显示屏，长宽比为 16:10，全高清分辨率，另外它还配备了前置和面向世界的摄像头。它周围有一个橡胶保险杠，以帮助它承受跌落，它使用康宁大猩猩玻璃来保护显示器。

与此同时，联想 13w Yoga 采用 AMD 锐龙 5000 系列处理器，最高可达锐龙 7，这是一款 13 英寸的可转换笔记本电脑，也采用 Windows 11。它有一个全高清网络摄像头和一个可选的 500 万像素全球摄像头，此外它还可以选择 4G LTE 连接，以便在任何地方保持连接。显示屏支持触摸输入，笔记本电脑还可以选配一个带车库的活动笔。联想 13w Yoga 还具有稍后升级的能力，支持高达 16GB 的内存和 512GB 的固态硬盘，使这一设备可以使用很长时间。

新设备将于 4 月份上市，联想 10w 平板电脑包括键盘的起价为 329 美元，而联想 13w Yoga 的起价为 749 美元。联想还宣布了一项名为 Lenovo NetFilter 的新的学校网络过滤服务，并与 VictoryXR 合作，为学生提供新的混合现实学习体验。
