第一组支持 UI 4.0 的好锁模块在这里

原文：https://www.xda-developers.com/the-first-set-of-good-lock-modules-with-one-ui-4-0-support-are-here/#0001-01-01

经过几轮测试，三星终于在昨天为 Galaxy S21 系列推出了基于 Android 12 的稳定 One UI 4.0 更新。如果你有一台 Galaxy S21，并且广泛使用 Good Lock，你可能会想知道你最喜欢的模块是否能在新软件上工作。由于每一个新的 Android 和一个 UI 版本的更新都会带来许多底层代码的变化，大多数好的锁模块都需要更新以确保兼容性。但好消息是，相当多的好锁模块已经更新了对 One UI 4.0 的支持。

正如 Twitter 用户和经常使用三星消息人士 @FrontTron 所发现的，几个好的锁模块现在支持一个 UI 4.0，包括 LockStar、QuickStar、ClockFace、MultiStar 等等。

下面是现在可以在一个 UI 4.0 上完全工作的好锁模块的完整列表:

  • 摇滚明星
  • 快速启动
  • 钟面
  • 多星
  • NavStar
  • 声音助手
  • 凯斯咖啡馆
  • 主题公园
  • 接得好

正如你所看到的，目前一个 UI 4.0 有九个好的锁模块。然而，随着三星更新更多模块，这个列表可能会在未来几周内增长。

对于不知情的人来说，Good Lock 是三星制造的应用程序或模块的集合。Good Lock 超越了大多数 OEM 皮肤提供的标准个性化功能集，并允许深度定制，让用户真正拥有自己的设备。Good Lock 目前仅在 15 个国家提供。然而，三星早在 7 月份就证实了它计划很快将该功能扩展到更多市场。

在相关新闻中，三星的 One UI 4.0 测试程序正在全速前进。上周，三星为 Galaxy S20 系列和 Galaxy Note 20 系列开放了测试程序，允许精选用户在公开发布之前试用 Android 12 的早期版本。
