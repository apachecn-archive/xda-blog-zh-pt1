Amazfit GTR 3 专业版、GTR 3 和 GTS 3 获得新的软件更新

Amazfit 正在为 GTR 3 Pro、GTR 3 和 GTS 3 推出新的更新。此次更新带来了一些新功能，并修复了已知的错误。

10 月，Amazfit 推出了 Amazfit GTR 3、GTR 3 Pro 和 GTS 3，刷新了其智能手表阵容。这三款智能手表都具有很高的性价比，并提供功能丰富的软件体验，可以与 Wear OS 驱动的手表相媲美。发布后不久，Amazfit 推出了新手表的重大软件更新，修复了许多漏洞，并优化了心率传感器。现在，该公司推出了一个新的产品线更新，带来了一些新功能，并完善了整体软件体验。

Amazfit GTR 3 Pro 评测:一款出奇优秀的智能手表

8.26.3.1 的 Amazfit GTR 3 Pro 正在获得固件版本(通过 Tizenhelp )，而 7.30.3.1 的 GTS 3 正在获得固件版本。在变化方面，最新的更新在 Zepp 应用商店中增加了一个 GoPro mini 应用程序，手表表面上的后台应用程序状态指示器，手动校准高度的能力等等。

请注意，在所有三款手表中，只有 GTR 3 Pro 拥有应用商店。因此，GTR 3 和 GTS 3 将无法访问 GoPro mini 应用程序或任何未来添加到 app store 的新应用程序。

以下是完整的更新变更日志:

  • amazfit gtr 3 pro:
    • 添加 GoPro mini app，在 Zepp 的 app store 下载。
  • 在表盘上添加后台应用程序状态指示器
  • 控制中心增加了锁屏功能
  • 添加手动高度校准
  • 提高系统稳定性
  • 修复一些已知的错误

如果你拥有这两款智能手表中的任何一款，请在未来几天留意新的更新。你可以在智能手机上的 Zepp 配套应用中手动检查更新。您也可以通过前往设置>更新来查看智能手表的更新。

Amazfit GTR 3 上个月接受了首次软件更新，带来了一系列变化，包括优化的心率和最大摄氧量算法，以及对音乐播放错误的修复。
