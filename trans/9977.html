Mac 工作室有迅雷 4 吗？

原文：https://www.xda-developers.com/does-mac-studio-have-thunderbolt/#0001-01-01

苹果最近推出了 Mac Studio，作为其最新的桌面平台。它将一个非常强大的苹果 M1 超处理器和高达 128GB 的内存塞进了一个紧凑的机箱，苹果公司表示，它比目前市面上最好的 MAC 电脑还要快。一些人可能会担心，因为它是由苹果硅驱动的，它可能缺乏 Thunderbolt 支持，因为 Thunderbolt 是英特尔技术。谢天谢地，你不用担心，因为 Mac Studio 至少有四个 Thunderbolt 端口。

它是这样工作的:Mac Studio 有两种主要版本:一种采用苹果 M1 Max 处理器，另一种采用苹果 M1 Ultra 处理器。两款机型背面都包括四个 Thunderbolt 4 端口。然而，如果你去与 M1 超版本，你也得到两个雷电 4 端口在前面。在 M1 Max 型号上，这些端口是 10Gbps 速度的标准 USB Type-C。

Mac Studio 还有哪些端口？

除了 Thunderbolt 4 端口，Mac Studio 还配备了稳定的 I/O 供应，因此您可以连接任何想要的东西。背面有两个 USB Type-A 端口、一个 10Gbps 以太网端口、HDMI 和一个支持高阻抗耳机的耳机插孔，因此您可以获得最佳的音频质量。这是在所有型号中可用的四个 Thunderbolt 4 端口之上的。

在前面，我们有两个上面提到的 USB Type-C 端口，如果你选择苹果 M1 超处理器，它也支持 Thunderbolt 4。此外，还有一个 SDXC 读卡器，所以你可以轻松地从相机传输照片和视频等。

由于有大量的端口，Mac Studio 支持的外部显示器数量是所有 Mac 中最多的——总共五个。您可以使用 USB Type-C 端口连接多达四个 6K 分辨率的 Apple Pro Display XDRs，并通过 HDMI 连接第五个 4K 显示器。这么小的机器推动了这么多像素。

你不应该期望它能够使用外部 GPU，因为 Apple Silicon 不能很好地与为 x86 架构设计的 GPU 一起工作。无论如何，你都不应该需要外部 GPU，因为苹果公司声称苹果 M1 Ultra 的集成 GPU 可以媲美 GeForce RTX 3090 的性能，至少在“选择行业标准基准”方面。这是目前桌面电脑最强大的 GPU，所以你已经得到了足够的能力。

如果你想买 Mac Studio，你必须准备好花一大笔钱，尤其是如果你想要前面的两个 Thunderbolt 端口。M1 Max 处理器的起价为 1999 美元，M1 Ultra 的起价为 3999 美元。这还不包括显示器或任何外围设备，所以你至少要多花几百美元。如果你选择苹果的 Studio Display，与 Mac Studio 一起发布，则需要额外支付 1599 美元。值得庆幸的是，还有其他很棒的显示器可以和 Mac 一起使用。

如果你是第一次接触苹果电脑，看看你需要知道的关于苹果操作系统最新版本的一切。或者，如果你正在从基于英特尔的 Mac 电脑升级，你可能也想看看苹果硅 Mac 电脑的最佳应用。

苹果 Mac 工作室

新的 Mac Studio 配备了超级强大的苹果 M1 超芯片组、大量端口和时尚紧凑的设计。M1 超型号有六个雷电端口，而 M1 最大版本有四个。

苹果工作室显示器

Studio 显示器是苹果公司 Mac Studio 的伴侣，提供 5K 分辨率和 1200 万摄像头的高端视觉体验。
