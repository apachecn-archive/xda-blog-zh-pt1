Khronos 集团发布 Vulkan Graphics API v1.3

原文：https://www.xda-developers.com/vulkan-graphics-api-v1-3-announced-by-khronos-group/#0001-01-01

2016 年 2 月，Khronos 集团发布了 Vulkan 图形 API 的首个版本。Vulkan 本质上被设计成 OpenGL 的一个继承者，但是增加了有效利用多核处理器的能力。这些年来还有一些其他的改进，我们日常使用的大多数主要操作系统，如 Android 12 和 Windows 11 ，都支持这个 API。1.1 版本发布于 2018 年 3 月，1.2 版本于 2020 年 1 月发布，现在 Khronos 正式发布了 1.3 版本。Vulkan 1.3 中添加的特性都不是可选的，这确保了该 API 版本的所有实现的一致性。

首先也是最重要的，Android 世界的一个大的更新是 Android Baseline Profile 2021 的引入。这将允许开发人员轻松了解设备在 Vulkan 渲染方面的能力，并可用于“宣传 Android 生态系统中大多数活跃设备支持的超过 Vulkan 1.0 的功能集，包括不支持和不定期接收驱动程序更新的设备。”谷歌表示，绝大多数设备不需要 OTA 更新来支持该配置文件。

同时，该公司宣布了 2022 年的 Vulkan 路线图。这是 Vulkan 路线图中第一个明确的里程碑。所有积极为智能手机、平板电脑、笔记本电脑、控制台和桌面平台开发中高端设备的 Vulkan 工作组硬件供应商都致力于支持这一里程碑，从 2022 年的几款产品开始。这个里程碑要求支持 Vulkan 1.3 以及一些被认为对目标市场至关重要的扩展。

与先前版本的 Vulkan 规范一样，1.3 版旨在 OpenGL ES 3.1 级硬件上加速，本质上意味着它应该受到所有支持旧版本的 GPU 的支持。

AMD 已经宣布在所有 AMD 镭龙 RX Vega 系列和 AMD RDNA 架构显卡上支持 Vulkan 1.3 和 2022 年路线图。ARM 已经宣布支持其 Mali GPUs，Google 已经宣布支持 Stadia，NVIDIA 也在 Linux、Windows 10 和 Windows 11 上提供即时支持。

Vulkan 工作组将于 2022 年 2 月 1 日举办“Vulkan 化网络研讨会”，该会议将用于提供有关 Vulkan 1.3、API 路线图和配置文件的更多深入信息。
