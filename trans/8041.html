Windows 11 回顾:修复过去十年的错误

原文：https://www.xda-developers.com/windows-11-review/#0001-01-01

经过三个月的预览版， Windows 11 终于面世了。有很多是新的。它有着多年来最大的视觉革新，然后它有了新的功能，比如小部件、团队聊天、快照布局等等。

现在的问题是，你应该升级你的电脑吗？我的答案将是一个艰难的是。在我看来，Windows 11 是一个赢家，微软终于抛弃了一些已经存在了大约十年的糟糕的用户界面选择。

但是能不能升级到 Windows 11？

好吧，让我们把这件事说清楚，因为它是房间里的大象，而且很重要。虽然 Windows 11 是 Windows 10 的免费升级，但它确实大大提高了硬件要求。使用 Windows 10 的很多人都没有资格升级。

变化最大的是 CPU 要求。您需要下列之一:

  • 英特尔第八代或更新版本
  • 英特尔酷睿 i7-7820HQ(这是唯一支持的主流第七代芯片，也是 Surface Studio 2 中的芯片)
  • AMD Zen 2 或更新版本
  • 高通骁龙 850 或更新

还有其他要求，比如最低 4GB 内存和 64GB 存储，这两项都是 Windows 10 要求的两倍。另一个大的是 TPM 2.0 然而，自 2016 年以来，这一直是新电脑的要求，所有受支持的 CPU 都在那之后开始发货。换句话说，如果你有一个受支持的 CPU，你也应该有 TPM 2.0。

我理解很多硬件不受支持的人对此感到不安，但公平地说，这是 Windows 7 问世以来第一次提出要求。尽管如此，Windows 10 将被支持到 2025 年 10 月。

如果你还在运行一台五年(或更老)的 PC， Windows 11 很可能值得为买一台新设备。

Windows 11 重新设计了 OOBE

  • Cortana 不见了
  • 它更漂亮
  • 你终于可以给你的电脑命名了

叮咚，女巫死了。我说的女巫是指 Cortana。不，我不讨厌 Cortana，但我讨厌数字助理成为开箱即用体验的一部分。这是不和谐的，坦率地说，这是可怕的。此外，如果有人抱怨，微软表现得好像他们是某种讨厌有特殊需求的人的恶棍。Windows 11 一直以它应该有的方式运行。如果您在一定时间内没有与 OOBE 互动，它会发出声音询问您是否需要帮助。

整个 OOBE 也被重新设计了，看起来漂亮多了。它现在有一个白色的背景，按钮上有圆角，它只是更图形化。你可能还记得，当 Windows 10 首次推出时，它只有这些扁平的蓝色屏幕，后来它被重新设计成更深的蓝色和更多的窗口。现在，好多了。

另一件要注意的事情是，你终于可以将你的电脑命名为 OOBE，这是 Windows 8 OOBE 中的一个东西，在 Windows 10 中被删除了。删除它可能看起来很疯狂，但你必须从构建操作系统的角度来考虑这个问题。当创建 OOBE 时，你倾向于从一定数量的屏幕或步骤开始，让用户体验，事情得到优先处理。但是能够再次命名一台 PC 真的很好。对于 Windows 10，你必须进入设置才能完成，然后你仍然需要重启。

全新的开始菜单、圆角和全新的 Windows 11 设计

  • 这是全新的设计
  • 活的瓷砖是死的
  • 东西居中
  • 像开始按钮这样的东西不是静止的

一旦你获得了开箱即用的体验，你会发现 Windows 看起来完全不同。在我看来，这是新 OS 最大的特点。我一直认为 Windows 8 和 Windows 10 很丑，我说的不仅仅是 Windows 8 中的糟糕想法。我说的是整体的设计语言，从亮色到尖角。

Windows 11 用悦耳的声音取代了刺耳的系统声音。

我不是设计专家；事实上，我不是那种会注意到设计中微小变化的人。我也不是那种你应该问什么东西看起来好不好的人。但对我来说，有两种流行的设计。一类是我称之为经典的，另一类是我称之为新潮的。对于一个新潮的设计，那是我们在十年或二十年后回顾的东西，想知道我们在想什么，即使它在当时看起来很好。经典设计是指即使过时了，仍然美丽的东西。

Windows 8 和 Windows 10 拥有时尚的设计。我觉得 Windows 11 更经典。它用圆角取代了尖角，这样一来，操作系统使用起来就舒服多了。刺耳的系统声音消失了，取而代之的是柔和的声音。

当然，还有更多。有了全新的开始菜单，Live Tiles 终于没有了。不记得活瓷砖了？我并不感到惊讶，即使你使用的是支持它们的操作系统。Live Tiles 起源于 Windows Phone，是一种在快捷方式中查看通知和其他应用信息的方式；它类似于一个小部件。不幸的是，它们在 Windows 上并不实用，我会从 Windows 10 的开始菜单中移除所有的磁贴。

“开始”菜单仍然显示我仍然讨厌的固定应用程序，以及推荐的应用程序和文档。我讨厌固定应用的原因是我用了太多的电脑。我没有时间去解开微软认为我应该使用的所有废话，并钉上我实际使用的东西。我只是把我想要的东西钉在任务栏上。令人沮丧的是，需要额外点击才能进入所有应用列表。

Windows 11 的视觉、听觉甚至触觉都更加愉悦。

任务栏和开始菜单也是居中的。这是一个很大的变化，似乎有点两极分化。我喜欢它，在 Windows 10 上使用过 TaskbarX 这样的实用程序来居中任务栏；然而，TaskbarX 将开始按钮留在了角落里。毕竟，这是有道理的。这是肌肉记忆，如果你把指针一直移到角落，你会得到开始菜单。几十年来一直如此。即使在 Windows 8 中，当开始按钮被臭名昭著地取消时，将鼠标移动到屏幕的左下角并点击会将你带到开始屏幕。

在 Windows 11 中，不仅按钮第一次移动了，而且它甚至没有停留在同一个位置。这就是当事物处于中心时的问题。当更多的图标被添加到任务栏时，开始按钮会进一步向左移动。

无论如何，这是对事物被居中的一大抱怨。个人觉得还好。我对事物的现状没有任何异议。

让我这么说吧。自从 6 月 24 日发布之前 Windows 11 的版本泄露以来，我几乎没有用过 Windows 10。每次我得到一个运行 Windows 10 的评测单元，都感觉是一种倒退。我在 Windows 10 上运行我的基准测试，然后立即安装 Windows 11。说真的，在过去的三个月里，我几乎没有用过 Windows 10。

捕捉布局和捕捉组

  • 现在，您可以在不同的布局中选择捕捉位置
  • 您可以恢复多组已捕捉的窗口

抛开设计不谈，我最喜欢的 Windows 11 功能变化叫做 Snap Layouts 。十多年来，Windows 一直在抢购。这就是你把一个窗口拖到屏幕的一边，它就会吸附到一边。您还可以捕捉到一个四分之一以将应用程序保持在一个象限中。Windows 10 引入了 Snap Assist，在你抓取一个应用程序后，它可以自动让你在屏幕的其余部分选择第二个应用程序。

Snap Layouts 将此提升到了一个新的层次。如果你悬停在一个窗口的最大化按钮上(或者点击 Win + Z)，它会显示一系列窗口布局，你可以选择你想要的。您还将选择您希望该应用程序出现在布局中的哪个窗口。非常棒。

Snap 布局最终让超宽屏幕成为一种享受。

超宽显示器也让生活变得更加美好。以前公司会送我超宽的 PC，我受不了，因为 Windows 10 真的不是为它做的。有了两个 16:9 显示器，我可以在每个屏幕上并排显示两个应用程序，总共四个应用程序。对于超宽，没有简单的方法可以做到这一点。Snap Layouts 现在可以让你同时拥有三个应用。我刚刚检查了惠普 Envy 34 一体机，当我把它送回来时，这是我第一次真正错过超宽一体机。

Snap Layouts 最初是 PowerToys 的一部分，PowerToys 是一个在 Windows 10 上运行的实用程序。不过，很高兴看到它进入了操作系统。

Snap Groups 是一个小的补充，但它仍然非常有用。当您最小化窗口时，您可以将鼠标悬停在图标上，在窗口预览中，您将看到每个窗口以及窗口组。以前，如果您有两个并排拍摄的应用程序并将其最小化，您必须分别恢复它们。现在，有一个选项可以打开你所有的窗口。

  • 小工具取代了新闻和兴趣
  • 他们只是暂时的第一党

你将在任务栏中看到一个新的 Widgets 图标，点击它将显示一系列第一方的 Widgets。不像过去的小工具，你不能把它们固定在桌面上。他们将呆在自己的小天地里。

它们取代了新闻和兴趣，这实际上是 Windows 10 中相当新的功能。尽管如此，Widgets 提供了比新闻提要更多的选择。在 Windows 11 中，您还可以添加 OneDrive 照片、Outlook 日历、Microsoft 待办事项、家庭安全等内容。它还带有你习惯喜欢的天气和运动的部分。

您可以四处移动它们、调整它们的大小、自定义它们等等。然而，Windows 11 Widgets 目前仅限于第一方应用和服务。这在未来会改变，但奇怪的是，微软并没有证实这一点。这也很遗憾，因为有很多第三方服务是我希望看到的。例如，用谷歌日历小工具代替 Outlook 日历，谷歌照片也是如此。但是，当然，这将我们带入另一个讨论，即谷歌是否真的会费心去做这些。

团队聊天将服务放在操作系统中，但它需要一些认真的工作

  • Windows 11 内置了微软团队
  • 你可以用它来发送短信
  • 你仍然需要完整的团队应用程序来开会
  • 微软需要解决一些大的棘手问题

微软宣布的 Windows 11 的一大特色是团队整合。叫做组队聊天，并不是 Windows 11 中我最喜欢的部分。这里有好有坏，但首先，有一点背景。

微软以前也这样做过。Windows 10 的第二个版本 1511 版将 Skype 集成到了操作系统中。它将 Skype 拆分为 Skype 视频、消息和电话应用。到下一次功能更新 1607 版时，微软已经推出了 Skype 应用程序的新 UWP 版本，取消了预览版中显示的功能，如 Messaging Everywhere(你认为我会在 Windows 11 的评论中提到 Messaging Everywhere 吗？我当然没有)。没过多久，Skype 在操作系统中的集成就被取消了，人们普遍认为这一直是个糟糕的想法。

所以我们在这里，和团队聊天。好消息是，开始时几乎不需要做什么工作。您可以向任何 Skype 联系人发送消息，也可以向任何人发送短信。尽管短信变得有点奇怪。它会向收件人发送一个在他们的手机上安装团队的链接，这样他们就可以选择安装或者只是回复短信。从我的经验来看，对于接受者来说，这有点令人困惑。

令人困惑的还不止这些。Teams chat 附带了一个微软团队应用程序，但如果你想点击链接加入微软团队会议，它不是你需要的那个。为此你需要标准的桌面应用程序。之后，在你的开始菜单中有两个几乎相同的应用程序，叫做微软团队。有点乱。

团队聊天显示了潜力，但是短信和会议导致了混乱的体验。

通知也不好。就我个人而言，我会关闭 Windows 中的通知功能。这比处理那些开始把通知硬塞给你的乱七八糟的应用程序要容易得多，因为不像 iOS 和 Android 这样的现代操作系统，Windows 不会在应用程序向你发送通知之前征求你的许可。每当我安装一台新电脑时，我都会在商店中检查更新，每次更新完成时，它都会发出通知。这是我关闭通知的通知。

我也很少发现通知有价值，这也是我很容易完全关闭它们的原因。我发现更有价值的是显示我有未检查通知的小徽章，比如一个消息应用程序的小“1”显示我有一封未读邮件。不幸的是，在团队聊天中，这个数字不会显示出来。测试时，我的大多数对话在每条消息之间有几个小时。我最后打开了通知，我已经讨厌它了。

这里有很大的潜力，但实际上，微软必须解决这个问题。拥有两个团队应用程序的问题是 2015 年 Skype 集成的直接问题。您可以使用内置应用程序做很多事情，但您仍然需要完整的 Skype 应用程序来获得完整的体验。令人困惑的短信问题也是一个痛点，因为它更像是一种将收件人拉进团队的方式，而不是给发件人一个适当的信息体验。

针对 Windows 11 和 Windows 10 有一个全新的微软商店

  • 新的微软商店接受所有的应用程序
  • 它还提供不同的店面
  • 你终于可以通过商店获得其他浏览器了
  • 未兑现的承诺包括新的娱乐板块和安卓应用

微软展示了一个全新的 Windows 11 商店，尽管事实证明，它也将进入 Windows 10。首先，它是完全重新设计的，这是你会在很多收件箱应用中听到的。但重新设计并不是大新闻。

更大的新闻是，微软正在接受所有在 Windows 上运行的应用程序进入微软商店。现在，又到了上历史课的时候了，因为事实证明，很多 Windows 11 仍然是关于修复 Windows 8 中诞生的错误。Windows 商店诞生于 Windows 8，问题是你必须重写你的应用程序才能进入商店。没有人会这么做，这个想法失败了。

然后是 Windows 10，微软承诺四桥。Project Westminster 让你将 web 应用打包成 UWP 应用，Project Astoria 承诺在 Windows 上运行 Android 应用，Project Islandwood 让你为 Windows 重新编译 iOS 应用，Project Centennial 让你将 Win32 应用打包到商店中。Astoria 从未在 preview 之外发货，Islandwood 从未起飞，因为微软真的跟不上 Swift。虽然桌面桥是关键，但更复杂的应用程序仍然没有进入商店。

现在，微软彻底放松了警惕。应用程序不再需要打包，对于那些应用程序，你可能甚至不会通过微软商店获得更新。有了 Windows 11，微软商店真的只是你要去的地方的一个门户。

第三方应用商店也是如此，它们现在占据了第一排的位置。宣布的第一个合作伙伴是亚马逊，现在已经推迟了。事实上，Windows 11 宣布的功能之一是 Android 应用支持，但除了微软之外，还没有人涉足这一领域。Windows 11 上的 Android 应用程序支持很快将向内部人士提供，但与此同时，另一个第三方应用程序商店合作伙伴将是 Epic Games Store。

你也可以从微软商店获得第三方浏览器，比如 Opera。微软过去对商店中的浏览器有类似于苹果的规则。他们很好，但是他们必须使用平台引擎，也就是 EdgeHTML。当 Edge 重建时，计划是将该政策改为 Chromium，但微软再次放松了警惕。如果你喜欢一个浏览器，厂商应该可以把它放在微软商店里。

最后，承诺的另一件事是提供不同服务内容的娱乐标签。不过，那封信从未寄出。它仍然只是微软电影和电视的购买和租赁。

Windows 11 带有全新的收件箱应用程序

  • 许多应用程序在视觉上有了很大的变化，有些还带来了新功能

计算器

仍然开源，微软说计算器应用程序已经用 C#重写。正如你对 Windows 11 重新设计的预期，按钮有圆角。你可以把它用作科学计算器，图形计算器，有一个编程模式，它可以在 100 个单位和货币之间转换。

具有焦点会话的时钟

时钟应用程序有一个叫做焦点会话的功能，这是非常重要的。这不仅仅是对新操作系统的重新设计。这是一种可定制的方式来设置时间段，让你专注于事情。

首先，有一个计时器。这可以让你设置你想要的会议时长，它会根据时长自动给你休息时间。还有一个“跳过休息”复选框。接下来，有一个仪表板显示每天的记录，你今天已经设置了多少小时的焦点会议，昨天多少小时，等等。你也可以设定一个每日目标。

这就是真正酷的地方。Focus Sessions 附带了 Microsoft To Do 和 Spotify 集成。这意味着你可以设置在会议期间播放的音乐，通过微软的待办事项，你可以选择特定的工作任务。

真正让我恼火的是，这不是一个新问题。自从微软终止了其内部 Groove 音乐服务后，Spotify 被认为是 Windows 用户的首选音乐流媒体服务。我理解做集成，因为这是微软的服务，但偏爱第三方服务令人沮丧。如果你看看像微软商店这样的东西，一个大的主题是让第三方开发者在 Windows 上拥有前排座位，所以也许我们应该停止在整个微软应用程序中展示 Spotify。

除此之外，这个功能真的很酷。我总是在寻找新的和创造性的方法来提高效率。

照片

在整个 Windows 10 时代，众所周知照片应用不好。随着 Windows 11 的推出，它得到了适当的重新设计。

首先，当你查看一张照片时，你会在底部看到一个电影胶片，文件夹中下一张和上一张照片的缩略图。显然，你可以随心所欲地点击这些，但这感觉像是 Windows 10 应用程序的一大进步。在顶部，有一个工具栏，你可以在那里找到常规的选项，如旋转、废纸篓、墨迹等等。

新的照片应用有一些基本的照片编辑工具，超级酷。这里的目标不是成为每个人的一切，而是，微软希望这是一个做它擅长的事情的应用程序。

我先讨论了个人照片视图，因为坦白地说，这是我启动照片的方式。我从来不会进入开始菜单点击照片。我在文件浏览器中双击一张图片，这就是我进入应用程序的方式。然而，如果你自己启动应用程序，你会看到一个新的主屏幕，有相册，人物和文件夹选项卡。

这真的是图片底部的丝带帮我做到的。这是一个很好的简单的滚动相机胶卷的方式，这是那些你会在其他平台上看到的功能之一，并且想知道为什么它不在 Windows 上。现在，它是。

颜料

微软画图正在进行很长时间以来的第一次大检修。事实上，就在不久前，Microsoft Paint 被弃用，这意味着它不再被开发，有朝一日将从操作系统中删除。在受到众多用户的强烈反对后，Redmond 公司承诺将其放入微软商店。但这并没有发生，因为该公司最终取消了 Paint 3D。

新的 Paint 应用程序仍然主要是一个重新设计。在启动新应用程序时，你会发现方形调色板现在是圆形的，有圆角，而且它更适合 Windows 11 的设计。很高兴看到这一点，因为我们已经看到了 Windows 8 和 Windows 10 中许多重大的设计变化，看着旧的、心爱的系统应用被遗忘令人沮丧。

你现在可能已经注意到，在某些应用的方向上摇摆不定是微软的一个主题。 Snipping 工具就是其中之一。该应用程序一度被官方弃用，并承诺将被 Snip & Sketch 取代。现在，我们回到 Snipping Tool，这是一个完全重建的应用程序。

这是一个令人兴奋的使用 Windows 的时代。

虽然它经过了重新设计，支持黑暗模式，但截图的方法没有什么不同。你可以选择画一个矩形来截取，截屏一个窗口，截屏整个屏幕，或者抓取屏幕的一个自由形式的部分。

你还会发现新的墨迹和编辑工具，它们存在于 Snip & Sketch 中，但不存在于旧的 Snipping 工具中。本质上，它充分利用了旧的剪贴工具和新的剪贴工具。

触摸键盘更漂亮

  • 触摸键盘支持主题

微软在 Windows 11 的触摸键盘中引入了一些快捷键。其中一个方法是主题。你可以选择任何预先制作的主题，或者你可以上传你自己的图片。

这是一种很好的触摸，它使触摸键盘更容易使用。对我来说，这是 Windows 中一直缺少的东西之一。有时，Windows 中的东西很好，但当你看到它们在其他平台上的表现时，就值得思考为什么微软没有做得更好。事实是，人们在 PC 上使用触摸的次数不如在智能手机或平板电脑上多，所以它被忽视了。当它被忽视时，人们不会使用它。这是一个真正的先有鸡还是先有蛋的问题。

Windows 11 给了 ARM 上的 Windows 急需的爱

  • x64 仿真终于来了
  • ARM 上的 Windows 现在有 Hyper-V

ARM 上的 Windows 11 有一些重大改进，这是旧闻了。这真是旧闻了。自 2020 年以来，Windows Insiders 一直在测试 x64 emulation 和 Hyper-V 对 ARM 上 Windows 的支持。它们最初将在 Windows 10 版本 21H1 中发布，当时那将是一个重大更新。然后，Windows 10X 将抢尽风头，因此 21H1 更新成为一个支持包，这些功能被推迟到秋天。Windows 10X 被废弃，Windows 11 带来了视觉上的大变革。

不过，对于 ARM 上的 Windows 来说，这是一个有价值的更新。ARM 设备已经能够运行原生 ARM 或 ARM64 应用，以及仿真 x86 应用。运行 x64 应用程序的能力一直是等式中缺失的一部分，特别是自从苹果推出 ARM PCs 以来，这些 PC 只能运行你使用的所有应用程序。

现在，软件会在那里。硬件将是一个值得关注的有趣领域。

结论:Windows 11 是一次值得的升级

对我来说，Windows 11 是一股新鲜空气。当然，它并不完美。如果“开始”菜单可以更加定制，而不是包含我不会使用的快捷方式和推荐应用，我会很高兴。例如，如果所有应用程序列表、快捷方式和小工具都合并成一个，那就太酷了。

Windows 11 是一股新鲜空气。

我从来没有真正喜欢过 Windows 10 的设计，或者 Windows 8 的设计。我是 Windows 用户，因为我喜欢 Windows 的功能。我喜欢我可以整天工作，当我完成时，我可以从我停止玩的地方继续玩 Forza Horizon 4 (很快 Forza Horizon 5 ！)，都在同一个硬件上。我不是一个狂热的游戏玩家，但有了 Xbox Game Pass 这样的廉价订阅，我可以在方便的时候随意玩各种游戏。

Windows 有我想要的应用程序。它允许我做我想做的事情。它有多种多样的外形规格，可以满足各种使用案例的需求。事实上，如果有一件事你可以在电脑上做，你也可以在 Windows 上做。这就是我喜欢这个操作系统的原因，也是为什么我在过去十年里一直使用它，尽管我并不真正关心它的设计。

但是现在，我喜欢这个设计。Windows 11 用起来感觉还不错。感官上更柔和。看起来更好，听起来更好，甚至感觉更好。对于 Windows 来说，这是一个激动人心的时刻。当然，有很多很棒的新功能，例如 Snap Assist，但最终，它只是更容易使用。

一旦你升级了，而且你完全应该升级，你可能会马上感觉到。如果你不这样做，我建议你在一周后回滚你的电脑。到那时，我保证你会感觉到使用 Windows 10 有多不和谐。
