Windows 11 内部人员现在可以尝试新的媒体播放器应用程序

原文：https://www.xda-developers.com/windows-11-media-player-available-insiders/#0001-01-01

微软推出一款新的媒体播放器应用，面向 Windows 11 内部注册开发者频道的用户。该公司最初在 Windows Insider 网络广播中展示了这款应用程序，然后在再次发布之前迅速隐藏起来。现在，它终于可用了，它的版本号是 11.2110.34.0。此应用需要 Windows 11 build 22471 或更高版本才能安装，因此如果您尝试在旧版本上加载它，它将无法工作。

新的媒体播放器应用程序是 Groove Music 应用程序的替代品，Groove Music 应用程序是微软在 Windows 10 和 11 中的默认音乐播放器。事实上，当您进行切换时，您的 Groove 音乐库将导入到新的媒体播放器应用程序中。除了音乐，新的媒体播放器应用程序还可以播放你的视频文件。它可以浏览你的音乐和视频库，创建播放列表，以及你所期望的一切。默认情况下，只包括 Windows 中默认的音乐和视频文件夹，但你可以向应用程序添加更多文件夹，以便显示来自那里的项目。

对于音乐，媒体播放器应用程序将在播放期间向您显示专辑封面和艺术家的图像，当您切换到迷你浮动窗口模式时，您也可以看到这些图像。当然，对于视频，您将在播放过程中看到实际的视频，并且迷你窗口模式允许您在做其他事情的同时继续观看。这里没有什么会让你惊讶的，它拥有你所期望的所有特性。但这是一种打开音乐和视频文件的新方式。

在这个新版本中，应用程序的界面有所不同。播放按钮现在是彩色的，并使用了类似于应用程序本身图标的渐变，所有图标都经过了调整，以匹配 Windows 11 的风格。这同样适用于整个应用程序的按钮和标题栏，标题栏现在也包括应用程序图标。

尽管这款新应用明显支持经典的 Windows Media Player，但它不会立即取代后者。Windows Media Player 已经有几年没有真正被支持了，但是你还是可以在 Windows 工具下找到。这是不会改变的，所以如果你因为任何原因仍然喜欢它，你可以使用它。

在 Dev 频道测试 Windows 11 的 Windows 内部人员应该会在微软商店获得新的媒体播放器应用程序作为更新。正如我们所提到的，它不能被非内部人员加载，因为它需要 Windows 11 build 22471 或更高版本才能运行。
