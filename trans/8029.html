Windows 11 深度探索:查看新的设置应用程序

原文：https://www.xda-developers.com/windows-11-settings/#0001-01-01

经过六年多的 Windows 10，我们终于在去年获得了 Windows 11 的重大更新。新的操作系统有很多新的东西，虽然它已经发布了一段时间，但对很多人来说可能还是新鲜的。这一次，我们来看看新的设置应用程序，这是 Windows 11 最重要的重新设计部分之一。

这个应用程序有一个全新的外观，一打开它就变得很明显。不再有简单的黑色或白色背景，为新的半透明云母材料让路。你不再会看到一个全屏的菜单，取而代之的是应用程序直接打开到系统部分，并有一个侧边栏用于访问其他部分。另外，到处都使用彩色图标，而不仅仅是跟随你的系统主题的平面图标。此外，不仅该应用程序针对 Windows 11 进行了重新设计，甚至在 Windows 11 2022 更新(版本 22H2) 中也做了更多的改变。让我们仔细看看这个应用程序及其各个部分。

浏览这篇文章:

系统

每当你在 Windows 11 中打开设置应用程序时，系统部分都是登录页面，正如名称所示，它是你访问核心系统功能设置的地方。这包括显示、音频和通知设置等等。

应用程序窗口的标题通常会显示一些主要的快捷键和控制，每个部分都会有所不同。在这种情况下，您可以看到笔记本电脑的名称、Microsoft 365 订阅的状态、OneDrive 文件夹的快捷方式以及 Windows Update 的状态。在侧面，所有不同的部分都可用，应用程序窗口的核心区域显示您当前部分的页面。让我们来看看它们。

显示

此页面可让您管理与 Windows 11 电脑显示器相关的各种设置。在顶部，您可以通过设置每个显示器之间的相对位置来调整多显示器设置的布局，或者选择是否在显示器之间重复显示或扩展以获得更多空间。通过扩展多显示器选项，您还可以选择如果应用所在的显示器断开连接，应用是否应该最小化，以及是否应该根据显示器连接记住窗口位置。您也可以从这里连接到无线显示器。

继续向下滚动，您可以调整显示器的亮度和夜间照明设置。该功能使屏幕具有黄色(或更极端的红色)色调，使您的眼睛在夜间或黑暗环境中更容易看到。您可以安排夜灯在特定时间打开，或根据您的位置跟随当前的日落和日出时间，或者您可以让它永久打开或关闭。如果您的显示器支持，您也可以更改 HDR 设置。

更进一步，你可以改变 Windows 11 的设置，如屏幕上文本和图标的比例，使它们变得更小或更大。可用选项取决于您的屏幕大小和分辨率。这可以用来给你更多的屏幕空间，或者在东西太小的情况下让项目更明显。您还可以调整显示分辨率和方向。

最后一部分包含高级显示设置，在这里您可以看到有关显示器的更多详细信息，如分辨率、刷新率、位深度以及它是否支持 HDR。您还可以在这里更改刷新率，如果您有一个高刷新率的显示器，这是一个重要的功能。最后，图形页面是您可以设置特定应用程序在特定 GPU 上运行的地方，如果您的 PC 有多个 GPU 的话。

声音

接下来，Windows 11 中有声音设置，这些更简单一些。第一部分指的是输出设备，因此您可以调整扬声器的设置。您可以更改默认输出设备、调整音量或禁用立体声。您还可以进入扬声器的高级设置，调整左/右音频平衡、音频质量，并启用 Dolby Atmos 等空间音频功能。

下一个部分是你的输入设备，比如麦克风。就像输出一样，您可以更改默认输入设备、调整音量，如果您有无线麦克风，还可以与新设备配对。你也可以测试你的麦克风——只要麦克风接收到声音，音量条就会显示动画。

最后一部分包含音频问题的疑难解答，以及对所有音频设备列表的访问。最值得注意的是，你还可以从这里访问音量混合器，这样你就可以改变单个应用程序的音量。

通知

下一页包含您的 Windows 11 通知设置，这是 Windows 11 2022 更新中显著重新设计的页面之一。在这里，您可以打开或关闭通知，禁用通知声音，或者更改通知是否应该在锁定屏幕上可见。还可以启用免打扰，选择何时应该自动启用，选择某些可以突破的 app。

在这些适用于所有 Windows 11 的通用设置下，您可以更改每个应用程序的设置。您可以停用特定应用程序的通知，或更改通知的行为方式，例如它们是否应该使用声音、显示横幅或在通知中心中优先显示。只有以前发送过通知的应用程序才会显示在这里，因此如果您没有立即看到所有应用程序，请不要感到惊讶。

焦点

焦点是 Windows 11 2022 更新中新的一页，都是关于 Windows 11 中新增的焦点会话。在最初的版本中，这些功能大多只是内置在时钟应用程序中，但现在，它们被更深入地融入到了操作系统中。您可以在此选择持续时间并开始专题讨论，还可以更改一些设置。

例如，您可以选择在专题讨论会期间是否打开“请勿打扰”,但您还可以更进一步。您可以停用应用程序的任务栏闪烁(这样它们就不会变成橙色来显示通知)，停用通知徽章，并选择是否要在时钟应用程序中查看焦点会话的计时器。

电源和电池

这一部分可以让你看看你当前和过去的电池电量，这样你就可以知道你的笔记本电脑充电后通常可以使用多长时间。在这里，你还可以改变 Windows 11 的各种电源设置，比如你的电脑在闲置一段时间后什么时候应该关闭屏幕，什么时候应该进入睡眠状态。您还可以更改电源模式来提高性能或优化电池寿命。

您也可以在此处启用电池节电，它会禁用后台任务和一些通知，以帮助您的电池持续更长时间。可以设置何时自动开启(默认情况下，当电池剩余 20%时会启用)，以及当电池节电模式开启时，屏幕亮度是否也应降低。如果你用的是台式电脑，电池相关的设置自然不会出现。

储存；储备

在这一部分，您可以看到存储的当前状态，包括您还有多少可用空间，以及有多少被什么占用。有应用程序、临时文件以及文档、视频和 OneDrive 文件等文件夹的类别。每个文件夹和应用程序列表都有快捷方式，因此您可以更具体地管理哪些内容占用了空间。

在存储使用率指示器下方，有一些附加设置。首先是存储感知，它会定期自动清理某些类型的数据。你可以设置它在一段时间后删除回收站中的内容，以及下载或下载的 OneDrive 文件(如果你有一段时间没有使用它们)。有一个关于清理建议的页面，向您显示可以删除以节省空间的文件。

您也可以通过点击临时文件页面手动删除某些不必要的文件。这包括旧的 Windows Update 文件、清空回收站、缓存缩略图等。你的下载文件夹也在这里，所以你需要小心不要意外删除任何东西。

最后，还有高级存储设置。这些功能包括查看有关其他硬盘的信息(主页将仅显示您的 Windows 硬盘)、设置新内容的保存位置(如果您有多个硬盘)、创建和管理存储空间、管理您的磁盘和卷(分区)、硬盘优化(硬盘碎片整理程序)以及数据备份选项。

附近共享

附近共享是 Windows 11 中引入的一项功能，它允许您通过本地无线连接与您附近的 PC 共享文件。在此页面上，您可以设置是否希望您附近的任何设备能够发现您的电脑进行共享(前提是只有您自己的设备能够发现)，或者完全禁用附近共享。您还可以设定在通过“附近共享”接收文件时想要存储文件的位置。

多任务处理

这一部分包括与如何管理 Windows 11 中同时打开的多个应用程序相关的设置，包括像 Snap Layouts 这样的新功能。你可以启用或禁用许多功能，例如 Windows 11 是否应该向你显示当前应用旁边的其他应用，或者当你将鼠标悬停在窗口控制中的最大化按钮上时，你是否希望看到快照布局选项。还有一些与虚拟桌面相关的设置，特别是你是否希望其他桌面的应用程序在其他桌面的任务栏上可见，以及何时使用 Alt + Tab 切换应用程序。

如果你使用微软 Edge，你还可以选择是否在 Alt + Tab 界面显示单个标签。最后，您可以启用或禁用标题栏窗口摇动手势，在屏幕上摇动窗口可以最小化所有其他窗口。就功能而言，这个页面自 Windows 11 首次亮相以来基本相同，但在 2022 年的更新中进行了视觉上的调整，所以看起来有点不同。

激活

在此页面中，您可以检查 Windows 11 许可证的激活状态，以及您运行的版本。您也可以输入不同版本的产品密钥，或者从 Microsoft Store 购买升级。

充当故障检修员

此页面允许您查找可能遇到问题的疑难解答步骤，以及更改推荐的疑难解答程序的设置。Windows 11 有时会建议您运行故障诊断程序，如果它检测到某些东西没有按预期工作，在这里您可以选择故障诊断程序是否应该自动运行和/或首先请求用户许可。您还可以在您的电脑上查看推荐的疑难解答的历史记录。

不过，这一部分最值得注意的部分是其他故障诊断程序页面，在这里你可以找到 Windows 11 中包含的所有故障诊断程序，看看它们是否可以帮助解决你遇到的特定问题。有针对互联网连接、音频、打印机等的疑难解答。

恢复

如果您想从故障诊断程序无法解决的问题中恢复，这是您应该去的地方。如果你的操作系统不能正常工作，这个页面会给你一些选择。你可以重置你的电脑以删除所有个人文件和应用，如果你打算将电脑出售给其他人，这也很有用。如果你感兴趣的话，我们有一个关于如何重置 Windows 11 的指南。

您也可以选择回到以前的 Windows 版本，但此选项通常仅在 Windows 重大更新(如从 Windows 10 升级到 Windows 11)后的 10 天内可用。最后，高级启动允许您重新启动电脑以访问高级选项，如选择不同的启动设备。这对于执行 Windows 11 的全新安装非常有用。

投影到这台电脑

Windows 11 允许您从其他设备投影内容，包括其他 Windows PCs 和支持 Miracast 的手机。此页面允许您设置其他设备每次想要投影到您的 PC 时是否需要等待许可，或者投影是否需要 PIN。

远程桌面

此页面允许您管理远程桌面设置，这意味着您可以使用另一台电脑通过 Microsoft 远程桌面远程连接到它。只有专业版或更高版本才能通过远程桌面访问，所以如果你有 Windows 11 家庭版，这个页面不会向你显示太多。

剪贴板

Windows 11 中的剪贴板有不少你可能不知道的功能。在此页面中，您可以启用或停用“剪贴板历史记录”,它会存储您拷贝的文本和图像，以便您以后可以更容易地粘贴它，即使您同时已经拷贝了其他项目。您还可以跨设备同步您的剪贴板，并选取是自动同步所有项目，还是想要手动同步特定的剪贴板项目。这也支持安装了 SwiftKey 键盘的 Android 手机。

关于

最后,“关于”页面显示了您需要的关于您的电脑的几乎所有信息。这包括名称处理器、RAM 数量、设备和产品 id 等等。它还包含有关 Windows 的信息，包括当前版本、版本、内部版本号以及 Windows 功能体验包的已安装版本。该页面上还有其他 Windows 11 设置的链接，包括激活、远程桌面、BitLocker 和设备管理器。

蓝牙和设备

转到蓝牙和设备部分，这自然是指连接到 Windows 11 PC 的外围设备和设备。此部分的标题包括已与您的电脑配对或连接的设备列表，如耳机或无线控制器。添加设备按钮允许您连接蓝牙设备、Xbox 无线控制器、无线显示器等等。您也可以在此页面打开或关闭蓝牙。

设备

点击本部分标题中的查看更多设备，或点击页面下方的设备，即可进入设备页面。在这里，您可以看到连接到 PC 的所有外围设备，包括显示器、鼠标、键盘等。此页面还允许您禁用或启用 Swift 配对通知，这旨在使连接过程与支持的无线设备更加无缝。

打印机和扫描仪

此页面允许您管理连接的打印机和打印服务。您可以查看每个的打印队列、打印测试页、运行疑难解答等。您还可以选择 Windows 是否应该管理您的默认打印机，以及是否应该通过计量连接下载打印机驱动程序。

电话链接

这本质上是 Windows 11 中 Phone Link 应用的快捷方式，以前叫你的电话。这个应用程序可以让您将 Android 手机连接到 PC，以同步通知、消息和从 PC 拨打电话。有些手机可能还允许您从 PC 上打开应用程序。否则，该页面允许您禁用与使用 Windows 11 的 Android 手机相关的建议。

摄像机

在这里，您可以管理连接到 PC 的摄像头，包括内置网络摄像头。您可以搜索联网的摄像头，也可以更改每个摄像头的设置，如亮度和对比度，以提高视频通话的图像质量。

老鼠

“鼠标”页面允许您配置鼠标设置。您可以设置左按钮或右按钮是您的主按钮，更改鼠标指针速度、滚动速度，以及当鼠标悬停在非活动窗口上时是否应该滚动。

您还可以访问高级设置来更改鼠标指针图标、增加其大小或更改其颜色，这些功能对于辅助功能非常重要。

触摸屏设备

在这个页面中，您可以自定义与触摸板特别相关的设置，如果您有一个精密的触摸板，还可以有更多的设置，您可能会这样做，因为 Windows 笔记本电脑已经被要求包含它们好几年了。您可以在这里更改大量设置，从连接外部鼠标时停用触控板开始。您也可以更改光标速度。

其余的设置是为了更有趣。您可以调整触控板对点击的敏感度，选取哪种类型的鼠标点击应与特定点击相关联，例如用两个手指点击作为右键点击，等等。也可以启用或禁用滚动手势，您可以相对于您的手势反转滚动方向，或者禁用缩放。

不过，最有趣的手势是有三个或四个手指的手势。您可以选择用三个或四个手指轻扫触摸板时发生的情况，或者只轻按触摸板时发生的情况。主页面给了你一些对大多数用户有意义的预置，但是你也可以进入高级手势页面自定义每个手势。可用的操作包括切换应用程序、调节音量、切换桌面等。

钢笔和 Windows 墨迹

在这里，您可以找到与活动数字笔相关的设置，前提是您的设备支持它们。您可以选择用哪只手书写来提高识别能力，更改快捷按钮的功能，或者查找您的笔(如果笔支持该功能)。您还可以自定义使用笔输入文本的手写面板，例如使用笔书写时文本使用的字体。

自动播放

自动播放是 Windows 的一项功能，当您插入闪存卡或内存卡时，它允许您的电脑以特定的方式运行。在此页面中，您可以选择插入设备时默认发生的情况。

通用串行总线

此页面可让您在 USB 设备未正确连接时关闭通知，还可选择当屏幕关闭时是否应关闭 USB 设备以节省电池电量。

网络和互联网

Windows 11 app 中的这一段设置包括了所有与互联网连接相关的连接。此部分的标题将显示您当前连接的网络、网络类型以及过去 30 天您在该网络上的数据使用历史记录。这些也是更深入的网络属性和您的详细数据使用的快捷方式。

在此之下，我们有以下选项:

无线保真

这允许您启用或禁用 Wi-Fi 连接，查看当前连接的属性，查找其他 Wi-Fi 网络，以及管理您已经添加到电脑的网络。您还可以查看 Wi-Fi 适配器的详细属性，如果您不想使用默认设置，也可以在这里设置 IP 和 DNS 配置。最后，您可以启用随机硬件地址，这使得网站和其他人更难跟踪您的设备的确切位置。

虚拟专用网络

在此页面中，您可以设置 VPN 配置文件以安全地访问公司资源，或者使其更难被在线跟踪。

移动热点

就像在电话上一样，此功能允许您与其他用户共享您当前的互联网连接。您可以通过 Wi-Fi 或蓝牙共享 Wi-Fi 或有线连接。在此页面中，您还可以设置 Wi-Fi 网络的名称和密码。

飞机模式

此页面允许您禁用所有无线通信，或者选择仅禁用 Wi-Fi 或蓝牙。

代理人

在这里，您可以手动或自动设置代理服务器。设置代理服务器有助于掩盖您的在线身份。

拨号上网

如果您还在使用拨号上网，本页允许您设置拨号上网。

高级网络设置

在这里，您可以查看有关您的连接的高级信息，例如连接已经运行了多长时间，通过网络发送了多少数据，以及它的链接速度。您还可以检查您的数据使用情况，管理您的硬件和连接属性，并将您的网络设置重置为出厂默认设置，这意味着所有已知的网络都将被遗忘。最后，该页面包括经典网络适配器设置和 Windows 防火墙的快捷方式，这些快捷方式可以在 Windows 11 的其他部分找到。

个性化

您可以在此自定义电脑的外观。此部分的主页显示了当前主题的预览，以及一些您可以快速切换到的主题。在此之下，您可以获得更多的定制选项。

背景

您可以在这里选择显示在桌面上的背景图像。您可以选取单个图像、纯色或幻灯片显示。如果选择单个映像，您还可以为每个显示器或每个虚拟桌面设置不同的映像。Windows 11 中包含了一些预设图像，但你可以使用自定义照片，如果长宽比不匹配，你还可以调整图像的格式以适应你的屏幕。

有了 Windows 11 2022 更新，你还可以选择 Windows Spotlight 作为桌面背景，它每天都会把你的背景换成新的东西。

颜色；色彩；色调

在这里你可以改变你想要的 Windows 11 的“模式”是亮还是暗。你也可以选择一种自定义模式，Windows 本身遵循一种模式，但应用程序遵循另一种模式。您还可以在整个操作系统中启用或禁用透明效果。

此外，您可以选取一种强调颜色，用于某些应用程序中的一些高亮显示、图标和带有链接的文本。除了一些默认颜色，您还可以使用颜色选择器、RGB 或 HSV 值或十六进制代码来设置任何想要的自定义颜色。如果需要，强调色也可以用于任务栏和“开始”菜单，以及标题栏和窗口边框。

你也可以从这里访问对比主题，尽管这些被认为是一个辅助功能，你可以在 Windows 11 设置应用的其他地方找到它们。

主题

此页面允许您为您的电脑选择一个总体主题。Windows 11 主题可以包括背景图像、颜色设置、声音和鼠标光标，当您更改主题时，所有这些都会随之更改。默认情况下包含一些默认主题，但是您也可以在 Microsoft Store 上找到更多主题。您也可以将自己的自定设置存储为主题，以便以后可以轻松应用。

此页面还包括桌面图标设置的快捷方式，因此您可以添加或删除文件夹(如“这台电脑”、“回收站”等)的图标。

锁定屏幕

在此页面上，您可以更改电脑锁定屏幕的背景，即登录帐户前显示的内容。您可以选取单个图像、幻灯片显示或 Windows spotlight，这是一组经常在互联网上更新的图像，因此您总能看到一些新内容。后者是默认设置。你也可以选择一个应用程序在锁定屏幕上显示详细的通知。

文本输入

Windows 11 的一大新功能是改变触摸键盘的外观，这就是你可以做到的。随着 Windows 11 2022 的更新，你还可以改变语音听写面板和表情符号/剪贴板面板的外观。相同的主题被用于所有这些事情。

默认情况下包括 16 个主题，您也可以使用自己选择的背景图像创建自定义主题。此外，您可以更改触摸键盘的特定设置，如整体大小、是否希望每个键都有背景以及键内文本的大小。

开始

在这里，您可以自定义开始菜单的行为。有了 Windows 11 2022 更新，你可以改变的第一件事就是你是想要更多的空间用于开始菜单中的固定项目，还是更多的空间用于推荐。在此之下，您可以选择最近安装的应用程序是否应该显示在“开始”菜单的“推荐”部分，或者您最常用的应用程序是否应该显示在所有应用程序列表的顶部。您还可以选择最近打开的项目是否应该显示在文件资源管理器的开始、跳转列表和快速访问页面中。

此外，您还可以自定义显示在“开始”菜单底部电源按钮旁边的文件夹。其中包括文稿、下载、音乐和其他文件夹，以及指向文件浏览器和设置应用程序的链接。

任务栏

任务栏页面允许您选择应该在任务栏上显示的图标，包括搜索、任务视图、小部件和与 Microsoft 团队聊天。您还可以选择是否希望在任务栏的角落看到笔菜单、触摸键盘或虚拟触摸板的图标，以及哪些图标出现在溢出菜单中，溢出菜单是当您单击任务栏角落附近的箭头时打开的弹出菜单。这是许多应用程序，如 Skype、Slack 和 Teams 在后台运行时驻留的地方。

最后是 Windows 11 任务栏本身的一些设置。你可以选择将任务栏图标居中对齐(Windows 11 的默认设置)或者像以前版本的 Windows 一样靠左对齐。还可以设置任务栏在几秒钟后自动隐藏，选择是否希望任务栏在所有显示器上可见，以及每个显示器上的任务栏上应该出现什么应用程序。最后，您可以通过单击任务栏的最右上角来禁用(或启用)查看桌面的快捷方式。

字体

您可以在这里管理安装在系统上的字体。您可以通过将字体文件拖到此页面顶部的区域来安装新字体，或者删除现有字体。一些字体也可以在微软商店买到。

设备使用

此页面允许您设置最常使用电脑的内容。这可以根据你的使用情况产生个性化的提示和广告。

应用程序

Windows 11 设置应用的这一部分允许您管理安装在您电脑上的应用和功能。这里没有标题菜单，所以我们将直接进入各个页面。

安装的应用程序

此页面包括你电脑上安装的所有应用程序，你可以看到它们占用的空间、首次安装的时间以及你拥有的版本。借助 Windows 11 2022 更新，您还可以从一些显示选项中进行选择，如列表、网格或平铺视图。

您可以点按给定应用程序旁边的三点图标来卸载它，以及修改安装(对于“经典”应用程序)或更改高级应用程序设置(对于现代应用程序)。这允许您启用或禁用摄像头、麦克风、位置和存储权限，终止应用程序，修复或重置应用程序，或者完全卸载它们。您还可以从三点菜单中将应用程序移动到不同的驱动器。

高级应用程序设置

在 Windows 11 2022 更新的新页面“高级应用设置”中，您可以更改各种设置，如是否可以从 Microsoft Store 或互联网上的任何地方获取应用，选择应用如何在设备之间共享体验，以及选择如果您不经常使用应用，是否应自动存档应用以节省空间。

默认应用程序

此页面允许您设置打开特定类型文件和链接的默认应用程序。与以前版本的 Windows 不同，你不能设置默认的音乐或视频播放器。除了浏览器之外，每个默认关联都必须基于每个文件类型来完成。您可以点按应用程序来查看它可以打开哪些文件类型和链接，然后选取您想要设定的默认设置。在浏览器的情况下，你会在顶部看到一个横幅，这样你就可以将你选择的应用程序设置为默认浏览器。

离线地图

在此部分，您可以管理 Windows 11 中地图使用的离线地图。您可以检查已下载的地图并下载新地图，更改离线地图的存储位置，以及选择是否应通过计量连接下载地图。您还可以检查地图信息更新并管理自动更新。

可选功能

Windows 11 2022 更新中添加了另一个新页面，在这里您可以管理 Windows 中的可选功能，但默认情况下不会启用或安装。您还可以禁用某些功能，如 Edge 的 Internet Explorer 模式。

网站应用程序

当应用程序能够处理网址时——比如在 Teams 应用程序中打开一个团队会议——你会在这里找到那个应用程序。您可以禁止链接在各自的应用程序中打开，这将允许它们在浏览器中打开。

视频回放

你可以在此页面上更改播放视频的应用的设置。这包括自动增强视频、降低视频分辨率以节省网络带宽以及选择是否要通过降低视频分辨率来优化视频播放以延长电池寿命的设置。

帐目

在此部分，您可以管理已添加到电脑的帐户。这包括您的 Microsoft 帐户，它在整个操作系统的许多地方都使用，但也包括其他电子邮件应用程序，例如，您可能已经在 mail 应用程序中添加的应用程序。此页面在 Windows 11 版本 22H2 中进行了重大修改，因此现在您还可以看到有关您的 Microsoft 365 订阅的信息(如果有)。微软也在努力增加对其他订阅的支持，如 OneDrive 独立计划和 Xbox 游戏通行证。

在此之下，您有各种与您的帐户相关的选项。

您的信息

此页面允许您查看您的 Microsoft 或本地 Windows 帐户信息和个人资料图片，以及更改该个人资料图片。根据您当前使用的帐户，您还可以切换到本地帐户或 Microsoft 帐户。

登录选项

此页面允许您为 Windows 设置不同的登录方法，包括 Windows Hello。根据您的硬件支持，您可以设置面部识别、指纹识别、PIN 或物理安全密钥作为登录方法。您也可以选择要求 Microsoft 帐户进行 Windows Hello 身份验证，这样您的 Microsoft 帐户密码不足以让您登录。

您还可以自定义设置，例如是否希望您的电子邮件显示在登录屏幕上，是否希望保存您的登录信息以便您的电脑可以在不要求身份验证的情况下完成更新，以及选择某些应用是否可以在您登录时自动重新启动。

电子邮件和帐户

在这里，您可以找到您添加到邮件应用程序的所有电子邮件帐户，无论它们基于哪个电子邮件提供商。您还可以在 Windows 11 的应用中查看您可能正在使用的 Microsoft 帐户。您也可以在这里添加或删除现有帐户。

家庭的

您可以在此管理 Microsoft family group 的设置。通过此页面，您可以查看家庭组成员并添加新成员，或者打开家庭安全应用，在该应用中您可以查看与家庭组相关的所有选项和设置。

Windows 备份

此部分允许您管理文件和 Windows 11 设置的备份。您可以选择是否要同步您的应用，以便您可以在另一台电脑上轻松获取它们，同步密码、语言偏好设置和其他 Windows 设置。此页面还包括 OneDrive 同步设置的快捷方式，允许您选择是否要将文档、图片和桌面文件夹备份到 OneDrive，以便随时随地访问它们的文件。

其他用户

此页面允许您将更多用户添加到您的电脑，以便您可以为不同的人或不同的目的拥有不同的帐户。

访问工作或学校

在这里，您可以管理依赖于 Microsoft 365 的公司或学校帐户。您可以添加新帐户导出管理日志文件，并访问您的企业或学校可能需要的其他管理功能。

时间和语言

在此部分，您可以管理 Windows 11 中的区域格式和语言设置，以及日期和时间。通过 Windows 11 2022 更新，你可以在页面顶部看到你当前的时区和地区设置，但没有更多信息。让我们来看看各个页面。

日期和时间

在这里您可以看到您所有的日期和时间设置。默认情况下，此信息通常与 Windows 时间服务器同步，但如果需要，您可以禁用它，您也可以选择您的时区或让 Windows 根据您的位置自动设置它(默认情况下，此设置是关闭的)。如果需要，您也可以手动同步当前时间，并选择是否希望在任务栏上看到其他日历(特别是中国农历)。

语言和地区

这个页面有几个组。首先是语言设置，您可以管理安装在 PC 上的语言(用于语法检查、文本到语音转换功能、语音识别和手写)，以及更改显示语言。这将更改遵循系统设置的所有窗口和应用程序中的文本。

第二组设置适用于您所在的地区。选择不同的地区可能会更改您在 Microsoft Store 上看到的内容，并且某些功能可能仅在某些地区可用。您还可以更改区域格式，例如，日期和时间的显示方式。

打字

在此页面上，当您在电脑上键入文本时，您可以调整 Windows 的行为。随着 Windows 11 2022 的更新，一些与触摸键盘相关的设置现在出现在这个页面上，所以你可以选择是否希望在你键入时听到声音，自动大写句子中的第一个字母，等等。您还可以选取是否希望在没有连接物理键盘时自动显示触摸键盘。

其余的设置与文本建议和语法检查相关。您可以选择在使用物理键盘键入时查看文本建议，如果安装了多种语言，则启用多语言文本建议，启用或禁用自动更正，并选择是否要突出显示拼写错误的单词。您还可以看到您的键入见解，它显示您使用自动完成单词、建议单词等的次数，让您了解 Windows 如何帮助您更快地键入。

演讲

此页面具有语音识别和语音转文本功能等设置。您可以选择要用于语音识别的语言，并在这里为其设置麦克风。你还可以选择在 Windows 11 中使用哪种语音来实现文本到语音转换功能，以及更改语音速度。最后，您可以通过此页面管理您的语音包并向您的电脑添加新的语音语言。

赌博

此部分包含 Windows 11 中游戏相关功能的一些设置，如 Xbox 游戏栏、游戏捕捉等。这一次也没有有趣的标题，所以我们将直接跳到各个页面。

Xbox 游戏吧

在这里，您可以启用或禁用使用 Xbox 无线控制器上的 Xbox 按钮打开 Xbox 游戏栏的功能。所有其他的 Xbox 游戏条设置都是通过游戏条本身来访问的。

捕获

此页面包含 Xbox 游戏栏中可用的录制和捕获功能的设置，从您希望将这些捕获保存在驱动器上的位置开始。记录发生的事情功能会在你玩游戏时自动记录你的游戏，所以你只需按下特定的组合键就可以保存刚刚发生的事情的视频文件。如果启用此功能，您还可以设置自动录制的时长(最长 10 分钟)。

否则，您可以选择手动开始的录制的最大录制长度，可以选择在这些录制过程中是否包括游戏和麦克风音频，选择帧速率和视频质量。您还可以选择是否希望鼠标光标在游戏截图中可见。

游戏模式

这个页面包含一个游戏模式的开关，它限制游戏过程中的后台活动，以帮助游戏更流畅地运行。默认情况下，此功能是打开的。

易接近

这一部分以前称为“轻松访问”,包含多种功能，有助于使您的电脑更易于使用，特别是在您有视力或听力障碍或行动不便的情况下。这些包括许多设置，所以我们将会更广泛地介绍它们。

视力

针对视力障碍的辅助功能包括:

  • 文本大小 -改变屏幕上文本的大小
  • 视觉效果 -启用或禁用可能导致可见性问题的视觉效果，如透明度和动画。您还可以将滚动条设置为始终可见，或者更改通知横幅从屏幕上消失的时间。
  • 鼠标指针并触摸 -您可以更改鼠标指针的颜色和大小，以便在屏幕上更容易找到。对于触摸屏，您也可以在触摸屏幕时触摸以查看触摸指示器，并选择使该指示器更大更暗。
  • 文本光标 -如果您在打字时找不到大的文本光标指示器，您可以启用它。您可以更改光标指示器的大小及其颜色。您还可以增加文本光标的粗细，使其更加清晰可见。
  • 放大镜 -启用或禁用放大镜放大您的屏幕，并调整默认情况下您希望看到的缩放级别。使用放大镜放大或缩小时，您还可以更改缩放级别的变化程度。其他设置包括反转屏幕上的颜色以及平滑文本和图像的边缘。
  • 滤色器 - Windows 包括滤色器，用于处理各种类型的色盲。您可以在此页面选择符合您个人需求的过滤器。
  • 对比主题 - Windows 还包括一些旨在提高视觉障碍者的可见度和舒适度的主题。Windows 11 包含了全新的主题，应该会更悦目。
  • 讲述人 -讲述人功能允许你大声朗读 UI 元素和文本。您可以在这里启用或禁用此功能，更改讲述人的声音设置、详细程度、光标等。

听力

对于有听力障碍的用户，可以使用以下设置

  • 音频 -启用单声道音频通道(如果只有一只耳朵有损伤，影响立体声时有用)，并选择接收音频通知时屏幕是否应该闪烁。
  • 字幕 -改变字幕在视频中的外观，带有一些预设主题和创建完全自定义主题的能力。

互动

对于行动不便的用户来说，Windows 提供了一些功能，可以通过减少移动来改善交互。以下是可用的设置:

  • 语音 -启用语音识别，通过语音访问和语音输入来控制您的电脑。
  • 键盘 -启用粘滞键等功能，过滤可定制灵敏度的重复按键，或启用按下 Caps Lock 或 Num Lock 等切换键的声音。也可以在这里打开屏幕键盘。
  • 鼠标 -启用鼠标键，可以使用 yout 键盘上的数字键盘移动鼠标，以及一些移动速度和加速度的设置。
  • 眼球控制 -对于支持眼球跟踪器的设备，如 Tobii 的设备，您可以仅通过眼球运动来控制窗口。此页面允许您启用和管理此功能。

隐私与安全

此部分包含与您设备的安全性相关的设置以及 Windows 11 的隐私设置，例如哪些应用程序可以访问某些类型的数据或物理设备。同样，这里没有大标题，所以我们将直接进入各个部分。同样，它们有很多，所以我们将保持这些选项相对简单。

安全性

第一组设置侧重于安全性，包括:

  • Windows 安全 -显示您当前安全状态的概述，例如是否检测到任何病毒或任何设置可能会给您带来风险。这些设置可以在 Windows 安全应用中更改。
  • 查找我的设备 -支持在地图上定位您的电脑。然后，您可以在 web 浏览器中转至您的 Microsoft 帐户设置来找到它。
  • 设备加密 -加密您的设备数据，以便其他用户无法访问您的文件，除非他们知道您的凭据。
  • 针对开发人员 -启用开发人员模式和测试可能需要的其他应用程序开发功能。

Windows 权限

第一批与隐私相关的设置与 Windows 知道的关于您的信息有关。它包括以下内容:

  • 常规 -启用或禁用您的广告 ID，以便应用程序更容易向您显示个性化广告，允许 Windows 根据您的语言列表推荐内容，让 Windows 通过跟踪您的应用程序启动来改进其在搜索和启动中的推荐，以及在设置应用程序中查看建议的内容。
  • 语音 -启用或禁用在线语音识别，它由的功能使用，如 Cortana 和语音打字，通过连接到互联网来准确识别您所说的话。禁用此项将禁用这些功能。
  • 墨迹书写&打字个性化 -选择您是否想要根据您键入的单词或用墨迹书写的单词创建个人词典。这可以通过记住您可能经常使用的自定义单词来改进键入建议。
  • 诊断&反馈 -选择您发送给微软的诊断数据的详细程度。使用 Windows 需要一些数据，但你可以选择发送更详细的数据，如有关你访问的网站和使用的应用的信息，以帮助 Microsoft 更轻松地识别和修复潜在问题。您还可以选择如何使用这些诊断数据——例如改进墨迹书写和打字。
  • 活动历史记录 -保存关于您正在使用的应用和文件的数据。这是 Windows 10 中的时间线功能所要求的，但现在没有明确的目的，所以你可能需要禁用它。
  • 搜索权限 -更改 Windows 搜索功能的搜索设置，例如调整 web 结果的安全搜索级别，或者允许从您的 Microsoft、工作或学校帐户访问，以便在 Windows 搜索中查看这些帐户的文件和其他结果。您还可以禁用或清除本地搜索历史记录，它用于改进搜索建议。
  • 搜索索引 -选择搜索索引的深度，允许它查找整个驱动器中的文件或只选择文件夹。您也可以从搜索结果中排除特定的文件夹。

应用权限

最后，我们有应用程序权限，这意味着每个应用程序可能需要访问的信息，以启用某些功能。这些设置中有许多可以完全禁用，或者仅针对特定应用程序禁用。这包括访问您的物理位置、网络摄像头、麦克风等。因为这些设置很多都是不言自明的，而且列表很大，所以我们不会全部列出来。

Windows Update

Windows 11 设置应用的这一部分，在 Windows 10 中称为更新&安全，现在专门关注 Windows Update 和 Windows Insider 程序。此部分的标题显示您当前是否是最新的，或者是否有任何更新正在等待，并且它包括一个用于检查更新的按钮。您还可以在一段时间内暂停更新(在 Windows 11 家庭版上最多五周)。这里的其他选项有:

更新历史记录

查看安装在您电脑上的更新历史记录，分为 Microsoft Defender Antivirus 的功能更新、质量更新、驱动程序更新和定义更新。还有一个“其他”更新的部分，包括微软每月推出的 Windows 恶意软件清除工具。

高级选项

在这里，您可以更改 Windows Update 行为的许多设置。首先，您可以选择在接收 Windows 更新的同时接收其他微软产品的更新，如 Office。“让我保持最新状态”选项使您的电脑在安装更新后尽快重新启动，但您也可以选择是否希望在需要重新启动以完成安装更新时收到通知。您还可以设置活动时间(当您频繁使用电脑时)，以避免在此期间意外重启电脑。最后，您可以设置是否希望通过计量连接下载更新。

这里的一些附加选项包括可选更新，其中可以包括 Windows 的驱动程序更新或更新预览。还有交付优化设置，这基本上允许您选择是否要使用点对点连接到其他电脑接收更新。这可以提高下载速度，但会增加带宽使用率，因为您还会向其他人发送更新。默认情况下，你的电脑将被设置为从与你在同一网络上的其他电脑下载更新，但不从互联网下载。您可以更改该设置或完全禁用它。

在交付优化页面，你也可以进入高级选项来设置 Windows Update 可以用来下载和上传 Windows 更新的带宽。您可以设定绝对带宽值，或者选取您想要使用的带宽百分比。

Windows Insider 计划

最后，本部分让您管理您在 Windows Insider 计划中的参与情况。您可以加入该计划以接收 Windows 11 的开发版本，该版本可能不稳定。你可以选择你想加入的频道，或者完全退出内部版本。要立即退出，你必须执行 Windows 11 的全新安装，但如果你在测试版或发布预览版渠道，你也可以选择在下一个公共更新发布后退出。这意味着在下一个主要更新完成之前，你会一直获得预览更新，但之后你将运行一个稳定版本的操作系统，而不会丢失数据。
