微软致力于新 Windows 11 版本的低价笔记本电脑

原文：https://www.xda-developers.com/microsoft-low-cost-laptop-windows-11-se/#0001-01-01

微软正准备推出一款面向年轻学生的新型低价笔记本电脑，试图与 Chromebooks 竞争。这些信息来自 Windows Central 的扎克·鲍登，他还声称这款代号为 Tenjin 的设备只是微软在这一领域竞争的开始。微软的 Surface 品牌通常更专注于高端产品，但该公司一直试图通过平价设备扩大其受众，如 Surface Go 3 和 Surface Laptop Go 。

Tenjin 旨在进一步扩展，为 K-12 学生提供最基本的功能。据报道，这款笔记本电脑将采用英特尔赛扬 N4120 处理器，8GB 内存，11.6 英寸显示屏，分辨率为 1366 x 768。对于端口，它将有一个 USB Type-A，一个 USB Type-C，一个耳机插孔和一个筒式充电器端口，这意味着你也不会看到类似 Surface Connect 端口的东西。

该报告中的另一个重要信息与操作系统有关。显然，天神将运行一个专门版本的 Windows 11 SE。这个版本将为像这样的低端设备量身定制，它可能是微软对 Chrome OS 的回答。随着 Windows 11 显著提高了大多数 PC 的系统要求，一个专门的版本可能是在入门级笔记本电脑上实现可靠 PC 体验的答案，比如这款。实际上，我们在发布操作系统之前已经看到了对 Windows 11 SE 的参考，但是 SKU 还没有实现。

至于微软低成本笔记本电脑的最终名称，该报告暗示，它可能会效仿操作系统本身，命名为 Surface laptop SE，尽管这似乎还没有确定。

据 Bowden 的消息人士透露，该设备基本定型，计划在年底前公布。当然，计划可能会改变。没有提供定价信息，但许多较便宜的 Chromebooks 都在 200 美元至 400 美元之间，所以价格应该在这附近。鉴于 Surface Go 3 已经售价 399 美元，微软可能希望这款低价笔记本电脑更加便宜。
