新的 Windows 11 预览版最终重新设计了音量弹出按钮

原文：https://www.xda-developers.com/new-windows-11-preview-finally-redesigns-the-volume-and-brightness-flyouts/#0001-01-01

今天是周三，像时钟一样，微软在开发频道为测试人员推出了新的 Windows 11 内部预览版。建造号是 22533 ，虽然它没有太多的变化，但它有一些显著的变化。

最值得注意的是，微软终于为你使用硬件调整设置重新设计了弹出按钮。这些包括音量、亮度、相机开/关、飞行模式和相机隐私。与之前的黑色不同，这些可以根据明暗模式进行调整。

新设计旨在反映 Windows 11 的更新风格。请记住，Windows 11 是在相对较短的时间内开发出来的，这要归功于微软利用了个人电脑市场在家工作的热潮。定于今年秋天到来的功能更新更像是一个成品，类似于最初的 Windows 10 更新(版本 1511)的预期。

当然，虽然你可能会说这种变化来得晚了几个月，但其他人可能会说这种变化早就应该发生了。

至于这个版本中的其他新功能，你现在可以从任务栏中搜索语音访问，并且可以像使用其他应用程序一样将其固定在开始菜单上。ime、表情面板和语音打字有 13 个新的触摸键盘主题。此外，你现在可以卸载时钟应用程序。

当然，还有一大堆修复和已知问题。

Windows 11 内部预览版内部版本 22533 修复

【常规】

  • 修复了内部人员在驱动程序或固件更新期间可能会看到错误 0x8007012a 的问题。
  • 解决了导致内部人员有时无法登录某些应用程序(如反馈中心)的问题。
  • 修复了 Windows 安全应用程序中漏洞保护描述的文本，因此它仅指 Windows 而非 Windows 10。
  • 修复了导致无法从某些相机和手机导入照片到照片应用程序的问题(它只会永远循环说迄今为止找到 0 个项目)。
  • 启动 Windows 沙盒，关闭它，然后再次启动它，应该不会再导致任务栏中出现两个 Windows 沙盒图标(其中一个不起作用)。

【任务栏】

  • Wi-Fi 图标现在应该更可靠地出现在任务栏中。
  • 如果你有多台显示器连接到你的电脑上，右击主显示器上任务栏上的日期和时间，explorer.exe 将不再崩溃。
  • 按住 CTRL 键并将鼠标悬停在任务栏的任务视图图标上不会再导致 explorer.exe 崩溃。

【设置】

  • 缓解了与在设置中使用云母相关的潜在问题，该问题影响了最近航班中设置应用程序的整体可靠性。
  • 修正了一个影响一些内部人员的问题，该问题导致当试图访问已安装的应用程序，启动应用程序和默认应用程序页面时设置崩溃。
  • 缓解了为应用程序添加动作时设置中的滚轮页面崩溃的问题。
  • 如果播放音频并重复点击快速设置中的音量滑块来改变音量，您应该不会再听到爆裂声。

【开窗】

  • 如果您将鼠标悬停在 ALT + Tab 或任务视图中被截断的窗口标题上，将会出现显示完整窗口名称的工具提示。

【输入】

  • 通过在候选窗口、表情面板和剪贴板上应用主题，改进了文本颜色和按钮的外观(在此之前，一些按钮/文本很难通过某些自定义背景颜色看到)。
  • 点击麦克风图标调用语音输入后，语音输入启动器不会再意外出现。
  • 对于拥有更新的输入切换器体验的内部人员来说，辅助工具如放大镜和讲述人现在应该可以更好地使用它了。

注意:在积极开发分支的 Insider Preview 版本中提到的一些修复可能会成为 Windows 11 发布版本的服务更新，该版本将于 2021 年 10 月 5 日正式发布。

Windows 11 内部预览版内部版本 22533 已知问题

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【设置】

  • 查看可用 Wi-Fi 网络列表时，信号强度指示器不能反映正确的信号强度。
  • 进入系统>显示> HDR 时，设置可能会崩溃。如果您需要在支持 HDR 的 PC 上启用或禁用 HDR，您可以使用 WIN + ALT + B 键盘快捷键。
  • 蓝牙和设备下有一个空白条目。

【微件】

  • 更改任务栏对齐方式会导致小工具按钮从任务栏上消失。
  • 当有多台显示器时，任务栏上的小工具内容可能会在不同显示器之间不同步。
  • 任务栏左对齐时，温度等信息不会显示。这将在未来的更新中得到解决。

和往常一样，如果你在开发频道，你可以在 Windows Update 中获得这个版本。如果不是，您可以在“设置”中的“Windows Insider 计划”选项卡中注册。
