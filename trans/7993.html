微软展示了新的 Windows 11 媒体播放器，然后将其隐藏

原文：https://www.xda-developers.com/microsoft-shows-off-windows-11-media-player/#0001-01-01

过去几周，微软一直在慢慢展示和发布为 Windows 11 设计的新应用。从剪切工具到绘画——这是昨天在 Dev 频道向内部人士推出的——许多 Windows 应用程序已经为 Windows 11 更新。在昨天的 Windows Insider 网络直播中，微软似乎不小心展示了另一个，该视频已经被下线。由 Windows 最新 发现，似乎有一个新的 Windows 11 应用程序被简单地称为媒体播放器。

该应用程序在播放视频的网络广播中展示，表明这可能是我们目前在 Windows 10 中的电影和电视应用程序的替代品。然而，也可能不止于此。在 Windows 最新抓取的截图中，我们看到了随机播放当前播放列表或跳过曲目的选项，这些选项在视频中并不常用。使用更通用的媒体播放器名称，这可能表明微软正在 Windows 11 中统一视频和音乐播放器应用程序。

该应用还具有与 Windows 11 相一致的新 UI 元素。播放/暂停按钮很大，使用了渐变颜色，图标和搜索栏等其他元素也使用了 Windows 11 的设计语言。一些以前在左下角的控件也被移到了应用程序窗口的右侧。

值得注意的是媒体播放器这个名称的使用，因为 Windows Media Player 曾经是在 Windows 上播放视频和音频的首选解决方案。事实上，它仍然包含在 Windows 11 中，但自 Windows 7 以来，它没有收到任何值得注意的更新。Xbox Music 和 Xbox Video 是 Windows 8 中的默认媒体应用，然后这两个应用分别被 Groove Music 和 Movies & TV 取代。

这特别有趣，因为我们看到微软拥抱了更多的传统应用程序，并为 Windows 11 进行了现代化，而不是试图创造全新的品牌和体验。在微软试图推广 Paint 3D 后，新的剪贴工具取代了 Windows 10 的剪贴&草图应用， Paint 正在进行改版。鉴于微软目前对 Windows 11 应用的做法，恢复媒体播放器的名称似乎是非常合理的。

我们很少看到这款用于 Windows 11 的新媒体播放器应用程序，随着微软似乎试图通过删除 Windows Insider 网络广播视频来隐藏它，很可能在不久的将来会有一个“更盛大”的展示。我们在 Windows 11 中看到的几乎所有大的新应用最初都是由 Panos Panay 在 Twitter 上取笑的，所以我们可能会在它推出之前看到类似的一瞥。
