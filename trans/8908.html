Stardock 推出 Fences 4 来帮助你整理你的 Windows 桌面

原文：https://www.xda-developers.com/stardock-fences-4-organize-windows-desktop/#0001-01-01

Stardock 是包括 Start11 在内的流行应用程序的开发商，它将推出 Fences 4，这是其软件的最新版本，可以帮助你整理你的桌面。如果你不熟悉它，fences 是一个应用程序，它可以让你在我们的桌面上创建区域(称为 Fences)，允许你将快捷方式和文件组织到这些不同的区域中。与桌面本身不同，你可以像在文件夹中一样在围栏内滚动，这样你的桌面看起来也更整洁。

Fences 4 中最大的新功能叫做 Peek，它实际上非常有用。使用键盘快捷键( Windows 键 + 空格键)，Fences 4 可以让你看到桌面上所有的栅栏都显示在你所有打开的应用程序的顶部。如果您想快速从围栏中抓取文件并在当前应用程序中打开，您可以使用 Peek 快速找到它，而不必最小化您的应用程序或打开文件资源管理器。

Fences 已经提供了一段时间的另一个功能，叫做文件夹门户，这使得它更加有用。文件夹门户允许您创建围栏，但您可以使用它们将 PC 上的任何文件夹带到桌面上，使其可以立即访问，而不是显示桌面上的项目。您可以将它与 Peek 结合起来，在做其他事情时快速访问常用文件夹。

栅栏还有很多功能，让它更有用，更容易使用。您可以为 fences 创建页面，因此您可以在任何给定时间在桌面上显示特定图标之间切换(有点类似于虚拟桌面)。围栏可以调整大小或完全折叠到标题栏中以节省空间，您可以设定自动规则将文件分配到特定的围栏，并拍摄当前布局的快照，以便在切换时将其应用到新的监视器或电脑。

栅栏还强调自定义，因此您可以调整栅栏的透明度，更改它们的颜色，切换半透明性，等等。应用程序的设置让你可以改变它的很多行为，甚至是 Windows 本身的一些部分。例如，您可以更改 Windows 键 + D 快捷键的行为，这样当您按住时，它会让您看到您的桌面，而不是像开关一样工作。

如果你有兴趣，你可以在 Stardock 的网站上了解更多。Fences 4 本身售价 9.99 美元，或者你可以购买 Object Desktop 套件来访问 Stardock 的所有应用程序。
