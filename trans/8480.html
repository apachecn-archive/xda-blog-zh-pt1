三星互联网 16.0.2 带来了对焦模式、标签组等等

原文：https://www.xda-developers.com/samsung-internet-16-0-2-beta-bottom-navigation-bar/#0001-01-01

三星互联网是最好的安卓浏览器之一。它速度快，安全，内置广告拦截器，提供大量定制，更重要的是，随着新的改进不断变得更好。最近一次三星浏览器更新带来了增强的搜索体验，对透明像素的保护，页面支持等等。三星刚刚发布了新的浏览器测试版，带来了一些新的增强和改进。

三星互联网 16.0.2 beta 中最显著的视觉变化(via 安卓警察 )是焦点模式，将地址栏带到了屏幕下方，这样更容易单手触及。这是一种可选模式，可以在浏览器设置中启用。

最新的测试版还增加了对选项卡组的支持，使得将一组相关的选项卡放在一起变得更加容易。您可以轻松地重命名、编辑或删除选项卡组。这和苹果的 Safari 浏览器在 iOS 15 上提供的功能差不多。

另一个重要的变化是支持 HTTPS 升级，这是作为一个实验室功能推出。这与 Chrome 的 HTTPS 优先模式非常相似。启用后，它会自动尝试将所有网页升级到 HTTPS。

最后，三星互联网 16.0.2 beta 还为添加对类似 Galaxy Z Fold 3 的可折叠手机的更好支持奠定了基础。这个版本引入了设备姿态 API，它承诺在可折叠文件夹上实现新的用户体验。该 API 目前隐藏在一个标志后面，可以通过访问 https://internet:flags 来启用。三星表示，它计划在即将到来的稳定版本中全面实现该 API。要了解关于设备姿态 API 的更多信息，请查看该文档。

三星互联网 16.0.2 测试版正在谷歌 Play 商店和银河商店推出。也可以从 APKMirror 抓取。稳定版本将在未来几周内发布，可能会在最新的测试版更新中引入许多功能，如果不是全部的话。
