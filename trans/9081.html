DuckDuckGo 正在开发一款桌面网络浏览器

原文：https://www.xda-developers.com/duckduckgo-desktop-web-browser/#0001-01-01

DuckDuckGo 最初是一个搜索引擎，目标是增强对谷歌和必应的隐私保护，但它已经慢慢扩展到其他领域。有一个内置隐私保护的 DuckDuckGo 移动浏览器，一个电子邮件过滤器，等等。现在，该公司似乎终于将其移动浏览器扩展到了桌面，尽管你还需要等待一段时间才能试用。

DuckDuckGo 在其年度综述中宣布，该公司正在开发一款桌面网络浏览器，以补充 iOS 和 Android 上现有的 DuckDuckGo 应用程序。帖子透露，“就像我们在移动设备上所做的一样，桌面版 DuckDuckGo 将重新定义用户对日常在线隐私的期望。没有复杂的设置，没有误导性的警告，没有隐私保护的“级别”——只是默认情况下在搜索、浏览、电子邮件等方面有效的强大隐私保护。不是‘隐私浏览器’；这是一款尊重你隐私的日常浏览应用，因为阻止公司窥探你的搜索和浏览历史从来都不是一个坏时机。”

该浏览器不会像大多数流行的替代网络浏览器(包括 Brave、微软 Edge、Vivaldi)一样，基于谷歌 Chrome 的开源版本 Chromium。DuckDuckGo 将转而使用由其运行的操作系统提供的渲染引擎，就像现有的移动应用程序一样——在 Windows 上应该是 Edge Chromium ( WebView2 )在 macOS 上应该是 WebKit (Safari)。DuckDuckGo 表示“早期测试发现它比 Chrome 快得多”，使用操作系统自己的引擎意味着 DuckDuckGo 不会负责快速修补安全漏洞，但这也意味着浏览器不会像 Chrome 或 Edge 那样功能齐全。

DuckDuckGo 没有提到桌面网络浏览器发布的具体时间框架，但该公司的 Twitter 账户称“我们不能等待 2022 年的到来。”现在微软再次向 Windows 用户施压 ( 并增加膨胀)，而 Brave Browser 仍然是可疑的，我们可以在网络浏览器中使用更多选择。Firefox 对大多数人来说仍然是一个很好的选择，可以跨桌面和移动同步，但 DuckDuckGo 似乎以“没有隐私保护的‘级别’”抨击了浏览器——这可能是指 Firefox 的 Enhanecd 跟踪保护。
