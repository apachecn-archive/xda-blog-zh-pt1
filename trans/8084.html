高通& Windows 11 拥有提升 Wi-Fi 游戏性能的完美配方

原文：https://www.xda-developers.com/qualcomm-windows-11-have-the-perfect-recipe-to-improve-wi-fi-gaming-performance/#0001-01-01

除了添加到 Windows 11 的一长串令人兴奋的功能之外，微软还宣布将 Wi-Fi 双站的原生支持引入 Windows 11 生态系统。紧随这一宣布之后，高通现在强调 Wi-Fi 双站与其高通 FastConnect 4 流双频同步(DBS)如何能够为低延迟游戏树立新的期望。

众所周知，高通的 4 流双频同步(DBS)是高通 FastConnect 移动连接系统独有的功能。它可以利用多个频段和天线，为游戏和其他应用提供低延迟和高速移动性能。通过同时利用 2.4 GHz 和 5 GHz 频段(或 6 GHz，如果可用)，原始设备制造商将能够为游戏提供高度可靠的连接。一个频带中的延迟问题可以很容易地在系统级解决。

Value 通过在 Steamworks SDK 中添加对 Wi-Fi 双站的支持，使游戏开发商和发行商更容易为游戏提供低延迟体验。该公司的热门在线游戏，如 Dota 2 和 Counter-Stike:Global Offensive(CS:GO)，都利用 Steamworks SDK 进行联网，现在将受益于改进的响应能力。这显然只能在支持高通 FastConnect 连接系统的 Windows 11 游戏设备上运行。

这种合作是否会让玩家对他们的 Wi-Fi 连接更有信心，还有待观察。然而，高通表示，Wi-Fi 双站及其 4 流 DBS 以有线级的响应能力释放无线游戏体验。在对现成的零售 Wi-Fi 6 接入点进行测试后，该公司相当有信心。

使用这项技术的优势不仅限于游戏。它也有望改善和满足企业 PC 行业不断增长的需求。它可以满足视频通话、协作和企业远程管理工具等应用对密集双向数据流量的需求。

这也是高通与宏碁、AMD 等其他合作伙伴合作的原因。提供高质量的连接解决方案。例如，宏碁的新游戏笔记本电脑采用高通的 FastConnect 6900 连接，提供以太网级 Wi-Fi。

高通科技公司产品管理高级总监 Miguel Nunes 表示:“移动自由是我们 Windows 设备的一个关键目标，这些设备由骁龙计算平台提供支持，我们的创新不断将用户从充电或连接对电线的依赖中解放出来。“通过今天的发布，我们很高兴高通科技能够通过 Windows 11 和高通 FastConnect 6900 为当今要求苛刻的用户解锁以太网级无线连接，从而进一步推进这种自由。”
