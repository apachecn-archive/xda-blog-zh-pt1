三星为Galaxy Note 20的One UI 3.0更新招募美国测试人员

原文：https://www.xda-developers.com/samsung-galaxy-note-20-android-11-beta-one-ui-3-0/#0001-01-01

三星正在美国为Galaxy Note 20的One UI 3.0 (Android 11)更新招募测试人员

三星已经开放了Galaxy Note 20和Galaxy Note 20 Ultra即将推出的基于Android 11的One UI 3.0更新的测试版注册。

三星在8月份为Galaxy S20系列推出了One UI 3.0开发者测试计划。后来，该公司开始向韩国、美国、德国以及今天早些时候英国的Galaxy S20用户提供基于 Android 11的定制软件的公测版。虽然该更新的稳定版本尚未登陆这些设备，但该公司现在正在美国为Galaxy Note 20和Galaxy Note 20 Ultra的One UI 3.0公共测试程序招募测试人员。

三星Galaxy Note 20论坛| |三星Galaxy Note 20 Ultra论坛

今天，三星的一名社区经理在三星的社区论坛上宣布了One UI 3.0测试版注册。虽然该公告没有强调更新中将包括的任何新功能和改进，但我们预计该版本不会与已经用于Galaxy S20系列的版本有太大不同。如果你在Sprint、T-Mobile上拥有Galaxy Note 20或Galaxy Note 20 Ultra，或者通过三星解锁，你可以通过打开三星会员应用程序，然后点击顶部的横幅来注册测试版。

基于Android 11的第一个UI 3.0公共测试版尚未在美国为三星Galaxy Note 20用户推出，但一旦推出，我们会让你知道。

我们预计测试程序将在未来几天内扩展到其他地区。关注XDA论坛，看看是否有人设法抓取OTA下载链接。
