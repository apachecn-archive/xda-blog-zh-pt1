HoloLens 2 用户获得 Windows 全息版 21H2

原文：https://www.xda-developers.com/windows-holographic-hololens-2-21h2/#0001-01-01

所有支持的 Windows 版本，包括新推出的 Windows 11 、今天都收到了累积更新。然而，一个版本的 Windows 正在获得一个新的功能更新，那就是 Windows 全息。微软增强现实耳机 HoloLens 2 的用户正在获得 Windows 全息版本 21H2 的更新，它包括一些重大变化。

此次更新的内部版本号为 20348.1432，这是一个有趣的数字，因为我们还没有看到任何 Windows 10 或 Windows 11 的内部版本。在 Windows 11 发布之前，我们确实在这个范围内获得了 Windows 10 版本，但它们从未对普通用户开放。

这不像 Windows 11 那样是一个大的更新，但这肯定是一个比即将到来的 Windows 10 版本 21H2 更值得的更新。类似的事情发生在 21H1 更新中，它实际上引入了一些 UI 变化，并引入了基于 Chromium 的 Edge 浏览器。这一次，大多数新功能都是面向商业用户的，但这是意料之中的，因为这主要是一款商业设备。

其中一个大的新功能被称为移动平台模式，这意味着能够在平台上跟踪船只和其他低动态运动的海洋船只。这项功能目前处于测试阶段，但对于需要在这些类型的船只上安装增强现实功能的组织来说，这应该是一个大消息。另一个重大变化是，现在可以配置 Microsoft Store，只允许组织的应用程序访问，防止用户安装公共可用的应用程序。

其他的变化没有这么华丽，但是对于 IT 管理员和一般的商业用户来说还是有一些好处的。它支持 PFX 证书文件，你现在可以直接在 HoloLens 上查看高级诊断日志，IT 管理员可以为应用程序更新配置智能重试，以及其他新功能。

如果你想获得 HoloLens 2 的 21H2 更新，你可以使用 HoloLens 2 中的设置应用程序检查更新。否则，您可以手动下载最新版本并使用高级恢复伴侣执行完全闪存更新，这将从设备中删除您的所有数据。
