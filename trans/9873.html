MDN Web Docs 推出新设计，尝试付费订阅

原文：https://www.xda-developers.com/mdn-web-docs-redesign/#0001-01-01

MDN Web Docs，以前被称为 Mozilla Developer Network，是 Web 开发的最佳文档来源之一。从新的 CSS 属性到 Chrome 扩展的最新 API，这是一个学习一切的极好资源，现在该网站有了一个全新的外观和感觉。MDN 也在考虑一个可能的付费订阅选项。

Mozilla 在一篇博客中写道，“2021 年年中，我们开始考虑现代化 MDN 的设计，以创建一个干净和有吸引力的网站，使浏览我们的 44，000 篇文章尽可能容易。我们希望为我们的用户创造一个更全面的体验，强调改进的导航性和我们所有页面的统一外观。”

更新后的网站有更一致的颜色和主题，布局改进，以及一个方便的深色开关，用于在浅色和深色布局之间切换。就像许多其他现代网站一样，当你滚动一个长页面时，页面内导航仍然可以访问，所以更容易知道你在哪里。MDN 团队还用更清晰的信息和图标更新了网站的浏览器兼容性图表。

还有一个新的徽标(在本文顶部可见)，这是一轮社区投票中的八个选项之一。这是一个“M”字母组合，末尾有一个下划线，暗示这是一个帮助编码的网站。老实说，我很希望旧的 Mozilla 恐龙标志能够卷土重来，但也许这只是我的想法。

最后，因为软件开发人员和文档专家也需要吃饭，重新设计的公告还提到了“MDN Plus ”,它可能很快就会出现。该团队写道，“在接下来的几个月里，我们将根据我们从希望定制其 MDN 体验的 web 开发人员那里收到的反馈，扩展 MDN，以包括一项高级订阅服务。”离线支持和推送通知是两个经常被提到的功能，可能是 MDN Plus 的一部分，但还没有任何东西是固定的(公开的)。
