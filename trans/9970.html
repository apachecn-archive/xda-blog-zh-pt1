苹果 M1 超发布日期，基准，规格，等等

原文：https://www.xda-developers.com/apple-m1-ultra/#0001-01-01

就在发布新的 M1 Pro 和 M1 Max 芯片几个月后，苹果决定在其春季“Peek Performance”产品活动上揭开苹果硅 SOC M1 家族的又一成员。被称为 M1 超，这种新的芯片是 M1 家族的第四个也是最后一个成员。M1 Ultra 是专门针对台式机的，它将在苹果的新 Mac Studio 中首次亮相。在本文中，我们将详细了解您需要了解的关于新的苹果 M1 超 SoC 的一切。

苹果 M1 超:发布日期和可用性

苹果在春季“Peek Performance”活动上宣布了新的 M1 超芯片，以及新的 M1 驱动的 iPad Air 和新的 iPhone SE (2022)。该公司还证实，新芯片将首次亮相 Mac Studio 机器，这也是在同一场活动中宣布的。

M1 超动力 Mac Studio 台式电脑起价 3999 美元，将于 3 月 18 日开始购买。值得指出的是，新的 Mac Studio 是唯一一台采用新的 M1 超芯片的电脑。虽然我们预计苹果未来会将这种新的 SoC 带到其他一些机器上，但目前还没有官方信息。

苹果 M1 超:什么是超融合架构？

那么，如何制造一款比 M1 Max 芯片更强大的新 SoC 呢？该芯片已经凭借其令人难以置信的一系列规格突破了芯片组领域的极限。根据苹果公司的说法，答案是通过使用 UltraFusion。这种新的封装架构本质上是借助一个非常高速的接口来连接两个 M1 Max 芯片。苹果公司直到现在一直对这个界面保密。它还使用硅插入物在两个芯片之间路由信号。苹果公司表示，它可以在芯片之间路由大量信号，从而允许两个 M1 Max 芯片之间的超宽、超高带宽连接。这是苹果在 M1 Ultra 主题演讲中展示的一个模拟动画:

正如你所看到的，动画显示了连接两个芯片的某种桥梁。虽然苹果公司表示，它正在使用硅内插器建立连接，但究竟是什么组件在这里发挥作用，使高带宽连接成为可能，仍有待观察。由于 UltraFusion，M1 Ultra 可以在两个互连的芯片之间提供令人难以置信的 2.5 TB/秒带宽。这也使得 M1 超生产两个独立的 GPU 作为一个单一的单位的应用程序。本文稍后将对此进行详细介绍。

苹果 M1 超:硬件概述

既然我们知道苹果是如何将两个独立的 M1 Max 芯片作为一个单元来运输的，那么新的 M1 Ultra 几乎将硬件增加了一倍就不足为奇了。这也延伸到规格表中的数字，这意味着 M1 超有两倍于 M1 最大 SoC 的 CPU 和 GPU 核心。神经内核、LPDDR5 内存通道甚至外设 I/O 的数量也是如此。

就 CPU 而言，M1 超提供了总共 20 个核心，其中包括 16 个高性能核心和 4 个高效核心。如果你不知道，性能核心是苹果的 Firestorm 核心，而效率核心是他们的 Icestorm 核心。这种混合内核架构与英特尔在新的第 12 代芯片中分配内核的方式非常相似。

虽然苹果没有透露内核的时钟速度，但我们认为苹果会将速度提升到比我们在 M1 Max 芯片上看到的更高。我们这么说是因为 M1 Ultra 是专门针对台式机的，这意味着苹果不一定要花太多精力来节能。更不用说，高效率的 Icestorm 内核也将继续努力控制功耗。据该公司称，M1 超性能优于 16 核英特尔酷睿 i9-12900K 台式机 CPU，同时功耗相对更低。

由于苹果在芯片上使用了两个 M1 Max 芯片，M1 Ultra 的内存通道数量增加了一倍。这意味着我们现在也在考虑双倍的内存带宽。M1 Ultra 有 32 个 LPDDR5 通道和 800 GB/秒的内存带宽，而 M1 Max 有 16 个 LPDDR5-6400 通道，总内存带宽为 408 GB/秒。对于那些关心规格表上的原始数据的人来说，M1 Ultra 现在有 128GB 的统一内存，而 M1 Max 只有 64GB。新芯片让 64GB 的统一内存听起来没什么大不了，这太疯狂了。

话虽如此，但值得指出的是，128GB 的统一内存仍然无法与许多其他高端桌面工作站相提并论。尽管如此，它肯定比大多数高端消费级电脑要高得多，所以这是需要考虑的。但是，相比 M1 Max 已经提供的性能，拥有两倍的内存带宽能提供任何有意义的性能提升吗？嗯，这是我们在拿到新芯片进行详细对比后才能知道的事情。

在 GPU 方面，M1 超有多达 64 个 GPU 核心，多达 8192 个执行单元。拥有 32 个 GPU 核心的 M1 Max 已经是大材小用了，所以可以肯定地说，M1 Ultra 应该能够毫不畏惧地处理任何类型的 3D 工作负载。我们过去见过多核 CPU，但多核 GPU 在硅行业并不常见。在大多数多 GPU 系统中，每个 GPU 都充当一个独立的单元，这主要是因为内部带宽要求非常高。

苹果基本上已经成功实现了令人难以置信的复杂，并制造了一种结合两个独立 GPU 的芯片。通过将 M1 超与 RTX 3090 进行比较，苹果做出了一些大胆的声明，3090 是最强大的独立 GPU 之一。但只有时间才能证明新的 M1 超在 GPU 工作负载中的表现。您可能已经知道，GeForce RTX 3090 以牺牲能效来提供令人印象深刻的性能而闻名。另一方面，与运行英特尔酷睿 i9-12900K 和 RTX 3090 的高端台式机相比，M1 超动力系统据说性能更好，功耗更低。我们必须仔细看看 M1 超，并将其与独立 GPU 进行比较，以展示我们对 GPU 性能的判断。

M1 Ultra 是最快的芯片吗？

M1 超据说是苹果迄今为止最强大的芯片。该公司已经连接了两个 M1 Max 芯片，使该芯片的硬件规格翻倍。一旦我们有机会测试它，我们将有更多的机会来谈论它的性能。

M1 超级跑车有多快？

按照典型的苹果风格，该公司没有公布 M1 Ultra 的官方时钟速度数据。然而，这款新芯片拥有 16 个性能内核和 4 个效率内核，可以轻松处理您交给它的工作负载。

苹果推出了采用新 M1 超芯片的 MacBook 吗？

不，没有 M1 超 MacBook，至少在写这篇文章的时候没有。新的芯片目前只属于新的 Mac Studio 台式机。

Mac studio 是 CPU 吗？

不，Mac Studio 是苹果新的台式电脑，运行在新的 M1 超芯片上。你也可以购买 Mac Studio 的一个变种，它装有稍旧的 M1 Max 芯片。
