运行 Windows 11 的不受支持的电脑仍在获取更新

原文：https://www.xda-developers.com/unsupported-pcs-running-windows-11-getting-updates/#0001-01-01

当微软宣布 Windows 11 时，人们很快发现了一件事，那就是它大大提高了最低硬件要求。具体来说，你需要漂亮的新 CPU。该公司最终表示你可以通过使用 ISO 在不支持的电脑上安装 Windows 11 但是，您不会获得更新。

事实证明，使用这种方法的人得到了更新。本周是补丁星期二，这一天所有受支持的微软产品都将获得更新。随着 Windows 11 的更新，不支持的设备发现自己和其他人一样。

尽管微软几乎说不支持的设备不会获得 Windows 11 更新，但这并不完全令人惊讶。雷德蒙德公司有一个支持页面，专门用于在不符合最低要求的设备上安装新操作系统。与此相关的关键短语如下:

“如果您继续安装 Windows 11，您的电脑将不再受支持，并且无权接收更新。制造商保修不包括因不兼容而对您的电脑造成的损坏。”

这基本上意味着微软不再插手你的个人电脑。它重视支持，这就是为什么它的所有产品都有生命周期终止日期。如果你现在在个人电脑上安装 Windows 10，你可以保证在 2025 年 10 月前得到更新。这就是说，如果你不满足 Windows 11 的要求，这并不能保证什么。您可能会在一段时间内获得更新，也可能不会，或者可能会完全停止获得更新。微软表示不担心，因为你的设备不受支持。

Windows 11 上不受支持的设备仍在获得更新并不特别令人惊讶，因为实际上只有一个生产渠道。Windows 11 在技术上能够支持的要求是双核 64 位芯片，最低时钟速度 1GHz。它不能在单核或 32 位处理器上运行。英特尔第八代或 AMD Zen 2 处理器的规则更加随意，如果它们没有内置到 ISO 中，那么它们没有内置到操作系统中也就不足为奇了。

更重要的是，你不应该接受今天的新闻，因为在你不支持的硬件上安装 Windows 11 是安全的。微软可以在它愿意的时候关闭这些更新。也许你不会得到下一年的功能更新，当 21H2 版本的 24 个月支持期结束时，你就没那么幸运了。Windows 11 上不支持的设备的未来非常不确定，但如果你坚持使用 Windows 10，你可以保证再有四年的更新时间。
