ReactOS，开源的 Windows 克隆，有了一个新的更新

原文：https://www.xda-developers.com/reactos-0-4-14/#0001-01-01

ReactOS 是一个免费的开源操作系统，旨在完全重新实现 Windows NT 内核和系统，并具有运行为 Windows 设计的应用程序、游戏和驱动程序的能力。它从 1996 年就开始开发了，现在可以下载自去年以来的第一个新版本。

ReactOS 团队在博客文章中写道，“我们花了一年多的时间来完善这个版本并修复回归。因此，ReactOS 0.4.14 不包含我们在 2021 年在博客和社交媒体上宣传的最新发展。它们可以在我们的夜间版本中找到。这是一个维护版本，请继续关注接下来的内容！”

ReactOS 0.4.14 在 Shell 中首次实现了“发送到”,该功能自 Windows 2000 以来就在微软的操作系统中提供，并允许您快速将文件共享给应用程序(例如，创建一封新的电子邮件，并将文件作为附件)。ReactOS 现在也有“打开文件位置”和“在这里打开命令提示符”选项，当右键单击一个文件或目录时，这很方便。Windows 软件兼容性也应该好一点，更新了来自 Wine 项目的 dll，并且提供了对引导 NEC PC-9800 计算机的初始支持。

此次更新还包括对操作系统的互联网控制消息协议实现、即插即用、用 C 编写的软件的运行时异常处理、引导程序、屏幕键盘和字体的改进。所有这些变化意味着 ReactOS 运行起来更流畅，消耗的内存比以前更少——在大多数情况下，整个操作系统在空闲时使用的内存应该少于 100MB。

对大多数人来说，ReactOS 离取代 Windows 还很远，但它是一项令人印象深刻的技术成就。该项目的 Twitter 账户经常发布与 ReactOS 一起工作的应用程序、游戏和个人电脑的例子。虽然大多数完全兼容的软件都很旧，如暗黑破坏神 2 和微软 Office 97 ，但一些现代应用程序如 VLC 媒体播放器也很实用。

你可以从官网了解更多关于 ReactOS 的内容并试用。
