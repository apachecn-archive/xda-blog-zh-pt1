三星再次暂停为其设备推出一款 UI 4

原文：https://www.xda-developers.com/samsung-halts-one-ui-4-rollout/#0001-01-01

虽然三星在过去几周已经向其几款设备发布了一款 UI 4 的稳定版本，但推出并不像该公司希望的那样顺利。几位早期采用者报告了在 Galaxy S21 系列、 Galaxy Z Flip 3 和 Galaxy Z Fold 3 的 Android 12 版本中面临的问题，由于这些问题，三星过去不得不暂时停止推出。虽然该公司已经发布了带有修复和改进的新版本，但似乎并没有解决所有的问题。因此，三星现在已经停止了其所有设备的 One UI 4 推出。

在三星社区论坛上最近的一篇帖子(viati zenhelp)中，一位版主写道:

“Android 12(One UI 4)更新后，由于部分设备与 Google Play 系统的兼容性问题，在极少数设备中发现了错误现象，因此暂时搁置了更新。谷歌在分析内容后正在采取行动。当谷歌的行动完成后，One UI 4 版本将被重新分发。我们正在努力尽快发布一个版本的 One UI 4。

我们计划在解决相关错误并准备好应对措施后重新分配 FOTA，以确保终端的稳定性，减轻消费者的不便并提高客户满意度。"

如果您还没有将您的设备更新到 One UI 4，我们建议等到三星解决所有问题并恢复推出。如果你已经安装了更新，也不必担心，因为主持人指出，这个问题影响了一小部分设备。

我们预计三星将在未来几天分享更多关于推出的信息。一旦我们了解到更多情况，我们会尽快通知您。
