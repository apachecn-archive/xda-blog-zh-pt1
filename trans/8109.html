首批支持 Android 12 游戏模式的游戏正在推出

原文：https://www.xda-developers.com/first-games-to-support-android-12-game-mode/#0001-01-01

Android 12 更新中改进最大的领域之一是游戏。游戏一直是谷歌这一次迭代的重点，该公司甚至在谷歌游戏开发者峰会上专门展示了 Android 12 的新游戏功能。除了“边下载边玩”这一通过在后台下载大型资源来让你更快进入游戏的功能外，还增加了游戏仪表盘。游戏仪表板提供了对一些有用工具的快速访问，以及用于向 YouTube Live 流媒体的小部件，显示 Google Play 游戏的成就，并更改性能配置文件。直到最近，不可能改变任何游戏的性能配置文件，因为游戏必须添加对 Android 12 的游戏模式 API 的支持。然而，现在第一批支持上述 API 的游戏开始推出了。

今天早些时候，我们注意到选择游戏现在支持游戏模式 API。我们注意到了这一点，因为终于可以从 Android 12 的游戏面板中改变性能配置文件了。我们可以选择将配置文件从使用游戏默认设置的“标准”更改为“性能”或“省电”选择“性能”应该最大化帧速率以提高游戏的流畅性，而选择“电池保护”应该降低帧速率以提高电池寿命。更改性能配置文件将导致游戏重新启动，以使新设置生效。

另一种判断游戏是否支持性能配置文件的方法是检查你是否可以从游戏的应用程序信息页面打开“游戏优化”设置。如果“游戏优化”是灰色的，并说“不可用于此游戏”，那么游戏还没有添加对游戏模式 API 的支持。在某些情况下，游戏可能只支持“性能”或“节电”模式，而不是两种模式都支持，目前我的设备上的《我的世界》就是这种情况。如果你根本看不到“游戏优化”设置，那么有可能是 Android 没有把游戏识别为游戏，因为它没有添加适当的 Manifest 条目。

我们已经看到为 【《我的世界》】狙击 3D 和 神庙逃亡 2 ，开放了性能配置文件设置，但可能还有其他游戏增加了对游戏模式 API 的支持。像 Genshin Impact 、 PUBG Mobile 和海绵宝宝:比基尼底部再水化之战这样的重量级游戏尚未添加对游戏模式 API 的支持。

当谷歌首次宣布游戏仪表盘功能时，他们表示该功能不会在 Android 12 测试版中提供。不过，启用该功能很容易，只需进入设置>通知>请勿打扰>日程安排，然后点击“游戏”日程安排的齿轮图标。这让你可以访问“游戏设置”，在这里你可以切换游戏面板快捷方式，以显示在游戏中，以及选择是否应在游戏中启用请勿打扰模式。

谷歌还表示，游戏仪表盘将只在运行 Android 12 的“精选设备”上可用，但我们不知道其他 OEM 厂商的手机是否会支持该功能。

值得一提的是，当我们试图改变《狙击手 3D》中的性能档案时，似乎游戏中什么都没有改变。在“性能”和“省电”模式下，FPS 都被限制在 60，游戏的图形预设也没有改变。有可能这个功能实际上还没有工作，或者它改变了我没有注意到的设置，所以我需要更多地使用它。
