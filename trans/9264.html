Visual Studio 2022 17.1 英寸版本即将推出新的 Preview 2 版本

原文：https://www.xda-developers.com/visual-studio-2022-17-1-preview-2/#0001-01-01

Visual Studio 是目前最流行的集成开发环境(ide)之一，支持许多不同的编程语言。最新的可用版本是 Visual Studio 2022 版本 17.0.3，但微软正在通过新的预览版发布 17.1。

微软在博客文章中表示，“我们继续迭代 Visual Studio 2022 版本的主题，并增强了现有功能，带来了新的生产力改进，并使您更容易将 Visual Studio 变成自己的。性能工作永远不会结束，我们努力在每个版本中做得更好。在 17.1 中，我们优化了 C++项目中的查找所有引用搜索，性能提升了 22%。”

17.1 更新包括在 Git 代码库中工作时的新特性，例如增强的分离头支持、将当前分支与其他分支进行比较的能力、快速检查拉请求以及检查任何远程分支的提交。当您在一个存储库中创建一个新的分支时，还有一个新的选项来跨其他存储库创建相同的分支。幸运的话，在 Visual Studio 7.1 中，您不必经常使用命令行，至少在 Git 中处理项目时是这样。

Visual Studio 7.1 还有一些保存文件的新功能。现在，您可以在每次保存文件时自动运行代码清理配置文件，方法是“工具”>“选项”>“文本编辑器”>“保存时运行代码清理配置文件”。还有一个新功能，当你切换到电脑上的不同应用程序时，可以自动保存你的文件。最后，还有一些特定于语言的改进。NET 和 C++，以及一个新的选项来设置自定义标签颜色的基础上的项目和文件扩展名，这应该有助于组织时，与多个项目在同一个窗口。

可以从微软的网站下载 Visual Studio Code 2022 17.1 预览版 2 。目前还不清楚最终的 17.1 版本将于何时发布。
