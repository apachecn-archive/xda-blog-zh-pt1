搭载 Chrome OS 100 的 Chromebooks 将迎来动画屏保

谷歌已经为运行 Chrome OS 100 的 Chromebooks 推出了动画屏保，未来还可能增加更多。

Chrome OS 要想赶上 Windows 和 macOS 还有很长的路要走，但它比以前做得好得多。它不断获得新的功能和改进，Chrome OS 100 是开发者发布渠道中可用的最新版本。虽然平台的每一次迭代通常都会带来新的改进和功能，但有一个特别之处刚刚到来。在开发者频道最新发布的 Chrome OS 中，Chromebook 用户现在可以使用动画屏保。

由关于 chrome book 的发现并通过在浏览器中转至chrome://flags#ambient-mode-animation启用的功能标志，可以被启用以访问一个动画屏保。这个名为“感受微风”的动画从你的谷歌相册中取出照片，并把它们挂在虚拟的晾衣绳上。这绝对感觉更像是一个演示动画。

目前，功能标志说明如下:“当进入环境模式时，启动动画屏保(相对于现有的照片幻灯片)。目前只有一个动画主题可用(感受微风)。”

这个屏幕保护程序使用了 Lottie，一种轻量级的动画文件格式。关于 Chromebooks 推测 Chrome OS 用户将能够使用 Lottie 编辑器来创建他们自己的动画屏幕保护程序，并且有可能通过 Lottie Marketplace 建立一个屏幕保护程序市场。很可能在未来的版本中会引入更多的动画屏保，但是现在，这是更新中真正能展示的。
