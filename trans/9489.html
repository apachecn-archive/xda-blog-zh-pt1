微软宣布下个月 Windows 11 将推出一系列新功能

原文：https://www.xda-developers.com/microsoft-announces-windows-11-features-february/#0001-01-01

微软正准备在 2 月份向 Windows 11 用户推出一些新功能，Windows 首席 Panos Panay 今天宣布。在一篇关于 Windows 11 现状的博客文章中，微软 Windows 和硬件部门的负责人也展望了未来，从 Android 应用开始。

然而，围绕 Android 应用程序的措辞有些混乱。以下是 Panay 在博文中所说的:

“下个月，我们将为 Windows 带来新的体验，包括公开预览如何通过微软商店在 Windows 11 上使用 Android 应用程序，以及我们与亚马逊和英特尔的合作关系(...)"

这很奇怪，因为 Windows 11 上的 Android 应用程序现在已经可以在预览版中使用了。如果你注册了 Windows Insider 测试版或开发频道，你可以安装 Android 和亚马逊应用商店的 Windows 子系统，并开始安装应用程序。只要稍加改动，你甚至可以在非业内人士的电脑上实现。最有可能的是，这意味着微软将向所有 Windows 11 用户提供 Android 应用，但你必须手动安装 Android 的 Windows 子系统，它仍将被标记为预览版。如果是这样的话，我们仍然不知道微软计划何时将这视为一个“稳定”的功能。

在最近的一篇文章中，我们自己的 Rich Woods 指出，我们在 Windows 11 上体验 Android 应用已经三个月了，并没有发生太大的变化。也许微软希望在未来进行一些性能改进。我们确实就此事联系了微软，如果有回音，我们会更新这篇文章。

除此之外，微软还宣布了一些新功能，Windows 内部人士已经尝试了一段时间。任务栏上有新的麦克风静音图标，首次出现在 Windows 11 build 22494 中，目前仅适用于微软团队的工作版本。另一个团队独有的功能是在会议期间从任务栏轻松共享窗口的能力，该功能首次出现在 build 22499 中。

此外，新的记事本和媒体播放器应用程序也将在下个月推出。新的记事本应用程序使用了新的现代设计，并首次支持黑暗模式，此外还包括一个新的查找和替换功能界面。新的媒体播放器应用取代了 Groove 音乐和电影&电视应用，它将成为 Windows 11 上音频和视频播放的主要目的地。

最后，微软还推出了任务栏上的窗口小部件面板的新入口点，直接在任务栏上显示当前天气。这让人想起了去年加入 Windows 10 的新闻和兴趣功能。

至于这些功能将于何时推出，很有可能会在 2 月份 Windows 11 补丁星期二更新时推出。我们不应该等太久才能确定。
