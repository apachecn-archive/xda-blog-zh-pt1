由于联邦航空局的安全担忧，美国电话电报公司和威瑞森推迟了他们的 C 波段 5G 部署

美国电话电报公司和威瑞森已经同意推迟 5G C 波段部署，以回应美国联邦航空局对可能的飞机设备干扰的担忧。

美国联邦航空局最近表示担心一些运营商部署的 C 波段频谱可能会干扰飞机安全系统，现在似乎美国电话电报公司和威瑞森已经决定推迟他们计划的 5G 部署作为回应。

消息通过传来，称两家运营商分别同意将新的 C 波段频谱部署推迟约一个月。对于外行来说，C 波段的频谱工作在 3.7 GHz 到 4.2 GHz 之间。它提供了一个很好的中间地带，介于深远但较慢的低频带和短程但快速的毫米波带之间。在二月和三月的拍卖中，美国电话电报公司和威瑞森公司都赢得了一部分 C 波段频谱。两家运营商都计划在今年年底和明年部署它。

由于这些担忧，这些计划现在被搁置，因为航空公司正在与联邦航空局讨论这些计划。美国电话电报公司显然同意将部署时间推迟到 1 月 5 日，而威瑞森则应交通部的要求同意推迟“一个月”。这一延迟肯定会影响美国电话电报公司的计划，该计划将于今年 12 月开始启用其 C 波段 5G+。

今年早些时候，联邦航空局提出了对 C 波段干扰的担忧。某些飞机安全功能，如恶劣天气着陆辅助和防撞，使用工作在 4.2 GHz 至 4.4 GHz 范围内的无线电高度计。联邦航空局担心着陆期间附近发射塔发射的 C 波段频率可能会对这些系统产生负面影响。运营商官员声称，证据不支持这些担忧，5G 网络不会干扰飞行设备。

美国电话电报公司明确表示，将继续与联邦通信委员会和联邦航空局讨论，以解决他们的担忧。该运营商表示，由科学和数据引导的讨论对于正确评估任何可能的干扰问题非常重要。
