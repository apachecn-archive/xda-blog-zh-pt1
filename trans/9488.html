微软提前推出了 Windows 11

原文：https://www.xda-developers.com/microsoft-ahead-of-schedule-windows-11-rollout/#0001-01-01

今天，在一篇博客文章中，微软谈论了过去一年的 Windows，以及我们将会看到的未来。该公司表示，Windows 现在每月有 14 亿活跃设备，尽管据推测，这一数字仅限于 Windows 10 和 Windows 11 。它还表示，Windows 11 分阶段推出的速度比预期的要快。原因是，不管你听说过什么，人们就是喜欢它。

该公司表示，人们接受免费升级的速度是 Windows 10 的两倍。当然，这并不意味着更多的电脑升级，因为 Windows 11 实际上只适用于一小部分设备。尽管如此，这仍然令人印象深刻，因为在 Windows 10 推出之前，微软花了几个月的时间在用户的电脑上安装广告软件，目的是欺骗他们说他们想要 Windows 10。Windows 11 的推出要微妙得多。

但这还不是全部。微软表示，Windows 11 拥有有史以来最高的客户满意度和最高的质量分数。人们在装有 Windows 11 的电脑上花费的时间增加了 40%，尽管这可能也与我们在过去几年中看到的在家工作热潮有关。

雷德蒙德公司还表示，Windows 11 为微软商店带来了三倍多的流量。这一点很有意义，因为商店现在已经以有意义的方式开放，任何人现在都可以在微软商店中放置应用程序。

所以，因为人们太喜欢 Windows 11 了，微软现在进入了推出的最后阶段，所以它比计划提前了很多。最初，该公司表示，分阶段推出要到 2022 年年中才能完成。

Windows 11 并非没有问题。还有个已知问题，比如与 VirtualBox 的兼容性。新操作系统在首次发布时相当仓促，仅用了大约 9 个月就完成了构建。Windows 10 的最初计划是，它将成为 Windows 的最后一个版本，所以像这样的重大用户界面改革将是对它的功能更新。相反，它变成了 Windows 11，因为它利用了在家工作的热潮和 PC 市场的暂时复苏。当然，发布时出现的大多数问题都已得到解决。
