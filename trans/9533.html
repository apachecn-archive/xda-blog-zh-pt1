Android 12L Beta 2 现可用于联想 Tab P12 Pro

在 Pixel 手机可以试用两周后，联想 Tab P12 Pro 现在有了适用于 Android 12L Beta 2 的测试版。

Android 12 仍在向许多手机和平板电脑推出，但谷歌已经在开发下一个主要版本， Android 12L 。即将到来的更新主要针对可折叠和平板电脑，由于谷歌不再生产 Android 平板电脑(RIP Pixel C)，主要的平板电脑测试设备一直是联想 Tab P12 Pro。在Android 12L Beta 2 在 Pixel 手机上发布两周后，该版本终于可以安装在联想 P12 Pro 上。

联想开发者网站现在有一个固件映像可供 Android 12L Beta 2 闪存使用，应该与 Pixel 设备上已经提供的类似。该映像的大小为 1.7GB，安装过程与联想 P12 Pro 上的 Beta 1 相同。联想表示，该版本包括 12 月 2021 日的安全补丁级别。

Android 12L Beta 2 包括新 API 的最终版本，这意味着 Beta 2 和最终公开版本之间只会有微小的差异(并且没有新的应用功能)。它修复了一些在第一个测试版中发现的错误，例如破碎的吐司信息和锁定屏幕上的时钟错位。与第一个测试版相比，没有显著的变化。

谷歌在 Android 12L Beta 2 发布时表示，“这是开始最终测试和开发工作的时候了，以确保你的应用程序的兼容版本将为用户获得这一更新做好准备。12L 此时将有一个标准的 API 等级(API 等级 32)。有了最终的 API，也是时候将重点转移到最终的兼容性测试上，并向用户和下游开发者发布更新的应用了。您还可以使用最终的 API 进行构建，并改进任何使用新 API 或特性的新代码。”
