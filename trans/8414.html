谷歌详述 Android 12L 发布时间表

原文：https://www.xda-developers.com/android-12l-rollou-timeline-pixel-devices/#0001-01-01

在昨天的 Android Dev 峰会上，谷歌正式发布了 Android 12L 的第一个开发者预览版，这是专为可折叠、平板电脑和大屏幕设备设计的 Android 12 的一个功能下降。

虽然 Android 12L 主要针对可折叠设备、平板电脑和 Chrome OS 设备，但它也将进军手机领域。谷歌表示，目前它的重点是可折叠和 Chrome OS 设备，但它将在 12 月开放 Pixel 系列的测试版注册，从第一个 Android 12L 测试版开始。

以下是哪些 Pixel 手机将在 12 月获得 Android 12L 测试版:

  • 像素 6
  • Pixel 6 Pro
  • 像素 5a 5G
  • 像素 5
  • 像素 4a (5G)
  • 像素 4a
  • 像素 4

除了谷歌 Pixel 系列，谷歌已经证实，新软件也将用于像 Galaxy Z Fold 3 和联想 Tab P12 Pro 这样的设备。Android 12L 的第一个测试版将于 12 月发布，第二个版本将于 1 月发布，第三个也是最后一个测试版将于 2 月发布。公开发行定于 2022 年 Q1，可能在 3 月左右。

正如 Mishaal Rahman 所指出的，尽管当前的开发者预览版是 API 级别 31，但一旦 API 最终确定，Android 12L 平台将使用 API 级别 32。然而，似乎开发者在 Google Play 上发布应用时并不需要针对新的 API 级别。

Android 12L 目前作为开发者预览版提供，您可以通过在 Android 模拟器中启动 Android 12L 系统映像来试用。新版本带来了几个 UI 改进和功能，为可折叠和平板电脑提供了更好的软件体验。新功能包括通知阴影和锁屏的两列布局、活动嵌入、改进的多任务处理体验等等。
