Chrome 和 Gmail 会收到更多您设计更改的材料

原文：https://www.xda-developers.com/chrome-gmail-material-you-changes/#0001-01-01

谷歌最近推出了 Chrome 94，它带来了大量的改进，包括一个新功能，可以对网页进行 T2 滚动全页截图。现在，我们了解到 Chrome 94 build 也为应用程序带来了一些新的材料。除了 Chrome，Chrome 操作系统上的 Gmail for Android 应用程序也进行了一些实质性的修改。

我们先来看看 Android 版 Chrome 的变化。据9 to 5 谷歌 报道，Chrome now 提供动态颜色支持，这意味着应用程序内的 UI 元素适应手机壁纸上的主色。例如，如果你的壁纸有一个大向日葵，应用程序将创建一个黄色的彩色调色板，并将其作为应用程序的背景，而不是传统的白色或黑色。

(图片:9to5Google)

此次更新还改变了 Chrome 应用程序中一些图标或按钮的外观。例如，Chrome 中的标签切换器 UI 发生了轻微的变化。选项卡现在显示为圆角，并且尺寸稍大。标签的上半部分列出了打开的网站，也支持动态颜色。在不支持动态颜色的旧设备上，标签背景是灰色的，被选择的标签背景是蓝色的。在 Android 12 上，标签页的背景采用之前打开的网页的主色。

Chrome OS 上的 Gmail 应用程序也更新了新的应用程序图标和您更改的材料。这是以前停留在 Gmail 的旧外观上，没有对整个工作区进行品牌重塑。据T5 9 t05 谷歌 T7 报道，现在，这一更新已经发布到版本为 2021.09.10.397347674 的应用程序中。

(图片:9to5Google)

令人惊讶的是，我们在 Android 和 iOS 上的 Gmail 应用程序中看到的 Google Meet 和 Chat 集成并没有包含在 Chrome OS 的这个版本中。应用程序中也没有切换黑暗模式的选项。这个版本只包括轻微的设计调整和你重新设计的材料。如果你有 Chromebook，你应该可以通过 Play Store 获得 Gmail 应用程序的更新。
