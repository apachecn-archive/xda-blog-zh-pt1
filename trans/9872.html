谷歌表示，2021 年，桌面网络应用安装量增长了近两倍

原文：https://www.xda-developers.com/chrome-desktop-pwa-installations/#0001-01-01

谷歌表示，2021 年，桌面网络应用安装量增长了近两倍

谷歌表示，自 2021 年初以来，PWAs(渐进式网络应用程序)的桌面安装量几乎增加了两倍。

在这一点上，Web 应用程序已经在桌面计算机上流行了十多年，但是在过去几年中，更新的 API 和功能使它们更加强大。谷歌也一直在推动 Chrome 中的网络应用程序在台式电脑上“可安装”，就像它们在 Android 的 Chrome 上一样，结果令人印象深刻。

谷歌今天在 Chrome OS 开发门户上的一篇博客文章中写道，“就像网络一样，进步的网络应用(PWAs)⁠)比以往任何时候都更有能力和强大。在开放网络的推动下，在最新的网络应用编程接口的支持下——更不用说在 Google Play 上轻松发现和赚钱了——pwa 使网络开发者能够创建用户可以安装和享受的可靠应用。这个用户群正在迅速扩大。自 2021 年初以来，桌面 PWA 安装量增长了 270%。”

考虑到许多人只是像普通网站一样在浏览器中使用渐进式网络应用程序，这一跳跃令人印象深刻——与 2021 年初相比，使用更像原生应用程序的网络应用程序的人数增加了近三倍。Chromebooks 的 Android 应用安装数量也在增加，2021 年将增加 50%，正如谷歌几个月前分享的那样。

Chrome(以及基于 Chrome 的浏览器)中的渐进式网络应用程序最近的大部分工作来自于项目 Fugu ，Chromium 团队将其描述为“弥补网络功能差距的努力，使新类别的应用程序能够在网络上运行。”Project Fugu 的一些新功能包括 Web Bluetooth API、WebUSB API、Web Share Target(允许 Web 应用程序出现在 Android 的共享菜单中)、Web Share API Level 2(允许图片和其他文件从 Web 应用程序共享到本机应用程序)、Contacts API 和几十个其他有用的功能。
