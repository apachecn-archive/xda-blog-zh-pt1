Windows 11 build 22000.469 发布了大量补丁

微软为 Windows 11 发布了一个新的可选累积更新，随着设置中的新页面，还有一堆修复。

微软将于本月发布可选的 Windows 11 累积更新，也称为 D 周更新。这是我们下个月看到补丁星期二更新之前的预览更新。

并且在变更日志中有很多修正。这并不罕见；这只是微软如何记录这些事情，即使这一个比正常的长一点。可选更新列出了所有的更改，然后它只是一种假设，这是所有捆绑到强制性更新，在每月的第二个星期二到达。

不过有一个变化，那就是在“设置”的“帐户”部分有一个新的“你的 Microsoft 帐户”页面。其余的都是修复，包括任务栏中的音量图标错误地显示为静音，蓝牙修复，等等。

当然，这里是修复的完整列表:

Windows 11 构建 22000.469 修复程序

  • 提高 ARM64 设备上应用安装的可靠性。
  • 约旦将夏令时从 2022 年 2 月开始，而不是 2022 年 3 月。
  • 解决了导致 ARM64 设备在休眠或从休眠中恢复时停止响应的问题。
  • 解决了在 Microsoft Edge Internet Explorer 模式下影响预测预渲染的问题。
  • 解决了有时会阻止您在输入法编辑器(ime)中输入字符串的问题。
  • 解决了在某些支持硬件加速蓝牙音频的设备上导致音频服务停止响应的问题。
  • 解决了通知客户 Windows 更新进度的文本对于日语不正确的问题。
  • 解决了应用程序不运行时影响图标的问题。在任务栏上，这些图标可能会显示为活动状态，就像应用程序正在运行一样。
  • 解决了可能导致 VPN 配置文件消失的问题。当您使用 Microsoft Intune 或第三方移动设备管理(MDM)工具在 Windows 11(原始版本)上部署 VPN 配置文件时，会出现此问题。
  • 解决了影响只与 Azure Active Directory (AAD)集成的应用程序的问题。这些应用程序将无法在加入 Active Directory 联合身份验证服务(ADFS)的计算机上运行。
  • 解决了在试图报告受信任的平台模块(TPM)信息时可能导致 Get-TPM PowerShell 命令失败的问题。该命令失败，并显示错误“0x80090011 Microsoft。Tpm.Commands.TpmWmiException，Microsoft。Tpm.Commands.GetTpmCommand "。
  • 解决了导致远程桌面协议(RDP)会话断开连接或服务器核心屏幕黑屏的问题。安装 AppCompat 功能时会出现此问题。
  • 解决了一个影响 windows.system.profile.retailinfo.dll 的问题。
  • 解决了在浏览文件和选择文件时影响文件资源管理器性能的一些问题。
  • 向家庭版和专业版的 Windows 设置中的帐户类别添加新的您的 Microsoft 帐户页面。
  • 解决了任务栏中音量图标不正确显示为静音的问题。
  • 解决了导致文件资源管理器和桌面上下文菜单停止工作的可靠性问题。
  • 解决了使用朝鲜语 IME 时无法将 Shift 键事件传递给应用程序的问题。
  • 添加 HelpWith 功能，该功能使用 Microsoft Bing 技术来建议与每个设置页面相关的帮助主题。
  • 解决了当设备拥有 Microsoft 帐户(MSA)时，触摸键盘无法出现在锁定屏幕上的问题。
  • 解决影响任务栏上徽章信息加载的问题，该问题有时会导致设备停止工作。
  • 解决了阻止某些选项出现在 Win+X 菜单上的问题。
  • 解决了当设备连接到多台显示器时导致设备停止工作的问题。
  • 解决影响任务栏自动隐藏功能的问题。当您将鼠标悬停在主显示器或辅助显示器上时，任务栏可能不会可靠地出现。
  • 解决了有时会阻止您使用简体中文 IME 的问题。
  • 解决了可能会阻止图标出现在辅助显示器的任务栏上的问题。
  • 解决了在安装打印机设备驱动程序时无法安装某些打印机配套应用程序的问题。
  • 解决了在设置中的蓝牙和其他设备页面上显示已连接蓝牙设备的过期电池百分比的问题。
  • 解决了阻止 IP 摄像机连接和流式传输到某些 DirectShow (DShow)应用程序的问题。
  • 改进了自动亮度算法，在所有支持的系统上，在低光照条件下提供更好的响应。
  • 解决了导致lsass.exe停止工作和设备重启的问题。当您在 NTDS 服务停止后查询 Windows NT 目录服务(NTDS)计数器时，会出现此问题。
  • 解决导致 WebDav 重定向器死锁的问题。当您尝试从本地 TfsStore 读取文件时，会发生此问题，这会导致系统停止响应。
  • 解决启用更新序列号(USN)日志时出现的性能退化问题。
  • 解决了在某些情况下无法应用组策略对象(GPO)“不允许在所有 NTFS 卷上压缩”的问题。
  • 解决了阻止 Robocopy 重试文件复制过程的问题。
  • 解决了一个已知问题，该问题可能会阻止某些图像编辑程序在某些高动态范围(HDR)显示器上正确呈现颜色。这通常会影响可能以亮黄色或其他颜色显示的白色。
  • 解决了导致 Windows 停止工作并生成错误“IRQL _ 非 _ 小于 _ 或 _ 等于”的问题。
  • 解决调用 WinVerifyTrust () 时发生的内存泄漏。如果对具有多个签名的文件的第一个签名验证失败，就会出现此问题。
  • 向 Active Directory 域控制器添加审核事件，以识别不符合 RFC 4456 的客户端。有关更多信息，请参见 KB5005408:智能卡认证可能导致打印和扫描失败。

与可选更新一样，您可以通过 Windows Update 检查更新，然后选择下载和安装选项来获取它们。如果您不选择安装它，这些修复将被捆绑到下个月的补丁星期二更新中，这是强制性的。
