谷歌用安卓-Windows 的邻近份额挑战隔空投送

谷歌正致力于在今年晚些时候将本地附近共享支持引入 Windows PCs。请继续阅读，了解更多信息。

除了一系列新功能使 Android 手机和 Chromebooks 比以前更好地协同工作外，谷歌还在开发一项新功能，以提供 Android 手机和 Windows PCs 之间的无缝互操作性。该公司在其 CES 2022 主题演讲中强调了这一即将推出的功能，根据提供的信息，该功能似乎是 Windows PCs 和 Android 手机的隔空投送替代品。

谷歌表示，即将推出的功能将让用户使用 Fast Pair 轻松地将他们的 Android 手机连接到兼容的 Windows PC。设置完成后，用户还可以连接蓝牙配件，同步短信，并与附近的 share 共享文件。

目前，谷歌尚未分享有关该功能的完整细节，但该公司表示，它正在与宏碁、惠普和英特尔等原始设备制造商合作，在今年晚些时候将这些功能引入 Windows PCs。

不为人知的是，谷歌一直致力于为 Windows PC 带来邻近共享支持。回到去年 7 月，附近的 Share 通过谷歌 Chrome 出现在 Windows 上。然而，该功能当时并不可用。不过，谷歌似乎已经搁置了这一实现，因为上面的视频显示了 Windows 上不需要谷歌 Chrome 的原生 nearyshare 实现。

如前所述，谷歌在 CES 2022 上宣布了一系列新功能。如果你错过了公告，你可以查看我们以前的报道，了解更多细节。
