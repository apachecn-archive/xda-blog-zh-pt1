苹果不会在第三方屏幕修复后禁用 Face ID

原文：https://www.xda-developers.com/apple-wont-disable-face-id-after-third-party-screen-repairs/#0001-01-01

苹果放弃数百万，不会在第三方屏幕修复后禁用 Face ID

在一波投诉之后，苹果公司决定撤销其在第三方屏幕修复后禁用 Face ID 的决定。

继 9 月份发布 iPhone 13 系列后，一次拆解透露了一个令人不快的惊喜。苹果最新旗舰手机在第三方屏幕修复后禁用 Face ID。这引起了网上的愤怒，因为许多小型独立商店依靠屏幕维修来维持生计，许多 iPhone 用户更喜欢第三方维修，因为他们更实惠。因此，这家市值万亿美元的公司决定取消这一限制。

这家库比蒂诺的科技巨头已经告诉 The Verge 它计划在未来的软件更新中取消这一限制。由第三方进行的屏幕修复将不再禁用 iPhone 的面部识别系统。这将允许没有苹果授权中心的国家的客户在他们破解屏幕时恢复正常使用他们的 iPhones。

苹果公司在 iPhone 13 系列中增加了一个微控制器。在屏幕修复之后，微控制器必须使用苹果的特殊修复工具将新屏幕与 iPhone 配对。独立的服务提供商无法访问它，所以手机检测到这一点。这会导致“无法在此 iPhone 上激活 Face ID”错误。

在未来的软件更新后，此错误将不再显示，在第三方屏幕修复后，Face ID 应该可以正常工作。苹果还没有具体说明更新将于何时到来，但我们很高兴它正在工作中。苹果有过限制 iPhone 用户获得第三方维修的历史。该公司曾在第三方触控 ID home 键维修后完全屏蔽了 iPhones。iPhone 在尝试重启时会显示错误。苹果最终决定只禁用触控 ID，而不是整部手机。

除了垄断维修行业之外，不清楚苹果为什么会采取这些苛刻的处理方式。该公司甚至可以通过撤销禁用 Face ID 的决定来避免一些潜在的诉讼。
