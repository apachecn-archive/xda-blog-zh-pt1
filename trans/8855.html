Microsoft Office 受 Windows 11 启发的设计现已推出

原文：https://www.xda-developers.com/microosft-office-windows-11-design-update/#0001-01-01

微软今天宣布，该公司已经开始向公众推出其 Windows 办公应用套件的新设计。微软最初在夏天宣布了这一新的 Office 应用设计，旨在与两个月前刚刚发布的 Windows 11 保持一致。

新设计的特点是更简洁的外观，更强调每个应用程序的强调色。相反，默认情况下，窗口的框架以及应用程序的其余部分将遵循系统的深色或浅色主题，尽管你可以改变这一点。功能区看起来也有点不同，它现在看起来更像一个弹出窗口的浮动面板。它还使用了圆角，这是典型的 Windows 11 风格。

微软表示，它还在努力增加对云母透明效果的支持，这是 Windows 11 的特点。如果你不熟悉它，这是你在 Windows 11 开始菜单、设置应用和操作系统中其他使用透明的部分看到的效果。

虽然新的 Office 设计受到了 Windows 11 的启发，但 Windows 10 用户也可以使用它，尽管云母效果对他们来说不可用。此外，Windows 11 用户显然会优先考虑这种设计。如果您正在运行新的操作系统，并且您在当前渠道中注册了 Microsoft 365 订阅或拥有 Microsoft Office 2021 永久许可证，那么您可以通过使用 Office 应用程序标题栏中的“即将发布”按钮来立即访问更新。如果你仍在运行 Windows 10，目前只有部分用户可以这样做，更新将随着时间的推移继续推出。微软表示，目前有 50%运行最新版 Office 的用户可以使用这一更新。

如果您正在尝试启用此功能，有一些事情值得注意。“即将推出”切换仅在 Word、Excel、PowerPoint、OneNote 和 Outlook 中可用，但 Outlook 不允许您启用新外观。您需要在其他四个应用程序中启用它，但一旦启用，它将应用于您的所有 Office 应用程序，包括 Access、Outlook、Project、Publisher 和 Visio。
