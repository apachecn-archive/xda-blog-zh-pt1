Chrome 95 将 Material You 带给每个人，增加了安全支付确认等功能

原文：https://www.xda-developers.com/chrome-95-material-you-for-everyone-secure-payments/#0001-01-01

上个月，谷歌推出了 Chrome 94，带来了一些新的改进，比如 HTTPS 优先模式、Chrome 共享中心、更简洁的设置页面等等。现在谷歌又带着新版本 Chrome 95 回来了。最新的 Chrome 版本带来了你在 Android 12 上为每个人改头换面的材料，安全支付确认，保存标签组的能力，等等。

虽然谷歌在过去几个月里一直在缓慢地用 Material You elements 更新 Android 版的 Chrome，但用户必须启用一些 Chrome 标志才能接触到新设计。但是 Android Police 报道称，在 Chrome 95 上，你重新设计的材料现在在 Android 12 上默认可以被每个人访问。

Chrome 95 还为 WebAuthn 引入了新的支付扩展，为在线支付提供一致、低摩擦、强认证体验。

“安全支付确认借助 WebAuthn 增强了网上支付认证体验。该功能为 WebAuthn 添加了一个新的“支付”扩展，允许银行等依赖方创建一个公共密钥凭证，作为在线结账的一部分，任何商家都可以通过支付请求 API 使用“安全支付确认”支付方法进行查询，”谷歌在一篇文章中写道。

接下来，Chrome 95 允许 web 应用程序将自己注册为自定义 URL 的处理程序。这将使 web 应用程序的行为更像本机应用程序。

在其他地方，最新的 Chrome 版本增加了保存标签组的选项。然而，这一功能目前被隐藏在一面旗帜后面。要尝试一下，你需要启用chrome://flags/# tab-groups-save并重启浏览器。
