NewPipe 0.21.14 带来了描述片段和一个新的“添加到播放列表”选项

开源 YouTube 播放器 NewPipe 已经收到了一个更新，现在可以向您显示描述片段，并允许您将视频添加到播放列表中。

NewPipe 是一个开源的 YouTube 客户端，可以让你在没有 YouTube 应用程序的情况下在智能手机上播放 YouTube 视频。如果你不喜欢 Android 上的 YouTube 应用程序，或者没有安装 Google Play 服务，这是 YouTube 应用程序的一个要求，那么这很有帮助。在没有谷歌服务的地区，或者在华为等品牌的手机上没有预装谷歌服务的地区，NewPipe 可能是 YouTube 应用程序的一个很好的替代品。它还会定期更新新功能，比如能够播放无数字版权管理的音乐和 YouTube 章节支持。现在，最新的 NewPipe 0.21.14 更新带来了描述片段和添加到播放列表选项等变化。

最新的 NewPipe 更新不仅引入了新功能，还带来了一些生活质量的改善和修复，这将进一步改善用户体验。第一个新特性是在搜索视频时支持描述片段。这实际上是显示在搜索结果中每个视频下的视频描述的一小部分。这有助于观众了解视频的基本概念。另一个新特性是支持打开 y2u.be 链接——基本上，链接使用这样的链接缩写。

一个有用的新增功能是“添加到播放列表”选项，它现在是 NewPipe 打开菜单的一部分。当你点击应用程序外部的链接并选择用 NewPipe 打开它时，你现在会看到一个选项，可以将视频添加到播放列表中，以便稍后观看。如果您有多个视频链接，并且想要一个接一个地排队回放，这可能会很有用。如果你已经从 GitHub 下载，NewPipe 还通过在应用程序本身中添加一个检查更新选项，使应用程序的更新变得更加容易。

最后，当你长按播放列表中的视频时，有一个新的标记为观看过的选项。如果您在一个播放列表中有多个视频，并且忘记了您已经观看了哪些视频，您可以使用此选项。除了这些新功能，该应用程序还增加了一些改进。其中包括弹出播放器打开时的 toast 通知、回放控件的改进布局和新的提要刷新体验。

你可以从其 GitHub 页面下载最新版本的 NewPipe。以下是 NewPipe 0.21.14 的完整变更日志，以便您更好地了解所有新功能和修复:

  • 新
    • 在共享菜单中添加“添加到播放列表”选项
    • 在更新设置中添加“检查更新”按钮
    • 添加对 y2u.be 链接的支持
    • 支持 PeerTube 短链接

  • 改进后的

    • 在队列长按菜单中增加了“显示频道详情”菜单项
    • 在所有 XML 资源中使用自定义文本视图和编辑文本
    • [提要]保存“显示观看的项目”开关的状态
    • 简单的回放速度控制改进
