AdDuplex 表示，Windows 11 的使用份额现已超过 16%

原文：https://www.xda-developers.com/windows-11-usage-share-adduplex-january-2022/#0001-01-01

尽管用户最初遇到了一些问题，但微软似乎在 Windows 11 上取得了相当成功的推出。为微软商店上的应用程序提供广告框架的公司 AdDuplex 最近发布了一月份的使用报告。自上次报告以来的两个月里，回到 11 月，AdDuplex 数据显示，Windows 11 的使用份额几乎翻了一番，达到 16.1%。11 月份，这一比例为 8.6%。

像往常一样，需要注意一些事情。该报告基于从大约 60，000 台正在调整包含 AdDuplex SDK v.2 的应用程序的 PC 收集的数据。从各方面来看，这是一个相对较小的样本规模。最重要的是，这里的百分比只是相对于运行某个版本的 Windows 10 或 11 的设备总数，因为只有这些应用程序才能拥有 AdDuplex SDK。这意味着我们不再考虑旧版本，如 Windows 7 或 8。

尽管如此，对于 Windows 11 来说，这些都是不错的数字，而且它们符合微软最近的声明，即 Windows 11 的采用比预期快得多。本周早些时候，该公司宣布人们升级到 Windows 11 的速度是升级到 Windows 10 的两倍，客户满意度也是任何版本的 Windows 中最高的。当你考虑到许多 Windows 10 PCs 不符合 Windows 11 的系统要求，无法正常升级时，这就更加令人印象深刻了。

除了 Windows 11，我们还看到 Windows 10 版本 21H2 也有显著增长，达到 12.1%的使用份额。如果用户的电脑不支持 Windows 11，这是用户正在升级的版本，而且它也可能在商业场景中被更多采用，因为它比 Windows 11 小得多。如果我们把这两个版本加起来(总计 28.2%)，它们是第二大最常用的 Windows 版本。

使用最多的版本仍然是 Windows 10 版本 21H1，占 28.6%，但这比 11 月份的 36.3%大幅下降，因此很明显用户正在快速更新 Windows 11。Windows 10 版本 20H2 也从 31.8%下降到 26.3%，所以越来越多的用户跳到了 Windows 的最新版本。

纵观过去的 Windows 趋势，Windows 11 并不是增长最快的 Windows 10 版本，但当你考虑到许多 PC 无法安装更新时，它已经做得相当不错了。当您添加 Windows 10 版本 21H2 时，最新的 Windows 版本正在以非常快的速度增长。

提醒一下，微软下个月将在 Windows 11 中增加对 Android 应用程序的支持，所以看看是否有更多的用户认为这是一个足够好的升级理由会很有趣，如果他们还没有升级的话。
