讨论:三星的One UI 3.0更新是否领先？

原文：https://www.xda-developers.com/discussion-samsung-good-software-updates-one-ui/#0001-01-01

如果你过去几个月一直在关注，你会注意到三星一直在通过软件更新来扼杀它。除了坚持不懈地向各种旗舰和旗舰杀手推出最新的基于Android 11的One UI 3.0更新之外，三星还以类似的待遇奖励了几款中端设备。除了这种值得称赞的及时性，该品牌还值得我们赞赏，因为它更新了其他品牌经常忘记的mid-rangers。因此，在本期我们的每周讨论系列中，我们问你，在软件更新方面，你是否认为三星在安卓制造商中处于领先地位？

虽然三星似乎处于行业的顶端，但几年前的情况肯定大不相同。对该品牌的全球声誉具有讽刺意味的是，三星还因推迟软件更新、限制旗舰设备的关键更新(如Android版本增量更新)以及通过其TouchWiz用户界面提供混乱的用户体验而受到指责，该界面后来被更新为三星体验。三星通过在2018年的第一次 One UI更新，对UX进行了重大改造，解决了人们对它的抱怨。不仅如此，最近几年，最新更新的频率和可用性也有了很大的提高。

三星于2020年10月开始发布第一个UI 3.0测试版，随后于2020年12月开始推出稳定的更新，首先是威瑞森上的 Galaxy S20系列。从那时起，一系列设备已经收到了稳定的 One UI 3.0 更新。该列表包括:

除了这些设备，三星最新的 Galaxy S21系列预装了一个UI 3.1 的，这带来了一些额外的功能，如改进的谷歌Home设备控制。 Galaxy Tab S7 也更新了一个UI 3.1，增加了一系列功能来增强协作和生产力。

然而，尽管如此迅速，三星仍在一个领域落后，那就是支持Android 上的无缝更新。这项功能是Android Nougat推出的，它允许在手机仍在使用时在后台安装新的更新。虽然安装更新通常只需要几分钟，但我们不能否认节省时间也是一个好处，尤其是在我们不断依赖手机的时代。在谷歌没有对制造商强制进行无缝更新之前，我们可以期待三星拥有自己的甜蜜时光。
