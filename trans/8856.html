如果你想下载 Chrome，微软 Edge 现在会说它“太过时了”

原文：https://www.xda-developers.com/microsoft-edge-chrome-2008/#0001-01-01

可以说，微软已经走得太远了，它把新的 Edge 浏览器强加给用户，让你在 Windows 11 上切换默认浏览器变得更加困难，还屏蔽了将某些 Windows 链接重定向到你最喜欢的浏览器的应用程序。但事实证明，事情总是会变得更糟，因为如果你试图使用 Edge 下载谷歌 Chrome，微软的浏览器现在会主动侮辱它。据 Neowin 报道，当你在微软 Edge Canary 中打开 Chrome 的下载页面时，你现在可能会看到一条消息，上面写着“这款浏览器太 2008 了”。这条信息自然会敦促你继续使用 Edge，而且也没有明显的方法可以忽略它。

事实上，你可能会看到两条不同的消息，另一条只是说 Edge 运行在与 Chrome 相同的技术上，“得到了微软的更多信任”。我们无法让这些消息显示在我们的终端上，但您可以在下面看到它们。

很自然，微软认为它的浏览器比竞争对手更好，但它很容易被认为这种行为是对用户的敌意。如果你使用必应搜索任何其他浏览器，微软已经试图阻止用户下载竞争对手的浏览器，Edge 的广告横幅占据了页面的很大一部分。这种额外的尝试，加上我们上面提到的微软的其他有争议的举动，似乎有些绝望。为微软说句公道话，谷歌也因在自己的网页上大力宣传其 Chrome 浏览器而闻名，但它没有像微软那样走得那么远。

与此同时，微软一直在将一些不必要的功能与 Edge 浏览器捆绑在一起。最近，该公司在浏览器中增加了“现在购买，以后支付”功能，这可能会鼓励用户花更多的钱。这一直是强烈反对的目标，最终可能会赶走一些用户。

我们只能希望尽快从 Edge 中删除这些消息，但事实是，这些消息是在它们本不应该出现的时候被推给用户的。
