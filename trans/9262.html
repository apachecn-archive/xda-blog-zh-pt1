Windows 11 照片应用程序获得了重新设计的编辑器和新的更新

原文：https://www.xda-developers.com/windows-11-photos-app-redesigned-photo-editor/#0001-01-01

微软推出了 Windows 11 照片应用的新更新，包括照片编辑器的重大重新设计。我们今天在 Windows 11 PC 上收到了照片应用程序的更新，运行的是生产版本(非内部人员)，所以它应该会向 Windows 11 上的每个人推出。

虽然微软为 Windows 11 推出了一款新的照片应用，并进行了一些设计调整和新功能，但照片编辑体验与 Windows 10 版本相比基本没有变化。现在，一个完全不同的界面正在推出。当您单击编辑按钮时，您将立即看到更改。裁剪工具不再显示在侧边，取而代之的是，您可以使用图像下方的控件旋转图片，包括垂直或水平翻转图像的功能。以前，您只能水平翻转照片。

裁剪控制也有所增强。现在，当选择裁剪的宽高比时，您可以立即选择横向或纵向。以前，你必须选择一个通用的宽高比，然后用一个单独的按钮选择是横向还是纵向。此外，当您调整裁剪区域时，您现在可以看到图像被裁剪到的确切分辨率。

另一个变化是，如果你试图编辑一个透明的图像，你会看到典型的方格背景，表明背景是透明的。这只发生在编辑图像的时候，当你只是在查看它的时候，它将会是简单透明的。

Windows 11 照片编辑器更新还包括调整选项卡中的更改。现在默认显示所有控件，因此您不必展开任何菜单选项。然而，“清晰度”滑块似乎已经被移除。你也不再有像红眼消除和斑点修复这样的功能，希望这只是一个疏忽，将在未来的更新中解决。同时，Filters 选项卡有一些新的过滤器，它们有新的名称，这应该会使每个过滤器的功能更加清晰。

最后一个变化是，在普通照片编辑器中，在图像上绘图的能力现在在一个名为“标记”的标签中。以前，这是一个完全独立的模式，所以如果你想进行一般的编辑，然后在图像上涂鸦，你必须做一件事，保存图像，然后返回去做另一件事。这将简化流程。

据我们所知，这些是此次更新照片应用程序的唯一变化，但如果您发现了其他不同之处，请告诉我们。如果你还没有收到更新，请查看微软商店是否有可用的更新。你拿到的 app 版本应该是 2021.21120..8011.0.

微软最近也开始在测试频道向内部人士推出记事本应用的重大更新。这个更新一个月前在 Dev channell 中首次亮相，它包括一个新的黑暗模式和其他 UI 变化，以配合 Windows 11 设计语言。最近对新的媒体播放器应用的更新也使得在非业内人士的电脑上安装成为可能，如果你下载 MSIXBUNDLE 安装文件的话。你可以利用这个网站来做到这一点。
