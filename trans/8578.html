谷歌浏览器将很快让你捕捉和编辑截图

原文：https://www.xda-developers.com/google-chrome-capture-edit-screenshots/#0001-01-01

谷歌 Chrome 最近一直在推出新功能，如新的安卓应用小工具、手机上更容易的密码访问，以及正在进行的搜索侧面板。谷歌在最近的 Chrome 94 更新中添加了在 Android 版 Chrome 中捕获截图的功能，但目前还没有桌面版的截图工具。谢天谢地，谷歌现在正在测试一种捕捉和编辑网页截图的方法。

最新的 Chrome 98 Canary builds 在 Chrome 中有两个与截图相关的标志://flags(viaTechdows):# sharing-desktop-screens(“允许从桌面共享中心截图”)和# sharing-desktop-screens-edit(“为在桌面上创建截图的用户启用编辑流程”)。一旦两个标志都设置为“启用”，Chrome 地址栏的发送菜单中就会出现一个截图项目。

一旦你按下截图选项，你可以拖动鼠标选择当前网页的一部分。释放鼠标捕获截图并将其保存到您的剪贴板，您可以单击“下载”按钮将其保存为文件。与 Firefox 的截图功能不同，在截图时没有滚动页面的选项。您也不能选择屏幕截图文件的确切位置，如果您点按“下载”按钮，它会一直保存到您的“下载”文件夹中。

屏幕截图弹出菜单上还有一个“编辑”按钮，但目前，这只是打开一个空白页面。目前还不清楚 Chrome 的屏幕截图编辑器将提供哪些标记选项，但至少可以绘制(或模糊)屏幕截图的部分。

一些基于 Chromium 的浏览器已经实现了自己的截图工具，比如微软 Edge 上的 Web Capture(确实支持滚动和基本编辑)。随着谷歌在 Chrome 上添加了一个截屏工具，基于 Chrome 的网络浏览器实现自己的类似工具应该会变得更容易。
