Chrome 正式推出“旅程”功能，为你的安卓小工具提供素材

原文：https://www.xda-developers.com/chrome-journeys-android-widgets/#0001-01-01

谷歌 Chrome 正式推出“旅程”历史功能和 Android 小工具材料

谷歌在 Chrome 上推出了一个新的“旅程”功能，将你的浏览器历史组织成类别，以及其他变化。

谷歌 Chrome 是世界上发展最快的软件之一，谷歌和许多其他公司都为其发展做出了贡献。Chrome 中定期出现新功能，现在谷歌宣布在桌面上推出 Chrome 新功能，可以帮助组织你的浏览历史。

谷歌周二在一篇博客文章(通过 Android Police )中宣布了“旅程”功能，该功能将你的浏览历史组织成类别，并突出显示给定类别/主题中(Chrome 认为是)最重要的页面。该公司表示，“当你在搜索栏中键入一个相关的词，然后点击“继续你的研究”或访问 Chrome 历史旅程页面，你会看到一个你访问过的相关网站的列表，并可以快速找到你离开的地方，无论是今天早些时候还是几周前。旅程甚至会考虑你与一个网站互动的程度，将最相关的信息放在最前面和最中心，同时还会为你提供有用的建议，让你下次尝试相关搜索。”

谷歌很快指出了旅程中内置的隐私保护。你可以删除单个项目或整个活动群，组织只发生在你的设备上——如果你登录到谷歌帐户，你的整体浏览历史仍然是同步的，除非你也关闭它。如果你根本不想看旅程，Chrome 也可以选择关闭旅程。新功能似乎与谷歌在 Topics API 上的工作有关，该 API 分析你的浏览器历史，以帮助网站创建有针对性的广告，但谷歌在其声明中没有证实这是否有关系。

旅程功能将在所有平台的 Chrome 桌面上推出，包括英语、德语、西班牙语、法语、意大利语、荷兰语、葡萄牙语和土耳其语。谷歌还为每个人推出了一些已经在测试中可用的其他功能，如额外的 Chrome 操作(如“分享这个标签”和“查看你的 Chrome 历史”)和你在 Android 上添加的材料。
