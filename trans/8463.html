Windows 11 现已面向所有 Windows 365 客户推出

原文：https://www.xda-developers.com/windows-11-is-now-available-for-all-windows-365-customers/#0001-01-01

这是微软 Ignite，该公司今年最大的活动。像往常一样，有大量的公告，正如你所料，有几个与 Windows 11 有关。最值得注意的是，所有 Windows 365 用户现在都可以使用新的操作系统。你可能还记得，当 Windows 365 宣布基本上是一个流媒体操作系统或云 PC 时，它向 Windows 10 提供了 Windows 11 的承诺。Windows 365 企业用户已经可以使用它，现在有新的工具可以帮助 Windows 365 企业用户过渡到最新版本的 Windows。这个现在有了。

这项服务还有其他几项改进。有一个新的可配置的宽限期。现在，有一种方法可以设置云电脑，使其在七天的宽限期后取消供应。它现在可以立即结束宽限期，这个更新是在预览。

现在将支持加入 Azure AD 的云电脑，所以你不需要自己的 Azure 基础设施。在 Microsoft Endpoint Manager 中，将会有关于 CPU 和 RAM 性能报告的新指标。显然，这是为了改善对云 PC 的支持。这两个更新现在都在预览中。

最后，windows365.microsoft.com 实际上也在进步。您将能够更改本地资源设置，如打印机和键盘，并且可以在会话期间编辑设置。这是在预览。

还有一些其他与 Windows 相关的改进正在进行中。微软网上 Excel 将在今年晚些时候支持通用打印，这意味着你可以在你单位的任何打印机上打印。Microsoft 365 的测试基础现已正式推出。这是为了更容易地测试具有功能更新的应用程序，说到 Windows 11 功能更新，你将能够使用 Microsoft Intune 管理这些更新。
