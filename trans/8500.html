苹果 ID 网站得到了一个逾期的翻新

原文：https://www.xda-developers.com/apple-id-website-gets-an-overdue-facelift/#0001-01-01

苹果 ID 网站在过去几年看起来都一样。幸运的是，苹果公司终于对它进行了一次迟到的翻新，彻底取代了旧界面。在过去，访客会在一个登录/启动屏幕上看到随机的人使用不同的苹果设备。现在它有了一个更现代的外观，与改版后的 iCloud 网站相匹配。

据 MacRumors 报道，新的用户界面(UI)于昨日上线。苹果用户的照片已经被一个苹果标志所取代，当你滚动经过它时，彩色的圆点会产生动画效果。登录后，你会看到一个卡片视图，里面整齐地列出了你的信息。

在左侧，您会找到与您的 Apple ID 数据和设置相关的部分列表，如个人信息，包括您的姓名、生日等。支付方式，顾名思义，包括你所有保存的银行卡和相关信息。在“家庭共享”中，您可以找到 iCloud 家庭的成员。至于设备，你可以检查并移除所有链接到你的 Apple ID 的 Apple 设备。

苹果最近也更新了 iCloud 网站。它用一个与苹果操作系统用户界面相匹配的更时尚的应用程序取代了过时的邮件网络应用程序。它还调整了 iCloud 网络应用的其余部分，以支持最新版本的 iOS 和 macOS 中引入的新功能。

通过appleid.apple.com可以访问新的 Apple ID 网站，通过icloud.com可以访问 iCloud 网站。苹果继续关注其网络应用，包括维护苹果音乐和苹果电视网络客户端。这可能是为了让他们更容易在更广泛的平台和设备上访问，从而增加订户的数量。
