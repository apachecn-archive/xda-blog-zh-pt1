最新的 StartAllBack 测试版使 Windows 11 黑暗模式更加一致

原文：https://www.xda-developers.com/startallback-beta-windows-11-dark-mode-control-panel/#0001-01-01

虽然 Windows 11 对用户界面进行了一些改进，但黑暗模式在整个操作系统中仍然不太一致，特别是一些传统元素仍然可用。谢天谢地，有第三方工具试图解决这个问题，比如 StartAllBack。StartAllBack 的最新测试版为 Windows 11 的更多部分带来了黑暗模式，包括控制面板。

微软多年来一直试图摆脱传统的控制面板，许多设置已经转移或正在转移到新的设置应用程序中。但是如果你还想用控制面板。StartAllBack 版本 3 . 2 . 9——目前只提供 beta 版——使其在黑暗模式下可用。所有的控制面板页面都被重新设计为黑暗模式，链接也做了调整，所以它们会跟随你的主题颜色，而不是总是蓝色。

此外，Windows 11 中的文件传输对话框也已更新，现在支持黑暗模式，为体验带来了更大的一致性。这一切都是在 Windows 11 的其他部分之上，由于 StartAllBack，这些部分已经可以在黑暗模式下使用，包括运行对话框和其他窗口对话框，如错误和警告消息。这些都是微软还没有更新的东西，所以很高兴看到第三方工具来解决这些问题。

StartAllBack 是 Start 菜单的替代品，去年 9 月作为 StartIsBack 的新版本首次亮相。它是为 Windows 11 从头开始构建的，除了取代开始菜单，它还可以恢复对文件资源管理器等的更改。它还恢复了控制面板中的一些经典小程序，包括旧式的个性化选项，以及 Windows Insider builds 中已移动到设置应用程序的一些选项。也许最重要的是，它比默认的开始菜单实现具有更好的性能，因为它使用更少的资源。
