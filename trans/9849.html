Chrome 的密码管理器正在开发一个“发送密码”按钮

原文：https://www.xda-developers.com/google-chrome-send-password-button/#0001-01-01

Chrome 的密码管理器可能很快会让你发送保存的密码

正如 Chromium 经常爆料者@Leopeva64 所发现的，谷歌在 Chrome 的内置密码管理器中添加了一个新的“发送密码”按钮。

谷歌 Chrome 的内置密码管理器通过记住我们所有的密码并提供跨网站和应用程序的自动填充，使我们的数字生活变得更加容易。而且还会变得更好。谷歌正在为密码管理器开发一项新功能，让用户可以轻松安全地共享密码。

正如经常使用 Chrome 的人所发现的那样，谷歌将很快在 Chrome 的内置密码管理器中添加一个新的“发送密码”按钮。该按钮可从密码管理器中每个保存的用户名/密码条目旁边的三点溢出菜单中访问。

 

“发送密码”按钮在 Chrome 中还没有功能，所以我们不知道它到底是如何工作的。Chrome 已经允许你复制密码，可以用来共享密码，或者在登录字段中手动粘贴密码。“发送密码”有可能做到这一点，但会更加无缝和安全。为了防止滥用，Chrome 可能会要求用户在共享密码时验证他们的身份，就像你试图复制密码时一样。

Chrome 频道还没有共享密码的功能。在进入稳定频道之前，它最有可能首先到达 Chrome Canary。我们会密切关注，并在它上线时通知您。

谷歌 Chrome 现在还允许你手动添加密码到密码管理器。然而，这个特性还没有在稳定版本中实现(目前只在 Canary 版本中可用)。此外，Chrome 的密码管理器也在开发一项新功能，可以让你在保存的密码中添加注释。
