Realme UI 3.0 测试版为 Realme GT Master Edition 发布

原文：https://www.xda-developers.com/realme-ui-3-0-android-12-beta-realme-gt-master-edition/#0001-01-01

上个月，Realme 为 Realme GT Neo 2、 Realme GT 和 Realme X7 Max 5G 发布了基于 Android 12 的 Realme UI 3.0 早期版本。该公司现在邀请 Realme GT Master Edition 的用户试用 Android 12 的早期版本。

基于 Android 12 的 Realme UI 3.0 实践:基于壁纸的主题，新的黑暗模式选项，等等！T3】

根据 Realme 社区论坛上最近的一篇帖子，Realme 已经为 Realme GT 主版本开放了 Realme UI 3.0 Ealy 访问计划。beta 计划允许精选用户在公开发布之前试用基于 Android 12 的 Realme UI 3.0 的早期版本。

如果你自豪地拥有一台 Realme GT，并想测试新软件，你需要将你的手机注册到测试程序中。为此，请前往设置>软件更新。在这里，点击右上角的齿轮图标 >试用版>立即申请并填写所需信息。如果你的申请获得批准，你将通过 OTA 收到 Realme UI 3.0 更新。由于测试程序的座位有限，你可能要抓紧时间。

由于这是测试版软件的常见问题，Realme GT Master Edition 的 Realme UI 3.0 可能会有点粗糙，所以不要指望一切都能顺利工作。此外，在安装更新之前，请务必备份您的重要数据。如果您遇到任何错误，您可以使用反馈日志包直接向软件开发团队报告。
