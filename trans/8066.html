Windows 11 现已全面上市——以下是立即获取的方法

原文：https://www.xda-developers.com/windows-11-generally-available-how-to-get-it/#0001-01-01

今天，微软宣布 Windows 11 正式上市，所以根据你在哪里，你现在就可以得到它。这可能看起来有点早，但该公司正在采取一种不同于往常的方法来推出。新操作系统不是在太平洋时间上午 10 点发布，而是在你们所在时区的 10 月 5 日推出。

Windows 11 有很多新功能。它有一个全新的设计，有一个居中的开始菜单和任务栏，圆角等等。动态磁贴也不见了，因为你现在只能在开始菜单中看到图标。它还配备了一个新的小部件面板，取代了新闻和兴趣，以及全新的收件箱应用程序。

到现在为止，你可能都知道了。你可能只是想知道现在如何获得微软的最新操作系统。

确保您的电脑受支持，或者使用新设备获取 Windows 11

自 2009 年发布 Windows 7 以来，Windows 11 首次提高了最低硬件要求。这意味着虽然这是一个免费的升级，但并不是每个人都可以使用。

首先，确保您的电脑满足 CPU 要求:

  • 英特尔第八代或更新版本(或酷睿 i7-7820HQ)
  • AMD Zen 2 或更新版本
  • 高通骁龙 850 或更新

如果你达到了这些要求，你就可以开始了。还有其他增加，如最低 4GB 内存和 64GB 存储，但任何现代 PC 都应该有这些。另一个大的是 TPM 2.0，但自从 2016 年以来，这一直是新电脑的要求，任何支持 CPU 的东西都应该有它。如果出于某种原因，你的电脑认为你没有 TPM，它可能已被禁用，你可以在这里查看我们的指南。

但如果不符合要求，你有两个选择。当然，你可以继续使用 Windows 10，该系统在 2025 年 10 月之前仍然受支持。有了四年多的支持，微软不会让任何人掉队。

你可以做的另一件事是买一台新电脑。我们已经列出了我们能找到的有资格升级的每一台 PC，以及一堆你可以购买的 Windows 11 就绪设备。

通过 Windows Update 获取 Windows 11

根据您的居住地，从今天开始，您可能会在 Windows Update 中看到它。如果你把这作为一个选项，这绝对是最简单的方法。不幸的是，这是一个分阶段的推出，将持续到 2022 年年中，所以大多数人不会在第一天就获得它。

基本上，如果你有一台全新的电脑，你会看到这个选项。如果它被宣传为可以升级到 Windows 11，那么你将是第一个看到它的人。

使用 Windows Insider 程序

你可以使用 Windows Insider 程序来获得 Windows 11。不要担心，我们不会引导你去得到一些不稳定的预览版本。这是你将要下载的真正的视窗。

  1. 转到设置->更新和安全-> Windows Insider 程序。
  2. 点击“开始”。
  3. 浏览提示，选择发布预览频道。
  4. 出现提示时，重新启动您的电脑。
  5. 在它启动后，返回到 Windows Insider 程序设置，以确保您在发布预览频道。
  6. 转到 Windows Update 选项卡并检查更新。

安装完 Windows 11 后，查看下一步该做什么的部分，或者如果你之前通过 Insider 程序安装了 Windows 11，也可以提前获得。

对于大多数想要并且有资格的人来说，媒体创作工具是一个不错的选择。这一点也不复杂，它只是强迫你的电脑下载并安装 Windows 11。也有很多选择。您可以选择全新安装新的操作系统，升级您的电脑，或者创建一个可以安装在另一台电脑上的 ISO。

首先，前往媒体创建工具下载页面，下载并运行它。之后，只需按照屏幕上的指示。当您进入询问您是否要升级这台电脑或创建 ISO 文件的屏幕时，选择升级选项，您就可以开始了。

如果您参加了 Windows Insider 计划，该怎么办

如果您加入 Windows Insider 计划只是为了测试 Windows 11，您现在可以取消注册。您已经运行产品构建有一段时间了，所以您不需要做任何事情。

这不适用于开发频道。如果你在预发布版本上，那就已经太晚了。为了回滚，您需要进行全新安装。

如果您使用的是测试版或发布预览版，您可以转到“设置”->“Windows Update”->“Windows Insider 程序”，然后选择停止接收 Insider 预览版的选项。这将把你的电脑转移到生产渠道，你将获得定期更新，如下周的补丁星期二更新。

如果你在测试频道，这一点尤其重要。虽然 Dev 通道是用于测试预发布的特性，但 Beta 通道实际上是测试新的特性更新。现在这个更新已经发布了，微软可能会很快将版本 22H2 转移到测试频道，或者可能要等几个月才能完成。如果你不希望这种情况发生在你的电脑上，只需取消注册。您随时可以在以后重新注册。
