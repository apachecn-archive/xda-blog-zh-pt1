Flutter 移动应用现在可以轻松使用谷歌移动广告

原文：https://www.xda-developers.com/google-mobile-ads-flutter/#0001-01-01

没有人喜欢广告，尤其是在屏幕空间有限的移动设备上，但它们通常是支付开发和服务器成本的必需品。谷歌的 Flutter 框架正在成为构建跨平台应用的一种流行方式，但与原生应用或网站相比，广告的选择更少。谷歌的移动广告 SDK 只提供了公共测试版，但现在它已经正式准备好供每个人使用。

“今天，我们很高兴地宣布，经过六个月的测试期后，用于 Flutter 的谷歌移动广告 SDK 现已正式推出。这意味着我们已经准备好让要求最苛刻的 Flutter 应用在生产中显示应用内广告，”谷歌宣布。SDK 支持谷歌的所有移动广告格式，包括横幅、插页、奖励视频、原生广告(与应用的外观和感觉相匹配)和应用开放广告。可以使用 Google AdMob 和 Google Ad manager 设置和管理广告。

尽管 Flutter 在桌面平台上可用——1.5 版本在 2019 年增加了初始的 Windows、Mac 和 Linux 编译——但新的广告 SDK 目前仅支持 Android 和 iOS。如果你是一名开发跨平台桌面 Flutter 应用的开发者，你将不得不使用另一个广告框架，或者在 WebView 容器中嵌入网络广告。

在过去的几个月里，谷歌对 Flutter 框架进行了许多改进。Flutter 2.2 是在今年早些时候的 Google I/O 期间发布的，在新项目中默认启用 null 安全，在服务人员编译的 web 应用中进行资源缓存，以及一个新的 iOS 和 Android 支付插件。最近， Flutter 2.5 于 9 月发布，在 Android 上提供了新的全屏选项，对 Material You 元素的初始支持，升级的 IntelliJ/Android Studio 插件，以及 Visual Studio 代码插件中的新命令。

关于谷歌移动广告 SDK 的更多信息可以在 Dart 包库上找到。
