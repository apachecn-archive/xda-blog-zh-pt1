记事本可能是下一个升级 Windows 11 的应用

原文：https://www.xda-developers.com/notepad-next-app-windows-11-facelift/#0001-01-01

上周，微软开始发布 Windows 11，并附带了一系列重新设计的应用。像许多新功能一样，许多应用程序更新似乎是在最后一刻才整合起来的。有所有新版本的照片、剪贴工具，甚至颜料都没有在发布前几周发布。现在，似乎记事本是 Windows 11 的下一个更新版本。

新应用的一些图片已经泄露，因为它们是在被删除之前由微软工程师展示的。这实际上是经常发生的事情。这是一个奇怪的问题，因为有这么多新的应用程序，他们正在内部测试，然后这些员工试图炫耀 Windows 11。一个解决方案可能是在发布时就准备好这些应用程序。

显然，在记事本中没有什么大的不同。它有一个设置页面，很酷，还有一个新的菜单栏。除此之外，一切都将保持不变。

记住，记事本是一个基本的文本编辑应用程序。有没有机会将有功能添加到它，使它像微软 Word 一样强大。这不是这个产品的用途。事实上，如果 Redmond 公司真的尝试给记事本添加富文本、共享对话框和其他有趣的新功能，我们可能会看到街头骚乱。这是一种几十年来功能相对没有变化的产品。

一个新的记事本应用程序实际上在六月就被取笑过，或者至少暗示过。除了应用程序上的圆角之外，没有任何东西真正显示出来。正如我们从 Windows 11 中看到的，几乎所有的应用程序都有圆角，无论它们是否针对新的操作系统进行了重新设计。它只是新 UI 的一部分。

现在根据截图，我们知道有事要来了。内部人士应该很快就会得到它。
