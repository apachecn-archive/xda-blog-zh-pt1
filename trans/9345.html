Galaxy A52s 5G 开始接收 Android 12 的稳定 One UI 3.0 更新

原文：https://www.xda-developers.com/stable-one-ui-4-0-update-goes-live-for-the-galaxy-a52s-5g/#0001-01-01

三星的 One UI 4 更新列车正在全速运行。在将其大多数旗舰产品更新为 Android 12 之后，该公司现在将注意力转向了中端产品组合。在过去的两周里，三星向多款中端手机推出了基于 Android 12 的稳定的 One UI 4.0 更新，包括 Galaxy A72、Galaxy Note 10 Lite 和 Galaxy A52 4G 。现在又有一款三星 mid-ranger 加入了 Android 12 的行列:Galaxy A52s 5G。

三星已经开始在多个欧洲国家的 Galaxy A52s 5G 设备中植入稳定的 One UI 4.0 更新(软件版本 A528BXXU1BUL7 )。到目前为止，该服务已经在法国、瑞士、卢森堡和波兰推出，更多的市场将很快跟进。如果你拥有一部 Galaxy A52s 5G，并居住在上述任何一个市场，请留意未来几天的 OTA 通知。或者，您也可以通过前往设置>软件更新>下载&安装来手动检查更新。在通常的 One UI 4.0 和 Android 12 功能中，此次更新还带来了 2022 年 1 月的安全补丁。

安装 One UI 4.0 更新后，Galaxy A52s 5G 用户可以期待许多令人兴奋的功能和变化，包括重新设计的快速设置面板、改进的黑暗模式、新的充电效果、改进的主屏幕小部件、改进的多任务处理体验等。要了解更多关于 One UI 4.0 的信息，请务必查看我们对三星最新 Android 皮肤的全面评测。
