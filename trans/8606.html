红米 Note 11T 5G 确认 11 月 30 日在印度上市

原文：https://www.xda-developers.com/redmi-note-11t-5g-india-launch-date/#0001-01-01

上个月，小米在中国推出了全新的红米 Note 阵容，包括红米 Note 11 5G、红米 Note 11 Pro 和红米 Note 11 Pro+。虽然小米没有分享关于新产品线后续国际发布的任何细节，但该公司现已证实，它将至少向印度市场推出一款机型。

在今天早些时候的一条推文中，小米印度总经理 Manu Kumar Jain 证实，该公司将于本月晚些时候在印度推出 Redmi Note 11T 5G。发射将在 11 月 30 日进行。

虽然这条推文没有进入更多细节，但多产的情报人员 Ishan Agarwal 报道这款手机将是中国红米 Note 11 5G 的更名版本。如果情况确实如此，我们可以预计这款手机将采用 6.6 英寸全高清+ 90Hz 显示屏，联发科天玑 810 SoC，50MP 主摄像头，5,000mAh 电池，支持 33W 快速充电。它还将提供双立体声扬声器、5G 支持、侧装指纹扫描仪、X 轴倾斜电机和搭载 MIUI 12.5 的 Android 11。根据 Ishan 和 91Mobiles 的说法，这款手机将有三种配置:6GB/64GB、6GB/128GB、8GB/128GB，以及三种颜色:哑光黑、星尘白和海蓝宝石蓝。

值得注意的是，小米子品牌 POCO 最近在欧洲以 POCO M4 Pro 5G 的别名推出了同一款手机。

红米 Note 11T 5G 将接替 7 月在印度推出的红米 Note 10T 5G。它配备了 6.6 英寸 90Hz 显示屏、联发科天玑 700 SoC、三摄像头和 5000 毫安时电池，支持 18W 快速充电。

虽然与前代产品相比，红米 Note 11T 5G 并没有实质性的升级，但它确实配备了更鲜艳的屏幕(DCI-P3 宽色域支持)、更快的 SoC、更好的主摄像头和更快的充电速度。
