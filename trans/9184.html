WSATools 可以让你在 Windows 11 上备份你的安卓应用

原文：https://www.xda-developers.com/wsatools-020-android-app-backup/#0001-01-01

WSATools 是一款粉丝开发的用于在 Windows 11 上安装 Android 应用的应用，它将很快允许你备份所有的 Android 应用，并轻松恢复它们。如果您出于任何原因需要为 Android 重新安装 Windows 子系统，这将使设置变得更加容易。

如果你还不知道，微软已经测试在 Windows 11 上运行 Android 应用程序的能力几个星期了，开发人员很快就开始围绕这一点进行开发。一个特殊的开发者 Simone Franco 创建了 WSATools，这是一个 Windows 11 应用程序，可以更容易地安装 Android 应用程序。微软官方支持亚马逊 Appstore 上的一些应用，但 WSATools 让你可以更容易地安装任何你能在网上找到的应用。

在即将到来的 0.2.0 版本更新中，WSATools 还允许你备份你已经安装的任何 Android 应用程序，包括与它们相关的用户数据。当您恢复备份时，不仅您的应用程序可以正常运行，而且您应该已经登录，就像从未丢失过它们一样。这是通过备份存储 Android Windows 子系统的整个虚拟驱动器(VHDX 文件)来完成的，所以这是一个相当基本的功能，但不是每个人都想通过其他方法来完成。

除了简单地安装应用程序之外，WSATools 自推出以来还增加了一些功能，如在 Android 的 Windows 子系统上安装和启用 Android debug bridge (adb ),减少了用户在安装 Android 应用程序之前的设置需求。该应用程序的目标是让那些可能不熟悉 Android 开发场景的人变得更容易。如果你喜欢用老派的方式做事，我们有一个关于如何在 Windows 11 上下载 Android 应用的指南。

我们目前还没有 WSATools 版本 0.2.0 推出 Android 应用程序备份的日期，但希望它会很快发生。与此同时，如果你是 Windows Insider，你可以在微软商店找到这款应用，并用它来安装 Android 应用。如果你不是，我们还有一个关于如何在没有内部版本的情况下使用 Android 的 Windows 子系统的指南。
