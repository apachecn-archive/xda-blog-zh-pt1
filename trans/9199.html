pinkphone 官方键盘套终于可以购买了

原文：https://www.xda-developers.com/pinephone-keyboard-case/#0001-01-01

Pine64 是一家技术公司，多年来一直销售基于 Linux 的硬件，包括入门级的 PinePhone 和即将推出的 PinePhone Pro。Pine64 已经与 PinePhone 社区合作了几个月，以创建一个官方键盘附件，并且在设计完成后的两个月，你终于可以购买一个来创建你梦想中的有趣大小的袖珍 Linux PC。

新的键盘外壳兼容 PinePhone 和即将推出的 PinePhone Pro，并取代了原来的后壳。由于 PinePhone 已经有了为配件设计的内部弹簧针，所以没有增加体积或摆弄蓝牙连接。它看起来更像一台微型笔记本电脑，而不是一部带键盘的手机。

Pine64 在其最新的博客文章中写道，“铰链采用 180°设计，不仅允许双手在表面上打字，而且在完全伸展时也可以舒适地用拇指打字。蚀刻的键帽可以很容易地重新定位，用于替代布局，如 AZERTY 或 QWERTZ。键盘外壳运行开放的固件，这意味着任何人都可以改变现有功能或添加新功能。”

不过，这款保护套不仅仅是一个键盘，它还有自己的 6000 毫安时电池和 USB Type-C 端口。你可以通过键盘的 USB 端口同时给键盘的电池和手机的电池充电，或者如果你不想处理两个电池，键盘在从手机吸取电力时仍然完全可用。两种电池电量都可以通过软件来检查，但这取决于每个 PinePhone 操作系统的维护人员添加所需的支持代码。

除了键盘外壳，Pine64 现在还为 PinePhone 和 PinePhone Pro 销售其他三种替代后壳。“PineDio LoRa”机箱是对原始后机箱(没有任何品牌)的替换，增加了一个用于 I/O 项目的 LoRa 模块。还有一款增加了无线充电功能，另一款增加了指纹传感器。这三个选项都很容易拆卸，Pine64 提供了大量的文档。

新款保护套可从 Pine64 在线商店购买。无线充电保护套 9.99 美元，PineDio LoRa 保护套 19.99 美元，指纹识别器保护套 24.99 美元，键盘保护套 49.99 美元。
