如何检查 Windows 11 中是否启用了基于虚拟化的安全(VBS)

微软已经证实，它正在与合作伙伴合作，在大多数新电脑上启用基于虚拟化的安全 VBS。下面介绍如何在 Windows 11 中检查是否启用。

Windows 11 现已正式提供下载，但似乎有一种挥之不去的想法，即该操作系统会对游戏性能产生负面影响。这很大程度上是由于微软实施了基于虚拟化的安全性(VBS)。至少，这是流行的 3DMark 和 PCMark 基准测试工具套件的开发商 UL 基准测试公司根据他们的测试提出的建议。

VBS 使用硬件虚拟化功能创建一个安全的内存区域，并将其与正常的操作系统隔离开来。操作系统可以使用它来托管许多安全解决方案，并保护它们免受影响系统的其他漏洞的影响。你可以在微软的支持文档中了解更多信息。根据 UL 基准，VBS 对游戏性能产生了负面影响。他们还证实，他们的基准测试的未来版本将提供 VBS 检测，以帮助您更好地理解。但与此同时，这里有一个快速教程，教你如何检查你的电脑是否启用了 VBS

如何检查 Windows 11 中是否启用了 VBS

  • 按“Win”键打开“开始”菜单，或者只需按 Windows 中的“搜索”按钮打开搜索栏。
  • 现在，键入“MSInfo32”并按回车键。
  • 当你在“系统信息”应用中向下滚动时，你会看到你的电脑是否启用了 VBS。
  • 你可以用同样的过程来看看 VBS 在 Windows 10 中是否也启用了。

从我们的发现来看，如果你以前没有玩过这个设置，VBS 将不会被激活。这是因为 VBS 在 Windows 10 中默认是不启用的。事实上，如果您从 Windows 10 升级或手动安装副本，Windows 11 中默认情况下不会启用它。然而，当您从 OEM 购买新的笔记本电脑或预装台式机时，情况就不一样了。正如 PCGamer 所指出的，微软已经确认将与合作伙伴合作，在明年的大多数新电脑上启用 VBS 和 HVCI。

如果 VBS 对游戏性能有负面影响，那么我们宁愿默认关闭它。但是现在，在确保游戏性能不会受到重大影响之前，这似乎是我们必须经历的额外一步。除此之外，在启动时点击安装按钮之前，你可能想知道一些 Windows 11 的其他已知问题。祝你好运！
