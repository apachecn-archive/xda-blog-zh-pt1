微软发布了另一个针对 Windows 11 中过期应用证书的修复程序

原文：https://www.xda-developers.com/windows-11-kb5008295-certificate-fix/#0001-01-01

微软最近承认，由于证书过期，一些内置的 Windows 11 应用程序可能在最近几天停止工作。一些应用程序受到影响，包括截图工具、提示和入门。针对这一问题的新修复程序现已在 Windows 11 的 KB5008295 更新的测试版和发布预览版频道中面向 Windows 内部人员推出。

这实际上是微软针对该问题发布的第二轮修复程序。几周前，微软推出了版本号为 22000.282 的 KB5006746。此更新修复了损坏的应用，包括输入法编辑器、触摸键盘和其他一些应用，但这是一个可选更新，你必须从 Windows Update 手动下载。另外，Snipping 工具仍然受到这个问题的困扰，如果你在 S 模式下使用 Windows 11，一些系统功能，如开始菜单和设置应用程序中的帐户页面也会被破坏。

用于 Windows 11 的 KB5008295 更新修复了剩余应用的过期证书和被破坏的系统功能。然而，同样的，这个更新并不是对每个人都适用。如果您想立即获取此更新，您需要在您的电脑上加入 Windows Insider 计划。此更新不会更改你的操作系统的内部版本号，因此如果你想确认已安装该修复程序，你需要在 Windows Update 中查看你的更新历史记录。

如果你现在就需要修复，加入发布预览版并没有太大的风险。它本质上与稳定的 Windows 11 版本相同，安装修复后你可以很容易地选择退出。然而，如果你能等的话，这个补丁可能会包含在下周二的补丁中。在那一天，微软将发布另一个累积更新，包含当前预览版中的所有修复，然后是一些。

自首次发布以来，Windows 11 遭遇了一些小问题，这在任何大型软件发布中并不罕见。这些问题包括在 AMD 处理器上的较低性能，这也已经通过前面提到的 KB5006746 在预览版中得到部分修复。另一个补丁可以直接从 AMD 获得，因为它与芯片组驱动程序有关。随着这些问题的解决，升级到 Windows 11 应该会更加引人注目。
