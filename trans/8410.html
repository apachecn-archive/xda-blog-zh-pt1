谷歌 Pixel 的附近通话功能可以让你在 Nest Hub 上接听电话

谷歌 Pixel 的附近通话功能可以让你在 Nest Hub 上接听电话

附近通话功能将允许您在家时直接从 Nest Hub 接收来电。我

昨天，谷歌发布了 Android 12L，这是为 Android 12 设计的特殊功能，适用于平板电脑、可折叠设备和 Chrome OS 设备。虽然新的更新为大屏幕设备提供了优化的体验，但它也让我们看到了谷歌正在为谷歌 Pixel 和谷歌 Nest 设备开发的新功能:附近通话。

正如米莎尔·拉赫曼所发现的，Android 12L 配备了一个名为“跨设备通信服务”的新应用，它提供了谷歌正在为 Pixel 手机和谷歌 Nest Hub 开发的新的附近通话功能。

通过链接您的手机，您可以在附近直接从 Nest hub 拨打电话。

顾名思义，当你在家的时候，附近通话将允许你直接从你的 Nest Hub 接收来电。此外，你还可以在 Pixel 和 Hub 之间快速转接电话，并通过语音命令开始通话。目前，这项功能还没有发挥作用，看起来是一项正在进行的工作。我们不知道谷歌计划何时向所有人推出它。我们会密切关注，如果有更多关于附近通话功能的细节出现，我们会通知您。

Android 12L 带来了几项 UI 调整和优化，使 Android 在大屏幕上更可用。它为通知阴影、锁屏和其他系统 UI 元素引入了两列布局，以充分利用额外的实际屏幕空间。Android 12L 目前作为开发者预览版提供，正式发布定于明年 3 月的某个时候。
