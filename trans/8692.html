Chrome 现在可以让你轻松删除网站存储的数据

原文：https://www.xda-developers.com/google-chrome-97-beta-delete-site-data-easily/#0001-01-01

谷歌最近开始在稳定频道推出 Chrome 96，带来了 Windows 11 风格的菜单。除了稳定版，该公司还发布了 Chrome 97 测试版，带来了一些新的变化，包括让用户更容易删除网站存储的数据。

在周四的一篇博客文章中，Chrome 团队宣布(via9 to 5 Google)Chrome 97 beta 的发布。最值得注意的是，最新版本简化了网站存储设置，允许用户通过导航到设置>隐私和安全>网站设置>查看权限和跨文件存储的数据来删除单个网站存储的所有数据。

谷歌表示，它将在设置>隐私和安全>cookie 和其他网站数据>移除更精细的控制。但是，web 开发人员仍然可以在 DevTools 中访问这些控件。

谷歌认为这一变化将为用户提供更清晰的体验，并减少用户意外破坏网站的机会。

“通过为用户提供删除单个 cookies 的能力，他们可以意外地更改网站的实现细节，并可能破坏他们在该网站上的体验，这是很难预测的。甚至更有能力的用户也冒着损害他们的一些隐私保护的风险，错误地假设 cookie 的目的，”谷歌在一篇博客文章中说。

除了这一变化，Chrome 97 beta 还带来了一些新的改进，包括支持 WebTransport API 和 CSS 媒体查询，以检查显示器是否支持 HDR。

Chrome 97 已经开始在测试频道推出，你可以从这里下载。稳定的更新将在接下来的几周内发布，虽然它可能不会包括所有上述变化。

谷歌 Chrome 也在测试一项名为“隐私指南”的新功能，该功能用一种易于理解的语言解释了浏览器的各种隐私和安全控制，以及它们如何影响你的浏览体验。该功能目前隐藏在一面旗帜后面，似乎是一项正在进行中的工作。
