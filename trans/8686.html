Chrome 96 带来了 Windows 11 风格的圆形菜单

原文：https://www.xda-developers.com/chrome-96-windows-11-style-menus/#0001-01-01

谷歌 Chrome 96 已经在稳定频道上发布，除此之外，它还为 Windows 11 带来了更新的设计。

9 月，提交给 Chromium Gerrit 的一个补丁显示，Chrome 正在努力添加 Windows 11 风格的菜单。不久之后，该设计在 Chrome Canary 上首次亮相，今天，它终于开始稳定发布(通过 Windows 最新 )。

正如你在下面附上的截图中看到的，新的设计用圆角取代了菜单和其他 UI 元素上的尖角。请注意，这些变化不是现成的；您需要启用一个标志来访问它们。

截图鸣谢: Windows 最新

您可以通过在地址栏中粘贴以下 URL 来检查新的设计:chrome://flags # win 11-style-menus并将标志设置为“Enabled”重新启动浏览器以使更改生效。如果你运行的是 Windows 11，将标志设置为“启用”就可以了。然而，Windows 10 用户需要选择“所有 Windows 版本”选项才能访问它。在 Chrome Canary 98 上，默认情况下界面是活动的。

Chrome 新的 Windows 11 风格设计在最新的稳定版本 96 版中可用。更新开始推出，应该很快就会登陆你的 Windows 设备。如果你不想等，可以从 Chrome 的网站抓取最新版本。除了 Windows 11 机器的这种小的视觉刷新，Chrome 96 似乎没有太多东西。

Windows 11 带来了巨大的视觉变革，包括全新的开始菜单、彩色图标、新的字体、半透明窗口等等。它还采用了微软新的流畅设计，将尖角换成圆角，使系统看起来更柔和。Chrome 加入了微软的 Edge 浏览器，该浏览器已经进行了视觉改造，以符合 Windows 11 的审美。
