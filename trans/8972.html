三星为 Galaxy Z Fold 3 和 Galaxy Z Flip 3 推出了另一个 UI 4 测试版更新

三星正在为 Galaxy Z Fold 3 和 Galaxy Z Flip 3 推出第四个 One UI 4 beta 更新，解决了稳定版本中发现的问题。

继上个月发布了 Galaxy Z Fold 3 和 Galaxy Z Flip 3 的三个 One UI 4 beta 版本之后，三星本月早些时候推出了稳定的 One UI 4 可折叠版本更新。然而，三星 foldables 的稳定 One UI 4 更新存在问题，用户报告了各种问题，如屏幕闪烁、性能不佳、黑暗模式问题等等。一些用户甚至声称他们的手机在安装过程中进入了恢复模式，从而对他们的设备进行了软屏蔽。为了解决这些问题，三星现在正在 beta 频道上为这两款设备推出另一个 UI 4 更新。

Galaxy Z Fold 3 和 Galaxy Z Flip 3 的第四个 UI 4 测试版已经开始向注册测试计划的用户推出。根据最近在三星社区论坛上的帖子，测试版更新(固件版本 ZUL4)修复了稳定版中发现的问题。帖子还指出，用户在安装更新时可能会遇到错误，并敦促他们在继续之前进行备份。

以下是 Galaxy Z Fold 3 和 Galaxy Z Flip 3 的最新一个 UI 4 beta 更新的变更日志(机器从韩语翻译而来):

  • 修正了更新后启动到安全模式或恢复模式的现象-当更新后启动时，所有更改为 16:9 或 4:3 纵横比的值都更改回应用程序默认值
  • Instagram，链接在脸书等一些应用中打不开
  • Kiwoom 证券应用程序无法运行
