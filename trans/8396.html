Jetpack Compose 获得您和 Wear OS 支持的材料

谷歌在今年的 Android 开发者峰会上宣布，Jetpack Compose 已获得 Material You 和 Wear OS 支持。点击此处阅读更多内容。

Jetpack Compose 是一个用 Kotlin 编写的声明式 UI 引擎，旨在取代或成为经典命令式 UI 引擎的可行替代方案，如 Android 的 XML 布局。本质上，Jetpack Compose 是一个强大的纯代码布局引擎，可以使 UI 实现更加容易。它于去年八月在 alpha 版本中首次发布，接着是今年二月在 beta 版本中的 T2 版本，并增加了更多内容。现在，在谷歌的 Android 开发者峰会上，该公司宣布 Jetpack Compose 将获得 Material You 和 Wear OS 支持。

Google 已经发布了第一个 alpha 化的 构成素材 3 ，其中提供了 素材 设计 3 样式组件和 主题 等素材你喜欢的个性化功能动态色彩。该公司还发布了第一个测试版的Jetpack Compose 1.1，具有 Android 12 的拉伸过度滚动、改进的触摸目标尺寸、实验性的懒惰布局动画等功能。

对于外行人来说，Material You 的动态颜色功能利用了 monet，这是 Android 12 中引入的一个新的主题引擎，目前仅用于 Pixel 手机——从你的壁纸中提取颜色，并生成丰富的彩色调色板。然后，应用程序可以以各种方式将这些颜色应用到它们的用户界面上，这就是包含你通常会做的内容的应用程序。动态颜色支持是 Material You 最令人兴奋的事情之一，因为它使每个应用程序的设计感觉更加个性化。它还能保持应用之间的一致性，让你的智能手机体验更具凝聚力。

更好的是，该公司还宣布 Jetpack Compose 将获得一个 开发者预览版 ，它也支持 Wear OS。它甚至有样本和文档来帮助你开始为 Wear OS 设备制作 ui。这是向开发者提供的另一个工具，它与 Wear OS 生态系统的其他附加功能密切相关，例如今年 8 月推出的 Watch Face Studio 。
