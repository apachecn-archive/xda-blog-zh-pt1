Pixel Watch 和 Pixel 6a 出现在美国运营商的库存系统中，即将推出

原文：https://www.xda-developers.com/google-pixel-6a-pixel-watch-carrier-inventory-system/#0001-01-01

谷歌即将推出的 Pixel 6a 和 Pixel Watch 在过去几个月里已经大量泄露。虽然该公司尚未透露这些设备的任何细节，但这两款设备现在已经在一家美国运营商的后端库存系统中被发现，表明即将推出。

知情人士告诉安卓警方，代号为“Rohan”和“Bluejay”的设备出现在一家未指明的美国运营商的库存系统中。这些代号与之前关于 Pixel Watch 和 Pixel 6a 的泄露相对应，“Rohan”指的是即将推出的可穿戴设备，“Bluejay”指的是平价的 Pixel 智能手机。

清单显示，谷歌将出售三种颜色的 Pixel 手表——灰色、黑色和金色——内置 32GB 存储空间。Android Police 推测，由于这家未透露姓名的运营商尚未销售非蜂窝智能手表，即将推出的 Pixel 手表可能会采用蜂窝模式。就 Pixel 6a 而言，库存清单表明这款手机还将有三种颜色可供选择——黑色、白色和绿色——以及 128GB 的板载存储。

尽管清单没有透露即将推出的谷歌设备的定价或可用性，但之前的报道表明，Pixel 6a 将于今年 5 月份的某个时候上市。鉴于 Pixel 手表与 Pixel 6a 同时被添加到运营商的库存清单中，可以肯定的是，智能手表可以与该设备一起推出。

如果你错过了我们之前对 Pixel Watch 和 Pixel 6a 的报道，这里有一个到目前为止我们对这些设备所知的一切的快速概述。根据之前的泄露，Pixel 手表将采用圆形设计，没有物理边框，并具有大量的健康和健身跟踪功能。这款手表可能会开箱运行 Wear OS 3 ，配备下一代助手，并装有 Exynos 芯片。另一方面，Pixel 6a 据传将像其更高级的兄弟姐妹一样采用谷歌张量芯片组，类似的设计，带有居中打孔切口的 6.2 英寸显示屏，以及显示器内指纹扫描仪。
