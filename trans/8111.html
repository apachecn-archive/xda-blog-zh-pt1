Vivaldi 4.3 带来了 web 应用程序支持、底层标签堆栈等等

原文：https://www.xda-developers.com/vivaldi-update-web-app-support-bottom-tab-stack-more/#0001-01-01

在过去的几个月里，Vivaldi 桌面和 Android 浏览器增加了几个新功能。最近，它收到了桌面上的折叠标签和命令链支持，以及在浏览器中打开外部链接的选项和 Android 上的新标签设置。随着最新的更新，Vivaldi 浏览器在两个平台上都有了更多的新功能。

Vivaldi 4.3 桌面版的新功能

Vivaldi 4.3 桌面版已经开始向用户推出新的功能。浏览器的内置捕捉功能已经得到了重新设计，最新更新的新图标有助于用户了解一切是如何工作的。它现在还提供了一个调整大小的选项，允许用户轻松地调整屏幕截图的大小。

Vivaldi 的同步功能也获得了更新的用户界面，使用户更容易启动和运行它。它现在更加直观，具有新的图标和颜色，可以吸引人们对重要功能的注意，还有一个下载备份加密密钥的选项，如果用户忘记了密码，可以让用户解密他们的同步数据。

此外，更新改进了下载面板，以显示下载速度随时间变化的信息图表，重新开始下载和查找下载文件的新按钮，以及突出显示关键统计数据的 UI 更改。

其他值得注意的变化包括 Vivaldi Translate 中的额外语言支持、Vivaldi Mail 上的拖放支持、Vivaldi Mail 和 Calendar 的 OAuth 登录更改以及 PWA 支持。最后，Vivaldi for desktop 现在还默认禁用了空闲检测 API。

Vivaldi 4.3 Android 版的变化

Vivaldi 4.3 for Android 带来了一个新的标签栈工具栏，它改进了之前更新中推出的新标签设置和标签栈。工具栏允许用户更改选项卡堆栈中的选项卡，打开/关闭堆栈中的选项卡，以及预览分组的选项卡。

与桌面版一样，Android 上的 Vivaldi Sync 也进行了 UI 刷新。新的用户界面使得设置同步更加容易。它有一个更直观的设计，以及新的图标和颜色，以改善帐户设置和恢复的经验。Android 版本还允许用户选择从同步设置中下载备份加密密钥，以便在忘记密码时解密他们的同步数据。

Android 上的 Vivaldi Translate 也获得了浏览器 4.3 版本对 68 种新语言的支持，使支持的语言总数达到 108 种。最后，Android 版本还默认禁用空闲检测 API。

Vivaldi 4.3 桌面版和安卓版已经向用户推出。如果你还没有收到更新，你可以去维瓦尔第的网站下载最新的桌面版本或安卓版本的谷歌 Play 商店。
