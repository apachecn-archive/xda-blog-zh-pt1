视频展示了 Windows 11 壁纸贴纸的样子

原文：https://www.xda-developers.com/windows-11-wallpaper-stickers-video/#0001-01-01

不久前，我们报道了微软正在为 Windows 11 开发壁纸贴纸，为用户提供更多个性化选择。虽然已经过去几周了，这个新功能还没有正式提供给 Windows 内部人员，但这并不意味着我们不知道它会是什么样子。Twitter 用户 Albacore 首先发现了 Windows 的桌面贴纸功能，现在他分享了一个视频。

视频显示了在设置应用程序中添加或更改桌面贴纸的选项，对于可以添加什么贴纸，似乎有很多选项。添加贴纸后，用户可以随意调整大小或在桌面上随意移动。视频还显示，即使你更换壁纸，贴纸也会留在 Windows 11 桌面上，所以直到你手动移除它们，它们才会消失。

贴纸选择有几点值得指出。首先，许多贴纸似乎是根据其中的一个特定字符进行分组的。然而，没有简单的过滤选项来查看专注于特定字符的贴纸，完整的列表似乎相对较长。这可能会使查找特定标签变得有些繁琐。这可能是微软正在做的事情，至少在标签选择器中有一个搜索栏，所以你有一些方法找到你想要的东西。

看看微软是否会允许安装第三方贴纸也可能很有趣。许多消息应用程序都有贴纸，它们通常包括一个市场，在那里你可以获得更多基于特定人物的贴纸。鉴于你可以从微软商店安装 Windows 11 主题，提供壁纸贴纸也是有意义的。然而，在视频中，没有迹象表明这是可能的。

在视频中，进入贴纸编辑模式也需要一秒钟，这可能不是理想的体验。这可能是微软在使其更广泛可用之前想要改进的东西。尽管如此，鉴于它们在 Windows 11 build 22563 中几乎功能齐全(但被隐藏)，如果壁纸贴纸在 Windows 内部人员的下一个版本中正式出现，我们也不会感到震惊。
