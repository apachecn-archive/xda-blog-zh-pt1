Microsoft Edge for Linux 现已正式推出

原文：https://www.xda-developers.com/microsoft-edge-for-linux-is-now-generally-available/#0001-01-01

距离微软的 Edge 浏览器在 Linux 上的第一个预览版已经过去了一年多，现在已经普遍可用了。如果你期待开源操作系统上 Edge 的稳定版本，它终于来了。

事实上，早在 2019 年初，雷德蒙德公司就开始戏弄它，并于当年秋天正式宣布了 Linux Edge。但即便如此，几乎过了整整一年，甚至连一个预览版都没有，然后又过了半年，才来到 Beta 通道。

这还不是全部，因为如果你在 Linux 上的 Dev 频道，会有一个特别的惊喜。如果你去 edge://surf 玩内置的 Surf 游戏，使用著名的 Konami 作弊码(上、下、下、左、右、左、右、B、A)，游戏里就有一个复活节彩蛋。可悲的是，Windows 和 macOS 用户不会看到它。你需要站在 97 号边缘。

Ignite 上的另一个微软 Edge 公告是一个新的 IE 模式改进。如果你不熟悉 IE 模式，这个想法是为企业提供一种在浏览器 6 月份退役之前从 Internet Explorer 过渡到 Edge 的方法。IE 模式所做的基本上是将 Internet Explorer 放在一个边缘浏览器标签中。当然，所有这些好东西都已经完成了，Internet Explorer 在 Windows 中不再可用。

有一个新的云网站列表管理体验，这意味着更容易列出哪些网站应该在 IE 模式标签中打开。您将不再需要在内部位置托管列表，这将简化企业的体验。

但是围绕 IE 模式的所有繁重工作都已经完成了。这只是让企业及其 It 部门能够轻松使用它的问题。

至于 Linux 上的 Edge，Debian 和 Ubuntu 有 DEB 格式，Fedora 和 openSUSE 有 RPM 格式。你可以在这里下载。该功能集与其他平台上的功能集基本相同，包括微软账户同步等。然而，它不包括像 IE 模式这样的 Windows 特有的功能。
