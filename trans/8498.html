由于证书过期，一些 Windows 11 应用无法运行

原文：https://www.xda-developers.com/some-windows-11-apps-arent-working-due-to-expired-certificates/#0001-01-01

如果你发现 Windows 11 上的某些应用和功能无法运行，你并不孤单。由于证书于 10 月 31 日到期，一些东西无法工作，因为微软已经开始警告用户关于。

受影响的应用程序包括 Snipping 工具、设置中的帐户页面和登录页面(如果你处于 S 模式)、触摸键盘、语音打字、表情面板、输入法编辑器 UI、入门和提示。

幸运的是，已经有一个可用的修复程序。你只需要去得到它。10 月 21 日，KB5006746 作为可选更新发布。然而，关于可选更新，你必须选择通过 Windows Update 来安装它们。它们不会自动安装。

你可以通过点击 Win 键并输入 winver 来检查你在哪个版本上。如果你看到你在构建 22000.282，你就可以开始了。如果你使用的是早期版本，你可以进入设置-> Windows Update，你应该会看到一个安装它的提示。

具体来说，此更新修复了触摸键盘、语音输入、表情面板、IME UI、入门和提示的问题。对于 Snipping 工具，微软其实是建议你用 Print Screen 键复制一张截图，然后粘贴到画图之类的 app 里。希望通过微软商店的应用程序更新可以解决 Snipping Tool 的问题。

然而，微软仍然只是说它正在为 Snipping Tool 和 S mode 开发一个合适的解决方案，并且它将在以后提供一个更新。

如果你不想安装可选的更新，好消息是下周二是补丁星期二。这意味着所有受支持的 Windows 版本都将获得更新，而且这些更新不是可选的。这意味着如果你什么都不做，它们会在某个时候被安装在后台。由于累积更新的性质，这些修复将被捆绑到该更新中。
