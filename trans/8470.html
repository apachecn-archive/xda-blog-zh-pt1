三星为 Galaxy S21 系列推出第四个 One UI 4 测试版

原文：https://www.xda-developers.com/samsung-one-ui-4-fourth-beta-rollout-galaxy-s21-series/#0001-01-01

三星上个月以 One UI 4 测试程序的发布拉开了其安卓 12 活动的序幕，允许 Galaxy S21 系列用户试用安卓 12 的早期版本。此后，这家韩国巨头发布了几个测试版更新，增加了新功能并修复了令人讨厌的 bug。第三代 UI 4 beta 于两周前发布，它修复了与虚拟内存、三星键盘等相关的错误。现在三星刚刚发布了 Galaxy S21 系列的新 One UI 4 beta，进一步细化了整体软件体验。

根据我们论坛上的用户报告，三星已经开始向 Galaxy S21、Galaxy S21+和 Galaxy S21 Ultra 推出第四个 One UI 4 测试版。更新重 807.27MB，携带固件版本g 998 bx Xu 3 zuk 1/g 998 box m3 zuk 1/g 99 xbx Xu 3 zuk 1。虽然最新的测试版没有引入任何重大变化，但它修复了测试版测试人员报告的许多错误和问题。

第四个 One UI 4 beta 带来了以下变化:

  • 移除操作“过度滚动”时的拉伸效果
  • 删除了第一次启动应用程序后出现的弹出窗口
  • 将屏幕分辨率设置为 WQHD 后，设备的性能略有下降
  • 警报响起时的丢帧问题
  • 在某些情况下，WIFI 和热点没有打开
  • 解锁设备时出现屏幕模糊问题
  • 拨打电话时生成两个电话记录的问题
  • 重启手机后的指纹识别问题
  • WIFLAP 搜索在某些情况下不起作用的问题
  • 在语音通话过程中重启电话
  • 问题:键盘被强制关闭
  • 当关闭所有最近的应用程序时
  • 主页被终止并重新启动
  • 修复了其他小问题

新版本已经开始在韩国和其他市场销售 Galaxy S21 手机。如果你是第三个 UI 4 测试版，你应该很快就会收到新的更新。One UI 4 打包了许多新功能和视觉变化，包括重新设计的快速设置面板、改进的黑暗模式、新的主屏幕小部件、新的充电效果、改进的多任务处理体验等等。要了解三星最新软件的更多信息，请务必查看我们对 One UI 4 的评论。
