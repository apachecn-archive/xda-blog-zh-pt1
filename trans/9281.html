PowerToys 0.53.1 增加了网络搜索和总是在顶部的工具

原文：https://www.xda-developers.com/powertoys-0-53-1-adds-web-search-and-always-on-top-tool/#0001-01-01

微软发布了 PowerToys 0 . 53 . 1 版本，为超级用户带来了一些新功能。这一次，该公司正在添加前一版本的变更日志中提到的“永远在顶部”的功能，以及一个新的网络搜索工具等。

从新的永远在顶部的工具开始，它做了几乎你所期望的。使用键盘快捷键( Windows 键 + Ctrl + T )，你可以强制活动窗口总是停留在其他窗口的顶部。您可以再次按下键盘快捷键来撤销它。您还可以在锁定的窗口周围设定边框，以便更容易识别设定为总是在顶部的窗口。

这个 PowerToys 0.53.1 中的另一个大升级是针对 PowerToys Run 工具的。这种对 Windows 运行对话框的替换现在也包括了搜索网页的能力。输入以开头的查询？？将在网页上搜索您在该命令后写下的任何内容。如果不方便的话，您还可以更改命令来触发 web 搜索。

这个功能的一个很大的好处是，它实际上使用你默认的浏览器和搜索引擎来打开搜索结果。如果你不喜欢 Windows Search 自动打开 Edge 和 Bing 的方式，这可能是一个受欢迎的功能。我们应该注意，使用 Vivaldi 作为默认浏览器时，不管默认设置如何，使用的搜索引擎总是 Google。这可能是兼容性问题或错误。

还有另一个值得注意的增加，那就是在文件浏览器插件中支持 g 代码文件。这允许您通过缩略图预览 g 代码文件(如果该文件包含缩略图)。

除此之外，0.53.1 版本还对 PowerToys 中包含的大量应用程序进行了修复、调整和其他改进。你可以在下面看到完整的变更日志:

PowerToys 0.53.1 的新功能

总是在上面

  • 欢迎加入这个家庭！快速Win + Ctrl + T，聚焦窗口切换到最上面。再次切换，它恢复正常。

颜色选择器

  • 调整颜色菜单的十六进制输入改进，包括支持不带 hashtag 的十六进制代码和短十六进制代码，如#CF0。感谢 @htcfreek ！
  • 更好的右下角屏幕覆盖检测

梦幻组合

  • 增加负空间边距
  • 修正了不能捕捉子窗口的问题
  • 修复了编辑器启动时清除键盘焦点的问题
  • 修复改善覆盖，以降低亮度和隐藏数字。感谢 @davidegiacometti

文件浏览器

  • 增加了对缩略图和预览面板的 g 代码支持。感谢 @pedrolamas

Image Resizer

电源重命名

  • 现在实现了行高亮+预览支持。感谢 @niels9001
  • 修正了 AltGR 输入问题
  • 改进的文件夹重命名支持
  • 在活动监视器上打开

  • 增加了网页搜索功能！将通过您的浏览器转到您最喜欢的搜索引擎。你也可以改变默认的动作键！感谢 @cyberrex5 的初步实现，感谢 @franky920920 和 @htcfreek 的支持

  • 二进制和十六进制数支持。感谢 @gsuberland
  • 在计算中使用阶乘的能力
  • PT Run 将不再显示在 Window Walker 结果中。感谢 @davidegiacometti
  • 修复日志/ ln 计算
  • 修复，使以前的结果清楚
  • 修复检测符号链接并防止递归循环
  • 修正触控板滚动速度过快的问题

  • 默认情况下禁用，因为这需要提升来注册虚拟摄像机。
  • 将静音摄像头和麦克风的热键(默认)从Win + N更改为Win + Shift + Q，以避免与 Windows 11 键盘快捷键冲突

设置

  • 多个可访问性，布局，图像，字符串和图标修复。感谢 @niels9001

跑步者

  • 改进了互斥支持，防止多个 PT 运行实例运行

安装程序

  • 注意:如果你有一个自定义的安装路径，新的安装程序在升级时会有一个视觉上的怪癖。它会显示默认的安装路径，但实际上会覆盖当前位置。我们正在研究如何解决这个问题。

  • 将许多 exe 和 dll 的命名与项目合并在一起
  • 拼写检查器更新。感谢 @jsoref
  • /dup 响应已添加
  • /reportbug /bugreport 将要求一个“报告 bug”zip
