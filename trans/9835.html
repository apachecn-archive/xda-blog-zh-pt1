Windows 11 将很快让连接蓝牙设备变得更加容易

原文：https://www.xda-developers.com/windows-11-bluetooth-quick-settings/#0001-01-01

Windows Insiders 现在可以在 Windows 11 上更容易地连接蓝牙设备

微软正在为 Windows 11 用户推出一项新功能，使从快速设置面板连接蓝牙设备成为可能。

微软正在让用户更容易将蓝牙设备连接到他们的 Windows 11 PCs，从运行 build 22563 的用户开始。现在，用户可以直接从任务栏上的快速设置面板连接到蓝牙设备，而不必打开设置应用程序。

到目前为止，虽然快速设置面板中有一个蓝牙按钮，但它所能做的只是启用或禁用蓝牙连接。如果你想连接到一个特定的设备，你需要右击图标，进入设置应用程序，寻找新的设备或连接到已知的设备。现在，蓝牙按钮的行为类似于 Wi-Fi 按钮。在图标旁边，有一个箭头，可以让您看到附近的蓝牙设备列表，并直接从面板连接到它们。

有趣的是，这似乎并没有以你可以下载的更新的形式出现。微软只是说它已经开始启用该功能，所以这可能是服务器配置的变化，现在正在推出。这不是微软的常见策略，因为大多数功能通常以新版本或应用更新的形式出现，但这里似乎不是这样。考虑微软可能远程启用的其他功能几乎更有趣，比如我们今天早些时候报道的新的 Windows 11 应用程序选择器。

除了新的蓝牙连接面板， Windows 11 build 22563 还包括一些有趣的新功能，包括平板电脑的可折叠任务栏和新的表情符号。这也是几周内第一个适用于 Surface Pro X 等 ARM64 电脑的版本。如果你来自以前的版本，你会注意到更多，包括一系列新的触摸屏手势，使 Windows 11 对平板电脑更加友好。这些是用 build 22557 添加的，由于一个严重的错误，该版本不适用于 ARM64 PCs。
