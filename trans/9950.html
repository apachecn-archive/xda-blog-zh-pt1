AMD 宣布了其新的 Threadripper Pro 5000 WX 系列 CPU

原文：https://www.xda-developers.com/amds-threadripper-pro-5000-wx-zen-3-workstations/#0001-01-01

在最初的 Threadripper 3000 Pro 系列发布两年后，AMD 终于用一套新的专注于工作站的处理器升级了产品线。这些新处理器将成为 AMD thread ripper Pro 5000 系列的一部分，它们基于该公司的 Zen 3 架构。我们总共有五个新的 SKU，所有这些都将是 OEM 专用的，至少目前是这样。

新的 Threadripper Pro 5000 WX 系列旨在弥合该公司在锐龙系列中的主流消费芯片与其 EPYC 处理器之间的差距。新的 TR 处理器在一个更适合工作站的平台上带来了像 EPYC 处理器一样的强大配置。与以往一样，TR 系列最大的卖点仍然是高数量的 CPU 核心。我们正在考虑在一个芯片上集成多达 64 个内核。您还可以获得多达八个 DDR4 内存通道。

由于内存带宽高，TR 5000 Pro 系列芯片现在可以支持高达 2TB 的 DDR4-3200 DRAM 和 ECC。然而，新系列是基于单插槽平台的，所以可以肯定地说，EPYC 机器仍然能够炫耀更多的内存。这次另一个显著的改进是内核基于 AMD 非常成功的 Zen 3 架构。由于采用了新的架构，与之前的 Zen 2 微架构相比，新芯片的 IPC 和频率都有所提高。

AMD Threadripper Pro 5000 WX 系列

正如我们前面提到的，堆栈顶部总共有五个处理器，其中包括 Threadripper Pro 5995WX。它配有 64 个内核/128 个线程，基本频率和睿频分别为 2.7GHz 和 4.5GHz。Threadripper Pro 5975WX 是下一款 32 核/64 线程处理器。这款处理器的基本频率为 3.6GHz。还有 TR Pro 5965WX，它是英特尔 24 核英特尔至强 W-3345 处理器的直接竞争对手。看看这两者如何比较会很有趣。

定价和可用性

宣布的新 Threadripper Pro 5000 WX 系列处理器仅面向 OEM，至少目前如此。你能得到这些新芯片的唯一方法是向联想购买一整套配备新芯片的工作站电脑。联想在推出新芯片的同时，还推出了 ThinkStation P620 的升级版。

这也是为什么我们没有任何关于这些新的 TR Pro 处理器定价的信息。将由原始设备制造商来决定他们的新工作站电脑应该收取多少费用。在撰写本文时，AMD 和联想都没有提供 TR 5000 Pro WX 或新款 ThinkStation P620 何时上市的信息。我们将很快讨论这些新 CPU 的性能，但新芯片看起来非常强大。也许是时候用这些新的工作站 CPU 更新我们的最佳 AMD 处理器系列了。
