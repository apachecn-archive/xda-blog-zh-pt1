WhatsApp Windows 测试版获得黑暗模式支持

原文：https://www.xda-developers.com/whatsapp-beta-windows-dark-mode/#0001-01-01

我们在去年 11 月首次看到的用于 Windows 10 和 11 的 WhatsApp 测试版应用程序正在进行重大更新，增加了黑暗模式支持。这似乎是这次更新最大和最值得注意的变化，因为我们还没有发现其他任何东西。

黑暗模式支持意味着你在 Windows PC 上使用 WhatsApp Beta 应用时会有更好的体验，尤其是在晚上。默认情况下，该应用程序将遵循您的系统主题，但您可以根据自己的喜好将其设置为永久亮或暗。实现似乎很彻底，从登录界面到设置面板都使用了新的黑暗主题。然而，在角落的反馈按钮边上有一些白色边框。

自从该应用首次出现在微软商店以来，Meta(或脸书)一直在对其进行一些改进。最近，设置 UI 被调整为更符合 Windows 11 的设计标准(相对于 Windows 10)。UI 的其他部分也得到了同样的处理，搜索框、消息列表项和消息气泡在最新版本中都有圆角。最初的版本在这些元素中的大部分仍然有尖锐的角落。此外，当你点击设置图标时，会有一个动画，让用户界面看起来更加生动。

如果你还没有，你可以看看如何安装新的 Windows 版 WhatsApp 测试版应用程序，该应用程序在微软商店仍然不可见。您可以使用下面的直接链接下载它。
