Firefox 96 改进了通话时的噪音消除、Android 上的书签等等

Mozilla 现在正在桌面平台和安卓系统上推出 Firefox 96，并一如既往地增加了新功能和安全性改进。

Firefox 是少数几个保留了非 Chromium 渲染引擎引擎的 web 浏览器之一，这使得 Mozilla 能够尝试新的 web 功能和低水平的性能变化，而大多数其他浏览器都无法做到这一点(无论如何，本质上不会成为一个分叉)。上个月推出了 Firefox 95，增加了新的安全功能和一些 macOS 增强功能，现在 Firefox 96 正式上市。

桌面平台上的 Firefox 96 对于视频和音频通话来说应该比以往任何时候都更好，因为 Mozilla 在噪音抑制、自动增益控制和回声消除方面“做出了重大改进”。许多通信平台都有自己的噪音消除技术，但如果你碰巧使用的服务没有，Firefox 至少应该有一点帮助。Firefox 现在在接收数据时默认使用“Same-Site = lax”HTTP 头，这提高了安全性。

Firefox 96 桌面变更日志

新功能

  • 我们在噪声抑制和自动增益控制方面做了重大改进，在回声消除方面也做了微小改进，为您提供更好的整体体验。
  • 我们还显著降低了主线程负载。
  • Firefox 现在将默认执行 Cookie 策略:Same-Site=lax ,这为跨站点请求伪造(CSRF)攻击提供了坚实的第一道防线。

修复

  • 在 macOS 上，Gmail 中的 command-click 链接现在可以像预期的那样在新的标签页中打开。
  • 我们的最新版本修复了视频间歇性丢失 SSRC 的问题。
  • 它还修复了 WebRTC 降级屏幕共享分辨率以提供更清晰的浏览体验的问题。
  • 此外，我们还修复了某些网站的视频质量下降问题。
  • macOS 上全屏分离视频已被暂时禁用，以避免一些问题与腐败，亮度变化，缺少字幕和高 cpu 使用率。
  • 各种安全修复

Android 上的 Firefox 96 并不是一个实质性的更新，但仍然有一些受欢迎的变化。主页为最近的书签提供了更高质量的图标，一些界面元素得到了修复或改进。没什么大不了的，但绝对是

火狐 96 安卓变更日志

新功能

  • 我们为最近访问的网站添加了历史亮点。
  • Firefox 现在可以在主页上显示更好的最近书签图片。

修复

  • “从剪贴板填充链接”在 Android 12 中得到改进。
  • 现在，在“跳回”中选择“搜索组”会切换到活动选项卡。
  • 我们修复了一个界面问题，私人标签显示为普通标签。
  • 历史记录删除按钮现在可以为屏幕阅读器正确标记。
  • 重复的历史项目现已合并。
  • 浏览书签时罕见的崩溃现已修复。
  • 已删除的搜索组现在会在与其选项卡交互后重新填充。
  • 您现在会发现，跳转回中的搜索组也不会显示在最近搜索中。
  • 你现在可以在搜索对话框后面滚动主页时取消键盘，以获得更清晰、更流畅的体验。

你可以从 Mozilla 的网站下载桌面平台的 Firefox，安卓设备的 Play Store 也有。Mozilla 最近也在微软商店为 Windows 个人电脑提供了 Firefox。
