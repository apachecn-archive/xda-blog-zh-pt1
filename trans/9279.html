微软 Edge 97 推出新的引用功能和更多功能

原文：https://www.xda-developers.com/microsoft-edge-97-released/#0001-01-01

微软正在向稳定渠道的用户推出其最新版本的 Edge 浏览器- Edge 97 。这次更新包括一些新功能，包括引文，微软两个月前开始与 Edge 内部人员在进行测试。

引用是 Edge 的一部分，它使得在学术工作中正确引用来源变得更加容易。这是一个可选功能，它作为 Collections 的一部分启用，允许用户在一个公共主题下以组的形式保存多个页面。在 Edge 中启用引文功能后，学生可以获得正确格式的给定页面的引文——APA、MLA 和 Chicago。用户可以设置他们正在查看的来源类型，例如是标准网页还是书籍，注明作者，引文文本将自动生成。

Edge 97 的另一项新功能是，当设备上存储有多个帐户时，可以使用当前的 Edge 配置文件自动登录工作或学校网站。到目前为止，如果你用一个工作或学校帐户设置了多个 Edge 个人资料，并且你访问了一个存储了不同帐户的网站，Edge 会要求你选择一个帐户登录。现在，浏览器将使用与当前 Edge 配置文件关联的帐户自动让您登录网站。

这个版本还附带了一些与安全相关的升级。Edge 97 将自动尝试将 HTTPS 用于支持它的域，因此用户可以与他们访问的网站建立更安全的连接，尽管这一功能正在分阶段推出。微软还增加了控制流保护(CFG)，这是一个针对内存损坏漏洞的额外保护层，仅适用于 Windows 8 和层的用户。

此外，Edge 现在将阻止网站中第三方框架中的 WebSQL 元素，微软将在 101 版本中完全移除对它们的支持。Chromium 项目还移除了 97 版对 WebSQL 的支持。最后，微软正在 macOS 上增加对微软端点数据丢失防护的支持。
