Discord down: Discord 现在有服务器问题

原文：https://www.xda-developers.com/discord-down-january-2022/#0001-01-01

当您使用的网站或服务开始出现加载问题时，您必须尝试猜测问题是出在您的家庭网络、您的互联网服务提供商还是您试图连接的实际服务，这总是很有趣。如果你现在有与不和谐相关的问题，不要担心，这不仅仅是你的问题，公司目前也有问题。

现在社交媒体上有许多关于 Discord 被关闭的报告，追踪器网站 Downdetector 在过去 30 分钟内收到了数千份关于该服务关闭的报告，大多数是在东部时间周三下午 2 点 30 分之后开始的。 Discord 还在 Twitter 上表示，“我们目前正在调查大范围的 API 中断，并正在尽快解决这个问题。”

Discord 的官方状态仪表板提到，太平洋时间中午(美国东部时间下午 3 点)左右发现了“大范围 API 中断”。后来的状态更新说，“我们已经确定了 API 中断的潜在问题，但正在处理我们的一个数据库集群上的次要问题。我们的整个电话响应团队都在线，并对问题做出响应。”Discord 表示，截至中午 12 点 30 分左右，它已经解决了所有数据库问题，但该公司仍在限制登录速度，以防止其服务器超载。

与最近其他大多数值得注意的中断不同，例如 11 月份影响 Spotify、Discord、Snapchat 和其他网站的问题，Discord 似乎是唯一受影响的平台或主要服务。然而，该公司尚未确定错误的原始原因，这可能与一家主要的云提供商有关。

既然“不和谐”已经(大部分)解决了导致最初中断的原因，服务应该很快就会重新开始工作。
