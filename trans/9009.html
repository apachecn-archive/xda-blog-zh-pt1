Windows 终端将成为 Windows 11 中默认的终端体验

原文：https://www.xda-developers.com/windows-terminal-default-experience-windows-11/#0001-01-01

微软正在改变Windows 11 的默认命令行体验，成为新的 Windows 终端应用。这将取代多年来一直作为标准的传统 Windows 控制台主机，这意味着您将获得更加现代和强大的开箱即用体验。

Windows 终端本身并不是一个新的应用程序。微软第一个在 2019 年宣布了它，它从去年 5 月开始作为稳定版本提供。这是典型命令行体验的一大进步，允许您在同一窗口的命令行上运行多个实例，同时将不同的命令行体验合并为一个。Windows 终端允许您在同一个应用程序中运行命令提示符、Windows PowerShell、Windows Subsystem for Linux 发行版等等。

不同的实例和配置文件可以在不同的选项卡中打开，但是您也可以并排平铺多个实例，以便它们同时可见。最重要的是，Windows 终端提供了大量的自定义选项，包括窗口的透明度设置、每个配置文件的自定义背景、选择默认配置文件等等。该应用的早期版本必须使用 JSON 文件进行配置，但微软已经为应用设置实现了 GUI，所以现在设置起来容易多了。事实上，Windows 终端经常获得更新，所以它自首次亮相以来已经变得好了很多。

随着 Windows 11 ，微软开始将 Windows 终端直接与操作系统捆绑在一起，所以你已经可以不用安装任何东西就可以试用了。Windows 11 还引入了选择您喜欢的终端体验作为默认体验的功能，因此，除了 Windows 控制台主机，您已经可以将终端作为您的默认终端体验。然而，你仍然不得不自己做这件事。

2022 年，微软将从注册 Windows Insider 计划的用户开始，将 Windows 终端设置为 Windows 11 的默认命令行体验。正如项目经理 Kayla Cinnamon 解释的那样，这一变化将“穿过内部计划的光环”,直到它到达所有 Windows 11 用户。这并不清楚这一变化是否会作为累积更新的一部分推出，或者我们是否必须等待明年秋天的功能更新，但现在没有什么可以阻止你改变你的默认终端。
