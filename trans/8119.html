如何给一个 CPU 贴导热膏？

原文：https://www.xda-developers.com/apply-thermal-paste-cpu/#0001-01-01

使用导热膏是电脑制造过程中的一个重要部分。该糊状物涂在 CPU 的集成散热器(IHS)上，以确保散热器和 CPU 冷却器的金属底座之间没有空气间隙。气隙是由于 IHS 和冷却器底座表面的微观缺陷而形成的。导热膏填充这些间隙，以便更有效地传热。通俗地说，导热膏帮助你的 CPU 时刻保持凉爽。

有几种不同的方法把浆糊放在 CPU 上。PC 构建领域的新来者可能会发现它势不可挡，尤其是因为一些糟糕的应用方法导致的可怕故事。好吧，如果你正在寻找一个可靠的热粘贴应用指南，以避免犯菜鸟的错误，那么你来对地方了。

要记住的事情

在我们开始之前，重要的是要确保你的冷却器和 CPU 的表面是干净的。如果你正在建造一台新的电脑，你的部件已经非常干净了，但是如果你要更换电池膏或者安装一个新的 CPU/冷却器，它们可能需要一些工作。然后，您必须小心地从相应的表面上去除旧的导热膏。为此，我们建议使用超细纤维布、棉签或含异丙醇(至少 91%)的纸巾。

一旦你完成了表面清洁，是时候确保你的 CPU 被锁定在适当的位置，你准备好安装 CPU 冷却器。导热膏的应用是散热器安装过程中的最后一步，也是安装散热器或阻水板之前的最后一步。如果你涂上浆糊，放上冷却器，然后意识到你忘了背板，你唯一的选择就是把它擦干净，重新开始。

如何使用热敏贴

正如我们前面提到的，在 CPU 的 IHS 上粘贴有几种不同的方法。如果一个人让你只涂一个点，而另一个人让你涂一条线，不要感到困惑。这在很大程度上取决于几个因素，如 CPU 芯片的尺寸、所用浆料的质量等。只要确保你得到充分的覆盖，并且浆糊均匀地涂在表面上。

如果你做到了以下几点，你就做得很好:

  • 有足够的浆糊盖住 CPU 芯片。
  • 浆糊没有溢出来。
  • 你没有创造一个厚层，降低效率，由于金属表面相距太远。

使用哪种应用方法

你可能已经见过人们使用所谓的“点方法”。这是一种简单的方法，只需要将少量的糊状物(比如说，一粒小米粒大小的糊状物)挤到 CPU 的中央。有些人喜欢在板上画一个晶体管大小的点。虽然这种方法对大多数消费级 CPU 都适用，但在本指南中，我们将尝试一种不同的方法。

做一条简单的线

这个方法相当简单，不是吗？直接在 IHS 的中心涂上一行浆糊。避免手动涂抹糊状物，当你将糊状物固定到位时，让较冷的压力起作用。任何具有良好一致性的优质导热膏都会均匀地附着在 CPU 上。你也可以直接在中间画一条线，在两边画两个小点，来制作“%”图案。

少即是多

当使用热敷膏时，越少越好。没有必要在 CPU 上倾倒大量的浆糊，以为它会神奇地降低你的 CPU 温度。我们只是想用浆糊填补 IHS 和冷却器的表面瑕疵，记得吗？如果你是第一次这样做，那么现在将是一个很好的时机再次移除冷却器，看看你的方法是否提供了一个均匀的分布。

只要你控制好热敷膏的量，你就可以随意试验。每个人都有自己的涂抹膏体的方式来达到想要的效果。如果你认为在放置冷却器后，你已经涂抹的东西没有均匀分布，你也可以向中心添加更多的小点。

运行稳定性检查

恭喜你！如果您已经深入了解了指南并遵循了每一步，那么您现在已经成功地将 thermals paste 应用到了 CPU。但是它成功了吗？如果您只是简单地更换粘贴、CPU 或冷却器，那么您可以直接启动到 Windows 并运行一些基准测试来查看您的 CPU 变得有多热。我们建议使用监控软件来获取一段时间内的温度读数。

如果您正在构建一台新的 PC，那么您可以在安装 Windows 之前完成其余组件的组装，并启动 BIOS 以查看 CPU 温度。如果它对于简单地坐在 BIOS 菜单中来说不是惊人的高，那么你可以继续进行。

我们建议运行 Cinebench R23 来查看 CPU 如何处理负载。像这样的基准测试和压力测试通常会迫使您的 CPU 达到最高频率。这意味着在这些运行过程中温度会相当高。只要系统没有在测试的几秒钟内崩溃，你应该没问题。或者，您也可以让 CPU 闲置，并检查闲置温度，看看是否看起来不错。

你应该多久更换一次导热膏？

热贴不会像你想的那么早就用坏。然而，更换导热膏不会损害你的系统，我们建议你每年检查一次。如果你在组装电脑一年后突然注意到 CPU 温度升高，那么很可能是因为灰尘颗粒堵塞了部件。

但根据经验，无论出于何种原因，在移除冷却器后，您都应该更换导热膏。在更换导热膏时，笔记本电脑不需要这么多关注。只要确保你从一个可靠的原始设备制造商那里买了一台好的笔记本电脑，质量会说明一切。

最后的想法

现在，您已经成功地将导热膏应用到了您的 CPU 上，我们想强调几件事情。在 GPU 上应用粘贴的过程也是一样的。事实上，您也可以为您的 GPU 使用相同的导热膏，尽管有些人可能会提出不同的建议。为 GPU 更换导热膏的频率也大致相同。根据显卡的不同，这个过程可能会很繁琐。

在您购买新电脑时，导热膏应该与购物清单上的其他组件一样优先考虑。它应该得到同样的待遇，我们认为人们应该在购买时做出明智的决定，就像人们如何搜索外设一样，如最佳显示器甚至是最佳机械键盘。然而，大多数人把热敏贴当作事后的想法，看到什么就拿什么。让我们改变这种情况，给予它更多的重视。
