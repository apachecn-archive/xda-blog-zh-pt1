Firefox 现在已经可以在微软商店上买到了，适用于 Windows 11 和 10

原文：https://www.xda-developers.com/mozilla-firefox-microsoft-store-windows-11/#0001-01-01

Mozilla 宣布 Firefox 浏览器现在可以在微软商店的 Windows 11 和 Windows 10 上下载。这给了 Firefox 一种下载浏览器的新方法，这可能比打开另一个浏览器并找到正确的网站更容易。

这是个大新闻，因为 Firefox 是首批利用微软商店新政策的浏览器之一。多年来，微软只允许开发者发布使用 EdgeHTML 的浏览器，这与旧的 Edge 浏览器使用的引擎相同。当 Edge 本身过渡到 Chromium 引擎时，这甚至更没有意义，但有了新的微软商店，任何类型的浏览器和应用程序都可以发布。

Firefox 并不是第一个出现在微软商店的知名浏览器，因为 Opera 现在就在那里。然而，它是第一个拥有专有 Gecko 引擎的浏览器，而其他大多数浏览器都在使用 Chromium。这包括微软自己的 Edge 浏览器，因此 Firefox 为那些想要以不同方式做事的用户提供了一个真正的选择。不仅如此，Firefox 实际上是通过微软商店本身提供和更新的，而不是使用外部服务器。从外部来源更新是微软商店的另一个新功能，这就是 Opera 在微软商店中的上市方式，但 Firefox 实际上正在经历微软对商店应用的批准过程。

事实上，Mozilla 一直处于在 Windows 11 上提供 Edge 替代品的最前沿。当微软让在新操作系统上切换默认浏览器变得更加困难时，Mozilla 对微软的方法进行了逆向工程，使得只需一次点击就可以将 Firefox 设置为默认浏览器。现在，下载 Firefox 也更容易了，在下载替代浏览器之前，你甚至不用打开 Edge 一次。

至于为什么应该考虑 Firefox，该团队强调了一些安全特性，如全面的 Cookie 保护、增强的跟踪保护以及对 HTTPS DNS 的支持。Firefox 也有一些方便的功能，比如多画中画模式，可以同时观看多个视频流。

虽然 Firefox 不是第一个进入微软商店的第三方浏览器，但仍有一些知名的浏览器没有出现。其中最引人注目的当然是 Chrome，这是最受欢迎的网络浏览器。Vivaldi 也是一个相对受欢迎的替代品，尚未在微软商店推出。希望随着时间的推移，越来越多的替代品会出现，因为 Windows 11 正式推出，而新的微软商店也将转向 Windows 10 。你可以使用下面的链接从微软商店下载 Firefox。
