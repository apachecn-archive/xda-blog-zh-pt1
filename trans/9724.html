有几十年历史的 OS/2 操作系统有了新的网络浏览器

原文：https://www.xda-developers.com/otter-browser-os2/#0001-01-01

每台电脑都需要一个功能强大的网络浏览器，但在运行 Windows、macOS 或 Linux 之外的系统的电脑上，这说起来容易做起来难。现在仍有少数人在使用 OS/2 操作系统，该操作系统由微软和 IBM 创建，于 2001 年结束主流开发，急需一款功能强大的网络浏览器。运气好的话，这个有几十年历史的操作系统可能拥有和现代智能手机一样的网络引擎。

OS/2 语音社区的主席 Roderick Klein 在的一篇公告文章中透露，新的 Otter 浏览器的公测版将于“二月的最后一周或三月的第一周”发布这款浏览器使用 Chromium 引擎——谷歌 Chrome、微软 Edge、Vivaldi 和其他大多数非 Firefox 或 Safari 浏览器使用的也是这个引擎。接口是用跨平台的 Qt 框架写的。

OS/2 是由 IBM 和微软在 20 世纪 80 年代末和 90 年代初联合开发的操作系统，旨在取代所有基于 DOS 和 Windows 的系统。然而，在 Windows 3.0 和 3.1 大受欢迎之后，微软决定专注于 Windows，让 IBM 继续自己的开发。IBM 最终在 2001 年停止了 OS/2 的工作，但是另外两家公司许可该操作系统在 IBM 停止的地方继续——首先是 eComStation ，最近是 ArcaOS 。

BitWise Works GmbH 和荷兰 OS/2 Voice foundation 于 2017 年开始致力于 Otter 浏览器的移植，因为在 OS/2 和 ArcaOS 上保持 Firefox 的更新版本变得越来越困难。2016 年的 Firefox 49 ESR 是 OS/2 上可用的最新版本，因为那是 Mozilla 开始用 Rust 代码重写 Firefox 的重要部分的时间，而且没有用于 OS/2 的 Rust 编译器。从那时起，主要的焦点一直是将 Qt 5.0 移植到 OS/2，其中包括 QtWebEngine (基于 Chromium)。这种努力还有一个副作用，那就是将来可能会有更多的跨平台端口。Otter 浏览器本身正式提供用于 Linux、macOS、Windows 和 BSD 系统。

帮助开发的 Steven Levine 在一篇公告中说:“Qt5 和 Otter 浏览器端口基本上是一个人的项目，得到了 bww bitwise works 其他员工和社区的帮助。Dmitriy[主要开发人员]正在做的是移植两个大型、复杂的项目，它们从未被设计为在 32 位平台上运行，也从未被设计为在 OS/2 上运行。在我看来，很少有开发者能在相同的时间内完成他所做的事情。在我看来，这个浏览器几乎可以每天使用。已经有很多对我很重要的网站，我现在可以用 Otter 访问，而不必在另一个平台上工作。”

尽管 OS/2 无论如何都不是一个现代的操作系统，但是, ArcaOS 发行版/fork 仍在开发中，主要面向遗留系统和复古爱好者。最新的更新是 2021 年 12 月的 v5.0.7，其中包括更新的驱动程序和内核修复。OS/2 端口开发者正在 OS/2 语音网站上接受捐赠。
