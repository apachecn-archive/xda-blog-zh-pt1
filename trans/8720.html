Galaxy Z Fold 3 和 Z Flip 3 的最新 UI 4 测试版将于 2021 年 12 月发布安全补丁

One UI 4 的第三个测试版现在推出了三星 Galaxy Z Fold 3 和 Galaxy Z Flip 3，并修复了一些错误。请继续阅读！

我们距离下个月还有几天时间，但这并没有阻止三星发布 2021 年 12 月的安全补丁。这一次，韩国 OEM 厂商为其最新的可折叠二人组捆绑了一套新的月度安全补丁和一个基于 Android 12 的全新 UI 4 测试版。 Galaxy Z Fold 3 和 Galaxy Z Flip 3 现在正在多个地区接收他们的第三个 One UI 4 beta 版本。

三星 Galaxy Flip 3 XDA 论坛 || 三星 Galaxy Fold 3 XDA 论坛

据我们论坛上的报道称，三星已经开始以固件版本 ZUKG 的形式向测试参与者发布新的软件更新。除了提升 Android 安全补丁级别之外，该版本在功能方面没有带来任何新的东西。然而，的变更日志 提到了几个重要的错误修复，这表明用于 Galaxy Z Flip 3 和 Galaxy Z Fold 3 的 One UI 4.0 的最终版本几乎准备在未来几周内推出。

到目前为止，测试版更新已经在韩国和印度上线。我们预计三星下一步将把推广范围扩大到美国。正如我们在之前的发布中所看到的，新的安全更新也应该很快会进入一个 UI 的稳定通道。

如果您希望亲自试用 One UI 4.0 测试版，您可以从三星会员应用程序中参与测试版的推出。请注意，参与该计划需要一个三星帐户，您可以通过访问该网站创建该帐户。如果 beta 计划不支持您所在的地区，您仍然可以通过手动刷新来试用三星发布的最新固件。为此，看看我们专门的 One UI 4.0 更新跟踪器。
