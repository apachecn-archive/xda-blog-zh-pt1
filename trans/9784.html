为 Realme X7 Pro 和 Realme 8 Pro 发布的基于 Realme UI 3.0 测试版

原文：https://www.xda-developers.com/realme-x7-pro-realme-8-pro-realme-ui-3-0-beta-program/#0001-01-01

在上周推出 Realme GT 系列的 Realme UI 3.0 测试版后，Realme 为两款新智能手机开放了 Android 12 测试版计划:Realme X7 Pro 和 Realme 8 Pro。测试程序允许用户在最终发布之前试用 Android 12 的早期版本。

根据 Realme 社区论坛上最近的帖子，Realme 正在邀请 Realme X7 Pro 和 Realme 8 Pro 的用户试用基于 Android 12 的 Realme UI 3.0 的测试版。Realme X7 将获得早期版本(只有有限的用户可以试用)，而 Realme 8 Pro 将获得公开测试版本(任何人都可以试用)。

如果你拥有这些设备中的任何一个，并且想体验一下新软件，你需要将你的手机注册到测试程序中。要参加，请前往设置>软件更新。从那里，点击右上角的齿轮图标T5 试用版>早期访问/公测版，填写所需信息。如果您的申请获得批准，您将通过 OTA 更新收到 Realme UI 3.0。

由于这是测试版，您可能会遇到错误和问题(您应该使用反馈日志包直接向软件开发团队报告)。)如果这是你日常使用的驱动设备，我们建议你不要安装测试版软件，等待最终稳定版发布。说到稳定版，Realme 还没有提供 Realme X7 Pro 和 Realme 8 Pro 的 Realme UI 3.0 公开发布的任何时间表。

Realme UI 3.0 是 Realme 自定义皮肤的最新版本。它带来了几个令人兴奋的功能和变化，包括新的流体设计，3D 图标，新的黑暗模式选项，浮动窗口 2.0，基于壁纸的主题，画板 AOD，等等。要了解更多关于 Realme UI 3.0 的信息，请务必查看我们对这款新软件的实际评测。

Realme 计划在未来几周内为更多手机开放 Android 12 测试版计划，包括 Realme 8、Realme 8i、Realme 7 Pro、Narzo 50A 和 Narzo 30。
