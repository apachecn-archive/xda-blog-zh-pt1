三星为 Galaxy S20 系列开放了一个 UI 4 测试程序

原文：https://www.xda-developers.com/galaxy-s20-users-can-now-try-out-android-12-via-one-ui-4-0-beta-program/#0001-01-01

上个月，三星以 One UI 4.0 测试版项目开始了其 Android 12 活动。虽然该计划目前仅限于 Galaxy S21 系列、 Galaxy Z Fold 3 和 Galaxy Z Flip 3，但三星现在邀请 Galaxy S20 用户试用最新的 Android 版本。

One UI 4 Beta 概述:动手体验三星 Android 12 更新中的每一项功能！T3】

三星今天(通过 SamMobiles )宣布，它已经为 Galaxy S20、Galaxy S20+和 Galaxy S20 Ultra 开放了 One UI 4.0 测试版程序，允许感兴趣的用户在公开发布之前试用 Android 12 的早期版本。该计划目前在美国对解锁的 Galaxy S20 机型开放。不过，三星没有透露有关运营商锁定机型的任何细节。也不清楚该公司计划何时将该计划扩展到其他地区和国家。

如果你拥有一台 Galaxy S20，并且居住在美国，你可以通过将你的设备注册到测试程序中来试用一个基于 Android 12 的 UI 4.0。为此，打开手机上的 Samsung Members 应用程序，点击主页上显示的 One UI Beta 计划横幅。一如既往，确保在安装测试版更新之前备份您的数据。

通过“一个用户界面测试计划”，比其他人更早体验 Galaxy S20 / S20+ / S20 Ultra 的新 Galaxy UX 和 Android 12 操作系统

对于 Galaxy S20 系列的公开发布，三星没有给出任何时间表。

回想一下，One UI 4.0 带来了许多新功能和视觉变化，包括重新设计的快速设置面板、改进的黑暗模式、新的充电效果、改进的主屏幕小部件、改进的多任务处理体验等等。要了解三星最新软件的更多信息，请务必查看我们对 One UI 4.0 的评测。
