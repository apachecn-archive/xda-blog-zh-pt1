谷歌 Pixel 7 设计在发布前几个月泄露

原文：https://www.xda-developers.com/regular-pixel-7-design-leak/#0001-01-01

紧随 Pixel 7 Pro 泄露事件之后，普通的 Pixel 7 现在也泄露了。泄露的渲染图让我们第一次看到了 Pixel 6 的直接继任者的设计。

Pixel 7 的 CAD 渲染来自于 OnLeaks 和 CARPH ，在智能手机泄露方面有着可靠的记录。正如我们昨天看到的 Pixel 7 Pro 泄漏，vanilla Pixel 7 的设计与去年的 Pixel 6 非常相似。然而，仔细观察会发现有一些关键的变化。最值得注意的是，黑色摄像头条现在延伸到了中框，使手机看起来更加无缝。

在正面，Pixel 7 配备了一个平板显示器，中央有一个用于自拍相机的打孔切口。四个角上都可以看到天线。在背面，这款手机配备了双摄像头和 LED 闪光灯模块。屏幕尺寸据说在 6.2 英寸到 6.4 英寸之间。

左侧装有 SIM 卡托架，音量键和电源键位于右侧框架上。同时，底部边缘容纳了 USB Type C 端口和扬声器格栅。据报道，这款手机的尺寸为 155.6 毫米 x 73.1mm 毫米 x 8.7mm 毫米(包括后置摄像头凸起时为 11.44 毫米)，比 Pixel 6 略小，更薄。作为参考，Pixel 6 的尺寸为 158.6 毫米 x 74.8mm 毫米 x 8.9 毫米。

总而言之，这款手机看起来和去年的型号没什么区别。请注意，我们距离 Pixel 7 系列的正式发布还有几个月的时间，谷歌很有可能在 Pixel 7 实际发布前的几个月对其设计进行一些重大修改。

我们目前还不知道 Pixel 7 的整体情况，包括它的摄像头硬件、电池容量和其他内部信息。上周的一份报告显示，Pixel 7 和 Pixel 7 Pro 的代号可能是“猎豹”和“黑豹”
