你很快就可以在 Windows PC 上看到三星手机上的最新应用

原文：https://www.xda-developers.com/samsung-recent-apps-your-phone-windows-pc/#0001-01-01

三星手机用户，包括那些购买了新的 Galaxy S22 系列的用户，将在 Windows 11 和 Windows 10 上获得一个新的手机应用功能。随着即将到来的更新，你将能够在 Windows PC 的任务栏上看到三星手机上最近的应用程序，从而更容易从你离开的地方继续。

在今天的 Unpacked 活动中，该功能被简要提及并展示，三星还发布了 Galaxy S22 系列以及 Galaxy Tab S8 系列平板电脑。你已经可以使用你的手机应用程序在你的电脑上打开三星手机的应用程序，但你必须手动打开应用程序。或者，你可以将应用程序固定在任务栏上，以便快速访问。

现在，你会在任务栏的通知区域(也称为系统托盘)看到一个你的手机图标，这可以让你看到你最近在手机上打开的应用程序。这样，你可以快速地拿起你正在手机上做的事情，并在你的笔记本电脑或台式机上继续。你可以在下面的图片中更清楚地看到这种体验，由微软的分析师奥特罗·迪亚兹在推特上分享。

三星和微软有着长期的合作关系，许多三星手机都预装了微软的服务和应用。三星还将其一些应用程序带到了 Windows 上，甚至三星笔记和微软的 OneNote 之间也有一些集成。与此同时，在 Your Phone 应用程序的功能方面，微软一直青睐三星手机。在微软自己的 Surface Duo 系列之外，只有三星手机支持像手机应用程序中的屏幕镜像这样的功能，后来演变成允许用户在自己的窗口中打开应用程序，就像它们是 PC 应用程序一样。

如果你没有三星手机，你很快就可以选择在 Windows 11 上运行 Android 应用程序，并为 Android 安装 Windows 子系统。微软将在本月的某个时候发布支持 Windows 11 的 Android 应用预览版，尽管它将只在美国正式推出。不过，这些应用不能依赖谷歌服务，除非你试图用非官方的方法在 Android 的 Windows 子系统上安装谷歌服务。
