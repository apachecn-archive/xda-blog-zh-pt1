据 AdDuplex 称，目前近 20%的个人电脑安装了 Windows 11

原文：https://www.xda-developers.com/adduplex-windows-11-usage-february-2022/#0001-01-01

AdDuplex 发布了 2 月份的 Windows 使用报告，让我们了解了 Windows 11 和其他版本操作系统的最新数据。根据该报告，Windows 11 现在安装在近 20%的 Windows PCs 上，比我们在 1 月份看到的 16.1%的使用份额有小幅增长。这些数字使 Windows 11 在最常用的 Windows 版本中排名第三，排在 Windows 10 版本 21H2 和 21H1 之后。

像往常一样，该图表的数据是从微软商店中使用 AdDuplex v2 SDK 的 5，000 多个应用程序中收集的。AdDuplex 为微软商店应用提供了广告框架，因此这些数据来自使用该框架的应用。通常，调查包括大约 60，000 台电脑，这是一个相对较小的样本量，但它确实让我们观察到 Windows 采用的总体趋势。同样值得注意的是，该调查只涵盖了 Windows 10 以后的版本，因此并没有代表 Windows 的每个版本。

如上所述，最新的 AdDuplex 图表显示，19.3%的 Windows PCs 正在使用 Windows 11，比 1 月份的报告略有增加。排在第二位的是 Windows 10 版本 21H2，如果用户的电脑不符合 Windows 11 的最低要求，用户可以获得该更新。该版本上个月在使用量方面排名第四，使用份额为 12.1%，但现在为 21%，与 Windows 10 版本 20H2 交换了位置。这一更新的高采用率可能是因为它非常小，但也是因为许多旧电脑无法获得 Windows 11。

Windows 10 版本 21H1 仍然是使用最多的 Windows 版本，使用份额为 27.5%。这与上个月 28.6%的报告相比只是略有下降。Windows 10 版本 20H2 目前已经下降到第四位，17.9%的 Windows 用户运行该版本。旧版本 Windows 10 的数字迅速下降:2004 版为 7.9%，1909 版为 2.5%，1903 版为 1.3%。Windows 内部人员占了 0.3%，这符合通常的数字。

如果你想知道为什么所有的图表现在都是蓝色和黄色，鉴于乌克兰最近发生的事件，AdDuplex 改变了颜色以示团结。

查看 AdDuplex 的趋势图，可以看到 Windows 11 和 Windows 10 版本 21H2 正在集体快速增长，与过去一些更快的更新保持一致。在同一时间段，Windows 10 版本 20H2 只有 20%的使用份额。当然，这次我们正在考虑两个更新，但它们是在同一时期发布的。单独来看，Windows 10 版本 21H2 和 Windows 11 在相同的时间内几乎达到了相同的增长速度。

Windows 10 版本 21H1 与新版本相比更加接近，在发布的四个月内，它也达到了大约 40%的使用份额。然而，在过去的一个月里，Windows 11 的使用增长似乎放缓了很多，尽管增加了一些大的新功能，包括对 Android 应用程序的支持。
