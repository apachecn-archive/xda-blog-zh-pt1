Windows 11 将很快提供 HDR 校准，就像 Xbox 一样

原文：https://www.xda-developers.com/windows-hdr-calibration-coming-soon/#0001-01-01

微软正在努力为 Windows 上的 DirectX 添加 HDR 校准功能，类似于 Xbox 游戏机上可用的功能。公司宣布将推出一款 Windows HDR 校准应用，允许用户校准他们的 HDR 显示器。

校准 HDR 很重要，因为每个屏幕的亮度水平都不一样。如果 Windows 无法正确检测面板的最大和最小亮度，可能会导致图像亮区或暗区的细节丢失，这种现象通常称为“裁剪”。本质上，这意味着某些颜色可能会超出显示器支持的最大或最小亮度级别，所有这些颜色都会变得单调。

Windows HDR 校准应用程序的工作方式与它在 Xbox 上的工作方式类似，遵循 HDR 游戏兴趣小组推荐的测试模式)。一个测试将确定仍然可见的最暗的可见细节，另一个将确定最亮的可见细节，最后一个将测试显示器的最大亮度。确定了这三件事，Windows 应该能够更有效地显示详细的 HDR 图像。

如果您在玩游戏或观看 HDR 显示器上的内容时发现任何问题，这可能会改变您的游戏规则。即使你还没有，尝试一下也是个好主意，这样你就能确保你的显示器能提供最好的效果。

昨天在 Dev 频道向内部人士发布的 Windows 11 build 22557 中包含了一个关于 HDR 校准的暗示。在设置应用中，在系统- >显示- > HDR 设置下有一个“HDR 显示校准”的链接，这导致了微软对该功能的发布。不过，在未来，你将能够从那里访问该应用程序。

微软还在 Windows 11 build 22557 中加入了另一项新的游戏功能，简称为“窗口游戏优化”。该功能可以改善延迟，并为在窗口模式(包括无边框窗口)而非全屏模式下运行的游戏启用自动 HDR 和可变刷新率等功能。这特别适用于 DirectX 10 和 DirectX 11 游戏，因为它们将传统演示升级为现代演示。DirectX 12 游戏已经使用了这种演示模式，因此应该不会受到影响。

微软没有明确表示 Windows HDR 校准是否只会在 Windows 11 上进行，或者是否也会在 Windows 10 上进行。然而，像自动 HDR 这样的功能是 Windows 11 独有的，所以在这里看到这种情况也不会感到惊讶。
