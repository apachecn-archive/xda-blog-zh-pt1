今天，Android 应用将登陆 Windows 11，有超过一千个应用

原文：https://www.xda-developers.com/android-apps-are-coming-to-windows-11-today-with-over-a-thousand-apps/#0001-01-01

上个月，微软表示 Windows 11 将推出一系列新功能，今天，该公司宣布推出。最大的新闻是，非业内人士今天可以在 preview 中开始使用 Android 应用程序。

Android 的 Windows 子系统在这里

当微软去年宣布 Windows 11 时，它说 Android 应用程序将在新的操作系统上运行。不幸的是，它没有及时发布，最终在今年晚些时候发给了内部人士。它最初只有 50 个应用程序，由微软和亚马逊精心策划。

现在，如果你满足硬件要求，任何人都可以使用它，它将提供一千多个应用程序。我是内部人员能够测试的小子集的一个大扩展，但它仍然是亚马逊应用商店的一小部分。我确实问过开发者什么时候能够提交一个应用程序并在 Windows 上使用，而不用经过重重关卡，亚马逊的一位发言人说公司正在尽快朝这个方向努力。

要在 Windows 11 上开始使用 Android 应用程序，请前往 Microsoft Store -> Library ->检查更新。商店应用程序更新和重启后，您可以搜索亚马逊应用商店。

Windows 11 在任务栏上获得了一些 Windows 10 的功能

人们发现 Windows 11 缺少的一个关键功能是在第二个显示器上显示时钟的功能。这是一个没有成功的特性，因为新的操作系统是在相对较短的时间内开发的。它回来了。如果你有第二台显示器，你将从现在开始看到时间和日期。

Windows 10 的另一个功能是任务栏上的天气。这是微软推出新闻和兴趣时加入 Windows 10 的。在 Windows 11 中，新闻和兴趣被小工具取代，但新闻和兴趣的入口，即天气指示器，消失了。现在又回来了。奇怪的是，它将你的小部件移动到屏幕的左下角，这感觉像是一个奇怪的举动，因为微软将任务栏中的所有内容都放在了 Windows 11 的中心。如果你想要右下角的肌肉记忆，它可能是开始按钮。

最后一个即将到来的是任务栏中的静音/取消静音按钮，这显然是用来打电话的。此外，如果你将鼠标悬停在任务栏中的图标上，你将能够共享该窗口。

任何安装了本月预览版更新的人都可以使用新的任务栏功能。如果你没有选择，你将在下个月的补丁星期二更新中得到它们。

随着微软继续为 Windows 11 更新传统应用程序，非业内人士终于获得了新的媒体播放器和记事本应用程序。微软表示，新的媒体播放器应用程序有一个全功能的音乐库，可以自动导入你所有的东西。

对于记事本，大的新功能是黑暗模式，它将默认使用你的系统主题。微软还表示，它使菜单变得更简单，尽管它们一开始并不太复杂。记事本并不以其复杂性著称；事实上，恰恰相反。记事本的简单是它的用户如此忠于它的原因之一。

若要开始使用新应用，只需在 Microsoft Store 中查看更新即可。如果你已经完成了获取 Android 应用的步骤，这些新应用将会在这个过程中自动安装。

雷德蒙德公司还表示，未来，它将会更频繁地发布 Windows 11 的新功能。它仍然会坚持每年进行一次大的功能更新，但是通过体验包放弃新的功能应该不会那么具有破坏性。
