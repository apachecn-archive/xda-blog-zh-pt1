微软 PowerToys 0.49 增加了“查找我的鼠标”和更多工具

原文：https://www.xda-developers.com/powertoys-049-find-mouse/#0001-01-01

微软的 PowerToys 收到了 0.49 版本的更新，这是我们有一段时间以来最大的更新之一。对于用户的稳定版本，这个版本包括两个主要的新工具，其中之一是新的视频会议静音工具。这个工具已经在 PowerToys 的实验版本中测试了一年多，但它最终被合并到稳定版本中。

如果你想知道它的作用，PowerToys 中的视频会议静音工具可以让你在系统范围内控制你的摄像头和麦克风。如果您使用多个会议平台或正在查看不同的窗口，此工具允许您使用键盘快捷键或浮动工具栏关闭每个应用程序的摄像头或麦克风。这样，您可以更容易地避免炫耀任何您不想要的东西或用大噪音打扰其他用户。

PowerToys 0.49 中的另一个新工具是 Find My Mouse。有了这个工具，用户可以通过双击键盘上的 Ctrl 键更容易地在屏幕上找到他们的鼠标光标。这将在鼠标光标周围创建一个聚光灯，当鼠标移动时，它也可以跟随鼠标几秒钟。这主要适用于大屏幕的多显示器设置或弱视用户。

你可能认为这个特性已经内置在 Windows 中了，事实也的确如此。但是原生的 Windows 突出显示只是一个圆圈，与这个版本相比消失得更快，也不那么明显。

这个版本的 PowerToys 的另一个大变化是 PowerRename 的新 UI。这个工具允许用户使用一个通用模板快速重命名多个文件，新的 UI 旨在与 Windows 11 中使用的设计语言更加一致。

除此之外，整个应用程序还有一些整体设计和可用性的改进。其中包括许多工具中屏幕阅读器的可访问性改进，包括 PowerToys Awake 、FancyZones 和颜色选择器。如果你感兴趣，你可以从 GitHub 或 Windows 11 上的微软商店下载 PowerToys 0.49 ，该软件最近才上市。它实际上仍然是从 GitHub 下载的，因为微软商店的应用程序现在可以来自外部来源。
