新的 OneDrive 文件点播体验现已在 macOS 上推出

原文：https://www.xda-developers.com/onedrive-files-on-demand-update-macos/#0001-01-01

微软宣布其新的 MAC OS one drive 文件点播体验将于今天推出。这种新体验建立在苹果的文件提供商平台上，提供了与操作系统更紧密的集成和更好的应用兼容性，以及其他改进。新的 OneDrive 文件点播体验于去年 6 月首次发布，同时还有其他一些改进。

OneDrive 文件点播对于 macOS 用户来说并不是一项新功能，它的目的是让你的 OneDrive 文件更容易访问，而不必强迫你将它们下载到你的 PC 上。有了这个新版本，这变得更加容易，因为 OneDrive 帐户现在将在 macOS Finder 应用程序中的侧栏上的“位置”下可见。这一切都要归功于新的苹果文件提供商平台，这是 macOS 的最新产品，确保了该功能将得到苹果的长期支持，因为它与 OS 深度集成。这也意味着 OneDrive 文件现在支持 macOS 功能，如文件标签、上次使用日期、文件系统标志、符号链接等。

当您在 Finder 应用程序中浏览 OneDrive 文件时，您还可以查看这些文件是保存到您的电脑中还是只能在线查看。如果您经常使用某些文件，您可以将它们设置为在您的电脑上始终可用，这样，如果您在另一台设备上进行任何更改，它们也会保持最新。否则，如果您想节省空间，您可以使用“释放空间”选项从本地存储中删除文件，同时将其保留在云中。

这个新的文件提供商平台的另一个好处是，它支持 OneDrive 的已知文件夹移动功能，这是 macOS 上以前没有的。这使您可以将某些重要的文件夹从本地计算机直接移动到 OneDrive，这样就可以在任何地方访问这些文件夹，还可以确保在您的设备损坏或停止工作时不会丢失这些文件。

不过，这种新体验也有一些需要注意的地方。首先，它只能在 macOS 12.1 或更高版本上使用，所以你需要安装最新的更新才能使用它。也许最重要的是，任何运行 macOS 12.2 以上版本的人都需要它。如果您更新到较新版本的 macOS，新的 OneDrive 文件点播体验将默认启用，并且不能禁用。

它也仅适用于使用 APFS 文件系统的驱动器，因此如果您仍在运行 HFS+驱动器，您需要将它们升级到 APFS 才能继续使用该功能。微软表示，只有“极少数”用户仍在使用带有 HFS+驱动器的 OneDrive sync，因此在这方面你不必太担心。

如果你想立即获得新的体验，你可以在 Mac App Store 上检查更新，如果你使用的是独立版本，也可以在应用程序内部检查更新。
