Windows 11 画图更新为圈内人带来了更统一的设计

原文：https://www.xda-developers.com/windows-11-paint-update-modern-dialogs/#0001-01-01

微软发布了 Windows 11 Paint 应用的更新，供该计划开发频道的内部人员使用。此次更新进一步更新了应用程序的 UI 元素，以与 Windows 11 的其余部分保持一致。

当微软首次推出新的 Paint 应用程序时，一些对话框，如颜色选择器和调整工具，仍在使用旧的设计。通过这次更新，这些对话框现在与应用程序的其余部分和 Windows 11 的其余部分保持一致，整体上看起来更加一致。虽然不是每个对话框都已经更新，但是图像属性 按钮仍然向你显示一个旧风格的窗口。

微软在此次更新中也做了一些修复。对于初学者，您现在可以按住 Shift 键并单击色样中的颜色来更容易地更改您的辅助颜色。该公司还修复了一些使用 IME 书写时文本会移动的问题，以及一些未在希伯来语等特定语言中本地化的文本。此外，此更新改进了对屏幕阅读器的支持。

微软似乎没有增加的一件事是对黑暗模式的支持，这是该公司在 Windows 11 Paint 应用首次出现时展示的。这也将有助于让 Paint 像其他 Windows 11 应用程序一样运行，但希望不会太远。

这是微软目前正在与 Windows Insiders 测试的两个应用程序之一，另一个是 Windows 11 的新媒体播放器应用程序。该应用程序是 Groove Music 和 Movies & TV 的替代品，它确实是一个音乐和视频播放器应用程序。当然，所有 Windows 11 用户都可以使用 Paint，只有这个更新是内部人士专有的，而媒体播放器是一个全新的应用程序。希望我们不用等太久就能把它们推向大众。同样的话也适用于 Android 的 Windows 子系统，该子系统仍然由测试和开发渠道的内部人士正式独占，尽管我们有一个关于如何让 Android 在任何 Windows 11 PC 上工作的指南。
