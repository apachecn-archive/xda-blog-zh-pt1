macOS 12.3 测试版中断了与 OneDrive 和 Dropbox 的文件同步

第一个 macOS 12.3 测试版打破了与 OneDrive 和 Dropbox 的本地文件同步，但实际上是故意打破的。

苹果周四发布了 macOS 12.3 的首个测试版，以及 iOS/iPad OS 15.4 的首个测试版。自去年发布以来，一些云同步服务一直无法在 macOS 12 Monterey 上可靠地工作，现在这些服务由于一个内核扩展的过时而完全中断。

macOS 12.3 Beta 版的发行说明(viaMAC rumors)称，“Dropbox 桌面应用和微软 OneDrive 使用的内核扩展不再可用。两家服务提供商都有替代目前处于测试阶段的这一功能的产品。”

Dropbox 尚未增加对 macOS 12.3 Monterey 的官方支持，一篇支持文章提到该公司“将在 2022 年 3 月开始推出测试版。”OneDrive 开始测试 macOS 12.1+的新点播文件功能，该功能基于更新的文件提供商 API 。本月早些时候，微软的社区帖子称，“因为新的体验与 macOS 更加融合，它将得到苹果的长期支持。按需文件的第一个版本是建立在一些现在已经过时的技术之上的。迁移到新平台使我们能够在未来几年支持这一功能。”

Dropbox 还没有使用新 API 的测试版，更新后的 OneDrive 体验似乎仍然局限于内部发布渠道。OneDrive 更新还将删除对旧的 HFS+文件系统的支持，并且由于新的 macOS 要求，OneDrive sync 文件夹将锁定到您的个人文件夹，不再将您的 OneDrive 文件夹存储在其他目录或外部驱动器中。

与此同时，Dropbox 仍在为 M1 本地支持进行更新，预计将于 2022 年上半年推出。 Google Drive 在 2021 年 10 月加入原生苹果 M1 支持，苹果自己的 iCloud 文件同步内置在 Finder 应用中，所以一直是 M1 原生的。
