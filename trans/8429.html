三星 Galaxy Z Fold 3 和 Galaxy Z Flip 3 现在可以通过 Android 12 获得一个 UI 4 测试版更新

三星已经开始为 Galaxy Z Flip 3 和 Galaxy Z Fold 3 推出基于 Android 12 的第一个 UI 4.0 测试版更新。请继续阅读！

在为 Galaxy S21 系列推出基于 Android 12 的 One UI 4.0 测试版之后，三星现在已经将测试版计划扩展到其最新的可折叠设备——Galaxy Z Fold 3 和 Galaxy Z Flip 3 。第三代 Galaxy Fold 设备的第一个 UI 4.0 测试版现在正在韩国和印度推出。

根据我们论坛和三星 社区 网站上的几篇 帖子，这两款设备的所有者现在可以通过三星会员应用程序注册 One UI 4.0 公测版，并随后获得更新提示。最初的测试版本为版本 F711BXXU2ZUJF (用于 Galaxy Z Flip 3)和 F926BXXU1ZUJF (用于 Galaxy Z Fold 3)，将稳定的 Android 12 代码库中包含的所有新功能都引入了可折叠文件夹，同时对三星自己的 One UI 皮肤进行了一些改进。值得注意的是，Thom Browne 版从现在开始被排除在测试版之外。

所有测试版的标准免责声明在这里也适用，即测试版软件往往包含错误。如果你正在为你闪亮的新 Galaxy Z Fold 3 或 Flip 3 寻找一个稳定的操作系统，完美地满足你的所有需求，你应该远离测试程序。三星的公共测试版通常比内部测试固件包更加完善，但仍有可能出现一些关键问题。

三星 Galaxy Flip 3 XDA 论坛 || 三星 Galaxy Fold 3 XDA 论坛

如果你有 Galaxy Z Fold 3/Flip 3 等符合条件的设备，但你还没有加入 One UI 4.0 测试版计划，你必须在三星会员应用程序中加入。当你进入应用程序时，点击“通知”按钮后，你应该会看到一个注册测试程序的选项。在接下来的屏幕上，点击“注册”按钮，然后等待几分钟，让应用程序处理您的注册。之后，前往设备上的软件更新页面，点击“检查更新”下载公开测试版。
