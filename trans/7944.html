Android 12.1 上手:先看看可折叠项的更新！

原文：https://www.xda-developers.com/android-12-1-hands-on/#0001-01-01

谷歌预计将在下周发布 Android 的下一个重大更新，Android 12，但在发布之前，我们已经听说了谷歌的后续更新计划。本月早些时候， XDA 首次报道称，谷歌计划在 2022 年的 Android 13 之前发布一个临时更新，我们暂时称之为 Android 12.1，以符合谷歌过去发布的命名惯例。现在，一个消息来源向 XDA 提供了一个 Android 12.1 版本，让我们可以亲自动手更新，让你先看看一些新功能。

Android 12.1 概述

如果你没有关注我们的报道，你可能想知道我们如何知道 Android 12.1 更新即将到来。我们的第一个提示来自提交给 Android 开源项目(AOSP) Gerrit 的代码更改，它将 Android 13 的 API 级别从 32 提升到 33。作为参考，Android 12 对应的 API 级别是 31，所以如果 Android 13 对应的是 API 级别 33，那么这就意味着 API 级别 32 一定对应某个临时版本。来自谷歌的评论表明，临时发布的代号为“sc-v2”，我们认为这是“雪锥 v2”的缩写顺便说一下，Snow Cone 是传闻中 Android 12 的甜点名称，这是谷歌几年前公开放弃的一个约定，但他们的工程师仍然在内部遵循。因此，我们把即将到来的临时版本称为 Android 12.1，但它最终可能会以其他名称上市。

下一件你可能会好奇的事情是，考虑到谷歌自 2017 年 Android 8.1 Oreo 以来就没有发布过一个点，为什么点发布甚至是必要的。虽然我们还没有完整的图片，但我们已经看到了谷歌正在添加新的 API 和功能的证据，旨在改善可折叠手机的体验。谷歌可能计划很快发布其即将推出的 Pixel Fold 设备，该设备带有新的操作系统版本和所有新的可折叠功能，因此他们不能只等待明年的 Android 13 发布，因为这将推迟手机的发布。由于 Android 12 已经在上个月达到平台稳定性，任何需要添加的新 API 都必须等到下一个版本，否则就是 Android 13。

我们不知道谷歌计划何时推出 Pixel Fold，也不知道谷歌计划何时发布 Android 12.1。我们怀疑这两者会齐头并进，但我们不能确定。我们所知道的是，更新仍在积极开发中，所以我们下面展示的所有功能都有可能发生变化。如果发生这种情况，我们将提供我们发现的更新。

Android 12.1 中的新功能

双窗格通知面板、设置和锁屏

在挖掘 Android 12 的早期版本时，我们了解到谷歌正在股票启动器应用程序和通知面板中添加双窗格布局，以及大小可调的锁屏。当时，我们认为这些用户界面的变化是针对平板电脑的，但谷歌在其任何博客帖子或开发者文档中都没有提到平板电脑的变化。事实证明，这些用户界面的改变是为了让 Android 在大屏幕可折叠手机上看起来更好。这是即将到来的通知面板的双窗格 UI 的新外观，以及双窗格设置应用程序和锁屏的第一次外观。

当设备的屏幕足够大时，快速设置和通知面板可以在 Android 12.1 上同时显示，快速设置在左半部分，通知在右半部分。相比之下，运行 Android 12 的大屏幕设备只能在状态栏完全展开的情况下显示快捷设置面板。Android 12 没有有效利用可折叠物提供的额外屏幕空间，但 Android 12.1 会。我们还了解到，当屏幕密度超过一定值时，Android 12.1 将显示一个 3 列快速设置面板，但我们无法让这一 UI 更改出现在我们有权访问的版本中。

同样，当设置应用程序打开时，Android 12.1 通过在左侧显示顶级设置和在右侧显示任何子屏幕，利用了许多可折叠文件夹的更大屏幕。

同时，锁屏也被一分为二，时钟和日期信息在左边，通知在右边。锁图标仍然居中，但现在要大得多。该图标可能需要一点工作，因为图标的颜色目前与白色背景冲突。不过，这个用户界面可能仍在开发中。

最后，正如我们之前提到的，锁屏 PIN/密码输入转移到侧面，以便更容易用一只手解锁设备。PIN/密码出现在用户触摸以关闭锁屏的屏幕一侧，但可以通过点击屏幕另一半的任何地方将其移动到另一侧。

任务栏

也许谷歌在 Android 12.1 中增加的改善可折叠手机体验的最重要的功能是任务栏。我们在早期的 Android 12 版本中看到了这个功能的一瞥，但是谷歌已经在其内部代码分支中完善了任务栏功能。任务栏集成了 Android 现有的多任务和分屏功能，它看起来与许多桌面操作系统非常相似。您可以使用任务栏在应用程序之间快速切换，拖放应用程序以分屏模式启动它，并按住任务栏以隐藏它。一旦启用，任务栏和启动器的应用程序 dock 就变成了同一个，每当用户退出应用程序时，任务栏就会无缝地转换到 dock 中。目前最多可以将 5 个应用程序添加到 dock/taskbar，尽管这很可能会在未来发生变化。

新的分屏和最近应用程序用户界面

Android 的分屏模式和最近的应用程序的用户界面也进行了轻微的调整，以便在像 foldables 这样的大屏幕设备上更好地工作。首先，分割屏幕的分隔线现在更加突出，使用深灰色而不是黑色，视觉效果更清楚地告诉用户分隔线可以拖动。较大的分隔线确实稍微覆盖了两个应用程序的一部分，但它还不足以分散注意力或掩盖重要的 UI 元素。不过，用户界面可能仍在开发中，所以我们可以预计在 Android 12.1 正式发布之前它会有所改变。

自从我们上次在 Android 12 中看到它以来，应用对这一功能已经取得了一些进展。我们仍然无法在主屏幕上添加应用程序对，但我们相信在 Android 12.1 达到稳定状态之前，该功能将完全可用。我们能够让它承诺的功能之一工作起来——即，双击分屏分割线来交换每个应用程序的位置的能力。我们还能够看到这两个分屏应用程序在最近的应用程序概览中作为一个任务出现，但我们无法找到一种方法来固定这一对。

谈到最近的应用概述，谷歌对 Android 12.1 的布局进行了微妙的调整。卡片的大小不再一致；相反，最近的一个被放大了，而其余的被缩小并排列成两行网格。在最近的卡片下面还有一个新的“分屏”按钮，可以启动分屏模式；以前，你必须长按最近应用概览中的应用图标才能进入分屏模式。

在分屏窗口中打开通知

Android 12.1 正在考虑一个名为“窗口通知”的新功能，尽管默认情况下尚未启用。启用后，此功能允许您在分屏窗口中启动应用程序活动，方法是长按通知，然后将其拖放到屏幕的任一半。代码更改表明，您还可以拖放抬头通知，但我们无法让它工作。我们也不能让这个功能在一个自由形式的窗口中打开应用程序，根据这个功能的描述，这个窗口也应该是工作的。

开源动态颜色，新的启动动画

当 Android 12 下周发布时，只有 Pixel 用户可以使用谷歌基于壁纸的主题系统，代号为“monet”这是因为主题系统不是完全开源的，但随着 Android 12.1 的发布，这种情况将会改变。正如我们上周首次报道的，谷歌已经将其“莫奈”主题系统的源代码添加到 Android 12.1 的 AOSP 内部分支中。这意味着三星、小米、OPPO 和一加等原始设备制造商将完全获得谷歌主题化系统背后的代码，允许他们将其纳入自己的 Android 12.1 分支。

以下是我们的消息来源上周与我们分享的一些截图。这些截图是在我们有权访问的同一个版本上拍摄的，它们证实了“monet”正在开源版本的 Android 上工作。

此外，我们了解到谷歌计划在 Android 12.1 中将动态着色扩展到 Android 的引导动画中。提醒一下，动态着色是谷歌所谓的 Material You 的基于壁纸的主题系统，代号为“莫奈”。由 XDA 查看的内部文档描述了如何将动态着色应用到引导动画中。新的动态着色渲染模式不是直接在引导动画文件中渲染 PNG 图像，而是将每个图像的 R、G、B 和 A 通道视为区域遮罩，根据动画的进度在开始和结束颜色之间进行插值。OEM 可以通过在启动动画的描述文件中添加一行来指定起始颜色，并从指定结束颜色的四个系统属性中读取来启用该功能，这些属性可能是在操作系统启动时用户更改壁纸时设置的。

我们没有一个动态着色的启动动画的工作演示，所以我们必须等待，看看它在实践中是否好看。Android 的股票启动动画这些年来只出现了少数微妙的变化，其中最值得注意的是在 Android 10 中添加了黑暗模式支持。最近，我们发现当系统应用更新艺术主线模块的 Google Play 系统更新时，引导动画可以显示进度条。

按住电源按钮的持续时间

近年来，Android 最有用的新增功能之一是设备控制，至少在我们看来是这样的。此功能可让您快速访问智能家居控制，而无需打开应用程序或轻按 widget。在 Android 11 中，设备控制被集成到电源菜单中，但在 Android 12 中，电源菜单不再包含设备控制(或快速访问钱包功能，就此而言)。)原因是谷歌想简化电源菜单，让长按电源键触发助手而不是电源菜单。

当启用“等待助理”功能时，按住电源按钮 500 毫秒将启动设置为默认助理服务的任何服务。在 Android 12 上，你不能改变按住电源按钮的持续时间，但在 Android 12.1 中你可以这样做。在“设置>手势>按住电源按钮”下添加了一个新的“按住持续时间”滑块，让您可以调整长按电源按钮手势的灵敏度。您可以选择从短(250 毫秒)到长(750 毫秒)，或者介于两者之间(350、500 或 650 毫秒)。

顺便说一句，如果你不喜欢谷歌在 Android 12 中对电源菜单所做的改变，门户网站 Kieron Quinn 的朋友制作了一个名为“ Classic Power Menu ”的应用程序，它恢复了旧的外观和感觉，并集成了设备控制和快速访问钱包。如果你有根接入的手机，那就去看看吧。

新壁纸

Android 12.1 的一个小的新增加是一个新的壁纸。这种壁纸包含在 AOSP 代码库中，但它在消费设备上可能可见，也可能不可见，因为 OEM 可以自由发布自己的壁纸。新壁纸取代了自 Android 7.0 牛轧糖以来 AOSP 发布的“粉色天空”图像。
