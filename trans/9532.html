谷歌 Chrome 测试关闭安卓所有标签页的警告

原文：https://www.xda-developers.com/chrome-android-close-all-tabs/#0001-01-01

谷歌 Chrome 仍然是 Android 设备上最受欢迎的浏览器，主要是因为它是大多数手机的默认选项。新功能经常被测试，通常首先出现在“金丝雀”不稳定发布频道，另一个刚刚出现的功能可能会避免人们意外丢失标签。

Android 上的 Chrome 在标签页上有一个“关闭所有标签”菜单选项已经有一段时间了，但似乎有些人可能会不小心点击它，因为谷歌已经开始测试一个新的“关闭所有标签模态对话框”标志( via TechDows )。当从 Chrome Canary 100 的 chrome://flags 页面启用该功能时，选择“关闭所有标签”菜单将显示一个新的确认弹出窗口。

对话框出现后，您可以按“取消”(或设备的后退按钮/手势)不关闭选项卡，或按“关闭所有选项卡”进行确认。如果你决定按下前进，所有标签将被清除，浏览器将关闭。重新打开浏览器将会打开一个新标签，通常会显示新标签页或谷歌移动搜索。

看到谷歌用确认对话框进行实验很有趣，而该公司大多数其他应用程序的清除/关闭操作(如在 Gmail 中存档电子邮件)只是显示一条带有撤销按钮的 toast 消息。然而，当撤销消息的计时器超时时，在内存中保存所有这些标签可能不是性能的最佳主意，即使 Chrome 已经在 Android 上让大多数标签在后台休眠。

如果你想亲自体验这个功能，你需要下载 Chrome Canary，因为它还没有渗透到其他发布渠道。也不清楚谷歌是否会保留这一功能，并最终向所有人推出，所以不要太执着。
