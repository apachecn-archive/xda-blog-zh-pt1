Galaxy S10 和 Galaxy Note 10 获得第三个 UI 4.0 测试版

原文：https://www.xda-developers.com/samsung-galaxy-s10-note-10-third-one-ui-4-beta-rollout/#0001-01-01

三星目前正在为其几款旗舰产品测试一款基于 Android 12 的 UI 4。本月初，韩国智能手机制造商为 Galaxy S10 和 Galaxy Note 10 系列推出了 One UI 4 beta 计划，允许用户在公开发布之前试用 Android 12 的早期版本。一周后，发布了第二个 UI 4.0 测试版，带来了大量的错误修复。现在，三星刚刚发布了两个系列的第三个测试版，进一步完善和改进了软件。

One UI 4 Beta 概述:动手体验三星 Android 12 更新中的每一项功能！T3】

在三星社区论坛的最近一篇帖子中，三星宣布为 Galaxy S10、Galaxy S10 Plus、Galaxy S10 5G、Galaxy S10e、Galaxy Note 10 和 Galaxy Note 10 Plus 推出第三个 One UI 4.0 测试版。最新更新解决了几个恼人的错误和问题，包括照片没有正确保存到媒体剪辑库，AKG N400 耳塞的连接问题，等等。

第三个 UI 4.0 测试版包含以下更改:

  • 修订历史
    • 与媒体相关的照片未保存(当捕获/图库/双信使不起作用时，等等。)
    • 改进了通知出现时系统变慢的问题
    • 过滤器经常出现故障
    • AKG N400 连接错误
    • 在我的文件/图库应用程序中找不到韩语文件名

三星 Android 12 Tracker:这里是所有官方 One UI 4.0 稳定版和 beta 版下载安装

第三次测试版更新已经开始在韩国的 Galaxy S1o 和 Galaxy Note 10 设备上推出。如果你正在运行第二个测试版，你应该很快就会收到 OTA。或者，您也可以通过前往设置>软件更新>下载&安装来手动检查更新。如果您不是 beta 计划的一部分，您可以使用三星会员应用程序将您的设备注册到该计划中。

除了 Galaxy S10 和 Galaxy Note 10，三星还在测试 Galaxy S20 和 Galaxy Note 20 系列基于 Android 12 的 One UI 4.0 更新。两个阵容最近都收到了第四个 One UI 4.0 beta 。
