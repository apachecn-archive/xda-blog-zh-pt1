三星 Galaxy Note 20 基于 Android 12 的 One UI 4 测试版上线

三星已经开始为 Galaxy Note 20 和 Note 20 Ultra 推出基于 Android 12 的第一个 UI 4 测试版更新。请继续阅读！

我们有一些好消息给三星 Galaxy Note 20 的用户:韩国 OEM 厂商似乎已经为 2020 年的 Note 系列设备开放了 One UI 4.0 测试版。One UI 4.0 是三星对 Android 12 的继承，它以的大量 UI/UX 增强为特色，比如基于壁纸的颜色主题。One UI 4.0 测试版程序已经可以在韩国、美国和其他地区的 Galaxy S21 、 Galaxy Z Flip 3 和 Galaxy Z Fold 3 上使用。最近，三星将去年的 Galaxy S20 系列加入了测试计划。现在，Galaxy Note 20 和 Galaxy Note 20 Ultra 也榜上有名。

XDA 论坛:三星 Galaxy Note 20 || 三星 Galaxy Note 20 Ultra

据几位 用户 报道，三星已经开始在英国推出基于 Android 12 的 Galaxy Note 20 系列测试版。那些注册测试程序的人将获得软件版本 N98xxXXU3ZUK1 作为第一个测试版本。它带有 2021 年 11 月日的安卓安全补丁。然而，bootloader 版本保持不变(v3)，这意味着你仍然可以通过手动刷新恢复到基于 Android 11 的旧 UI 3.0 版本，使用相同的 bootloader 。

请记住，One UI 4.0 测试版程序只是一个测试版程序，用于在上述设备上评估 Android 12。虽然您有机会试用三星最新的固件版本，包括其改进的用户界面和许多新功能，但在向公众推出之前，请注意这可能会带来系统不稳定和功能损坏的风险。通过测试阶段获得的反馈将用于稳定版本的工作。

如果您希望亲自试用 One UI 4.0 测试版，您可以从三星会员应用程序中参与测试版的推出。请注意，参与该计划需要一个三星帐户，您可以通过访问该网站创建该帐户。
