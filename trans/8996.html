微软发布 Windows 11 build 22000.376——以下是新内容

原文：https://www.xda-developers.com/microsoft-releases-windows-11-build-22000-376-heres-whats-new/#0001-01-01

今天是礼拜二，你知道这意味着什么。今天，每一款受支持的微软产品都获得了更新，当然，这也包括 Windows 11 版本 21H2。与所有星期二补丁更新一样，这是强制性的。

那些在 Windows 11 上的人将会看到 KB5008215 ，这使得内部版本号达到 22000.376。这里可以手动下载，以下是亮点:

实际的修复列表并没有说太多:

奇怪的是，微软实际上为这次更新制作了一个关于变更日志的视频。在视频中，它说它添加了新的流畅 2D 设计的表情符号 13.1。事实上，视频中的内容比变更日志中的要多得多。

可悲的是，这是意料之中的事。微软整个月都会发布 C 周和 D 周的更新，这些都是可选的。由于更新是累积性的，如果你没有更新，那些补丁和特性就会被加入到强制性的补丁星期二更新中。当这些更新发布时，那是微软真正谈论它们的时候。因此，当涉及到实际的补丁星期二更新时，变更日志有些不透明。

您现在可以通过手动安装或在 Windows Update 中检查更新来获取此更新。然而，如果你不这样做，它仍然是一个强制性的更新。这意味着在某个时候，它会在一夜之间安装完毕。一如既往，本月晚些时候将会有一个新的可选更新。

至于 Windows 10，对 2004 版的支持在今天结束。它仍然有它的最终补丁，当然，所有更新的都有补丁。如果你还在使用 2004 版本，这个月你一定会升级到新的版本。
