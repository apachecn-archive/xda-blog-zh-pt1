新的 Windows 11 媒体播放器现已在测试频道推出

原文：https://www.xda-developers.com/windows-11-media-player-beta-channel/#0001-01-01

微软公司宣布【Windows 11 的新媒体播放器应用程序现已在测试版渠道向 Windows 内部人士开放。早在 11 月份，该应用首次在开发频道向内部人士推出，它是一个统一的媒体播放器体验，取代了 Groove 音乐和电影&电视应用。

新的媒体播放器应用程序似乎利用了流行的 Windows Media Player 应用程序，该应用程序多年来一直是 Windows 的核心部分。在 Windows 8 中，微软试图将音乐和视频分成单独的应用程序(最初是 Xbox Music 和 Xbox Video)，并将其引入 Windows 10，然后是 Windows 11。现在，他们被带回同一个应用程序。

除了将音乐和视频播放功能合并到一个应用程序中，Windows 11 的新媒体播放器还有一个全新的设计，更符合 Windows 11 的其余部分。所有的图标都被重新制作，媒体播放器本身在中央有一个突出的播放按钮，比它在 Windows 10 应用程序中更加突出。

它可以让你添加任何你想要扫描音乐和视频文件的文件夹，创建播放列表，以及你期望从媒体播放器应用程序中获得的所有功能。它还支持视频和音乐文件的画中画模式，因此您可以在做其他事情的同时查看内容。当播放音乐文件时，应用程序将显示当前播放歌曲所在的专辑封面。

如果你担心失去经典的 Windows Media Player，不要担心，它仍然藏在开始菜单的 Windows 工具文件夹中。尽管微软多年来一直在尝试开发各种替代产品，但它并没有完全扼杀传统媒体播放器。

最近，微软宣布媒体播放器是 2 月份 Windows 11 的众多新功能之一，所以它现在只在测试版渠道提供有点令人惊讶。就在上周，对 Android 应用程序的支持在发布预览频道中可用，这通常是全面可用之前的最后一步。公平地说，新的媒体播放器应用不应该需要太多的测试。你也可以在预览版中查看我们的 Windows 11 功能列表，了解微软为未来准备了什么。
