微软为 Windows 11 开发的新画图应用正在向内部人士推出

原文：https://www.xda-developers.com/microsofts-new-paint-app-for-windows-11-is-now-rolling-out-to-insiders/#0001-01-01

上个月，微软展示了为 Windows 11 开发的新画图应用。它是众多被取笑的应用之一——包括剪贴工具、照片等等——但有一段时间，Paint 在行动中消失了。现在，终于在 Dev 频道向 Windows 内部人士推出了。

微软称之为“经典应用的现代旋转”，这基本上概括了这一点。随着 Windows 11 的推出，Paint 迎来了多年来第一次真正的视觉革新。它添加了圆角和云母之类的东西，这是一件大事。自从 Windows 8 时代以来，我们就有过这种不和谐的体验，一些应用程序针对新平台进行了重新设计，而 Paint 等传统应用程序却被遗忘了。

它不仅仅是云母和圆角。它有一个新的工具栏，正如你所看到的，它有全新的图标。彩色框现在是彩色圆圈，并且有新的下拉菜单，比如画笔。

事实上，这主要是一种视觉上的更新，采用一种熟悉的产品，并使其符合现代设计。不过有一个新功能，那就是一个新的文本工具。这是为了让你更容易地编辑文本。

新的 Paint 应用程序正在 Windows Insider 计划的开发频道上向 Windows 11 用户推出。也可以肯定地说，当 Windows 11 在 10 月 5 日发布时，这仍将是预览版。新操作系统仅在一周内发布，对于一个新应用程序来说，这似乎是一个很短的时间来通过测试和发布预览版渠道。

好消息是你不必等待功能更新或类似的事情。新的 Paint 应用程序应该会作为现有应用程序的更新通过微软商店发布，只要它准备好了。

这是微软为 Windows 11 戏弄的最后一个新应用。据推测，未来还会有其他收件箱应用程序更新，但现在，重点是 10 月 5 日新操作系统的发布。
