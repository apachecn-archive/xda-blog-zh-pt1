微软 Edge 正在测试一个现代 RSS 源，可以让你关注 YouTube 的创建者

微软 Edge 正在测试一种现代的 RSS 订阅源，这种订阅源可以让用户轻松关注 YouTube 的创建者。该功能是受控卷展栏的一部分。

在这个有影响力的人、博客作者和内容创作者的时代，每一种追踪我们最喜爱的在线 T2 名人的新方法都很重要。最新的增加来自微软，通过它的 Edge 网络浏览器。最新的金丝雀版本包括一个新的“可追随网络”功能——至少在的 Windows 上。它允许用户直接在 Edge 中轻松关注精选的 YouTube 内容创建者。值得注意的是，虽然这是一个受控的，有限的展示的一部分，并不是所有的金丝雀用户都可以访问。

由于 Reddit 用户 u/Leopeva64-2 已经发现了(通过 The Verge )，微软 Edge 现在允许一些用户直接关注一些 YouTube 内容创作者。访问 YouTube 页面时,“新建”按钮会出现在地址栏中。然而，它还不支持所有的频道。当用户关注一个博客时，他们最近的帖子就会被添加到 Edge 的收藏中。这使得他们可以轻松地跟上创作者的最新内容。

目前，只有少数 Edge Canary 用户可以访问这个可跟踪的网络功能，因为它是受控推出的一部分。很明显，这项功能还处于早期阶段——因为它还不能支持所有的内容创作者。微软可能会扩展这一功能，以支持除 YouTube 之外的其他网站的 RSS 源。

如果你已经使用了 Edge 的金丝雀版本，但还没有看到它的特性，那你就无能为力了。随着它变得更加稳定和完善，微软可能会向更多用户推出它。如果公司决定完全放弃这个项目，这个功能也有可能不会公开。微软还没有正式分享关于 Followable Web 的任何细节，所以没有太多关于测试版功能及其更广泛的可用性的信息。
