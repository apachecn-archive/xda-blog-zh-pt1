LG 即将推出的生活电视将成为您家庭的标志性产品

原文：https://www.xda-developers.com/lg-object-lg-standbyme-tv-launch/#0001-01-01

为了挑战三星的框架电视，LG 周二详细介绍了其新的生活电视系列。新的 LG Object 和 LG StandbyMe 电视采用了独特的设计和非常规的外形。

从 LG Object TV(型号 65Art90)开始，它采用了 65 英寸的有机发光二极管 Evo 面板，首次在 2021 年的 G1 系列中推出。这款电视看起来像一块帆布，设计用于靠墙放置，倾斜角度不超过 5 度。它配有一个织物罩，可以用遥控器升高或降低。全视图模式会将外壳完全降低，以完全露出显示屏，从而获得最佳的观看体验。同时，Line View 部分暴露显示屏，以显示时间和日期、音乐播放器和其他信息。

LG Objet TV 像一块画布，以其极简风格和中性色彩的巧妙运用，无缝融入任何房间，给任何室内空间带来即时的精致和视觉和谐，LG 在博客帖子中说。

除了具有令人惊叹的有机发光二极管显示屏，LG Object TV 还配备了 80W 的 4.2 声道音响系统，“提供电影院般的音频”这种可互换的织物套由丹麦纺织品创新者 Kvadrat 设计，有三种颜色:Kvadrat 米色、Kvadrat 红木色和 Kvadrat 绿色。

接下来是 LG StandbyMe 电视(27Art10)，该公司称之为“独一无二的娱乐中心”这款电视有一个 27 英寸的液晶面板、一个内置电池和一个带轮子的可移动支架。LG 声称 StandbyME 一次充电可持续 3 小时。它可以垂直和水平安装，并可以倾斜、旋转和转动，以获得最佳的观看体验。

LG StandbyME 配有一个可拆卸的底座，可以让你将手机放在电视上，并将其用作便携式网络摄像头。用户还可以通过 USB 或 HDMI 连接笔记本电脑或个人电脑，将电视用作外部显示器。此外，您可以使用移动屏幕镜像从 Android 或 iOS 设备传输内容。

LG 尚未透露 LG Object 或 LG StandbyMe 的定价和确切发布日期。我们期待在 CES 2022 上了解更多关于新电视的信息。
