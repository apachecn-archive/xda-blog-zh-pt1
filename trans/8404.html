Windows 11 build 22489 附带了一个新的 Microsoft 帐户设置页面

微软正在发布 Windows 11 Insider 预览版 build 22489，所有的新内容都是新的微软账户设置页面。

今天，微软向开发者频道发布了 Windows 11 内部预览版本 22489。像往常一样，没有多少新东西。事实上，虽然 Beta 频道现在能够测试 Android 应用程序支持，但 Dev 频道甚至还没有，尽管它被指定为首先获得新功能的频道。

此版本中的关键新功能是“设置”中的一个新页面，该页面显示了您的 Microsoft 帐户的详细信息。显然，你还会找到一些链接，让你可以管理你的账户。

虽然 Dev 频道在稳定性方面有很多风险，但在回报或新功能方面，目前提供的很少。即便是新的微软账户设置页面，目前也只对 Dev 频道的一小部分内部人士开放，以防这个功能让你垂涎三尺。

您还会发现一些比这更小的变化。Connect 应用程序现在被称为无线显示，这是一个更好的描述。在设置中。应用和功能将被分为已安装应用和高级应用设置页面。除了一长串修复和已知问题之外，就这些了。

Windows 11 内部版本 22489 修复程序

【任务栏】

  • 辅助显示器上的应用程序图标现在应该更可靠地绘制，而不是空白。
  • 停止了 explorer.exe 崩溃时，有时会发生使用桌面弹出上下文菜单。
  • 修正了当关闭桌面弹出时 explorer.exe 崩溃的问题。

【文件浏览器】

  • 在文件资源管理器中右击一个驱动器时，快速访问是一个顶级选项。
  • 我们改进了上下文菜单的启动性能。
  • 进行了一些修复，以帮助提高 explorer.exe 使用文件浏览器时的可靠性。

【开窗】

  • 在任务视图中关闭窗口现在应该看起来不那么紧张了。
  • 做了一些工作，以帮助解决一个问题，导致闪烁的应用程序窗口时，调整某些应用程序在最近的开发频道建设。

【设置】

  • 解决了在某些情况下进入 Windows Update 后导致设置崩溃的问题。
  • 添加了搜索触摸键盘设置时搜索结果中缺少的空格。
  • 修正了在轮子设置中自定义选项时设置崩溃的问题。
  • 如果动画关闭，使用 X 消除通知将不再有动画。
  • 修正了最近播放音乐时，媒体控制有时在快速设置中不显示的问题。也被认为影响了硬件介质密钥的使用。
  • 快速设置中 Wi-Fi 选项的工具提示不再会飞到屏幕顶部。

【其他】

  • 缓解了导致任务管理器中的进程选项卡有时为空白的潜在问题。这也被认为是导致 UAC 最近开放非常缓慢的根本原因。
  • 解决了 Xbox Game Pass 游戏无法安装的问题，错误为 0x00000001。
  • 修复了 PowerShell 中的 get-winevent 因 InvalidOperationException 而失败的问题(问题#60740 )。
  • 在最后几次飞行中缓解了 mousocoreworker.exe 的高碰撞事故。
  • 我做了一些工作，试图改善图标和文本都存在的情况下，通知按钮的文本布局。
  • 如果已经卸载了提示应用程序，入门应用程序将不再崩溃。
  • 修复了在以前的版本上更新时导致某些设备使用 SYSTEM _ SERVICE _ EXCPTION 进行错误检查的问题。
  • 做了一个基本的更改，以帮助修复导致一些用户在启动时看到意外的“坏图像”错误消息对话框的问题。

注意:在积极开发分支的 Insider Preview 版本中提到的一些修复可能会成为 10 月 5 日正式发布的 Windows 11 版本的服务更新。

Windows 11 内部版本 22489 已知问题

【常规】

  • 在这个版本中，你会注意到在主 Windows Update 设置页面下 Windows Update 、恢复和开发者的链接。您需要再次单击 Windows Update 来检查更新。“恢复”和“开发人员”链接不应出现在“设置”中的“Windows Update”下。这些问题将在未来的版本中得到解决。
  • 从版本 22000.xxx 或更早版本更新到使用最新开发频道 ISO 的较新开发频道版本的用户可能会收到以下警告消息:您尝试安装的版本是 Flight Signed。要继续安装，请启用航班签名。如果您收到此消息，请按下启用按钮，重启电脑，然后重试更新。
  • 一些用户可能会经历他们的屏幕和睡眠超时减少。我们正在调查更短的屏幕和睡眠超时对能耗的潜在影响。

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【文件浏览器】

  • 尝试重命名桌面上的项目在此版本中无法正常工作。如果您打开文件资源管理器，导航到桌面文件夹，并尝试从那里重命名，它应该可以工作。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。
  • 我们正在修复一个问题，该问题导致工具提示在悬停在任务栏角落后出现在意外位置。

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【快速设置】

  • 我们正在调查来自内部人士的报告，即音量和亮度滑块在快速设置中显示不正确。

在某一点上，开发频道将再次开始获得实际的新功能。事实上，功能更新现在是一年一次，而不是一年两次，所以微软可以再等一段时间来公开测试新东西。或许，这家位于雷德蒙德的公司正等着像它对 Windows 11 所做的那样，发布一个新功能的大公告。
