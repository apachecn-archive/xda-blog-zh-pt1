谷歌新的 Roboto Serif 字体让阅读更加舒适

原文：https://www.xda-developers.com/google-roboto-serif-font/#0001-01-01

Google 和 Commercial Type 推出了一种新的可变字体，名为 Roboto Serif。开源字体旨在让眼睛更容易阅读，并在各种屏幕尺寸和印刷中提供理想的阅读体验。

Roboto Serif 有一个最小的设计,只有“一点点的衬线”该字体是在经过大量不同形状和比例的实验以提高其易读性后从头开始创建的。Roboto Serif 有四个可变轴和六种光学尺寸(微型、小型文本、文字、卡片、显示和海报)。您还可以调整字体的宽度、高度、大小和等级。谷歌提供了九个现成的版本，包括薄、超轻、普通、中等、粗体和黑色。

Roboto Serif 是一种可变字体系列，旨在创造舒适、流畅的阅读体验。它体积小巧，功能强大，由于其重量和宽度涵盖了各种光学尺寸，因此可在任何地方使用(即使是应用程序界面)。

Roboto Serif 是 Roboto 超家族的最新成员，该家族包括 Roboto Sans、Roboto Slab、Roboto Mono 和 Roboto Condensed。Roboto Serif 的最小化设计和多功能性使其有别于 Literata 等其他可变 Serif 字体，后者主要用于屏幕阅读。在支持的浏览器中，字体利用 CSS 属性“font-optical-sizing”来提供改进的屏幕阅读体验。谷歌表示，这种字体在任何尺寸、任何格式下，从屏幕到印刷品，都可以轻松阅读。

“我们的目标是制造一种你可以用于长篇新闻或小说的字体——一些非常长且复杂的东西；一段身临其境的文字，你可以在手机上阅读，而不想抱怨它，”商业类型的格雷格·加兹多维茨说。
