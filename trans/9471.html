英特尔 Arc DG2 显卡的内存规格泄露

原文：https://www.xda-developers.com/intel-arc-dg2-memory-specs-leak/#0001-01-01

在 CES 2022 上，我们听说有几款笔记本电脑将搭载英特尔 Arc 独立显卡，这是该公司多年来首次大举进军 GPU 市场。英特尔对其即将推出的 GPU 保持沉默，但 Twitter 用户@ 9550 pro(viaWindows Central)可能已经让我们一窥这些英特尔 Arc 卡的规格，特别是关于它们的内存。

在今天发布的一条推文中，该用户分享了一个表格，其中包含一些关于英特尔 Arc 系列各种据称 SKU 的信息。这是专门针对笔记本电脑的 GPU，所以台式机可能会有一些差异。该推文包括一些 SKU，包括一个具有 512 个执行单元(eu)的高端配置，一个具有 384 个执行单元，然后是 256 个执行单元，最后是两个具有 96 个或 128 个执行单元的入门级型号。顶层配置有 16GB 的 GDDR6 RAM，带有 8 个 VRAM 模块和一个 256 位接口。

最大内存带宽为 512 GB/s，似乎与其他高端笔记本电脑 GPU 一致，整体内存容量也是如此——16GB 是英伟达的 RTX 3080 Ti 和 3080 笔记本电脑 GPU 上的内存，AMD 尚未在其任何笔记本电脑 GPU 上提供这么大的内存。低端型号只有高达 112GB/s 的带宽，略低于 AMD 的镭龙 RX 6500M。

当然，当谈到 GPU 性能时，内存只是等式的一部分，所以还有很多我们不知道的。这里看到的高端 SKU 中的 512 个执行单元是英特尔处理器中集成显卡的一大进步，英特尔处理器只有 96 个，但这不足以得出任何有意义的结论。这也很难与竞争对手的 GPU 进行比较，因为他们使用其他测量来指示整体性能。我们必须等待这些 GPU 实际可用，才能知道它们的性能如何。

我们所知道的是，英特尔 Arc GPUs 将支持光线跟踪和英特尔的 XeSS(超级采样)技术。它还将支持深度链接，允许具有独立和集成 GPU 的 PC 同时使用这两种 GPU 来执行编码任务，以进一步提高性能。宏碁的新 Swift X 将是首批搭载英特尔 Arc 的笔记本电脑之一，但尚未确定发布日期。
