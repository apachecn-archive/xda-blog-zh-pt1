谷歌发布 Jetpack Glance Alpha，实现超级简单的小工具开发

原文：https://www.xda-developers.com/jetpack-glance-alpha/#0001-01-01

Jetpack Compose 可以为 Android 应用程序创建界面，而无需处理冗长的 XML 文件和其他笨重的资源。相反，它允许开发人员用 Kotlin 代码(与应用程序的其余代码一起)创建界面元素，具有更简单的 API 和完整的材料设计支持。现在，随着 Jetpack Glance Alpha 的推出，谷歌正在将同样的易用性引入小部件。

谷歌在的一份声明中写道，“Android 12 对许多 Android 用户的一个关键功能——应用程序小工具进行了修改，使它们更有用、更漂亮、更容易被发现(84%的用户至少使用一个小工具)。今天，我们通过发布 Jetpack Glance 的第一个 alpha 版本，使构建它们变得更加容易，这是一个基于 Jetpack Compose 运行时构建的新框架，旨在更快更容易地为主屏幕和其他表面构建应用程序小部件。”

Jetpack Glance 允许应用程序开发人员使用类似于 Jetpack Compose 的声明式基于 Kotlin 的 API 来创建小部件。这个第一个 alpha 构建包括一组初始的 Glance 组件(框、行、列、文本、按钮、LazyColumn、图像、间隔)，一些预定义的动作(actionStartActivity、actionRunCallback 等)。)，以及与现有的 RemoteViews 的互操作性。

Jetpack Glance 在 10 月份的 Android 开发峰会上被提及，但是现在开发者们已经可以试用了。GitHub 上有的代码样本，Glance 兼容 Android Studio 最新的稳定版本。

最近几个月，谷歌推出了 Jetpack Compose 的其他几项改进。对 Material You 和 Wear OS 的支持于 10 月到来，用于创建 Android 汽车应用的 Compose 于 3 月发布，Jetpack Compose 本身于 7 月离开测试阶段。将 Jetpack Compose 带到 web 上的工作也正在进行中，这将允许一个具有单一代码库的应用程序跨 Android、macOS、Windows、Linux 和 web 浏览器运行。
