新的 Razer Book 和 Blade 15 高级型号预装了 Windows 11

原文：https://www.xda-developers.com/razer-fully-embraces-windows-11-with-new-laptops-and-compatibility-for-its-laptop-product-lines/#0001-01-01

Windows 11 正式来了。因此，我们现在希望笔记本电脑原始设备制造商推出运行最新操作系统的新机器。继宏碁发布之后，Razer 现在准备通过新的 Razer Book 和 Blade 15 Advanced 型号全面拥抱 Windows 11。此外，该公司表示，它正在扩展其所有笔记本电脑产品线与 Windows 11 的兼容性。

搭载 Windows 11 的全新 Razer Book

Razer 以 Razer Book 标志着其对生产力笔记本电脑领域的进军，现在新款笔记本电脑正在更新，可以开箱运行 Windows 11。是的，Razer Book 是今天发布的两款预装新操作系统的笔记本电脑之一。有了这台新机器，用户现在可以在具有可靠规格的硬件上体验 Windows 的最佳性能。

新款 Razer Book 笔记本电脑没有太大变化。它们仍然采用第 11 代英特尔酷睿处理器，并获得了 EVO 认证。您可以获得英特尔的 Iris Xe 显卡、触摸屏选项以及轻薄的外形，从而提高移动办公效率。唯一改变的是，最新的机型现在将预装 Windows 11，并且在官网上的全新价格为 999.99 美元起。顶级 4k 型号现在也有了，存储容量增加了一倍。

全新 Razer Blade 15 高级版

新上市的 Razer Blade 15 Advanced 也获得了 Windows 11 的待遇。事实上，这些运行新操作系统的新型号从今天开始就可以使用了。它们带来了我们对 Razer 旗舰机型的期待一样的顶级游戏体验。其他一切都保持不变，因此您仍然可以使用 Nvidia Advanced Optimus 和 G-Sync 支持的 240Hz QHD 或 360Hz FHD 面板选项。

Razer Blade 15 Advanced 系列拥有市场上一些最强大的游戏笔记本电脑，最高可配 8 核英特尔酷睿 i9-11900H 和配备 16GB GDDR6 VRAM 的 Nvidia GeForce RTX 3080。Windows 11 的变化对这些产品的定价没有影响，现有版本的起价为 2299 美元。如果你已经购买了其中一台笔记本电脑，那么从今天开始，你仍然可以升级到 Windows 11。

除了新的笔记本电脑，Razer 表示，与 Windows 11 的兼容性将扩展到其所有笔记本电脑产品线，届时将向现有笔记本电脑用户提供免费升级。所有 Razer 软件和外围设备也完全兼容新的操作系统。如果你想知道使用新操作系统是什么感觉，那么你可能想看看我们对 Windows 11 的全面回顾，它揭示了 Windows 这些年来的变化。
