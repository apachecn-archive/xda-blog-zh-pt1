Windows 11 愿望清单——这是我们希望在新操作系统中看到的！

原文：https://www.xda-developers.com/windows-11-feature-wishlist/#0001-01-01

上个月，我们列出了 15 个我们希望在 Windows 11 中看到的功能。操作系统现在是官方的，在很大程度上，我们知道我们会得到什么。其实还挺多的。它从一个大的视觉改革开始，包括中央开始菜单和任务栏，圆角，新的透明云母材料 UX，重新设计的通知中心，等等。

但那只是外壳。Windows 11 也通过亚马逊应用商店获得了对安卓应用的支持。这将被内置到重新设计的微软商店中，对开发者的限制更少，机会更多。现在是时候重温我想看的特性了，看看交付了什么，然后看看我现在想看什么。

我们希望在 Windows 11 中看到的功能:

动态壁纸

我们从 iOS 和 Android 中窃取了一些功能，我对此没有意见。这个愿望清单很大程度上与让 Windows 成为一个更现代的操作系统有关，可以与 Android 和 iOS 这样的操作系统竞争。随着 Android 领域的可折叠操作系统和苹果领域的 M1 的出现，这两个“移动”操作系统都在寻求占领消费者个人电脑市场的空间。如果 Windows 想在消费市场再生存 10 年，它需要进行调整。

所以我们来看看 Windows 11 中的一些原生动态壁纸支持。Android 已经拥有它十多年了，所以如果单核智能手机可以做到这一点，Windows PC 也可以。微软甚至在 Xbox 游戏机上支持它。我们确实有办法让第三方替代品在 Windows 上工作，但现在是时候获得官方支持了。

将该功能放在装有 Windows 11 的现代笔记本电脑上应该不成问题。很容易说，由于 ARM 处理器，它可以在 Android 上完成，但我们现在也有 Windows 上的 ARM。与十年前相比，我们今天也有了更高效的英特尔芯片。

没发生

如果你想要一个原生支持动态壁纸的微软操作系统，你仍然需要一个全新的 Xbox 系列 X|S，在 Windows 11 上，你仍然需要一个第三方工具，不像十年前的 Android 手机。

电动玩具集成

好吧，我不实际上希望看到 PowerToys 成为 Windows 11 的一个功能。我只是想看到微软通过 PowerToys 提供的一些功能集成到操作系统中，供所有用户使用。这是有先例的。多年前，当 PowerToys 首次推出时，它是为高级用户设计的，最终一些功能被集成到了操作系统中。

这里有一些例子。花式区域让您创建窗口布局。事实上，我考虑在这里放上适当的超宽显示器支持，但花式区应该涵盖它。在 Windows 上，你只能并排放置两个应用程序。分屏对于普通的 16:9 显示器来说很好，但对于超宽显示器来说就不那么好了。

其他功能，如快速调整图像大小的 Image Resizer 和批量重命名文件的 PowerRename，将是 Windows 11 中的伟大功能。这个想法是为了帮助人们更有效率，对吗？

部分发生

Windows 11 附带了一个名为 Snap Layouts 的新功能，类似于 Fancy Zones。它为您提供了多种窗口布局供您选择，并使用 Snap Assist 来填充其他位置。不幸的是，其他有趣的 PowerToys 功能没有入选。

免费的 Microsoft Office

等等，我刚才是说微软 Office 免费吗？我知道，这听起来很疯狂，但请听我说完。我是微软 365 家庭用户，我不打算很快取消。首先，微软已经有了免费版的 Office，也就是网络应用。这其实挺好的，会符合大多数人的需求。微软可以简单地将它们作为 pwa 安装在 Windows 11 上。

目前，大多数个人电脑会为你提供微软 365 的免费试用，这最终是微软希望你购买的。让我们允许人们从 PWAs 开始，如果他们需要，让他们升级。它可以伴随着从 Windows 11 中移除广告和膨胀软件。

没发生

我真的没想到会这样，但那会很好。与其说是提供免费 Office，不如说是让人们更容易选择免费版 Office。很多人甚至不知道网上办公是免费的。我很乐意看到这成为默认，但这始终是一个白日梦。

智能家居控制

这个既简单又复杂。作为 Windows 11 中的一个集成功能，您应该能够控制您的智能家居设备。我对这个愿望清单的一个关键想法是，我从来不想说人们应该使用他们的手机来完成任何特定的任务。不，我目前使用的设备，任何设备，应该可以做我当时需要做的任何事情。

这就是事情变得复杂的地方。对于一个强大的智能家居控制中心，你需要像谷歌助手，亚马逊 Alexa，甚至苹果的 Siri 这样的东西。微软 Cortana 不打算删除它，原因超出了本文的范围。

但最重要的原因是，微软只是没有与智能家居供应商的合作伙伴关系。谷歌和亚马逊是这样做的，但微软不会在自己的操作系统中给他们最好的空间。

没发生

这是另一个白日梦，原因是微软只是没有它的平台。我们确实在任务栏中有一个新的快速设置部分，所以可能是可扩展的。我们仍有希望。

快速键

微软不在乎你用什么操作系统，只要你用它的服务。也就是说，在 Windows 上可用的内容和你在 iOS 或 Android 上可以找到的内容之间存在一些奇怪的差异。Outlook 就是一个很好的例子。

键盘是另一个。Windows 10 有触控键盘，没问题。团队投入了大量的工作，并不断推出新功能。但是这里有一个想法——用 SwiftKey 就可以了。SwiftKey 在 iOS 和 Android 上都非常受欢迎，如今，微软拥有了它。它真的很好，我不仅希望在 Windows 上看到它，而且它将在 Windows 和它的移动产品之间带来一些急需的一致性。我们稍后会回到一致性上来。

事情发生了

好的，当我说它发生时，你仍然没有在 Windows 中看到任何 SwiftKey 品牌。不过，你可以为触摸键盘提供更多定制选项，这很好。根据它的位置，你还可以使用滑动手势。

内置屏幕录制

是的，你已经可以用 Windows 10 录制你的屏幕了。好吧，以一种迂回的方式，你可以。通过 Xbox 游戏栏，你可以选择录制游戏，也可以告诉它某些应用程序是游戏，并录制那些游戏。但是有些应用程序不能使用这种变通方法。

如果你想记录你的整个屏幕和屏幕上发生的任何事情，你现在需要一个第三方应用程序。Android 和 iOS 都内置了屏保，Windows 11 也该效仿了。微软真的必须知道 Windows 上对屏幕录制的需求，所以如果这能以一种适当的方式发生就好了。

没发生

我们有一个全新的快速设置部分，这将是一个内置屏幕记录器的完美位置，就像我们在 macOS，iOS 和 Android 上一样。但它不在那里。你仍然必须使用第三方应用程序或有限的 Xbox 游戏酒吧录像机。

更好的收件箱应用

Groove Music、照片、邮件、日历、Skype、天气、OneNote、电影&电视和 Edge 都是 Windows 10 内置应用程序的例子。现在，问问你自己你实际上用了多少。OneNote 棒极了，Edge 是我的默认浏览器，因为它是由 Chromium 制成的。但对我来说，我尽量远离其他人。

有了 Windows 11，让我们做得更好。作为捆绑应用程序的邮件和日历可能是最好的例子。它们实际上应该是 Outlook 应用程序，尽管它们与 Outlook 家族中的其他应用程序没有任何相似之处。Outlook 在 Android 和 iOS 上非常棒，所以也许我们可以将 UX 带到 Windows 方面。

Groove Music 感觉像是微软过去提供订阅音乐服务时的遗物。它现在只是一种默认的音乐播放器，但它看起来仍然像是没有真正意义的东西。电影和电视是另一个处于同样困境的公司，因为没有一个头脑正常的人从微软购买电影或节目。除了 Xbox 和 Windows 之外，这项服务在任何地方都不可用，所以电影和电视应用程序最终真的只是一个默认的视频播放器，就像 Groove 一样，该产品最终感觉格格不入。

让我们在 Windows 11 中拥有更好的内置应用程序。曾经有一段时间，微软专注于做好这方面的工作。让我们再做一次。

发生了

默认情况下，不会安装 Paint 3D、Skype 和 3D Viewer 等应用程序，所以这只是一个开始。设置和微软商店正在进行重大改革，它们都比 Windows 10 的前辈好得多。至于邮件&日历、地图等其他应用，我们还得拭目以待。

无缝更新

当你走出发烧友的圈子，很明显每个人都讨厌 Windows 更新。无论是大的功能更新还是质量更新，大多数人只想打开电脑，开始工作，并在完成后关闭电脑。

但说到现代操作系统，无缝更新是微软应该注意的事情。这是我们已经在 Android 和 Chromebooks 上看到的东西。Android 是一个更好的例子，因为当你的手机更新时，后台会发生很多事情，重启后也会发生很多事情。

让我们去掉重启电脑安装和更新与启动 Windows 11 之间的所有额外时间。该重新启动时间应该与任何重新启动时间相同。让更新以更无缝的方式完成，不会分散消费者对他们需要做什么和何时需要做的注意力。盯着蓝色的“Windows 正在更新”屏幕看几分钟并不是开始一天工作的好方法。

微软正在尽最大努力

当 Panos Panay 发布 Windows 11 时，他说更新小了 40%,而且安装速度更快。Windows 11 甚至会尝试预测你的机器重新启动需要多长时间，帮助你更好地选择何时按下按钮。

这是一个开始，但也不是我写这篇文章时的想法。我梦想 Chrome OS 或 Android 如何更新。所有的更新都是在您使用电脑时进行的，然后重新启动所花的时间不会比常规重新启动所花的时间长。离线时间尽量少。

Xbox 游戏

在 Windows 10 的整个生命周期中，微软表现出对游戏的重新关注。游戏主管菲尔·斯潘塞这些天甚至在微软的高级领导团队中，这不能说是关于 Windows 和设备主管帕诺斯·帕奈。不过，这也说得通。Windows 无疑是最可行的 PC 游戏平台，除此之外，微软还有一台游戏机。这还不是全部，因为最重要的部分是它有一个云后端。

微软有个叫 Xbox Play Anywhere 的东西。这些游戏可以在 Windows 10 或 Xbox 主机上玩，所以你购买一次，然后在任何地方玩。随着微软商店和 Xbox Game Pass for PC 中游戏的增多，PC 游戏也重新受到关注。你甚至可以在你的电脑上玩 Xbox 游戏。

让我玩 Xbox 游戏吧。在准备光晕:无限的时候，你可以在你的 PC 上玩光晕:主首席收集，但是你不能玩光晕 5 ，因为那只在 Xbox 上。这将真正成为 Windows 11 的一个独特功能。

虽然这看起来很简单，因为微软拥有这两个平台，它可以在技术上把它们结合在一起，但事实并非如此。视频游戏许可变得很奇怪，所以将真正的 Xbox 游戏带到 PC 上可能还有其他问题。与此同时，我们只能通过 Xbox 游戏流来玩。

游戏流媒体即将登陆 Windows

Xbox 云游戏将在 Windows 11 中加入 Xbox 应用程序，因此你将能够玩很多你想玩的游戏。这与原生运行它们不太一样，但仍然非常棒，而且降低了硬件要求的门槛。

活瓷砖的死亡

活瓷砖已经慢慢消亡，但坦率地说，我想在这方面全力以赴，并说所有的瓷砖都需要消失。它们是 Windows Phone 时代的遗留物。

如果你不熟悉，实时磁贴是应用程序的方形或矩形快捷方式，可以动态显示不同的信息。例如，您可能有一个电子邮件应用程序，它有一个显示您最新电子邮件预览的动态磁贴。在 Windows Phone 8 上，它就像是你主屏幕上的一个通知中心。

我喜欢 Windows Phone，但我从来没有使用过动态磁贴。我总是发现通知出现得太慢，或者我从来不想等待通过磁贴来循环显示它的信息。我也总是发现他们太忙了，有太多东西要搬来搬去。

Windows 10 将这一点带到了桌面上，在桌面上它们更加无用。它们位于开始菜单中，所以你必须打开开始菜单来阅读这些东西。活瓷砖慢慢走了，瓷砖变得更有型了一点，通透，和主题很搭配。

但是摆脱他们。任何有尖角的东西都需要在 Windows 11 中消失。就像其他操作系统一样，给我们简单的图标点击。

事情发生了

活的瓷砖是死的。故事结束。很好的解脱。

一键卸载

这个很简单。在 Windows 10 中，有两种方式卸载应用程序。如果该应用程序来自微软商店，您可以在开始菜单中右键单击它，然后单击“卸载”。对于其他所有东西，这是大多数应用程序，你必须进入控制面板，找到应用程序列表，双击你想要卸载的应用程序，并通过软件供应商为自己的卸载过程设计的任何体验来工作。

对于所有应用程序，请使用第一种方法。让我点击右键，然后卸载。让它成为 Windows 11 的一个特色吧。

商店应用更现代，因为它们是打包的，无论是 Win32 还是 UWP 应用。这使得它们很容易被删除，就像在现代操作系统如 Android 和 iOS 上一样。来自网络的未打包的 Win32 应用程序会在你的系统中到处传播文件，并不断改变注册表等。

对此有几种解决方案。一个是确保所有的应用程序都被打包，可能是在安装过程中。另一个是让 Windows 11 更聪明地清理被卸载的应用程序。如果是后者，我的第二个要求是，操作系统应该确保所有的东西都消失了。允许软件供应商创建自己的卸载体验的痛点是，该公司可以选择在您的系统上留下任何它想要留下的东西。

这是一个现代操作系统应该解决的问题，用户不应该考虑这个问题。

没发生

新的微软商店向更多的应用程序开放了店面，而且准入门槛也低得多。但是，让我们想出一个办法，让操作系统找出卸载。

不再有广告

Windows 8 是第一个拥有应用商店的操作系统版本，但并不成功。没有多少公司为此开发，那些开发了的公司也不会更新他们的应用程序。当 Windows 10 出现时，微软宣布了一个巨大的胜利，称糖果粉碎传奇将进入该操作系统。

公告的另一半是糖果粉碎传奇将被预装在 Windows 10 的每个安装程序中。那是 Windows 中广告的开始。大量的其他应用程序紧随其后，而且它通常因 PC 安装而异。在过去，膨胀软件来自原始设备制造商，但随着 Windows 10 和原始设备制造商制造签名 PC，膨胀软件实际上来自微软。

这应该在 Windows 11 中停止。唯一预装的应用程序应该是收件箱应用程序，如邮件、日历、OneNote、照片等。

除此之外，我们还需要更少的广告。每次我安装新电脑时，都会在 Windows 中收到通知，要求我激活 Microsoft 365 试用版。即使我使用我的 Microsoft 帐户登录，并且我已经是订阅者，这种情况还是会发生。让我们也带回纸牌和扫雷舰，因为当它们在微软商店的时候，它们装满了广告。

让我们回到操作系统只是操作系统的 Windows 7 时代。它是用户和你想要访问的应用程序之间的一个网关。不要再试图向我推销东西，不要再给我弹出窗口试图让我使用新功能，让我打开 Edge 做我的工作。

简单点说吧。如果它有一个行动要求，或者是一个弹出窗口，我必须清除才能回到我的工作流程，我不想在 Windows 11 中使用该功能。

预装 app 较少

这是一个白日梦，我写的时候就知道，但如果我不把它列入清单，我就不会诚实。Windows 11 似乎确实摆脱了大量预装的第三方应用，但我们仍处于早期测试阶段。现在，如果微软可以在 Edge 的新标签页上放下广告，在我关闭它们时停止发送操作系统通知，等等，那就太好了。

更好的触摸控制

Windows 8 是一个似乎专门为触摸屏设计的操作系统。Windows 10 出来的时候，目标是取 Windows 7 和 Windows 8 之精华，放在一个产品里。但是，如果你是一个平板电脑用户，你可能会看到微软缩小了太多。

Windows 11 应该对支持触摸的电脑有好处，这是我们已经知道即将到来的一个功能。如果你不相信我 Surface Pro 7 的触摸体验不如 iPad，那么你就没有试过 iPad。但这就是 Windows 的问题。iPadOS 是为了获得触摸优先的体验而从头开始构建的。Windows 不是。Windows 是为鼠标和键盘设计的，后来也适应了触摸操作。

我们在 Windows 10 中有类似平板模式的东西，开始菜单全屏显示。如果你点击一个文本框，触摸键盘就会出现；好吧，如果你幸运的话。老实说，如果微软想开始更好的触摸控制，团队需要放下笔。Windows 平板电脑和敞篷车确实需要大量使用笔，但笔应该是增强体验的东西，而不是必需品。确保它可以很好地用手指操作，然后添加笔功能。

幸运的是，这是我们在 Windows 11 中肯定会看到的一件事。

事情发生了

Windows 11 在很大程度上以更好的触摸控制为中心，我写这篇文章时就知道这一点。它支持三个和四个手指手势，这与您已经从 Precision touchpad 手势中了解的手势相同。这是一种更好的体验。不过，我还是不认为我会喜欢它胜过 iPad。

一致性

一致性似乎总是 Windows 的致命弱点。这个问题可以追溯到 Windows 8 时代，那时我们有独立的桌面和城域环境。尽管 Windows 10 清理了一些东西，但一致性问题变得更糟了。

对于 Windows 10，我们仍然有传统的 Win32 应用程序，我们有 Windows 8 Metro 应用程序，但我们也有新的 UWP 应用程序。他们看起来完全不同。然后情况变得更糟。

Windows 10 发布几年后，微软发布了流畅的设计系统。Fluent 将分阶段推出，这基本上意味着它将随着时间的推移而发展。它有丙烯酸透明的元素，整个想法是，这是一个设计系统，可以在 2D 和 3D 环境之间很好地工作。毕竟，UWP 的应用程序应该可以在传统电脑和混合现实耳机之间的任何设备上运行。

结果不太好。Fluent 如期出货，几乎没有人为它更新应用；可悲的是，这甚至包括微软。你可能会认为，随着 Windows 新设计语言的发布，微软至少会在第一天就准备好自己的应用程序。它没有。甚至部分 Windows 直到后来才获得第一阶段的元素。这是一个奇怪的问题，微软将进入流畅的第二阶段，但仍将在各种产品中推出第一阶段。

在 Windows 11 中，一个关键的特性就是一致性。这也可以归结为我上面提到的更好的收件箱应用程序。由于 Windows 11 很大程度上是 Windows 10 版本 21H2，我真的希望它不只是使用 Windows 10 的旧 UWP 收件箱应用程序。我希望它们的设计符合 Windows 11。

这似乎正在发生

圆角出现在整个操作系统中，如果你有专为 Windows 10 打造的东西，你会知道的。微软已经通过内部渠道更新了 Office 和 Edge，而且似乎进展得相当快。

但是，从我自己的测试来看，它似乎工作得很好。我没有注意到使用 UWP 应用或 Win32 应用时的行为变化。它似乎只是工作。

为了让它工作

好吧，剩下的就扔了吧。让它工作。每当我的电脑上出现 BSOD，它就会向微软发送一条消息，告诉我发生了什么。使用 Windows 11，检查每一个错误报告并修复每一个错误。就是这样。这是 Windows 11 的一个功能，也是最好的功能。

别让我重装驱动程序。事实上，我再也不想考虑设备管理器了。这是我在使用新款 iMac 时注意到的第一件事。我去打印，我的打印机就在那里。在 Windows PC 上，我去打印，然后我必须选择添加新打印机的选项，然后它启动设置，然后我选择再次添加打印机的选项，然后它在我的网络上搜索打印机，然后我必须安装打印机，最后，我必须重新启动我试图打印的文档的打印对话框，因为 Windows 不够智能，无法自己添加打印机。体验是如此漫长和不必要的繁琐，它让你想知道普通用户是如何习惯它的。

不要误解我。Windows 能工作几乎是个奇迹。仅 Windows 10 目前就安装在超过 13 亿台设备上，这些设备包括成千上万种不同 CPU、GPU、RAM、存储、制造商、外设等配置。显然，苹果只需要与少数系统合作。是的，Windows 的时间都在工作。但当它不这样做的时候，它是绝对令人恼火的。

这就是为什么我对 Windows 11 的第一要求是它应该能够正常工作。不要再试图用我根本不想要的花里胡哨的东西来赢得我的心，比如画图 3D、故事混音等等，只要让事情运转起来就好了。当我想工作的时候，Windows 11 需要成为一个不妨碍我的平台。

我们必须等等看

我已经在太多的电脑上安装了 Windows 11，已经数不清了。它太棒了，我喜欢它的一切。我对 Windows 10 没有这种感觉。它似乎很棒，而且似乎是迄今为止最好的 Windows 版本。至于它推出时是否真的有用，我们还得拭目以待。

续！

Windows 11 现在已经正式发布，任何拥有兼容硬件的人都可以试用。除了已经宣布的，我不期望有太多的改变，但是一切皆有可能。此外，Windows 11 将获得每年的功能更新，所以即使我们的愿望不会随着最初的发布而实现，但在未来可能会实现。

文件浏览器中的选项卡

这是我在发表这篇文章后拍了拍我的额头的那些特征之一，因为它应该在那里。其实每个 app 都应该有标签页。Office 为什么没有选项卡？我跑题了。

我不知道有多少次我打开了多个文件资源管理器实例，只希望它们是一个窗口。更奇怪的是，这是人们普遍认同的事情之一。如果微软这样做了，就会有人制造一个“人人都喜欢”的迷因。出于某种原因，它就是做不到。

每隔一段时间，就会发生一些事情，从根本上改变我们使用计算机的方式。主流互联网是一个，然后宽带互联网是另一个。第一台个人电脑就是其中之一，然后最近我们有了现代智能手机。但是一个从根本上改变了我们使用电脑方式的软件变化是标签浏览。感觉 tab 一直都在这里。但是现在我们要问为什么没有更多带标签的东西。这很有意义。

关掉控制面板

几十年来，控制面板一直是 Windows 的一部分。在 Windows 8 中，微软推出了一个设置应用程序，复制了一些功能，在过去十年中，该公司一直在逐步取消控制面板的部分功能，并将它们转移到设置中。是时候扔掉这个东西了。

微软有一个习惯，试图同时做这么多事情，以至于它从来没有真正完成任何事情。我们从流畅的设计中看到了这一点。将控制面板移植到一个新系统不应该需要十年时间，要经历 Windows 的三个主要版本。我肯定这是在雷德蒙某人的待办清单上。

还有，如果你杀不死控制面板，就现代化它(哦等等；这是设置)并使其支持黑暗主题。这也适用于注册表编辑器，设备管理器，任务管理器，以及其他 Windows 的古老部分，它们已经很久没有被触及了。

经典模式

在抱怨了操作系统中的遗留组件后，也许这似乎违反直觉，但 Windows 11 应该有一个经典模式。这是以前存在于 Windows 中的东西。您通常可以将 UX 切换回之前版本的外观和感觉。这对企业尤其有利。

Windows 8 没有做到这一点。它将新的、不和谐的环境强加给用户。Windows 10 也没有经典模式，因为坦白说，谁愿意回到 Windows 8？Windows 11 应该给用户切换到 Windows 10 UX 的选项。如果事实上，有一个注册表调整你可以做旧的开始菜单回来，但微软删除了它。

在这一点上，似乎我们只能等待一些定制实用程序。

改变 Windows 11 版本的工作方式

我几乎呼吁用这个来消除 Windows 11 SKUs，我们知道这是不会发生的事情。毕竟，它是唯一一个拥有家庭、专业、工作站、企业和教育等 SKU 的操作系统。除了令人困惑的事实之外，它还产生了一个非常现实的问题。

大多数电脑都配有 Windows 11 Home。你可以花 3000 美元买一台游戏电脑，这就是你所能得到的。但正如我们所知，人们并不总是为了游戏而购买游戏电脑。他们用它们来做视频编辑、软件开发等等。但即使买了一台超级高端电脑，你仍然需要支付 99 美元升级到 Windows 11 Pro，让你可以运行 Hyper-V。

Windows 11 Pro 比 Windows 11 Home 附带的大多数功能对最终用户来说都不重要，尽管 Windows Sandbox 可能很重要。尽管如此，这是一个抓住你的时刻。你出去花了几千美元买了你能找到的最好、最强大的电脑，回家后你意识到你需要花钱设置一个 Hyper-V 虚拟机。

您手机的 iOS 支持

微软的 Your Phone 应用程序是该公司在没有自己的移动平台的情况下，试图将你的手机和个人电脑连接起来。苹果和谷歌都提供桌面和移动平台之间的集成，因此微软正试图重新创造这一价值主张。但是，在 iOS 上就不行了。

在这一部分，我们都指责苹果锁定了操作系统，并说微软只是尽了最大努力。如果其他第三方供应商不能与 iOS 兼容，这个借口会更有效。

但是他们是。这就是戴尔移动连接的用武之地。戴尔实际上已经设法在 iOS 和 Android 之间实现了功能对等，包括文件共享和屏幕镜像等。没错；如果您用 iPhone 录制视频，并且曾经有过将视频传输到 Windows PC 的噩梦般经历，戴尔移动连接的速度大约与隔空投送一样快。当然，它也发送和接收文本。

同样，流行的借口是苹果的产品太受限制，这不是微软的错。但事实并非如此，因为戴尔做到了，而且做得很好。

如果你愿意，可以称之为 iPadOS 的一个特性，因为我有一种感觉，这就是它将如何发展。iPadOS 14 在用户界面的一个专门部分添加了重新设计的小工具，而在 iPadOS 15 中，苹果认识到了自己的错误，并允许 iPad 用户将小工具钉在他们的主屏幕上，就像 iPhone 用户已经可以做的那样。

窗口小部件在 Windows 11 中回归了。任务栏中有一个专用的 widgets 按钮，可以打开一个专用的 widgets 部分。你可以重新组织它们，调整它们的大小，自定义它们，添加更多，删除一些，但是你不能把它们固定在桌面上。

这似乎是一个常识性的特征。我敢打赌，一旦微软完成了这项工作，它的下一步将是让用户把小工具钉在桌面上。
