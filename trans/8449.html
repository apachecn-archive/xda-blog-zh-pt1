红米 Note 11、华为 Watch GT 3 和 Xperia Pro-I XDA 论坛开放

原文：https://www.xda-developers.com/xda-forums-redmi-note-11-series-huawei-watch-gt-3-xperia-pro-i/#0001-01-01

不出所料，Techctober 充斥着智能手机公告。虽然 Pixel 6 显然是 10 月份最大的发布会，但在月底还有多个有趣的发布会。小米发布了其最新的 Redmi Note 11 系列，华为推出了一对带有鸿蒙系统 2.1 的智能手表，索尼用新型号刷新了其 Xperia Pro 阵容。

我们很高兴地宣布，XDA 论坛现已对上述三种设备开放。

小米 Redmi Note 11 系列

上周，小米推出了新的红米 Note 11 系列，包括红米 Note 11 Pro+，红米 Note 11 Pro 和红米 Note 11 5G。Redmi Note 11 Pro+是三款手机中功能最丰富的，提供高达 120W 的快速充电支持，据称可以在 15 分钟内将手机充满电。Redmi Note 11 Pro 与 Pro+没有太大区别，两者只是在充电速度、电池和 SoC 方面有所不同。最后，Redmi Note 11 5G 是新系列中最便宜的产品，配备了 6.6 英寸液晶显示器，刷新率为 90Hz，联发科天玑 810 SoC，最高 8GB 内存和 128GB 存储，50MP 主摄像头，以及 5,000mAh 电池，支持 33W 快速充电。要了解更多细节，请务必查看我们的红米 Note 11 系列的发布报道。

华为手表 GT 3

华为 Watch GT 3 是中国华为 Watch 3 的改装版。这款智能手表有 46 毫米和 42 毫米两种尺寸，运行鸿蒙系统 2.1。这两款机型都配备了 AMOLED 显示屏、传感器阵列、GPS、5 ATM 防水、蓝牙 5.2 和无线充电。

华为 Watch GT 3 XDA 论坛

索尼 Xperia Pro-I

新发布的索尼 Xperia Pro-I 是去年 Xperia Pro 的直接继承者。Xperia Pro-I 售价 1800 美元，是目前最贵的不可折叠手机之一。但索尼表示，它的目标不是主流消费者，而是专业摄影师和创作者。这款手机的主要亮点是它的摄像头硬件，具有一个 12MP 主摄像头和一个巨大的 1.0 英寸传感器，以及两个 12MP 超宽和长焦摄像头。
