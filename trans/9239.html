谷歌 Pixel 3 系列获得了一个令人惊讶的更新，修复了紧急呼叫错误

谷歌正在为 Pixel 3 和 Pixel 3 XL 推出一个特殊的错误修复更新，以解决紧急呼叫错误。请继续阅读！

谷歌一贯在每个月的第一个星期一发布新的 Android 安全公告，在过去几年中，他们几乎每个月都这样做。按照计划，2022 年 1 月的 Android 安全更新公告于昨天上线，该公司开始推出支持 Pixel 设备的新更新(没有 Pixel 6 家族)。新的补丁还为安卓系统的一个问题提供了永久修复，该问题可以阻止紧急呼叫。虽然 Pixel 3 和 Pixel 3 XL 没有获得新的安全补丁，但谷歌已经决定反向移植紧急呼叫漏洞修复程序，并为这两部手机推出一个特别的更新。

Pixel 3/3 XL 最初于 2018 年发布，但去年 10 月标志着主要软件更新的结束。谷歌告诉9 点到 5 点谷歌T5】计划为这些设备进行最后一次更新。然而，新的版本并不是最终的版本。软件版本为 SP1A.210812.016.A2 ，现在谷歌 Pixel 3 家族的解锁版和威瑞森 变种都可以使用，作为一个体积不到 500 KB 的微型 OTA。

值得注意的是，911 呼叫故障最初是由 Pixel 3 用户引起注意的，这可能是谷歌官方(T2)将 Pixel 3 和 Pixel 3 XL 列为有资格接受错误修复更新的原因。这个版本的工厂和 OTA 图片也已经发布。请记住，它仍然是 Android 12，它仍然在 2021 年 10 月的安全补丁级别。

不幸的是，Pixel 3 和 Pixel 3 XL 用户没有获得任何在上次 Pixel 功能下降中详细介绍的新功能。但是，如果其中任何一个对你来说听起来不错，你可能会考虑升级你的设备，因为它们已经正式达到了它的寿命。
