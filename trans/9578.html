想在微软 Surface Duo 上使用 Windows 11？它正在到达那里

原文：https://www.xda-developers.com/windows-11-microsoft-surface-duo-in-development/#0001-01-01

最初的微软 Surface Duo 与 Android 10 一起出厂，手机的软件体验以漏洞百出而闻名。然而，一些用户设法从中获得乐趣，我们已经看到了继任者的出现，即微软 Surface Duo 2 T1。不过，它仍然是一款基于安卓系统的设备，没有官方支持的方式在上面运行 Windows(或 Windows Mobile )。然而现在，一位名叫 Gustave Monce 的开发人员在微软 Surface Duo 上首次向我们展示了 Windows 11。

请记住，这只是 Windows 11 的安装程序，而不是 Windows 11 本身。看来 Monce 还没有在微软的 Surface Duo 上推出完整版的 Windows，不过 T2 看来是他们的目标。Monce 曾经在 Lumia 950XL 上引导 Windows 10X，之后又在 Lumia 950 XL上引导 Windows 11。您甚至不太可能希望在日常驱动程序中完全安装 Windows 11，因为它不太可能在安装后完全用作手机。

Windows 作为常规 x86 PCs 的操作系统而闻名，但微软的操作系统也可以在几种具有非传统外形的设备上找到。得益于 Windows 11 的 64 位仿真能力，ARM 上的 Windows 也在稳步扩张。微软并没有太多地关注移动设备，尽管 Renegade 项目旨在简化 Android 设备上 ARM64 Windows 11 和 Windows 10 的启动。微软 Surface Duo 上的 Windows 11 目前处于非常早期的开发阶段，但这绝对是一个良好的开端。

至于推文中显示的版本，它是 Windows 11 内部版本 22523 ，尽管最新的内部版本是 22543 。Monce 使用旧版本的原因只是因为他可以使用现成的安装程序。我们联系了 Monce，他也向我们确认了初步的固件实现已经完成。经过一些润色之后，他将着手实现 Windows 驱动程序本身。

我们可能还需要一段时间才能在微软 Surface Duo 上看到这一点，但这肯定是一个值得关注的有趣项目。当你可以在手机上安装 Windows 桌面时，谁还需要 Windows Mobile 呢？
