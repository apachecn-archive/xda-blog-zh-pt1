Chromebooks 的 Steam 可能很快就会到来

原文：https://www.xda-developers.com/steam-chromebooks-might-come-soon/#0001-01-01

Chromebooks 非常适合工作，通常价格实惠，重量轻，电池续航时间长。Chrome 操作系统的功能对于大多数人来说刚刚够用，但当涉及到游戏时，他们就犹豫了。Steam 是 Windows、Mac 和 Linux 上最受欢迎的游戏平台之一，即使你有一部最好的 chrome book，你仍然不能玩任何游戏。如果你真的想的话，当然有办法让现在就在你的 chrome book上运行 Steam，但不如原生支持好。这种情况可能最终会改变。

根据一份提交给 Chromium Gerrit T1 的合并声明(由 T3 Android Police T5 发现), Chrome OS 的原生 Steam 最快将于本月发布。Chrome OS 的 Steam 支持代号为“Borealis ”,虽然我们知道 Valve 从去年年初就想在 Chrome OS 上推出 Steam,但从那以后我们没有听到更多消息。现在 Android Police 表示，Chrome OS 98 已经添加了标志——应该随时会发布。

据报道，Chrome OS 上的 Steam 将通过 Chrome OS 的 Linux 兼容性来启用。使用克罗斯蒂尼可以在 Chrome 操作系统上运行 Steam Linux 客户端，但官方支持将为用户提供更好的体验。谷歌似乎致力于在 Chrome OS 上提供良好的游戏体验，因为该公司最近被发现也在为该平台准备新的“游戏模式”。

Android Police 还发现承诺将会有一个与 Proton 的游戏兼容性报告系统。Proton 是一个兼容层(由 Valve 带头)，允许基于 Windows 的游戏在基于 Linux 的操作系统上运行。它基于 Wine，并集成到基于 Linux 的机器上的 Steam 客户端中。

无论哪种方式，这都是比目前可用的解决方案更好的在 Chromebook 上玩游戏的方式。Chromebook 游戏最可行的时候是去年年底支持 Steam Sync 的 GeForce NOW 平台，即使这也需要 T2 稳定快速的互联网连接，更不用说订阅了。
