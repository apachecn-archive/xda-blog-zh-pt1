谷歌发布 Project Relate，这是一款为有语言障碍的人设计的语音识别应用

原文：https://www.xda-developers.com/google-project-relate-app-for-people-speech-impariments/#0001-01-01

为了帮助有语言障碍的人更容易与谷歌助手和其他人交流，谷歌宣布了一个新的 Android 应用程序，名为 Project Relate。

正如谷歌在博客文章中指出的，有数百万人的语言因 ALS、脑瘫、中风、创伤性脑损伤或帕金森氏病等疾病而受到影响。因此，这些人通常很难被其他人和语音识别服务理解。目前处于测试阶段的 Project Relate 应用程序旨在应对这些挑战。该应用程序有三个主要功能:聆听、重复和助手。

  • 收听:通过收听功能，Relate 应用程序实时将你的讲话转录成文本，这样你就可以将文本复制粘贴到其他应用程序中，或者让人们阅读你想告诉他们的内容。
  • 重复:您可以使用重复功能，用清晰的合成声音重复您说过的话。我们希望这在面对面的交谈中，甚至当你想对你的家庭助理设备说出命令时，特别有帮助。
  • 助手:从 Relate 应用程序中直接与你的谷歌助手对话，这样你就可以轻松地处理不同的任务，比如开灯或播放歌曲。

谷歌目前正在澳大利亚、加拿大、新西兰和美国寻找会说英语的人来试用这款应用并接收反馈。测试人员将被要求记录 500 个短语，这些短语将用于更好地理解用户独特的语音模式，并提供对上述功能的访问。

谷歌表示，为了开发这款应用，它与许多有语言障碍的人密切合作，包括谷歌的品牌经理奥布里·李，他的语言受到肌肉萎缩症的影响。

Project Relate 建立在 Project Euphonia 的努力和研究基础上，这是谷歌语音和研究团队的一项广泛努力，旨在使语音识别技术更容易为语言障碍者所用。
