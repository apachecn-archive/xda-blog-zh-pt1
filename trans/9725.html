Windows 11 build 22557 增加了开始文件夹、触摸手势等等

原文：https://www.xda-developers.com/windows-11-build-22557-touch-gestures-start-folders/#0001-01-01

在两周没有新版本之后，Windows 内部人士今天可以期待 Windows 11 build 22557 了。距离上一个版本发布已经整整两周了，这个版本中有很多新的东西。这是迄今为止功能最丰富的版本之一，对开始菜单、平板电脑触摸导航等进行了改进。

一个大的新变化是能够在开始菜单中创建文件夹。现在，您可以将一个应用程序的图标拖放到另一个应用程序上来创建一个新文件夹，这样就可以更容易地在固定的应用程序列表上一目了然地显示多个应用程序，而无需滚动页面或转到应用程序列表。这是 Windows 10 上的一个功能，但 Windows 11 最初没有它，所以它是一个受欢迎的功能。

最大的功能之一是新的触摸手势，尤其是如果你是平板电脑或可转换用户。微软增加了新的触摸手势，使 Windows 11 更适合这种设备，这是 Windows 很久以来一直缺少的东西。其中包括从任务栏中间向上滑动以打开开始菜单(向下滑动以关闭它)，在开始菜单本身中，你可以向右滑动固定的应用程序区域以查看你的所有应用程序列表，或者从推荐区域向右滑动以查看“更多”页面。您可以从通知区域向上滑动以打开快速设置面板(向下滑动以关闭它)。

此外，微软已经更新了从右边滑动以查看通知中心的动画，所以现在它会随着你的手指滑动，而不是以固定的速度打开。此外，现在有一个“全屏抓手”。如果你在运行全屏幕应用程序时不小心从屏幕两侧向内滑动，现在你会看到一个小手柄，这样你就可以再次向内滑动以打开通知中心。这样，你就不会不小心遮住了你正在使用的应用。

Windows 11 build 22557 还附带了一个新的请勿打扰功能，这似乎是微软所谓的聚焦辅助的新名称。当您打开“请勿打扰”时，通知将被静音，您将不会再看到它们在屏幕上弹出。但是，您仍然可以在通知中心看到您的通知。你可以为“请勿打扰”设置规则，使其自动生效，就像聚焦辅助一样。

然而，微软也有一个新功能，简称为 Focus，它有点像是时钟应用程序中的 focus 会话与勿扰模式的合并。使用 focus，您可以随时从通知中心启动 Focus 会话，设置计时器，然后开始。当焦点启用时，你会在屏幕上看到一个计时器，请勿打扰将被启用，任务栏应用程序的通知徽章将被禁用。应用程序也不允许在任务栏上闪烁。因为它集成了 Clock 应用程序的 Focus sessions 功能，所以您还可以播放音乐来帮助您集中注意力，并从 Microsoft To Do 上的待办事项列表中访问任务。

Windows 11 build 22557 中的另一个新功能是实时字幕，这与你已经从 Android 和一些谷歌应用中知道的差不多。当您播放任何带音频的内容时，Windows 11 将使用设备上的语音到文本技术将音频转录为文本，因此有听力障碍的人可以更容易地跟上，而不需要依赖预先制作的字幕(可能可用，也可能不可用)。目前只有英语(美国)的直播字幕。

文件资源管理器在这个版本中也得到了一些大的改进。首先，快速访问页面现在允许您将文件固定在最近文件列表的顶部，这样您就可以快速找到重要的文件。如果您使用 OneDrive，现在还可以直接在文件资源管理器中查看您的云配置文件上有多少可用空间。

Windows 11 build 22557 的另一大改进与对齐布局有关。现在，您可以将窗口拖到屏幕顶部附近来查看各种可用的对齐布局，这样您就可以轻松地选取您想要的布局以及您想要当前窗口在该布局中的位置。然后，捕捉辅助使您可以轻松完成布局设置。

在这个版本中还有一个新的任务管理器，这是一些用户几周前在以前的版本中发现的。这个新的任务管理器支持黑暗模式，并且有一个更好地遵循 Windows 11 设计语言的新设计。任务管理器也有一个新的效率模式，强制应用程序使用更少的电力，这样你就可以在其他应用程序中提升性能或节省电池电量。

在这一点上，设置应用程序中的电源设置页面已经被调整，因此睡眠和屏幕关闭设置的默认定时器是不同的，以降低能耗。然而，如果你只是升级到这个版本，这些设置不会改变，只对那些干净安装它的人。此外，该页面现在将建议更改某些设置以节省电能。例如，如果您禁用了屏幕关闭计时器(因此屏幕总是打开的)，系统会提示您重新启用它。

最后，该版本对讲述人与 Microsoft Edge 的协作方式进行了一些改进，因此阅读和浏览网页更加自然和高效。还有一个新的 PowerShell 模块，可以让您为 Windows 11 提供语言功能。

除此之外，Windows 11 build 22557 还进行了大量较小的改进。我们总结了以下一些亮点:

  • 你现在可以将内容拖放到任务栏上的应用程序中(奇怪的是，这个功能在 Windows 11 的最初版本中被删除了)。
  • 在 Microsoft Teams(工作或学校)中共享窗口时，您现在会看到正在共享的窗口的任务栏指示器。
  • 拥有多个颜色配置文件的电脑现在可以通过快速设置面板中的新操作在它们之间切换。
  • 当投射到无线显示器时，你会在任务栏的通知区域看到一个投射图标。
  • 充电图标已更新为使用闪电图标，而不是电缆图标。
  • 文件资源管理器现在可以预览文件夹中的文件。
  • 如果您从文件资源管理器向 Outlook 共享文件，您现在可以直接在小弹出窗口中撰写邮件，而无需打开 Outlook 应用程序(需要 Outlook 桌面应用程序和 Outlook 桌面集成)。
  • 在 Windows 搜索框中搜索应用和设置现在更快、更准确。
  • 在 Alt + Tab 任务切换器中，快照组现在显示在桌面背景后面，以便于识别。
  • 将抓取的应用程序移动到不同位置时的过渡现在更平滑了。
  • 旋转显示器时的过渡更快、更平滑(对于平板电脑和敞篷车)。
  • 云母现在被用于更多的地方，包括运行对话框。
  • Windows 11 Pro 现在也需要互联网连接和微软账户才能设置，类似于 Windows 11 Home。

最后，Windows 11 build 22557 有大量的修复，太多了，无法在原始文本中列出。如果你感兴趣，可以看看下面的:

Windows 11 内部版本 22557 中的修复

【常规】

  • 修正了一些内部人员正在经历的重复出现的 backgroundTaskHost.exe。
  • 修正了在某些游戏中自动 HDR 不能正常激活的问题。
  • 缓解了 lsass.exe 坠机事件，该事件导致一些内部人士在最近的航班登录屏幕上看到坠机事件。
  • 解决了最近飞行中使用 Xbox 无线适配器连接的音频设备音频明显失真的问题。

【任务栏】

  • 如果你按住 CTRL 键不放，任务视图不会再崩溃。
  • 在辅助显示器的任务栏上拖动文件不再会导致 explorer.exe 崩溃。

【文件浏览器】

  • 更新了磁盘属性对话框中的“存储使用”按钮，现在称为“详细信息”,范围更广，以适应按钮文本被截断的语言。
  • 改进了删除回收站中文件的性能。
  • 修复了蓝光图标，使其与其他驱动器和卷图标对齐。

【输入】

  • 解决了一个问题，IME 候选窗口出乎意料地没有按照你的偏好亮或暗的模式。
  • 修正了一个问题，导致一些元素的语音打字设置弹出是不可见的黑暗模式。
  • IME 候选窗口现在应该如预期的那样出现在 UAC 提示符的用户名字段中。
  • 更新了术语“半角字母数字/直接输入”,在日文 IME 上下文菜单中只显示“半角字母数字”。
  • 修正了在假名输入模式下输入平假名后，如果在平假名之前有一个半角字母字符，它会意外地变成全角。
  • 解决了一个罕见的问题，输入切换器可能会意外地列出同一个条目两次。
  • 缓解了导致输入切换器中有时出现意外黑盒的问题。
  • 更新了输入切换器，现在对于拥有 4 个以上键盘的人来说，它可以变得更高一点。
  • 修正了输入切换器内存泄漏导致 explorer.exe 反复快速使用后崩溃的问题。
  • 修正了在输入切换器中点击更多键盘设置时 explorer.exe 崩溃的问题。
  • 修复了一个问题，表情面板可能会显示“你离线”，即使你有网络连接。
  • 修复了乘法表情符号和逆时针箭头按钮表情符号显示错误表情符号设计的问题。
  • 更新了个性化>文本输入设置页面中的预览，以反映您的桌面壁纸而不是默认壁纸。
  • 做了一些工作来提高媒体播放时停止媒体键的可靠性。

【语音接入】

  • 我们修复了 Visual C++运行时不匹配导致语音访问在某些电脑上无法运行的问题。

【搜索】

  • 修正了一个高命中率的搜索崩溃。
  • 点击“开始”中的搜索框，现在会立即切换到搜索，而不会显示向上的搜索窗口动画，就像当你按下 Windows 键并开始键入时一样。

【设置】

  • 解决了讲述人无法正确读取系统>存储元素的问题。
  • 修复了时间和语言>打字>触摸键盘下的“尺寸和主题”链接。
  • 做了一些工作，以帮助提高性能时，更新强调颜色或模式。
  • 我们做了一个更改，以帮助减少某些语言和文本大小的设置中组合框的剪辑。
  • 修正了导致 IME 设置页面上的图标有时是白底白字的问题。
  • 缓解了在预览字体时导致安装按钮在某些语言中出现截断文本的问题。
  • 修复了一个潜在的崩溃，导致预览视频在系统>显示> HDR 有时失真。
  • 解决了一个潜在的问题，当你滚动窗口时，设置可能会崩溃。
  • 调整了快速设置的 Wi-Fi 部分中的 Wi-Fi 图标，以便为安全网络显示的锁现在与 Wi-Fi 强度有一点分离，使其更容易区分。

【Windows 聚光灯】

  • 将 Spotlight collection 的名称更新为 Windows Spotlight，因此与锁屏保持一致。
  • 进行了更改，以帮助 Windows Spotlight 更可靠地更新图像。
  • 如果由于某种原因，桌面上没有新的 Windows Spotlight 图像，它现在将返回显示默认的 Windows Spotlight 背景图像(Whitehaven Beach)，而不是显示纯色背景。
  • 右击桌面上的 Windows Spotlight 的“了解此图片的更多信息”,现在将总是显示所有选项，如果它们当前不可用，则灰显。
  • 当悬停在“了解此图片的更多信息”时，工具提示现在应该更一致地与当前图像对齐。

【开窗】

  • 解决了导致文件资源管理器和其他一些应用程序中的最小化、最大化和关闭按钮在某些情况下不可见的问题。
  • 缓解了使用丙烯酸树脂拖动窗口和其他使用丙烯酸树脂的场景时导致性能影响的问题。
  • 截图时，你不会再看到短暂可见的黄色边框。
  • 我们已经做了一些改进，以解决窗口意外重新定位的反馈。

【网络】

  • 做了一些工作来解决可能导致某些 VPN 连接失败的问题。如果您继续遇到问题，请在网络和互联网>下用捕获问题与 VPN 客户端连接提交反馈。
  • 修正了在上次飞行中唤醒你的电脑后导致 Wi-Fi 速度下降的问题。

【旁白】

  • 退格时,“讲述人”现在会更一致地通知您何时用退格删除字符。
  • 修复了导致讲述人无法在反馈中心阅读所选文本的问题。

在这个版本中仍然有一些已知的问题，这些问题在开始之前总是值得研究的。一个主要的已知问题是，由于一些问题，这个版本不能用于 ARM64 PCs。如果你有一个像 Surface Pro X 这样的设备，你必须等待更长时间才能获得这些功能。其他问题列举如下:

Windows 11 内部版本 22557 中的已知问题

【常规】

  • 在企业版上体验设备设置(OOBE)时，第一次尝试时会跳过网络添加屏幕。作为一种解决方法，当用户看到“命名您的计算机”选项时，请重新启动并重新启动 OOBE。现在，网络添加屏幕将按预期出现。

【开始】

  • 在“开始”中打开文件夹时，文件夹中的应用程序名称可能会短暂模糊。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。

【文件浏览器】

  • 快速访问中的搜索可能不起作用。
  • 我们正在修复显示 OneDrive 存储的弹出菜单中有关图标大小、视觉错误和文本剪辑的问题。

【聚焦】

  • 任务栏图标和工具提示可能与焦点状态不匹配。
  • 时钟应用程序集成需要一个应用程序更新，今天开始提供。大多数用户应该会自动获得此更新，但你可以通过导航到微软商店 > 图书馆并点击获得更新来手动检查应用更新。
  • 在应用程序中配置焦点会话时，时钟应用程序尚未更新 Windows 焦点状态。这将在未来的应用程序更新中解决。

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【微件】

  • 任务栏左对齐时，温度等信息不会显示。这将在未来的更新中得到解决。

【直播字幕】

  • 全屏模式下的某些应用程序(如视频播放器)会阻止显示实时字幕。
  • 位于屏幕顶部附近并在运行实时字幕前关闭的某些应用程序将在位于顶部的实时字幕窗口后重新启动。当应用程序有焦点时，使用系统菜单(ALT +空格键)将应用程序的窗口进一步下移。
  • 当实时字幕位于顶部时，触摸无法到达最大化应用的最顶部(例如标题栏窗口管理按钮)。

【任务管理器】

  • 我们意识到，一些子流程上缺少效率模式图标。
  • 一些设置选项不会保留。
  • 我们正在解决有关图标大小和视觉缺陷的问题，在大多数情况下，解决方法是调整任务管理器的大小或从“设置”>“显示”>“显示分辨率”中降低分辨率设置。

微软为这个版本提到的另一件事是，它现在来自一个名为 ni_release 的新开发分支。这不会影响你在每个版本中获得的特性，但是，微软实际上警告说，它将停止在新版本的未来博客帖子中提及这些变化。

如果您想尝试所有这些功能，请查看如何加入 Windows Insider 计划以及频道如何工作。如果不是，本周早些时候，Windows 11 也为非业内人士提供了新功能，包括对 Android 应用程序的支持。
