有人对 iPhone X 进行了逆向工程，安装了一个 USB Type C 端口

原文：https://www.xda-developers.com/iphone-usb-c-reverse-engineering/#0001-01-01

iPhone 上的 USB C？已经是现实了。有点儿

一位 DIY YouTuber 对他的 iPhone X 进行了逆向工程，安装了一个 USB Type C 端口。该调制器使用了一个带母 USB C 端口的柔性 PCB。

苹果是唯一一个仍然没有切换到 USB Type C 端口的主要智能手机品牌。相反，Cupertino 巨头使用其专有的照明连接器，迫使 iPhone 用户携带额外的电缆。虽然我们不知道苹果何时会放弃iPhone上的 Lightning 连接器，但一名 DIY YouTuber 已经对他的 iPhone X 进行了逆向工程，在其上安装了 USB Type C 端口。

这一令人印象深刻的壮举是由具有电子背景的瑞士机器人学生肯尼·皮隆内尔实现的(via AppleInsider )。在 YouTube 短片中，Pillonel 让我们第一次看到了带有 USB Type C 连接器的 iPhone。这个 40 秒的片段展示了一个经过修改的 iPhone X，它具有一个通过 USB Type C 电缆充电的 Type C 连接器。改进的连接器也支持数据传输。请观看下面的视频，亲自体验一下:

皮隆内尔表示，他对苹果 C94 连接器进行了逆向工程，并制作了自己的 PCB，带有一个母 USB C 端口，他设法将其安装在 iPhone X 中。YouTuber 尚未澄清通过修改后的 USB C 端口为 iPhone 充电是否有任何负面影响，如充电速度变慢。

Pillonel 已经为这个 DYI 项目工作了一段时间。早在 5 月份，他在他的 YouTube 频道上发布了一个概念验证视频，展示了 USB-C iPhone Mod 以及他将所有这些放在一起所面临的挑战。在上个月的一个快速更新视频中，他透露他已经订购了一个柔性 PCB，并正在实现他的目标。现在他终于做到了。如果你有兴趣了解更多关于这个项目的信息，你可以看看他的博客帖子，里面详细介绍了技术方面的东西。皮隆内尔说，他将很快在他的频道上发布一个完整的视频，展示他是如何实现这一壮举的。
