微软发布 Windows 11 build 22000.556——以下是新内容

原文：https://www.xda-developers.com/microsoft-releases-windows-11-build-22000-556-for-patch-tuesday/#0001-01-01

微软发布 Windows 11 build 22000.556，修复了重置后保留文件的错误

微软今天正在更新 Windows 11，为补丁星期二构建 22000.556，像往常一样，它包含了一堆重要的修复。

今天是这个月的第二个星期二，所以今天是补丁星期二，所有受支持的微软软件都会在这一天得到更新。除了几个版本的 Windows 10，今天还有一个版本的 Windows 11 获得了补丁。

如果你使用的是 Windows 11 版本 21H2，这是唯一一个发布的版本，你会得到 KB5011493 ，使内部版本号达到 22000.556。你可以在这里手动下载。

只有一个亮点:

完整的修复列表也不是很长:

  • 解决当您尝试重置 Windows 设备及其应用程序包含包含重解析数据的文件夹时发生的已知问题，例如 Microsoft OneDrive 或 Microsoft OneDrive for Business。当您选择删除所有内容时，从 Microsoft OneDrive 本地下载或同步的文件可能不会被删除。某些设备在安装此更新后可能需要七(7)天才能完全解决问题，并防止文件在重置后持续存在。为了立即生效，您可以使用 Windows Update 疑难解答中的说明手动触发 Windows Update 疑难解答。

变更日志很短并不奇怪。周二补丁更新倾向于这种方式，在 C 周和 D 周更新中有更长的变更日志。当然，这些修复被合并到周二的补丁更新中，因此在更新的名称中有“累积”一词。

一个得到修复的问题是一个 bug，在您重置 Windows PC 后，文件可能仍然存在。此问题仅影响 OneDrive 中同步的文件，并且是在上个月发现的。

像往常一样，这个更新是强制性的。这意味着，虽然您现在可以通过 Windows Update 手动安装，但如果您不安装，它将在某个时候自动安装。
