Flutter 2.10 带来了对构建 Windows 应用的稳定支持

原文：https://www.xda-developers.com/flutter-2-10-windows-apps/#0001-01-01

谷歌已经在 Flutter 上工作了几年，目标是创建开发者梦想中的跨平台软件框架。Flutter 可以用来为 Android、iOS、Linux、Windows、macOS 甚至 web 构建应用程序——所有这些都有一个共享的代码库和可视化设计。Flutter 2.10 现已推出，其中包括对 Windows 的稳定支持。

Flutter 现在已经有一段时间支持 Linux，Windows，macOS 的测试版了，但是 Windows 是第一个达到稳定状态的。谷歌周四宣布，“今天标志着这一愿景的重大扩展，首次发布支持 Windows 作为应用目标的产品，使 Windows 开发者能够受益于移动开发者一直享受的相同的生产力和能力。”

Windows 上的 Flutter 应用程序可以使用 iOS 和 Android 上大多数常见的 Flutter 框架，但它们也可以根据需要使用 Win32、COM 和 Windows 运行时 API。谷歌还更新了一些常见的 Flutter 插件来支持 Windows，如相机、文件选择器和共享偏好——所有这些插件都支持桌面平台有一段时间了，但谷歌现在认为它们已经可以生产了。微软的流畅设计在 Flutter 中也有的实现，所以 Windows 上的 Flutter 应用不需要像拇指一样突出。

谷歌还提到了微软对这项工作的贡献，称“微软的几个团队为今天的发布做出了贡献。特别是，我们要感谢 Fluent 设计团队为 Windows 上的 Flutter 应用程序图标化所做的贡献。该 fluentui_system_icons 软件包已被授予 Flutter Favorite 状态，作为其质量的信号。”

尽管这是桌面 Flutter 平台第一次被贴上稳定的标签，但 Flutter 已经在桌面上站稳了脚跟，尤其是在桌面 Linux 上。谷歌和 Canonical(Ubuntu Linux 的开发者)一直在合作，让 Linux Flutter 应用易于通过 Snap Store 分发，Ubuntu Linux 的安装程序是目前正在 Flutter 中重写的。
