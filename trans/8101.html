微软 Edge 集成了微软编辑器，用于拼写检查和语法纠正

Microsoft Edge 浏览器现在获得了一个内置版本的 Microsoft Editor 来检查您的拼写和语法错误。

如果微软 Edge是你的首选浏览器，由于一个新的更新，从现在起你将有一个改进的打字体验。正如情报人员 Leopeva64-2 所发现的，微软现在已经将其流行的服务——微软编辑器——集成到 Edge 浏览器中。如果你不知道微软编辑器，它是一个插件，检查你的内容拼写错误和语法错误。尽管微软编辑器已经作为 Edge 附加组件网站的一个扩展提供了一段时间，但微软现在宣布他们正在将该服务直接集成到 Edge 浏览器中，以使体验更加无缝。

该功能仍在测试中，所以目前美国的内部人士可以使用该功能，并且仅支持英语。如果您在 Windows PC 上使用 Edge 的 Dev 版本或 Canary 版本，您应该有资格获得此更新。启用该功能后，在文本字段中键入内容会导致拼写错误的单词下面出现红色曲线。如果有语法错误或者微软编辑器认为你可以用更好的方式写一个句子，你会看到一条蓝线。

如果您希望看到建议的改进和更正，请单击带下划线的项目，您将看到一个选项，要么接受建议，要么忽略它。微软表示，他们正在努力将这一功能扩展到其他语言和地区。因此，如果你希望该功能支持你的首选语言/地区，前往菜单>帮助和反馈>发送反馈并让微软知道。

这是微软的一个很好的举措，对于键入大量内容的 Edge 用户(比如我！)但是不能访问第三方扩展来检查拼写和语法错误。如果你也经常打电子邮件，你肯定会喜欢这个功能。
