Dynabook E10-S 是首批 Windows 11 SE 笔记本电脑之一

原文：https://www.xda-developers.com/dynabook-e10-s-windows-11-se/#0001-01-01

微软今天早些时候宣布了 Windows 11 SE ，作为面向 K-8 学生的 Windows 11 的专门版本。这是一个精简的、云优先的 Windows 11 版本，旨在运行在低端硬件上，从而在教育领域与 Chromebooks 竞争。微软将在推出新操作系统的同时推出自己的 Surface Laptop SE，但合作伙伴也在加入。Dynabook 将推出搭载 Windows 11 SE 的 Dynabook E10-S。

与其他 Windows 11 SE 笔记本电脑非常相似，Dynabook E10-S 是一款相对低端的设备。它由英特尔赛扬 N4020 处理器提供支持，这是一款低功耗双核处理器，具有 4GB 内存和 128GB 固态硬盘存储。考虑到 Surface Laptop SE 使用的是 eMMC 存储，使用固态硬盘是这款机型的一大优势。

它配备了分辨率为 1366 x 768 的 11.6 英寸高清显示屏，还包括一个用于通话的 720p 高清网络摄像头。至于声音，它使用双立体声扬声器和双麦克风，具有噪音抑制功能，因此在这台机器上通话应该听起来相当清晰。

Dynabook E10-S 还拥有丰富的端口选择，包括两个 USB Type-A 端口(其中一个始终开启)、一个 USB Type-C 端口(带数据、电源和显示输出)、以太网、耳机插孔和 microSD 读卡器。这使得学生可以连接他们可能需要的任何东西。

这种设计是让它成为 K-8 学生笔记本电脑的另一个重要因素，因为它应该是耐用的。它的设计旨在更容易抓握，并且由于边缘周围都有橡胶保险杠，它可以抵抗从高达 30 英寸的地方跌落。此外，它还有一个防泼溅键盘，以防在使用笔记本电脑时发生任何事故。Dynabook E10-S 重 2.5 磅，厚度约为 0.78 英寸(19.9 毫米)。

Dynabook E10-S 将是 2022 年初首批搭载 Windows 11 SE 的设备之一，售价 289 美元。Windows 11 SE 没有微软商店或 Widgets 窗格，所以它的局限性要大得多，但它应该由学校的 it 部门配置必要的应用程序。如果你需要 Windows 11 的全套功能，还会有一个带有 Windows 11 Pro Education 的版本。你还可以在 Dynabook 的网站上找到这款笔记本电脑的 Windows 10 版本。
