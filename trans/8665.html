新的 Windows 11 版本包括改进的主题和可定制的表情符号

原文：https://www.xda-developers.com/windows-11-build-22504-theming-emoji/#0001-01-01

又过了一周，又有一个 Windows 11 内部更新，这次是 build 22504。在昨天推出新的媒体播放器应用程序之后，这个版本包括对操作系统某些部分的主题化的一些改进，以及定制一些表情符号的能力。

在主题化方面，微软已经做了一些改进，使得 Windows 11 上触摸键盘可用的主题现在可以在更多的输入法中看到。如果您使用输入法编辑器(ime)、表情符号面板或语音输入，它们现在会反映您为触摸键盘设置的主题。这些主题选项可以在设置应用的个性化部分找到，它们将出现在一个名为文本输入的新页面中。

至于表情符号，微软正在使个性化定制表情符号中包含多个人的人脸成为可能。对于情侣、牵手的人和家庭，你可以在发送表情符号之前选择每个角色的皮肤和头发颜色。这项功能正在向内部人员的一个子集推出，所以现在可能不是每个人都可以使用。

抛开这两个变化，Windows 11 build 22504 相对来说比较小。继几周前推出的静音图标之后，您现在可以使用键盘快捷键来激活它。还有一个更新。NET 框架，增加了对 ARM64 设备的原生支持。微软也在这样做，现在默认情况下，在 11 英寸或更小屏幕的平板电脑上，应用程序可以全屏启动。这也适用于平板形态的敞篷车。

除了更大的变化之外，这个版本还包括了针对操作系统各个方面的常见修复列表。您可以在下面看到完整的列表:

Windows 11 内部版本 22504 中的修复

【开始】

  • 修正了一个问题，这个问题被认为是图标有时在开始的推荐部分显示不正确的根本原因(显示错误的或通用的图标)。
  • 如果你打开开始并立即按下 Shift + F10 或上下文菜单键，上下文菜单现在应该与搜索框对齐。

【任务栏】

  • 修正了一个问题，导致 explorer.exe 崩溃在最近的建设，如果有问题加载电池图标在任务栏中。
  • 做了一些改变，以帮助任务栏中的时钟更可靠地更新，包括解决最近的一个问题，即它在远程桌面上根本不更新。
  • 缓解了可能导致日期和时间从任务栏末尾被剪掉的定位问题。
  • 如果您已经更新了严重和低电量通知定义，任务栏中的电池图标现在将与之对齐，而不是使用默认值来显示警告。
  • 右键单击任务栏中的日期和时间将会关闭通知中心(如果它是打开的)，这样您就可以看到上下文菜单选项。

【输入】

  • 这个版本的键盘灯(例如，大写锁定灯)现在应该可以正常工作了。
  • 我们根据反馈对表情符号搜索关键词进行了一些调整，包括改善白天、夜晚和眉毛上扬的搜索结果。请继续在反馈中心的输入和语言>表情面板下与我们分享您的反馈。
  • 修正了在 WIN +中导致 kaomoji 列表的问题。使用中文 IME 访问时为空白。
  • 修复了表情面板搜索结果可能导致单个 gif 在某些查询中重复显示的问题。
  • 缓解了导致 ime 无法在某些游戏中运行的问题。
  • 解决了一个问题，在设置>个性化>输入下更改设置的选项“显示文本建议并在我选择文本建议后添加空格键”可能会意外地不显示。
  • 如果您在使用语音打字(WIN + H)时停顿很长时间，现在应该可以更可靠地恢复收听。
  • 修正了语音输入中实际听音状态和麦克风视觉之间的一些不一致。
  • 使用语音输入时，如果移动插入符号，文本将不再意外重复。

【开窗】

  • 选择快照布局选项应该不再随机将窗口放在其他显示器上。

【设置】

  • 快速设置中的亮度和音量滑块不再会随机变得不可见。
  • 修正了阻止手动调整活动时间的问题。
  • 解决了在快速设置中连接到网络时可能导致触摸键盘遮挡密码输入字段的问题。

【其他】

  • 修复了一个缩放问题，该问题导致更新提示对话框(以及该样式的其他对话框)被切断，并且在 DPI 更改后无法正确绘制。
  • 改进了右键单击桌面底部时上下文菜单的位置。
  • 缓解了与 TCPIP 相关的问题，该问题可能会导致唤醒设备时进行错误检查。
  • 修复了一个问题，高优先级通知(如警报)有时会在您没有参与的情况下意外取消，导致它在下次正常优先级通知到来时随机重新出现。

正如您所料，这个版本还附带了一个已知问题列表。这一次的列表相对较短，但是请记住，总会有这里没有列出的问题。这些仅仅是微软知道的。

Windows 11 内部版本 22504 中的已知问题

【常规】

  • 从版本 22000.xxx 或更早版本更新到使用最新开发频道 ISO 的较新开发频道版本的用户可能会收到以下警告消息:您尝试安装的版本是 Flight Signed。要继续安装，请启用航班签名。如果您收到此消息，请按下启用按钮，重启电脑，然后重试更新。
  • 我们正在调查一些电脑无法安装新版本或其他更新的问题。PC 可能会报告错误代码 0x80070002。如果您遇到此问题，请重新启动您的电脑，然后重试。
  • 安装此版本时，某些设备可能会执行错误检查，错误代码为 0xc1900101-0x4001c。如果你点击这个，在设备回滚到以前的版本，你可能要暂停更新，直到我们发布一个修复。

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。

【开窗】

  • 在任务视图中的不同桌面之间来回悬停鼠标将导致显示的缩略图和内容区域意外缩小。

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

虽然它没有绑定到新版本，但微软也宣布它开始为你的手机应用程序推出新的用户界面。这在 9 月份的 Surface 活动中被取笑过，它的特点是更快地获得通知和改进的导航。

随着下周感恩节的到来，微软也花时间澄清在那一周不会有新的开发频道。我们应该在接下来的一周恢复正常，尽管随着圣诞节的临近，另一个假期很快就会到来。
