微软发布带有新表情符号的 Windows 11 版本 22000.348

原文：https://www.xda-developers.com/microsoft-releases-windows-11-build-22000-348-new-emojis/#0001-01-01

今天，微软发布了一个新的可选累积更新，用于 Windows 11。然而，与大多数累积更新不同的是，这个更新实际上有一个新特性。它拥有七月份宣布的新表情符号。嗯，算是吧。

当新的表情符号宣布时，设计团队展示了这些美丽的 3D 角色，看起来和你在 iOS 上找到的一样。但是当它们开始在预览中出现时，它们实际上是平的，并且比用户所认为的更缺乏视觉吸引力。事实证明，漂亮的表情符号是团队和 Flipgrid 等事物的专属，而 Windows 11 正在获得 2D 表情符号(在此插入关于不一致的笑话)。

微软表示，新表情符号的目标是“个性化和熟悉化”，这是 Windows 11 的关键设计原则。还有更多表情符号，包括表情符号 13.1 集。

当然，这并不是这次更新的全部新内容。这毕竟是一个 C 周的更新，使得它成为一个可选的累积更新，有大量的修复。

如果你在 Windows 11 上，你会看到 KB5007262 ，这使得内部版本号达到 22000.348。这里可以手动下载，以下是亮点:

  • 更新了在使用输入法编辑器(IME)复制和粘贴文本时导致 Internet Explorer 停止工作的问题。
  • 更新了在通知区域显示 iFLY 简体中文 IME 图标背景不正确的问题。
  • 更新了阻止显示文件资源管理器和桌面快捷菜单的问题。当您选择使用单击打开项目时，通常会出现此问题。
  • 改进任务栏上图标的动画性能。
  • 更新影响蓝牙音频设备的音量控制问题。
  • 更新了导致文件资源管理器在您关闭文件资源管理器窗口后停止工作的问题。
  • 更新了某些视频显示不正确的闭路字幕阴影的问题。
  • 更新了从设备中自动删除塞尔维亚语(拉丁语)Windows 显示语言的问题。
  • 更新了当你悬停在任务栏图标上时导致闪烁的问题；如果您应用了高对比度主题，就会出现此问题。
  • 更新了一个问题，即在某些情况下，当您使用任务视图、Alt-Tab 或“捕捉助手”时，键盘焦点矩形不可见。
  • 更新了一个问题，当你戴上耳机时，可能会导致 Windows 混合现实启动。即使您关闭了“当我的耳机的存在传感器检测到我戴着它时启动混合现实门户”选项，此问题也会发生。
  • 更新了一个问题，该问题可能会导致您的设备在插入打印机后报告检测不到打印机。
  • 更新了一个可能会导致设备上的音频暂时丢失的问题。
  • 更新了导致某些可变字体显示不正确的问题。
  • 更新了使用 Meiryo UI 字体和其他垂直字体时以错误的角度显示字母或字符的问题。这些字体经常在日本、中国或亚洲其他国家使用。
  • 更新了导致某些应用程序停止响应输入的问题。此问题出现在带有触摸板的设备上。
  • 添加一个选项，让您选择是否在 Windows 功能更新后的第一个小时自动打开聚焦助手。
  • 更新影响 Xbox One 和 Xbox 系列音频外围设备并在您将它们与空间音频一起使用时发生的音频失真问题。
  • 更新了 Windows 表情符号的几个方面。作为迭代和持续工作的一部分，我们对此版本进行了以下改进:
    • 将 Segoe UI 表情字体的所有表情更新为流畅的 2D 表情风格
    • 包括对表情符号 13.1 的支持，它:
      • 更新表情词典
      • 增加了在所有支持的语言中搜索表情符号 13.1 的功能
      • 更新表情符号和更多面板，以便您可以在应用程序中输入表情符号

这些也只是亮点。修复列表甚至更长。

Windows 11 构建 22000.348 修复程序

有几种方法可以安装此更新。首先，您可以通过从上面的链接下载更新来手动安装。您也可以通过 Windows Update 获得它，方法是检查更新，然后选择安装它。但是它是可选的，这意味着它不会自动安装。如果你不接受，这些修正和改变将会在下个月的补丁星期二更新中推出，这是强制性的。
