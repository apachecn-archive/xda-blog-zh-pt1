新的 Windows 11 版本将更多选项从控制面板移至设置

原文：https://www.xda-developers.com/windows-11-insider-build-22523/#0001-01-01

微软正在向 Dev 频道注册的内部人士推出其 Windows 11 的每周版本，这次我们将获得 build 22523 。在上周的相当加载的构建之后，Windows 11 build 22523 有点驯服，但它仍然包括一些很酷的新功能供尝试。这个版本也适用于 ARM64 PCs，而上周的版本不是这样的，所以如果你有一台 ARM PC，你会有更多的新功能。

对于初学者来说，当你使用 Alt + Tab 切换应用程序时，你现在会看到列出了你的快照组，就像当你将鼠标光标悬停在任务栏上的某个应用程序上一样。这个功能以前已经为一些 Dev 频道内部人员提供，但现在每个人都可以使用它。如果你还不知道快照组，我们已经写了一篇关于如何使用它们的深度探讨。

另一个值得注意的变化是，如果您在文件资源管理器中打开此 PC 页面，您现在可以从命令栏中的三点图标添加媒体服务器。如果您有媒体服务器，也可以用这种方法删除。

最后，微软正在慢慢地将越来越多的功能从经典的控制面板转移到新的设置应用程序，这一努力始于 2015 年的 Windows 10，仍然有一段路要走。现在，如果某个更新给你带来了问题，你可以直接在设置应用中卸载 Windows updates。另一方面，设置应用程序中的已安装应用程序列表现在将有一个链接指向控制面板中的经典程序和功能页面，因此您可以更容易地访问经典程序管理界面。

除此之外，这是通常的一系列修复，这次有一个相当大的列表。

Windows 11 build 22523 中修复了什么

【任务栏】

  • 修复了一个与文本输入初始化相关的问题，该问题可能会导致外壳(例如，开始菜单和搜索)在 ARM64 PCs 上变得无响应。
  • 电池图标工具提示不应再意外显示高于 100%的百分比。
  • 当有很多打开的应用程序时，应用程序图标不再会与辅助显示器上的日期和时间重叠。

【文件浏览器】

  • 做了一些工作来解决当使用 F2 重命名 OneDrive 文件时，有时在按 Enter 后会导致失去键盘焦点的问题。

【聚光灯集合】

  • 启用聚光灯采集后，你的第一张图像(在怀特黑文海滩后)应该会更快到达。
  • 向 spotlight 集合上下文菜单项添加图标。

【输入】

  • 提高了调用语音输入的可靠性。
  • 修复了当对比主题启用时，我们的文本输入体验(语音输入，表情面板等)的边界不能正确绘制的问题。
  • 缓解了笔菜单进程的间歇性崩溃，如果它已启动，然后在启动前立即关闭。

【微件】

  • 我们修复了使用悬停打开插件板时导致链接无法正常打开的问题。

【设置】

  • 当设置窗口变小时，设置内容不再会被截断。
  • 当打开组合框时，设置不再偶尔崩溃，这影响了某些设置，如为笔设置自定义点击动作的能力。
  • 解决了当尝试连接新的蓝牙设备时，蓝牙设备中的“添加设备”选项无声崩溃的问题。
  • 添加了一些关键字，使语音访问功能出现在设置搜索结果中。

【其他】

  • 修正了导致 ARM64 电脑在之前的飞行中遇到内存管理错误的问题。
  • 修复了当尝试使用某些应用程序时导致 DWM 崩溃(导致屏幕反复闪烁)的问题。
  • 缓解了导致某些应用程序在讲述人运行时挂起的问题。
  • 在检查 narratorquickstart.exe 的属性细节时添加了一些缺失的信息。
  • 解决了讲述人无法响应 UIA 事件(如通知、实时区域或文本事件)的问题。

像往常一样，微软指出，其中一些修复可能会通过累积更新进入当前版本的 Windows 11，所以你不需要等待明年秋天的功能更新。

当然，在 Windows 11 build 22523 中也有一些已知的问题，你应该在开始之前了解这些问题。其中，微软意识到在最近的版本中更新驱动程序和固件可能会出现问题。完整列表如下:

Windows 11 内部版本 22523 中的已知问题

【常规】

  • 您可能无法登录某些应用，如反馈中心。重新启动你的电脑应该可以解决这个问题。
  • 我们正在调查一些内部人员在最近的版本中看到驱动程序和固件更新失败的报告，错误为 0x8007012a。

【开始】

  • 在某些情况下，使用“从开始”或任务栏搜索时，您可能无法输入文本。如果遇到此问题，请按键盘上的 WIN + R 启动“运行”对话框，然后关闭它。

【任务栏】

  • 切换输入法时任务栏有时会闪烁。
  • 网络图标有时在任务栏中不见了，而它应该在那里。如果您遇到这种情况，请尝试使用任务管理器重新启动 explorer.exe。
  • 如果你有多台显示器连接到你的电脑上，右击主显示器任务栏上的日期和时间，explorer.exe 就会崩溃

【搜索】

  • 单击任务栏上的搜索图标后，搜索面板可能不会打开。如果出现这种情况，请重新启动“Windows 资源管理器”进程，并再次打开搜索面板。

【设置】

  • 查看可用 Wi-Fi 网络列表时，信号强度指示器不能反映正确的信号强度。
  • 进入系统>显示> HDR 时，设置可能会崩溃。
  • 蓝牙和设备下有一个空白条目。

【聚光灯集合】

  • 如果您正在使用 spotlight collection，当前图像当前不会在升级时迁移，这可能会在升级到此版本后给您留下黑色的桌面背景。这应该在下一次飞行中解决。

【微件】

  • 更改任务栏对齐方式会导致小工具按钮从任务栏上消失。
  • 将入口点悬停在辅助显示器上时，Widgets board 可能没有正确的分辨率。
  • 微件面板可能暂时空白。
  • 当有多台显示器时，任务栏上的小工具内容可能会在不同显示器之间不同步。
  • 任务栏左对齐时，温度等信息不会显示。这将在未来的更新中得到解决。

【语音接入】

  • 某些文本创作命令，例如“选择该内容”或“删除该内容”，可能无法在 Windows 应用程序中正常工作。
  • 对某些标点符号和符号(如@符号)的识别不准确。

微软表示，这是我们将在 2021 年获得的最后一个 Windows 11 版本，原因很明显。假日季正在全面展开，因此该公司将休假一段时间，新的版本将于 2022 年回归。这使得确保这个版本中的已知问题不会妨碍您的体验变得更加重要，因为您会被它们困扰一段时间。
