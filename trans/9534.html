美国联邦航空局表示，威瑞森和美国电话电报公司可以在机场附近部署更多的 C 波段 5G

原文：https://www.xda-developers.com/c-band-airport-rollout/#0001-01-01

威瑞森和美国电话电报公司上周开始在美国的 C 波段频谱上推出 5G，最终给了两家运营商 T-Mobile 中波段 5G 成功的答案。由于联邦航空管理局(FAA)和一些航空公司的担忧，首次展示被推迟，因为它可能会干扰一些飞机和控制塔中的设备，这短暂地导致了旅行中断。现在，联邦航空局、威瑞森和美国电话电报公司显然已经解决了他们的问题。

美国联邦航空局周五宣布，“美国联邦航空局、威瑞森和美国电话电报公司已经就一些措施达成一致，这些措施将使更多的飞机能够安全地使用主要机场，同时也使更多的塔台能够部署 5G 服务。FAA 赞赏与无线公司的强大沟通和合作方式，这些公司提供了关于无线发射机确切位置的更精确数据，并支持对 5G C 波段信号如何与敏感的飞机仪器相互作用进行更彻底的分析。FAA 利用这些数据确定，可以安全、更精确地绘制 5G 信号减弱的机场周围区域的大小和形状，缩小无线运营商推迟天线激活的区域。”

这应该意味着威瑞森和美国电话电报公司的 C 波段 5G 网络将开始在美国更近的(或直接在)更多主要机场出现。该消息也是在 FAA 验证波音 737 Max 上的高度计在 C 波段 5G 塔附近飞行是安全的后一天发布的，即使在 FAA 没有特别指定为不受干扰的安全机场也是如此。

一些航空公司和美国联邦航空局最初担心 C-Band 5G 会干扰一些无线电驱动的高度计，这些高度计由飞机用来确定起飞和着陆期间它们离地面有多近。美国联邦航空局很快证实大多数波音和空客飞机没有受到影响，但截至周五，该机构仍阻止波音 747-8、747-8F 和 777 飞机在没有明确标记为安全的机场降落(如果美国联邦航空局说某个机场是安全的，则没有任何适用)。据美国联邦航空局称，这包括美国境内的约 336 架飞机和全球范围内的 1714 架飞机。

尽管只有三架(已知的)飞机明显受到 C 波段 5G 的影响，但它们是航空业中最重要的一些机型。多年来，波音 777 一直是广受欢迎的宽体飞机，在阿联酋航空、联合航空、法国航空、Cathaway Pacific、美国航空和其他公司的机队中占很大比例。
