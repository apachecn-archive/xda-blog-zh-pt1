Windows 10 现在会自动安装 Windows 11 电脑健康检查应用程序

原文：https://www.xda-developers.com/windows-10-now-auto-installs-the-windows-11-pc-health-check-app/#0001-01-01

微软在宣布 Windows 11 的时候，也推出了一款名为 PC Health Check 的新工具。这个想法是让你知道你的电脑是否有资格升级到新的操作系统。它检查各种需求，如 CPU、内存、TPM 等等。

虽然雷德蒙德公司已经表示，它永远不会强制用户升级到 Windows 11，但似乎不能对 PC 健康检查应用程序说同样的话。更新 KB5005463 在 Windows 10 版本 2004 及以上安装 app。此外，更新是自动安装的，所以大多数用户会在某一天醒来并拥有它。

如果这种行为听起来很熟悉，那是因为确实如此。真的很难不将 Windows 10 的情况与之相提并论。早在 2015 年，微软就有一款名为 Get Windows 10 的应用，通常简称为 GWX。在一个看似无害的举动中，Windows 7 和 Windows 8.1 发布了一个更新，在个人电脑上安装了 GWX。

GWX 的目标是让你“预订”你的 Windows 10，这显然是一个数字产品，不需要预订。一段时间后，GWX 会越来越频繁地出现，微软甚至有时会改变应用程序的行为，所以你之前取消 Windows 10 优惠的一些事情现在会确认你想要它。

当时看起来很无辜，因为这是史无前例的。Windows 10 是一个新的领域，Windows 以前从来没有免费升级过。显然，从那时起，微软已经失去了很多信任，如果你担心 Windows 11 PC Health Check 会在后台自动安装在你的 PC 上，那也是可以原谅的。

Windows 11 的要求实际上有点严格，所以如果你的电脑不合格，你不必担心任何事情。对于 Windows 11，你需要有一个英特尔第八代或更新的 CPU，AMD Zen 2 或更新的 CPU，或者高通骁龙 850 或更新的 CPU。

PC Health Check 应用程序还能做其他事情。你可以查看电池容量，存储容量，管理启动应用程序，它可以给你提示，以获得更好的性能。显然，主要功能是检查你是否有资格使用 Windows 11。
