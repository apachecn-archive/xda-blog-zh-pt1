Windows 11 深度学习:使用提示应用程序了解 Windows 功能

原文：https://www.xda-developers.com/windows-11-tips/#0001-01-01

Windows 11 现在正在推出，它带来了许多新的东西。新的操作系统将成为 Windows 历史上的一个高潮，在我们的评论中，我们发现它修复了 Windows 10 的许多错误。但是任何新的大版本都会带来一些不确定性，有很多东西需要学习。在这篇文章中，我们将看看 Tips 应用程序，这是 Windows 11 中包含的许多应用程序之一，您可能想了解这些应用程序。

我们已经编写了 Windows 11 中许多其他应用和功能的深入指南，你可以在本文底部查看。但是现在，让我们把注意力放在 Tips 应用程序上。如果你是 Windows 新手，或者你刚刚收到了一个主要功能更新(如 Windows 11 本身)，Tips 是一个很好的去处。这款应用包含了关于你可以在 Windows 11 中做什么以及如何做的各种信息。

这款应用其实没什么特别的。当你第一次在 Windows 11 上启动提示时，你会看到一个主屏幕，上面有一个大标题，下面有几个类别。标题将突出显示最新 Windows 版本中的新内容，现在，这就是 Windows 11。如果您刚刚安装了新的更新，这是一个理想的去处，因为它将向您显示自上次使用您的电脑以来的最大变化。例如，现在，它突出显示了新的小部件窗格、居中的开始菜单、与微软团队聊天等等。

但这将在您收到另一个 Windows 11 重大更新时进行更新，这应该大约每年发生一次。不时回来查看是个好主意，因为它可能会教你一些新东西，尤其是在一次大的更新之后。

如果您想探索更深入的信息，您可以使用标题下方的类别来查找更多提示。您将看到五个类别，这是微软认为您最可能关心的类别。您可以了解如何导航到 Windows 11 的不同部分，了解如何在 widgets 面板中添加或删除 widgets，并了解如何定制 PC 的外观，等等。当您打开其中一个类别时，向导将在主窗口上弹出，您可以使用弹出窗口底部的箭头查看所有可用的提示。

如果您选择查看所有可用的提示，总共有 18 个主题。其中包括与性能、文件管理、新的 Edge 浏览器等相关的提示。一些提示可能还包括指向网页的链接，以教您更多有关某些功能的信息。发布时可用的类别包括:

  • Windows 11 有什么新功能
  • 在 Windows 中四处走动
  • 关于小部件的一切
  • 快捷键
  • 个性化您的电脑
  • 保持安全
  • 提升电脑性能
  • 文件管理
  • Microsoft Edge 的提示
  • 创造终极桌面
  • 赌博
  • Windows 适合您
  • 了解微软 365
  • 触摸手势
  • 改善电脑运行状况
  • 触摸键盘
  • 手写手势
  • 用你的笔

如果你刚买了一台新电脑，它具备了你以前没有的功能，这些建议也会很有帮助。例如，如果你有一台带活动笔的电脑，不妨看看“使用你的笔”一节，了解如何充分利用它。
