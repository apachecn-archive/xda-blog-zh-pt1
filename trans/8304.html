Visual Studio 代码编辑器现在作为 web 应用程序提供

原文：https://www.xda-developers.com/visual-studio-code-editor-now-available-as-a-web-app/#0001-01-01

Microsoft Visual Studio 代码是目前最好的代码编辑器之一。它是完全免费的，可以扩展成千上万个扩展，并且可以在所有主要的桌面平台上工作。尽管它是基于 web 技术的(它是一个电子应用程序)，Visual Studio 代码从来没有在 web 浏览器中提供完整功能的版本，直到现在。

微软推出了基于网络的 Visual Studio 代码版本，可在 vscode.dev 获得。它看起来和感觉上完全像桌面应用程序，能够安装扩展并与 GitHub 或微软帐户登录同步您的设置。您可以登录到远程存储库，或者上传单个文件并在完成后下载它们。如果你的浏览器支持文件系统访问 API (比如 Chrome、Edge 和 Opera 的桌面版本)，你就可以在你的设备上处理本地文件夹，而不需要任何上传或下载。

“随着 vscode.dev 的推出，”Visual Studio 代码团队在博客中说，我们开始最终实现我们最初的愿景，即构建一个可以在浏览器中完全无服务器运行的开发工具。“重要的是，这使得 Visual Studio 代码首次可以在 iPads 和 Android 平板电脑上运行，或者在安装桌面软件受到限制或阻止的环境中运行(如企业或教育环境)。这也意味着你可以在 chrome book 上运行 Visual Studio 代码，而无需启用 Linux 应用程序支持，或者从一开始就没有获得 Linux 支持的旧 chrome book。

尽管这是一个令人印象深刻的端口，但 web 上的 Visual Studio 代码有一些限制。大多数扩展现在无法工作，因为它们需要被更新为不依赖于 Node.js 代码或本地可执行文件的 web 扩展。 Luna Paint Image Editor 和 GitHub Issue Notebooks 是在 VS 代码中工作的两个扩展的例子。微软还集成了主题支持，使用特殊的 URL 进行简单的测试——例如，打开vscode.dev/theme/sdras.night-owl会带你到安装了夜猫子主题的 VS 代码。

微软还计划在网上对 VS 代码进行更多的更新。“将 VS 代码引入浏览器，是对产品最初愿景的实现。这也是一个全新的开始。拥有浏览器和互联网连接的任何人都可以使用的短暂编辑器是未来我们可以真正在任何地方编辑任何内容的基础。”
