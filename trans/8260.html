Windows 11 只允许管理员在特定情况下打印

原文：https://www.xda-developers.com/windows-11-admins-print/#0001-01-01

Windows 11 发布已经有几个星期了，正如你可能知道的，它和一起出现了一些已知的问题。其中一个问题影响了 AMD 处理器，降低了这些 CPU 中 L3 缓存的性能。在最初的 Windows 11 版本中，许多已知问题已经通过最近的Windows 内部人员累积更新得到了修复，但仍有一个已知问题未被发现。正如在微软支持页面(通过 出血计算机 )上指出的，在某些情况下，只有电脑管理员才被允许在 Windows 11 上打印文档。

公平地说，这个问题可能只会影响一部分用户。实际情况是，当使用打印服务器打印文档时，用户每次尝试使用 Windows 11 打印时，都可能会被要求管理员权限。这种情况只会发生在使用与 PC 所在时区不同的打印服务器时，所以对大多数人来说这不是一个大问题。不管怎样，如果你碰巧受到了影响，这可能是生产力的一个主要障碍。

这个打印问题在 Windows 11 中并不是全新的。在 9 月补丁周二更新之后，Windows 10 用户受到了类似问题的影响。然而，随着10 月补丁周二发布，这个问题在上周得到了解决。然而，Windows 11 用户没有收到该补丁。甚至最近在测试版渠道发布的 Windows 内部人员更新似乎也没有解决这个问题，或者至少，微软没有提到它。

微软在其支持页面上详细介绍了这个问题，并表示预计将在 10 月下旬推出修复程序。通常，Windows 会在月底收到可选的更新，然后这些更新会滚动到下一个月的星期二补丁程序发布中，这是强制性的。因此，除非您愿意手动更新，否则您可能要等到 11 月才能通过 Windows Update 获得修复。不过，至少你有能力在接下来的一两周内解决这个问题。

微软没有为该问题提供任何变通办法，因此任何受影响的人都必须简单地处理安全提示，直到修复程序推出。
