仅限今天:只需 279 美元，即可获得这款宏碁 Chromebook Spin 713

百思买今天有宏碁 Chromebook Spin 713 的英特尔酷睿 i3/4GB RAM 型号在售，现在你只需 279 美元就可以买到。

Chromebooks 仍然不能做 Windows 或 macOS 笔记本电脑可以做的所有事情，但由于其更简单的操作和 Android 应用程序支持，它们已经成为受欢迎的选择。宏碁 Chromebook Spin 713 一直是更好的 13 英寸机型之一，仅在今天(11 月 9 日)，配备英特尔酷睿 i3 CPU 和 4GB RAM 的 Chromebook Spin 713 在百思买以 279 美元的价格出售。

在售的型号配备了 13.3 英寸 2256 x 1504 触摸屏，第十代英特尔酷睿 i3-10110U 处理器，64GB eMMC 存储，4GB 内存和背光键盘。这款笔记本电脑还有 USB Type-C 端口和全尺寸 HDMI 连接器，可以派上用场。尽管显示屏是触摸屏，但你不能将笔记本电脑向后弯曲，像使用一个巨大的平板电脑一样使用它——这确实是一台笔记本电脑，而不是 2-i-1 可折叠的。

宏碁 Chromebook Spin 713

这款中端 Chromebook 拥有英特尔酷睿 i3 CPU 和 4GB RAM。它的售价比建议零售价低 250 美元，但已经在 300 美元上下浮动了一段时间。

由于英特尔酷睿 i3-10110U 只是一个双核/四线程处理器，笔记本电脑只有 4GB 内存，这不会是最快的 Chromebook。然而，仍然有足够的能力来处理一些 Chrome 标签和运行 Android 应用程序。这也是配有适当英特尔酷睿 CPU 的 Chromebook 的最低价格——任何更便宜的产品通常都配有赛扬或中档 ARM 处理器，这些处理器在 Chrome OS 上要慢得多。3:2 的高分辨率显示屏在这个价位也相对少见。

谷歌表示，这款 Chromebook (CP713-2W)最早将于 2028 年 6 月前接收 Chrome OS 更新，因此你可能会在官方支持结束前准备好硬件升级。这款 Chromebook Spin 713 是基本学校作业、视频通话、典型生产力工作和媒体流的绝佳选择。只是不要试图在 Linux 容器中运行 Android Studio SDK 或编辑视频，那样可能会太慢而没有用。
