您将在手机上收到 Realme UI 3.0 的早期版本

原文：https://www.xda-developers.com/realme-ui-3-0-early-access-roadmap/#0001-01-01

在昨天的 Realme GT Neo 2 发布会上，Realme 简要介绍了 Realme UI 3.0——基于 Android 12 的最新版本软件皮肤。现在，该公司已经分享了该版本的早期访问路线图，强调了您何时可以在 Realme 设备上收到更新。

根据更新时间表，Realme UI 3.0 的早期版本将于本月开始在 Realme GT 上推出。Realme GT Master Edition、GT Neo 2、X7 Max 5G 和 8 Pro 将在 12 月前获得早期访问版本。明年第一季度，Realme 将推出 Realme X7 Pro 5G、X50 Pro 5G、Realme 8、Realme 8i、Realme 7 Pro、Narzo 50A、Narzo 30、Realme C25 和 Realme C25s 的早期版本。最后，Realme X7 5G、Realme X3、X3 SuperZoom、Realme 8 5G、8s 5G、Narzo 30 Pro 5G 和 Narzo 30 5G 将在 2022 年 Q2 接受更新。

Realme 指出，上面提到的发布时间表只针对印度的第一波早期访问。这意味着最初只有少数用户会在该地区收到 Realme UI 3.0 早期访问版本。Realme 目前还没有分享国际市场的发布时间表。

Realme UI 3.0 包含了几个新功能。它包括谷歌在 Android 12 中引入的大部分变化，以及新的 3D 图标和更宽敞的 UI 布局的视觉刷新。它还包括 AOD 个性化支持、性能改进和浮动窗口 2.0 支持。此外，它还包括各种隐私和安全方面的更改，应该可以帮助您更好地管理您的数据。有关更多信息，请继续关注我们对 Realme UI 3.0 的实际预览。

Realme 尚未确认何时开始向其设备推出 Realme UI 3.0 的稳定版本。一旦我们有了更多的信息，我们会尽快通知你。
