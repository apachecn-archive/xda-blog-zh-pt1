在最新的 Windows 11 版本中发现了设计流畅的任务管理器

原文：https://www.xda-developers.com/windows-11-task-manager-fluent-design/#0001-01-01

微软今天早些时候推出了 Windows 11 build 22538 ，如果你只是看该公司的 changelog，该版本没有任何突破性的新功能(尽管有一些受欢迎的改进)。然而，在这个版本中休眠是一件更有意义的事情。正如 Twitter 用户 FireCube (最初由 UWP 社区 Discord 服务器上的一名用户发现)所分享的，新的 Windows 11 版本带有一个新的流畅设计灵感版本的任务管理器。

在目前的状态下，新的任务管理器并不是对现有用户界面的彻底改造。相反，通常位于任务管理器窗口顶部的选项卡现在是侧面，窗口的外壳使用了新的云母半透明材料。还有一些常见操作的突出按钮，如创建新任务、结束任务等。然而，当您查看包含流程、服务等列表的列表本身时，它们看起来与当前迭代中的完全一样。然而，他们现在处于黑暗模式，这也是任务管理器的一个新功能。你可以在下面看到新的设计，由 Twitter 用户 Xeno 的测试版分享。

正如 FireCube 所指出的，如果你运行的是 Windows 11 build 22538，你可以自己启用这一功能，但你需要一个名为 ViveTool 的程序，它允许你启用隐藏的 Windows 功能。然而，这可能会严重影响系统的稳定性，所以除非您对可能出现的任何问题都感到满意，否则不应该这样做。这是一个命令行工具，所以在提取。ZIP 文件，您需要使用命令提示符运行它(不是 Windows PowerShell)。在存储 ViveTool 的文件夹中运行命令提示符，然后输入以下命令:

  • vivetool 添加配置 35908098 2
  • vivetool 添加配置 37204171 2
  • vivetool addconfig 36898195 2
  • vivetool addconfig 36898195 2

完成后，你应该可以在你的电脑上看到新的任务管理器，设计流畅。

没有迹象表明我们什么时候会看到新的任务管理器，而不需要使用黑客的变通方法来启用它。微软当然仍在努力完善这一体验，当它准备好的时候就会推出。希望，这意味着 UI 的其余部分也将被重新设计，以匹配新的流畅设计，而不是这些冲突的设计语言。
