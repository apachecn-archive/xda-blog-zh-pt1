任天堂 Switch 控制器驱动程序现已推出 Linux 内核 5.16

原文：https://www.xda-developers.com/linux-kernel-5-16/#0001-01-01

Linux 内核是无数设备和操作系统的核心，包括 Android 手机和平板电脑、Chromebooks、桌面 Linux 发行版等等。新版本通常每隔几个月发布一次，现在 5.16 版本可供试用。

Linux 内核的创造者和主要维护者 Linus Torvalds 在内核邮件列表 ( via omg！ubuntu！ )，“从【v5.16 发布候选 8】开始这里就没有很多了，这并不意外。由于假期，我们有额外的一周时间，这并不是说我们有很多最后一分钟的事情需要解决。因此，这主要包含一些驱动程序修复(主要是网络和 rdma)，一个 cgroup 凭据使用修复，一些核心网络修复，几个最后一刻的恢复，以及一些其他随机噪声。”

也许这个版本中最重要的变化是一个叫做“futex2”的新内核系统，是“快速用户互斥”的缩写。它允许应用程序创建互斥、信号量、条件变量和其他快速执行的同步机制。这个新特性可以提高运行在 Wine 兼容层的游戏(以及原生 Linux 游戏)的性能，但 Wine 还没有实现这一点，所以我们必须等待，看看结果如何。

这个版本也有新硬件支持的常见组合。任天堂 Switch 游戏机和 Pro 控制器现在无需任何额外的驱动程序或调整即可工作，还有 2021 苹果魔法键盘。AMD 图形驱动程序现在可以与 DisplayPort 2.0 一起工作，据传闻即将推出的镭龙显卡将包含 display port 2.0，对英特尔 Alder Lake S graphics (DG1)的支持现在被认为是稳定的。

如果你感兴趣的话，在这个版本中还有数百个其他的变化— 内核黑客有一个详细的日志。内核应该在未来几天开始作为一些桌面 Linux 发行版的更新出现，但一些操作系统和发行版(如 Ubuntu)将暂时搁置。例如，Canonical 还没有确认哪个 Linux 内核版本将包含在下一个 Ubuntu 22.04 更新中。
