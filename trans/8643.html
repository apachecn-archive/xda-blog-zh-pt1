如何在 Windows 上安装新的 WhatsApp 应用

原文：https://www.xda-developers.com/how-to-install-new-whatsapp-windows/#0001-01-01

最近， WhatsApp 为 Windows 10 和 11 推出了新的测试版应用，采用了全新的设计和功能。这个新的应用程序是使用原生 Windows 技术构建的，而不是使用旧的基于网络的应用程序，它包括视频聊天等新功能，以及绘制图片并与其他用户分享的能力。不过，如果你从未在个人电脑上使用过 WhatsApp，设置它可能会是一个复杂的过程。在本指南中，我们将向您展示如何为 Windows 安装和设置新的 WhatsApp 应用程序。

在开始之前，你需要做的第一件事是确保你运行的是 Windows 10 build 18362 或更高版本，或者 Windows 11。早于此版本的 Windows 版本不再受支持，并且您很可能已经运行了其中一个版本。如果你想确认，同时按下键盘上的 Windows 键和 R ，然后键入 winver 。按下 Enter 键，将弹出一个窗口，显示您当前的 Windows 版本，以及内部版本号。

如果你符合要求，你就可以开始了。

如何为 Windows 安装新的 WhatsApp 应用程序

如果你准备安装新的 WhatsApp for Windows，你可以从微软商店下载开始。目前，该应用程序处于测试阶段，处于隐藏状态，所以你不能通过自己在微软商店搜索来轻松找到它。相反，您可以使用以下链接下载该应用程序:

从微软商店下载 WhatsApp Beta】

该链接会将你带到一个微软商店网页，然后你可以点击获取来打开你电脑上的微软商店应用。有些浏览器可能会自动打开应用程序。然后，在微软商店窗口中再次点击获取T5】开始安装。当应用程序在微软商店上时，你不需要微软帐户来下载它，因为它是一个没有成熟内容评级的免费应用程序。

应用程序安装后，它会显示在您的开始菜单中，您可以启动它。

如何设置新的 WhatsApp

一旦你启动了 Windows 版 WhatsApp 应用，点击开始，你会看到一个二维码(一个由黑白点组成的方块)和一系列说明。你需要在手机上安装 WhatsApp 来执行这些步骤。

新的 WhatsApp Windows 测试版应用程序使用了新的多设备测试功能，允许你使用 WhatsApp 帐户使用多种设备，即使你的手机离线。在打开 WhatsApp 应用程序的手机上，点击右上角的三点图标(在 Android 上)或右下角的设置选项卡(在 iOS 上)，然后点击链接的设备。点击显示多设备测试版的选项，然后点击屏幕底部的加入测试版。

返回上一屏幕，点击链接设备。你需要输入手机的安全码或任何你用来解锁手机的方法，然后 WhatsApp 会打开你的相机。将摄像头对准电脑屏幕上的二维码，它就会开始将你最近的信息下载到你的电脑上。

完成后，您将看到您当前的所有聊天记录，然后您就可以开始使用该应用程序了。

现在，您可以开始给朋友和家人发送消息，甚至在手机不在线的情况下也可以在 WhatsApp 上进行语音和视频通话。脸书——Meta，我们指的是 T1——已经逐渐为 Windows 提供了更好的支持，去年推出了一款原生 Messenger 应用。我们还不知道这个应用程序何时会完全可用，但一旦它做到了，所有这些步骤都应该是一样的。
